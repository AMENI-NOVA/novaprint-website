Type de prépresse

Choisissez le type de prépresse parmi ceux proposés dans la liste. Cette saisie est obligatoire pour pouvoir calculer un devis.

Opérations de prépresse

La fenêtre de saisie des opérations de prépresse n'est visible que si vous avez accès à au moins une opération de prépresse.

Pour ajouter une opération de prépresse au devis, cliquez sur le bouton « Ajouter » du cadre « Opérations supplémentaires ».

Une nouvelle fenêtre est affichée, vous permettant de pointer l’opération à ajouter au devis.

Les opérations sont regroupées par famille. Cliquez sur le ‘+’ se trouvant à gauche de chaque famille pour voir les opérations qu’elle contiennent.

La liste des familles et opérations est restreinte à celles auxquelles vous avez accès.

Pour certaines opérations, vous devez indiquer le nombre d'opérations prévues pour ce devis. Ce nombre apparaît ensuite dans le tableau récapitulatif dans la colonne "Nombre" et peut ensuite être modifié directement à cet endroit.

Consulter mes commandes

Précédent   Suivant

Lorsque vous choisissez la fonction "Consulter mes commandes" une fenêtre similaire à la liste des devis s'affiche. Elle contient les commandes vous concernant.

 

Deux types de commandes sont affichés dans cette liste :

¨   les commandes que vous avez demandées en utilisant Print to B (qu'elles soient ou non acceptées par l'imprimeur),

¨   les commandes que l'imprimeur à enregistrées pour vous. La présence de ces dernières dépend de la configuration informatique de l'imprimeur.

 

Cette liste vous permet juste de consulter les commandes vous concernant. Aucune fonction n'est disponible.

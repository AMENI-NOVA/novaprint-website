Papier

Précédent   Suivant

La saisie des informations pour le papier est identique pour tous les types de devis.

 

Vous pouvez sélectionner une catégorie de papier pour laisser Graphiplus optimiser entre tous les papiers de cette catégorie, ou imposer un papier en particulier.

Le choix d’une catégorie ou d’un papier peut s’effectuer de deux manières différentes :

¨   Saisie du code de 4 caractères désignant la catégorie ou le papier. Cette méthode est la plus rapide, à condition de connaître le code.

¨   Choix dans la liste : cliquez sur le bouton <…> se trouvant à droite de la zone de saisie pour faire apparaître la liste des catégories. Choisissez une catégorie ou un papier, puis cliquez sur le bouton <Sélectionner>.

Après avoir indiqué le type de papier, Graphiplus explore le fichier et affiche dans une liste déroulante tous les grammages disponibles pour ce papier, parmi lesquels vous pouvez choisir.

Papier fourni

Si vous fourninnez le papier, le prix final ne comportera pas de prix papier (sauf le prix de la manutention). Le prix du papier sera cependant pris en compte lors de l'optimisation.

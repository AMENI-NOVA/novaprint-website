Saisie des livraisons

Pour indiquer un point de livraison, cliquer dans la zone de saisie de la colonne "Point de livraison". La liste des points de livraison enregistrés est alors affichée, dans laquelle vous pouvez faire votre sélection.
Les points de livraisons sont présentés par famille, s'il existe au moins 2 familles.

Saisissez dans la colonne suivante, la quantité à livrer pour ce point de livraison.

La colonne "Poids (Kg)" indique le poids en fonction de la quantité indiquée et du poids estimé de l’exemplaire à partir des informations saisies pour le devis.

Lorsque vous avez sélectionné un point de livraison, une nouvelle ligne vierge est ajoutée au tableau pour vous permettre d’indiquer un autre point de livraison. La colonne "Quantité" propose le solde du nombre d’exemplaires non assignés à un point de livraison.

Le nombre de points de livraisons n'est pas limité pour un devis.

Pour supprimer un point de livraison d’un devis, cliquez dans une des cases de la ligne du point de livraison à supprimer, puis appuyer sur ALT + SUPPR ou cliquez sur le bouton <Enlever> se trouvant au-dessous du tableau.

Conditionnement par point de livraison

Vous pouvez saisir des conditionnements différents pour chaque point de livraison. Pour cela, déplacez le curseur jusqu'à la colonne "Conditionnement", puis appuyez sur <Entrée> ou cliquez sur le bouton <...>.

Une fenêtre apparaît alors et vous permet de saisir les conditionnements pour ce point de livraison. La saisie est exactement la même que pour les conditionnements généraux (voir le paragraphe précédent) qui sont utilisés pour initialiser les zones de saisie.

La case à cocher "Conditionnements détaillés pour ce point de livraison" vous permet en la décochant d'annuler les conditionnements spécifiques d'un point de livraison.

Lorsqu'aucun conditionnement spécifique n'a été saisi pour un point de livraison, le calcul est fait à partir des conditionnements généraux. A l'inverse, vous pouvez utiliser la saisie des conditionnements d'un point de livraison pour indiquer qu'il n'y a aucun conditionnement pour ce point.

Exemplaires non livrés

Si tous les exemplaires ne sont pas livrés, le solde est automatiquement affecté à une ligne nommée "(départ usine)". La somme des quantités indiquées dans la colonne "Quantité" doit être égale à la quantité initiale saisie pour le devis, sinon le calcul du devis n’est pas autorisé.

Modification de la quantité

Si vous avez affecté tous les exemplaires à un seul point de livraison, puis que vous modifiez la quantité totale du devis, la quantité à livrer sera automatiquement ajustée en fonction de la nouvelle quantité à deviser.

Optimisation sur plusieurs quantités

Si vous avez demandé une optimisation sur plusieurs quantités, les quantités à livrer pour chaque point de livraison seront ajustées au prorata de chaque quantité à optimiser.

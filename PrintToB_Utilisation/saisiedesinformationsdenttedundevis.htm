Informations d’entête communes à tous les types de devis
Référence

Vous disposez de 200 caractères pour entrer un texte libre qui vous aidera à identifier le devis ultérieurement. La référence peut contenir plusieurs lignes.

Devis de prépresse

Cette case à cocher n'apparaît que lorsque vous chargez un devis de prépresse existant. En la décochant, vous pouvez transformer le devis de prépresse en devis d'impression. Vous pouvez alors compléter la saisie puis relancer le calcul.

Produit

Vous pouvez sélectionner un type de produit parmi ceux paramétrés par le fabricant.

Quantités supplémentaires

Ce bouton vous permet de faire apparaître 3 zones dans lesquelles vous pouvez saisir des quantités supplémentaires.

 

Le devis est calculé pour chacune des quantités saisies.

Les quantités supplémentaires sont enregistrées comme des variantes.

Informations d’entête spécifiques aux devis de brochures
Format fini

Il est à entrer en cm sans importance d'ordre (21/29,7 est identique  29,7/21 pour le logiciel).

Si des formats standards ont été paramétrés, le bouton <...> vous permet de choisir un format dans la liste.

Le format proposé par défaut pour les nouveaux devis est le format standard défini par défaut, ou 21 x 29,7 si aucun format standard n'a été paramétré.

Orientation

Définit si la brochure est à la française ou à l'italienne (piquage ou collage sur le grand ou sur le petit côté).

Brochage

Choisissez un type de brochage parmi ceux qui sont proposés.

Informations d’entête spécifiques aux devis de dépliants et d’affiches/fiches
Sens du papier

Cette option vous permet de préciser si le sens du papier a une importance. Si c’est le cas, vous devez indiquer si le grand côté du format à plat du dépliant doit être :

¨   dans le grand côté du format de la feuille.

¨   dans le petit côté du format de la feuille.

Il s'agit du format de la feuille livrée par le papetier. Si le programme choisit de refendre (et donc de changer de sens), il en tient compte.

Le grand côté de la feuille de papier correspond toujours à la 2ème valeur saisie pour le format. Par exemple, si on a saisi 92/65 comme format de papier, c’est 65 qui est considéré comme le grand côté.

Informations d’entête spécifiques aux devis d’affiches/fiches
Assemblage des fiches

Si vous cochez cette option, un coût d’assemblage feuilles sera calculé pour toutes les fiches du devis (tous les modèles de tous les éléments du devis).

Le calcul de l’assemblage des fiches n’est fait que s'il y a au moins 2 modèles dans un élément de devis, ou si le devis comporte plusieurs éléments.

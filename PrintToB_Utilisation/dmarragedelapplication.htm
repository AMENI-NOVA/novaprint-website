Démarrage de l’application

Précédent   Suivant

Que vous soyez un client ou un employé (commercial), vous devez saisir votre nom de connexion et votre mot de passe pour vous connecter au serveur d’applications.

Graphiplus se lance automatiquement avec les fichiers de l’imprimeur auxquels votre connexion est rattachée.

 

Graphiplus vérifie si la connexion se rapporte à un client ou un employé.

Si aucune correspondance dans les connexions n’est trouvée, vous ne pouvez pas utiliser l’application.

 

Si vous êtes un employé et que la connexion est définie pour tous les employés, vous devez vous identifier parmi la liste des utilisateurs enregistrés dans le fichier de l’imprimeur, sinon, vous rentrez directement dans l’application. Dans le cas d'un employé, l'utilisation de Graphiplus est décrit dans le manuel de référence.

Les fonctionnalités possibles sont les suivantes :

¨   "Consulter mes devis" : affiche la liste de vos devis.

¨   "Consulter mes modèles de devis" : ouvre la liste des modèles de devis définis pour vous

¨   "Calculer un nouveau devis d'impression" : ouvre l'écran de saisie d'un nouveau devis d'impression

¨   "Calculer un nouveau devis de prépresse" : ouvre l'écran de saisie d'un nouveau devis de prépresse

¨   "Consulter mes commandes" : ouvre la liste de vos commandes

¨   "Visualiser l'avancement de mes commandes" : ouvre une fenêtre contenant l'état d'avancement de vos commandes qui sont validées par l'imprimeur.

 

Pour utiliser chacune de ces fonctionnalités il faut cliquer sur son libellé ou dans la barre d'outils en haut de l'écran.

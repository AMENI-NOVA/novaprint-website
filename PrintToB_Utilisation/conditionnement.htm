Conditionnement

Précédent   Suivant

Indiquez le ou les conditionnements que vous souhaitez. Vous pouvez cumuler plusieurs conditionnement différents.

Pour chaque type de conditionnement, vous pouvez indiquer le nombre d'unités par paquet, ou laissez 0 pour que Graphiplus calcule automatiquement le nombre de paquets en fonction du poids moyen enregistré pour le conditionnement sélectionné.

Attention, dans le cas d'un cumul de plusieurs conditionnements, à partir du 2ème type de conditionnement, le nombre d'unités par paquet représente un nombre de paquets du conditionnement précédent. Ceci signifie qu'il faut saisir les conditionnements du plus petit au plus grand, par exemple : Film rétractable + Cartons + Palettes

Il est également possible de saisir des conditionnements par point de livraison. Les conditionnements saisis ici seront valables pour les points de livraison pour lesquels aucun conditionnement spécifique n'a été saisi.

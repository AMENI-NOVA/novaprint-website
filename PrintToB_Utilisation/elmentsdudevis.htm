Eléments du devis

Précédent   Suivant

Chaque type de devis peut être décomposé en éléments homogènes :

¨   Intérieurs, couverture et encarts pour les brochures

¨   Modèles pour les dépliants, les fiches et les affiches

Pour ajouter un nouvel élément à un devis, cliquez dans la liste de gauche à n’importe quel endroit, et appuyez sur la touche INSER.

Pour supprimer un élément d’un devis, cliquez dans la liste de gauche sur une des lignes qui compose l’élément à supprimer, et appuyez sur la touche SUPPR.

Elément de brochure

Type d'élément

De même que les devis sont regroupés en types, les éléments d'une brochure peuvent être de 4 types différents :

¨   Cahier : Pour tous les éléments de ce type, on considère que le pliage est fait en plis croisés.

¨   Couverture : Ce type d'élément est traité de la même manière qu'un encart. Il faut cependant différencier la couverture car dans certains cas le pliage de celle-ci ne doit pas être compté :

·    Brochures 2 piqûres à cheval, si l'encarteuse piqueuse peut plier la couverture (Voir fichier machines).

·    Brochures en Dos carré collé ou Couture fil, le pliage étant fait en même temps que l'encollage à chaud.

¨   Encart broché : Les encarts brochés (comme d'ailleurs les couvertures) sont des éléments qui sont traités comme des dépliants (Voir chapitre Dépliant). Ils sont pris en compte dans le calcul du brochage

¨   Encart jeté : Les encarts jetés sont calculés comme les encarts brochés, mais ne comptent pas dans le calcul du brochage.
Contrairement aux couvertures et aux encarts brochés, le format d’imposition d’un encart jeté ne comprend ni rogne, ni dépassant, quel que soit le paramétrage du chemin de brochage utilisé.
Si l’encart jeté fait au moins 4 pages, le pliage est toujours calculé, même si le chemin de brochage utilisé ne comporte aucune plieuse.

Nombre de pages

Pour les éléments de type 'Cahier', le nombre de pages doit être un multiple du plus petit cahier possible en fonction du type de brochage sélectionné.

Graphiplus vous avertit si le nombre de pages saisies n'est pas compatible avec le type de brochage que vous avez sélectionné. Vous devez alors, soit :

¨   modifier le nombre de pages

¨   sélectionner un autre type de brochage permettant de réaliser le cahier comportant ce nombre de pages

Format à plat (couverture et encart)

Un format à plat calculé automatiquement vous est proposé.

Si vous modifiez le nombre de pages de l’intérieur, le format à plat de la couverture sera automatiquement recalculé en fonction de la nouvelle épaisseur de la brochure. Si vous travaillez sur un devis qui a été rappelé, Graphiplus vous demandera d’abord de confirmer le recalcul automatique du format, au cas où celui-ci aurait été modifié initialement pour tenir compte par exemple d’un rabat .

Lors du calcul ce format sera augmenté de la rogne sur 4 côtés, et du dépassant suivant le chemin de brochage utilisé pour le calcul (ceci ne concerne pas les encarts jetés).

Le format à plat est automatiquement recalculé si vous modifiez le format fini, l'orientation, le type de brochage ou le nombre de pages de l'intérieur.

Elément de dépliant

Formats

C’est le format à plat qui est pris en compte pour optimisé l’impression.

Si des formats standards sont paramétrés, le bouton <...> vous permet de sélectionner un format dans la liste.

Pliage

Vous devez indiquer le nombre et le type des plis. Vous avez la possibilité de mixer des types de plis différents, par exemple : 2 plis roulés + 1 pli croisé. Dans ce cas, Graphiplus considèrera pour le calcul, qu'il s'agit de plis croisés (ici 3 plis croisés), mais sur la lettre de confirmation, le libellé aura la même précision que lors de la saisie du devis.

Pour saisir les plis, indiquez le nombre de plis du premier type, puis le type de ces plis. S’il y a plusieurs types de plis différents, utilisez les zones suivantes. Il est possible de mixer 3 types de plis différents.

Elément d’affiche/fiche
Format fini

Il est demandé à chaque nouvel élément. On peut donc avoir dans un même devis des fiches de format différent. Cependant, Graphiplus n'amalgamera pas les différentes éléments.

Si des formats standards sont paramétrés, le bouton <...> vous permet de sélectionner un format dans la liste.

Activer / désactiver l’utilisation d’un tarif

Précédent   Suivant

Une ligne est en erreur si parmi les tarifs utilisés, ils ne sont pas tous à zéro (ce qui signifie que l’opération est sous-traitée) ou ils ne sont pas tous différents de zéro (ce qui signifie que l’opération est interne)

Pour déterminer les tarifs que l’on souhaite utiliser, il suffit d’activer ou pas l’utilisation d’un tarif.

Pour cela, cliquez sur les boutons     dans la barre d’outils ou dans le menu <Liens>< Tarifs>.

Le nombre de boutons ou d’options dans le cas du menu reflète le nombre de tarifs actuellement définis dans l’application.

Codage des stocks

Précédent   Suivant

Les codes stocks comprennent 28 caractères répartis selon:

¨   Code famille: 12

¨   Code article: 12

¨   Code possesseur: 4

Toutes les saisies de code de stock sont accompagnées d’une liste d’aide.

Les lots sont repérés par un codage sur 30 caractères.

¨   Le code généré par Graphistock est un simple numéro d’ordre sans signification spéciale précédé du préfixe "@0". Il est attribué automatiquement aux lots entrés si l’utilisateur n’a pas entré d’autre code reconnu pour ces lots.

¨   Le code IFRA est une codification utilisée par les fabricants pour identifier les bobines de papier journal. Le code est formé de 16 chiffres. Le 14ème chiffre code les caractéristiques du papier, les chiffres 13, 15 et 16 définissent le fournisseur, les chiffres 9 à 12 donnent le poids en Kg de la bobine. Graphisoft peut reconnaître ces codes et les utiliser pour initialiser les informations liées au lot comme le code article, la quantité et le fournisseur. Le code IFRA inscrit sur une bobine par le fabricant est reconnu sans modification par Graphistock. Mais Graphistock lui adjoint le préfixe "@1" pour l’identifier comme tel dans les affichages internes.

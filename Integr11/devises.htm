Symbole monétaire

Entrez le symbole correspondant à la devise principale. Ce doit être le même que dans les paramétrages régionaux du système (Bouton « Démarrer », « Paramètres – Panneau de configuration – Paramètres régionaux »). Le programme utilise cette information pour vérifier que le système est bien dans la même monnaie que vos fichiers.

Notez que changer la devise principale ici ne convertit pas les montants de vos fichiers. Il faut utiliser un programme spécifique (moulinette de conversion) pour convertir tous les montants enregistrés dans la nouvelle devise.

Autres devises

Indiquez ici les devises avec lesquelles vous pouvez être amené à travailler.

Cette liste vous permettra ainsi de définir qu’un client utilise telle autre devise plutôt que votre devise principale.

Cliquez <Ajouter> pour ajouter une nouvelle devise à la liste et <Supprimer> pour l’effacer.

‘Définir par défaut’ permet d’avoir un double affichage des montants dans le dossier technique lorsque la devise du devis est la devise principale : Les totaux du devis seront bien sûr affichés dans la devise du devis, mais aussi dans cette devise de préférence.

Ainsi les utilisateurs, nouvellement passés à l’Euro par exemple, peuvent faire afficher les totaux dans leur ancienne devise.

 

Le bouton <Propriétés> permet de modifier les informations concernant une devise.

Les informations de cette devise vont permettre d’afficher, de convertir et définir les arrondis les plus adéquats pour les montants des devis exprimés dans cette devise.

Symbole monétaire international

Choisissez ou saisissez les trois lettres définissant le code unique (ISO 4217) de la devise

Si la devise est connue du système, les informations qui suivent seront remplies avec celles obtenues du système.

Pays

Entrez le nom du pays où cette devise est utilisée. Cette information permet une présentation plus claire de la devise dans les listes.

Symbole monétaire

Ce symbole sera utilisé pour représenter la devise dans les montants.

Format positif, Format négatif

Indiquez le format d’affichage des montants positifs et négatifs en choisissant dans la liste.

Symbole décimal, Nombre de décimales

Le symbole sera utilisé pour séparer les décimales de la partie entière. Les montants exprimés dans cette devise présenteront le nombre de décimales indiqué.

Symbole de groupement des chiffres, Groupement des chiffres

Le symbole sera utilisé pour séparer les groupes de chiffres dans la partie entière des montants.

Taux de conversion

On peut l’exprimer soit sous la forme 1 devise = .... devises principales, soit 1 devise principale = ... devises. Choisissez la forme la plus appropriée.

Arrondir les montants des devis en devise aux ...

Comme dans le paragraphe précédent « Arrondi des devis et type de mille en plus », vous indiquez ici les paramètres d’arrondis des montants des devis exprimés dans cette devise.

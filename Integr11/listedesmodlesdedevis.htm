Liste des modèles de devis

Précédent   Suivant

Pour ouvrir la liste des modèles de devis, cliquez sur le menu Graphisoft de la fenêtre principale, déplacez la souris jusqu’à Graphisoft, puis cliquez sur Liste des modèles de devis.

 

Créer un modèle de devis

La création d'un modèle de devis s'effectue par la fenêtre de saisie d'un devis optimisé en cliquant sur le bouton <Modèle de devis>. Le modèle de devis peut être enregistré que le devis ait été calculé ou non. Les modèles de devis enregistrés à partir d'un devis non calculé, ne peuvent pas être utilisés pour créer des commandes.

L'enregistrement d'un modèle de devis n'est possible que pour les utilisateurs ayant le droit correspondant.

Vous pouvez aussi créer un modèle de devis à partir d'un autre modèle. Après avoir modifié le modèle initial, cliquez sur le bouton <Modèle de devis>. Le programme vous demande si vous désirez remplacer le modèle que vous avez chargé; répondez Non pour enregistrer alors un nouveau modèle.

Ouvrir un modèle de devis dans le devis optimisé

Pour ouvrir un modèle de devis, sélectionnez-le dans la liste et choisissez l'option Ouvrir du menu.

Le modèle est ouvert dans la fenêtre de saisie d’un devis.

Vous pouvez alors modifier ou compléter le devis, le calculer, puis l'enregistrer soit :

¨   comme un nouveau devis

¨   comme un nouveau modèle de devis

¨   en remplacement du modèle de devis que vous avez ouvert.

Modifier un modèle devis

Ouvrez le modèle à modifier dans la fenêtre de saisie de devis. Effectuez les modifications que vous désirez, puis cliquez sur le bouton <Modèle de devis>. Le programme vous demande alors si vous désirez remplacer le modèle initial ou créer un nouveau modèle. Répondez Oui à cette question.

Attention, si vous chargé un modèle de devis calculé et que vous le modifiez, n'oubliez pas de le recalculer pour qu'il puisse toujours être utilisé pour créer des commandes.

Descriptif et commentaires du modèle de devis

Comme pour un devis ordinaire, il est possible de saisr des descriptifs et des commentaires pour un modèle de devis. Ces descriptifs et commentaires peuvent être ou non transmis au devis créé à partir du modèle. Le choix est laissé à l’utilisateur lorsqu’il enregistre le devis. Dans le cas de la création d’une commande directement à partir du modèle, les descriptifs et/ou commentaires sont automatiquement transmis au devis qui donnera lieu à la commande.

Propriétés d'un modèle de devis

La fenêtre des propriétés permet de saisir ou modifier :

¨   Certaines informations générales :

·    Référence

·    Produit

·    Deviseur

·    Commercial

·    Prix de vente et/ou coefficient :

Remarque : vous pouvez modifier le prix de vente d'un modèle de devis calculé sans avoir besoin de le recalculer. Le programme ajuste le coefficient en conséquence. Après la modification, il est possible de créer des commandes à partir du modèle, le prix de vente des nouvelles commandes étant le nouveau prix de vente du modèle.

·    Mille suivant et/ou coefficient correspondant

·    Unité du prix unitaire (0 si pas de prix unitaire défini)

¨   Les adresses de livraison pour les points de livraison

¨   Les informations complémentaires

Ces informations seront transmises aux commandes qui seront générées à partir des modèles de devis.

Remarque : vous pouvez aussi saisir des adresses de livraison et des informations complémentaires pour les devis (voir "Propriétés d'un devis" dans le chapitre "Traitement des devis et utilitaires").

Supprimer un modèle de devis

Pour supprimer un modèle de devis, sélectionnez-le dans la liste, et choisissez l'option Suppression du menu.

La suppression d'un modèle de devis n'est possible que pour les utilisateurs ayant le droit correspondant.

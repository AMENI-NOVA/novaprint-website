Macros : enregistrement des options

Précédent   Suivant

Il est possible d'enregistrer le paramétrage d'une importation dans une macro.

Charger une macro

Ouvrez la liste "Macros", sélectionnez le nom de la macro souhaitée et cliquez sur le bouton .

L'écran est mis à jour avec les paramètres enregistrés dans la macro.

Enregistrer une macro

Saisissez (ou sélectionnez) le nom de la macro dans la liste "Macros" et cliquez sur le bouton .

Si le nom saisi correspond à une macro existante, celle-ci est modifiée. Si le nom saisi n'existe pas, la macro est ajoutée.

Supprimer une macro

Sélectionnez le nom de la macro dans la liste "Macros" et cliquez sur le bouton .

Importer une macro depuis un fichier

Bouton :

Permet de charger un paramétrage d'importation enregistré dans un fichier (format xml).

Une fois chargé, vous pouvez enregistrer ce paramétrage dans la base de données en cliquant sur le bouton Enregistrer une macro.

Bouton :

Permet d'enregistrer le paramétrage d'importation dans un fichier (au format xml).

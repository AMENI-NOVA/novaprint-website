Actions

Précédent   Suivant

Il s’agit des actions possibles à effectuer liées au destinataire d'une note (voir "Actions liées aux destinataires d'une note" dans le chapitre "Notes" ou à un message (voir "Actions liées aux messages" dans le chapitre "Messagerie").

Pour ajouter une action, sélectionnez "Actions..." et choisissez "Nouveau" dans le menu "Paramétrages" ou le menu contextuel

Pour renommer ou supprimer une action, sélectionnez-la puis choisissez "Renommer" ou "Supprimer" dans le menu "Paramétrages" ou le menu contextuel

Pour accéder aux propriétés d'une action, sélectionnez-la puis choisissez "Propriétés" dans le menu "Paramétrages" ou le menu contextuel

 

L'option "Action à effectuer à une date donnée" oblige la saisie d'une date lorsqu'on choisit l'action lors de l'envoi d'un message.

Lors de l'envoi d'un message, si l'on choisit une action pour laquelle "Une réponse est obligatoire pour cette action" est cochée, la case "Réponse obligatoire" est cochée automatiquement et n'est pas modifiable.

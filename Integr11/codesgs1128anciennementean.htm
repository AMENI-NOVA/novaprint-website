Codes GS1-128 (anciennement EAN)

Précédent   Suivant

Vous pouvez ajouter des codes GS1 qui permettent de définir dans un code plusieurs informations sur les colis. Chaque type d’information est identifié par un code numérique.

Il peut contenir les informations suivantes :

-   SSCC (code 00)

-   Numéro de dossier (10)

-   Date du bon de livraison (11)

-   Quantité dans le colis (37)

-   Numéro de commande client (400)

-   Référence article client (241)

 

Lorsque vous ajoutez le champ GS1-128, vous indiquez dans le format les identifiants des informations souhaitées entre parenthèses.

Ex : (00)(10) pour obtenir le SSCC et le suivi du numéro de dossier

Des exemples non exhaustifs sont présentés dans la liste déroulante. vous pouvez composer votre propre code avec les identifiants proposés.

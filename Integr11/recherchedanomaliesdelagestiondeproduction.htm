Recherche d’anomalies de la Gestion de production

Précédent   Suivant

Cette fonction est accessible uniquement dans l'Intégrale.

Le but de cette fonction est de détecter automatiquement les anomalies possibles dans les données enregistrées dans les données de gestion de production de l'intégrale, en fonction de critères donnés par l'utilisateur.

Il peut s'agir par exemple de s'assurer d'un temps minimum de présence pour les employés ou de chercher les fiches de travail pour lesquelles l'écart entre le temps réel et prévu dépasse un certain pourcentage.

La recherche d’anomalie en cascade permet d’effectuer plusieurs recherches qui sont effectuées les unes après les autres.

A la fin de la recherche, le rapport d'anomalies apparaît à l'écran. Il est alors possible de le consulter et de l'éditer sur une imprimante.

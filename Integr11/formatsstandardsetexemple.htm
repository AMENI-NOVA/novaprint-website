Ce tableau permet de saisir les coefficients à appliquer suivant le format, sur la refente ou la coupe 4 faces (les coefficients ne sont pas utilisés pour la coupe 3 faces).

Ce tableau est composé de 7 coefficients indépendants pour différents formats standards. Pour A4, le coefficient est toujours égal à 100%, pusiqu'il s'agit du format de référence des prix saisis dans le tableau des tarifs.

Le premier coefficient (origine) ne correspond pas à un format réel, mais est utilisé pour calculer le coefficient des formats inférieurs à A5.

En face de chaque coefficient est affiché le prix refente et coupe 4 faces (au tarif 1), calculé en fonction du prix de référence (format A4), et du coefficient saisi pour ce format. Si vous avez saisi vos tarifs en temps, les valeurs affichées ici sont des temps.

Il est possible dans ce tableau (dernière ligne), de saisir un format exemple, pour obtenir le prix (temps) calculé en fonction des coefficients enregistrés.

Les courbes affichées sous le tableau représentent les prix (temps) au tarif 1 en fonction du format.

Le format utilisé pour le calcul du coefficient est :

¨   le format des feuilles avant refente, dans le cas d'une refente

¨   le format fini des feuilles après la coupe, dans le cas d'une coupe 4 faces

Þ  Pour calculer le coefficient suivant un format donné :

Soit le format LARGEUR/LONGUEUR (ex : 65 / 92) On détermine entre quels formats de référence il se situe, ici entre A1 et A0 (A1=4989,6 cm2, A0=9979,2 cm2, 65x92=5980 cm2)

Pour A1, le pourcentage est PA1, pour A0, il est PA0.

Soit Px, le pourcentage à appliquer au prix du format A4, pour obtenir le prix pour le format 65x92 :

Px = PA1 + ( [ (65x92 / 4989,6) - 1 ] x [ PA0 - PA1 ] )

Soit PrixA4, le prix de coupe pour le format A4

Le prix recherché pour le format 65x92 est :  Prix65x92 = PrixA4 x ( Px / 100 )

Ne pas oublier de pondérer ce prix en fonction du grammage.

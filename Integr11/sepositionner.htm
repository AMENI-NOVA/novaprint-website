Se positionner

Précédent   Suivant

Pour se déplacer d'une ligne à l'autre dans la liste des employés, il faut utiliser les touches [Flèche haut] et [Flèche bas].

Pour se déplacer rapidement, il faut taper les premières lettres de son nom ou de son code en entier.

Le libellé à gauche de la zone de saisie indique ce qu’il faut saisir.

Le choix entre le nom et le code dépend d’un paramétrage de la liste des employés.

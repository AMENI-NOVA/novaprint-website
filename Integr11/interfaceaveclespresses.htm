Interface avec les presses

Précédent   Suivant

Cette interface permet d'envoyer toutes les informations, connues par Graphiplus, pour initialiser les presses.

Ces informations sont transmises soit automatiquement (par exemple, lors de la transformation d'un devis en commande), soit manuellement depuis la liste des commandes.

 

Les presses, de leur coté, peuvent envoyer l'état d'avancement d'un travail. Cela permet d'alimenter automatiquement la gestion de production avec les informations suivantes :

¨   Temps de calage, roulage et lavage,

¨   Nom de l'opérateur (ou des opérateurs),

¨   Nombre de bonnes et mauvaises feuilles,

 

Voir le détail des données échangées la partie Annexe de ce document.

 

L'interface avec les presses répond aux normes JDF suivantes, définies par le CIP4 :

¨   JDF version 1.2 et 1.3

¨   Base Conformance Levels ICS version 0.5-v14 (Base_L1-1.0)

¨   MIS base Conformance ICS version beta 0.5 niveau 1 (MISBC_L1-1.0)

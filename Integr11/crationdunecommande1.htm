Création d’une commande

Précédent   Suivant

Lors de la création d’une commande, si on converti un devis avec un prix sur barème, alors la commande reprendra par défaut ce même prix sur barème.

Remarque : Si le barème du devis n’est pas calculable, un message vous en averti et vous pouvez le corriger, mais l’enregistrement de la commande est tout de même possible avec le prix actuel qui vous est proposé ; L’accusé-réception de commande affichera le prix sur barème avec le détail effectivement utilisé.

Vous pouvez créer un prix sur barème ou intervenir sur les opérations composant le prix sur barème avec le bouton « Prix sur barème » dans l’écran de confirmation.

Si un barème est déjà défini, il est chargé et vous pouvez le modifier, voire le supprimer. Si aucun barème n’existait, vous pouvez en créer un pour imposer le prix de vente d’après celui-ci.

En cas de changement de quantité et si le paramétrage des opérations des barèmes a changé, le barème peut devenir incalculable en l’état. Vous devrez alors intervenir pour le corriger.

Remarque : Si un prix sur barème est défini alors il ne peut pas y avoir d’unité de quantité et donc pas de prix unitaire.

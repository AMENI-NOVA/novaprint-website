Edition d’un rapport

Précédent   Suivant

Une fois que vous avez sélectionné votre rapport, une fenêtre de saisie des paramètres du rapport s’affiche. Cette fenêtre est différente ou ne pas apparaître suivant le rapport demandé. Une fois les paramètres saisis, appuyez sur « OK ». La fenêtre d ‘édition du rapport s’affiche.

Si un paramètre est mal renseigné, la fenêtre de saisie des paramètres reste affichée et le message « Valeur non valide » s’affiche au-dessus du paramètre non renseigné correctement.

 

 

Lorsque les paramètres de l’édition sont correctement sélectionnés, la fenêtre de l’édition s’affiche :

Elle se décompose en 3 parties :

¨   la barre d’outils

¨   le volet de droite affichant le rapport

¨   le volet de gauche affichant le rapport dans ses grandes lignes. Lorsque vous cliquez sur un groupe dans cette arborescence, le logiciel "saute" immédiatement vers la partie du rapport contenant les informations appartenant à ce groupe, ceci permettant de naviguer rapidement à l'intérieur de rapports longs. Cette arborescence permet pour certains rapport d’afficher plus de détails pour un élément dans un autre onglet.

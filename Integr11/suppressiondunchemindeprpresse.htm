Suppression d'un chemin de prépresse

Précédent   Suivant

Pour supprimer un chemin de prépresse, vous devez le supprimer de chaque type de prépresse auquel il appartient. Ouvrez le type de prépresse auquel appartient le chemin que vous voulez supprimer, puis choisissez Supprimer dans la barre d'outils ou dans le menu contextuel.

Si vous supprimez un chemin qui n'appartient qu'à un seul type de prépresse, un message vous propose de le supprimer définitivement.

¨   Si vous répondez Oui, le chemin sera effacé et n'apparaîtra plus dans la liste des chemins possibles de la mise à jour des types de prépresse.

¨   Si vous répondez Non, le chemin sera conservé et pourra plus tard être à nouveau lié à un type de prépresse.

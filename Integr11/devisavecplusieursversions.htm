Devis avec plusieurs versions

Précédent   Suivant

Graphisoft permet de saisir des devis pour des imprimés constitués de plusieurs versions, par exemple :

¨   Une brochure composée d’une version avec un encart et d’une version sans encart

¨   Un imprimé en plusieurs langues avec une version par langue : par exemple une brochure pour laquelle il y a des changements de plaques au noir sur l’intérieur pour chaque langue, et une couverture commune à toutes les versions.

 

Pour saisir un devis de plusieurs versions, vous pouvez :

¨   Ajouter une version au devis en utilisant l’option de menu correspondante

¨   Modifier la quantité à imprimer d’un élément du devis

¨   Saisir des changements de plaques pour un des éléments du devis

Lorsque vous effectuez une de ces 3 actions, la saisie de devis passe automatiquement en mode multi-versions, ce qui permet alors de modifier les différentes versions

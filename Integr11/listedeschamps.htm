Dossier technique

La plupart des champs renvoient une seule information (par exemple "Raison sociale", ou "Nombre de poses").

Certains champs renvoient un texte pré-formaté regroupant plusieurs informations (par exemple "Papier détail (pré-formaté)"). Dans ce cas, le nom du champ comporte la mention "(pré-formaté)".

§   Format papier machine : Format de la feuille en machine (voir aussi "Format d'imposition" dans la rubrique "Imposition").

S’il y a une refente avant impression, le format papier machine sera par défaut le format brut refendu. Pour un 65x92 refendu en 2, le format papier machine sera par exemple 46x65, le format d’impression pouvant être plus petit.
Ce format sera éventuellement ajusté en fonction des opérations effectués ou des matériels utilisés après l’impression.
S’il y a une opération de pelliculage/vernis, le calcul de l’opération est fait par défaut sur le format d’impression pour en minimiser le coût. Dans ce cas, le format papier machine sera ajusté à ce format.
De même si après l’impression, on utilise une machine typo dont le format maximum est inférieur au format brut refendu, le format papier machine sera ajusté en fonction du format maximum de la machine typo, sauf si une refente est nécessaire entre l’impression et l’opération typo.

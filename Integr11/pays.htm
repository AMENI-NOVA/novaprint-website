Pour ajouter un nouveau pays, sélectionnez la rubrique "Pays et secteurs", puis cliquez sur Nouveau dans le menu Paramétrages ou le menu contextuel, ou utilisez le bouton correspondant de la barre d’outils supérieure.

Pour modifier le nom d’un pays, sélectionnez-le dans la liste, puis cliquez sur Renommer dans le menu Paramétrages ou le menu contextuel, ou utilisez le bouton correspondant de la barre d’outils supérieure. Le nouveau nom ne doit pas correspondre à un pays existant ou supprimé. Lorsque vous modifiez le nom d’un pays, tous les éléments rattachés à l’ancien nom (secteurs, sociétés) sont automatiquement liés au nouveau.

Cochez la case "Pays par défaut" pour le pays dans lequel vous vous trouvez. Le pays par défaut sera automatiquement sélectionné dans les listes de sélection d’un pays.

Ordre d’affichage de l’adresse

Cette information est utilisée pour que l'adresse de la société soit correctement éditée en fonction du pays auquel elle appartient.

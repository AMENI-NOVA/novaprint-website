Filtres

Précédent   Suivant

Pour plus de facilité, vous pouvez sélectionner les stocks à lister en agissant sur les critères de sélection.

Exemples

¨   Vous ne voulez voir que les familles dont le code commence par « CM », Entrez « CM* » dans le champ « Code famille » et cliquez « Rafraîchir la liste » ou tapez <Entrée>.

¨   Vous cherchez un Couché Moderne en 45 x 64 : Tapez « CM* » dans le Code famille et « ????  45  64 » dans le code article. En sélectionnant tour à tour les stocks affichés, vous pourrez consulter les quantités disponibles.

¨   Vous cherchez un stock présent immédiatement : Sélectionnez « Stock au moins » et indiquez une quantité. (Attention, le critère s’applique sur la quantité en unité principale sans tenir compte de l’unité.)

¨   Vous cherchez un stock disponible rapidement (réservations déduites) : Sélectionnez « Disponible au moins » et indiquez une quantité.

¨   Vous cherchez un stock disponible après les approvisionnements  (réservations déduites et commandes incluses) : Sélectionnez « Libre au moins » et indiquez une quantité.

Le bouton pas de critères permet de réinitialiser le filtre pour voir tous les stocks. (Cliquez « Rafraîchir » pour actualiser l’arborescence).

Optiplus

En version Optiplus, un filtre permet de sélectionner l’entrepôt où est situé le stock. L’entrepôt des stocks listés est affiché derrière le nom de son possesseur.

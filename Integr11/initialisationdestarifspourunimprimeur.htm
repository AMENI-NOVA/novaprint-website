Initialisation des tarifs pour un imprimeur

Précédent   Suivant

Le calcul des devis n'est possible pour un imprimeur à un certain tarif que si ce tarif a été initialisé pour l'imprimeur

L'initialisation du tarif n'est pas nécessaire si vous utilisez l'assistant au paramétrage.

Pour initialiser un tarif pour un imprimeur, ouvrez les rubriques Calcul des devis, Optiplus, puis double-cliquer sur Initialiser un tarif "Imprimeur".

1. Sélectionnez l'imprimeur dont vous initialiser le tarif. L'imprimeur doit avoir été créé au préalable.

2. Sélectionnez le tarif à initialiser

3. Sélectionnez l'option à utiliser pour l'initialisation des tarifs :

·    Si vous saisissez manuellement les paramétrages de l'imprimeur, cochez l'option "Saisie manuelle des coûts"

·    Si vous utiliser l'échange de fichiers XML, cochez "A partir d'un fichier de paramétrage" et indiquez l'emplacement et le nom de ce fichier (voir " Paramétrage d'un imprimeur par l'échange de fichiers XML" pour plus de détails sur la procédure à suivre).

4. Cliquez sur <Initialiser> pour initialiser le tarif pour cet imprimeur.

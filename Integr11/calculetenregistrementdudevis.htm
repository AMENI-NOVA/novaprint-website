Optimisation des devis

Le programme de calcul optimise le devis pour chaque imprimeur sélectionné ou sur tous les imprimeurs si aucun n'a été sélectionné.
Le nombre maximum d'imprimeurs pris en compte pour le calcul ne peut dépasser le nombre d'imprimeurs maximum enregistré dans la clé électronique.

Au fur et à mesure du calcul, les imprimeurs consultés apparaissent dans la liste arborescente.

Si au moins une solution a été trouvée pour l'imprimeur, la mention "Calcul terminé" apparaît en face de son nom.

Si aucune solution n'a été trouvé pour l'imprimeur, la mention "Calcul impossible" apparaît en face de son nom. Dans ce cas, elle peut être complétée par "(pas de papier)" s'il n'y a aucun tarif pour le papier sélectionné pour cet imprimeur.

Résultats du calcul

Lorsque la meilleure solution a été trouvée pour tous les imprimeurs sélectionnés, un tableau récapitulatif est affiché, présentant le total et le mille + pour les imprimeurs pour lesquels au moins une solution a pu être calculée.

Le total et le mille + affichés dans le tableau incluent :

¨   Les coefficients de conjoncture

¨   Les marges acheteur

¨   Le coefficent global

Le récapitulatif du devis est affiché pour l'imprimeur sélectionné (dans le tableau ou dans la liste arborescente).

Les boutons d'enregistrement et d'affichage du dossier technique s'applique à la solution retenue pour l'imprimeur sélectionné.

Enregistrement du devis

C'est la solution de l'imprimeur sélectionné dans le tableau ou la liste arborescente qui est enregistrée.

Le programme enregistre aussi en annexe avec chaque devis, le tableau du total et mille + des autres imprimeurs calculés. Ces informations sont visibles dans les propriétés du devis.

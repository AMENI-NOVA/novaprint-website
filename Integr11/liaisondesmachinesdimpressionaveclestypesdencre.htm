Liaison des machines d’impression avec les types d’encre

Précédent   Suivant

Chaque machine d’impression doit être liée à au moins un type d’encre pour pouvoir être prise en compte dans l’optimisation du calcul des devis.

La liaison s’affectue dans les caractéristiques de chaque machine, dans l’onglet « Types d’encre », en cochant les types d’encre autorisés pour la machine, parmi les types d’encre compatibles avec le type de machine et pour lesquels un tarif est défini (y compris dans le cas ou le tarif est de 0).

Exemple : Presses offset feuilles

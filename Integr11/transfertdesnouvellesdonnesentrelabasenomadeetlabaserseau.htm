Transfert des nouvelles données entre la base nomade et la base réseau

Précédent   Suivant

Le but de ce traitement est de transférer les nouvelles données entre la base nomade et la base réseau.

Les deux sens de transfert sont possibles :

¨   De la base nomade vers la base réseau;

¨   De la base réseau vers la base nomade.

 

Seules les données suivantes sont transférées :

¨   Les nouvelles notes avec leurs pièces jointes

¨   Les nouveaux clients et les clients modifiés

¨   Les nouveaux devis et leurs lettres de confirmations

Les autres données ne sont pas transférées par exemple, nouvelles factures, nouvelles commandes, modification d'un devis (descriptif ou commentaire), …

 

Le principe de fonctionnement est le suivant :

1. Cliquez sur Connexion, pour vous connecter à l'autre base de données.

2. Choisissez le sens du transfert avec les liste De et Vers.

3. Sélectionnez le type des données que vous souhaitez transférer : notes, devis, …

4. Sélectionnez la date de début du transfert.
Par défaut cette date correspond à la dernière fois où vous avez mis à jour la base nomade avec les données du réseau. Voir "Copier toutes les données utiles à une utilisation nomade".

5. Si nécessaire, choisissez une macro pour les devis. Cela permet de filtrer de manière précise les devis à transférer. "< Par défaut >" signifie qu'il n'y pas de filtre.

6. Cliquez sur "Transférer".

7. Une fois le transfert effectué, les données sont transférées dans la base de destination.

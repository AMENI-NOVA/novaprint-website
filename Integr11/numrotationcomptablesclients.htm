Numérotation comptables clients

Précédent   Suivant

Les numéros de comptes clients peuvent être attribués automatiquement par Graphisoft. Les comptes doivent avoir un format numérique, ce qui permet de les incrémenter.

Le paramétrage vous permet d’activer la numérotation automatique et de définir le format des comptes qui seront alors automatiquement créés de type « Auxiliaires ».

Les sociétés existantes doivent avoir des comptes vérifiant le format donné : Les comptes doivent être « Auxiliaires »,  numérique et formatés ur le nombre de chiffres indiqué. Dans le cas contraire, l’activation ne sera pas possible.

Remarque :

-   Lorsque la numérotation automatique des comptes est activée, les zones de saisie permettant de les saisir manuellement sont désactivées.

-   Le prochain numéro de compte est défini comme le plus grand numéro trouvé augmenté d’une unité.

-   Seuls les numéros comptables client sont concernés par ce paramétrage. Les comptes des fournisseurs et sous-traitants ne sont pas attribués automatiquement.

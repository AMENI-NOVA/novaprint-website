Produit

Le produit est initialisé avec celui enregistré dans le devis dont est issue la commande, mais il peut être modifié (voir "Produits fabriqués" dans "Paramétrages généraux").

Acompte versé

L’acompte versé peut être modifié tant qu’aucune facture n’a été émise pour le dossier.
Cet acompte sera pris en compte dans la première facture du dossier.

Quantité pour le prix unitaire

Si vous saisissez une quantité différente de 0 pour le prix unitaire, Graphiplus calculera le prix unitaire en fonction de cette unité, et l’affichera sur la lettre de confirmation, à la place du mille suivant.

Le prix unitaire calculé apparaît aussi à la fin du calcul dans le tableau récapitulatif.

Vous pouvez aussi saisir, modifier ou remettre à zéro cette information, dans le devis simplifié.

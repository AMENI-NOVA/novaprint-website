Calcul d’une solution pour un chemin de prépresse comportant une flasheuse

Précédent   Suivant

Dans tous les cas, les chemins pour lesquels le format maxi de la flasheuse est plus petit que le format d'1 bout (page ou modèle) ne sont pas pris en compte dans l'optimisation.

Le format d'imposition comprend les doubles coupes, le blanc tournant, la gamme Bruner et le dépassant, mais pas la prise de pince.

Flasheuse plaques

¨   Format d'imposition < format flasheuse :
Calcul flashage à partir du format d'imposition .
Calcul éventuel du montage films, copie et plaques matière (en toute logique, ces étapes ne devraient pas se trouver dans un procédé comportant une flasheuse avec une sortie plaques, mais si c'est le cas, les coûts correspondant seront tout de même calculés en fonction de la machine d'impression).

¨   Sinon : ce chemin est impossible.

Flasheuse pouvant sortir des films montés

¨   Format d'imposition < format flasheuse :
Calcul flashage pour tous les bouts (pas de montage manuel).

¨   Format d'imposition > format flasheuse :
Si le chemin ne comporte pas d'étape "Montage films", on considère qu'il est impossible dans ce cas, puisque le montage entier ne peut être flashé en une seule fois.

·    Format d'1 pose < Format flasheuse

§  Si pas d'étape "Duplis" dans le chemin :

Si la flasheuse sort des films en bobine :

Optimisation du flashage pour flasher en une seule fois le maximum de poses déjà montées, et minimiser le nombre de bouts de films restant à monter après le flashage. Par exemple dans le cas d’une imposition en 8 poses, le flashage pourra être fait par 4 poses montées  (suivant la laize de la flasheuse).

Calcul montage manuel pour les bouts montés et flashés.

Si la flasheuse sort des films en planche :

Calcul du flashage pour tous les bouts,

Calcul montage manuel pour monter les x poses.

§  Si étape "Duplis" dans le chemin :

Calcul flashage pour tous les bouts de la 1ère pose

Calcul des duplis (optimisation répétex) et du montage pour les x poses.

·    Format d'1 pose > Format flasheuse :
Possible uniquement dans le cas d'une brochure ou d'amalgame de fiches ou dépliants.
Calcul du format à flasher et du nombre de bouts à monter :

§  Si brochure :

. Format du 4 pages < Format flasheuse : flashage par 4 pages montées,

. Format du 4 pages > Format flasheuse : flashage par 2 pages,

Toutes les pages doivent être montées.

§  Si amalgame de fiches/dépliant : flashage par 1 modèle

Tous les modèles doivent être montés.

Calcul du flashage (à partir du format à flasher précédemment calculé).

§  Si pas d'étape "Duplis" dans le chemin :

Calcul flashage pour tous les bouts,

Calcul montage manuel.

§  Si étape "Duplis" dans le chemin :

Calcul flashage pour tous les bouts de la 1ère pose (ou pour 2 poses si imposition basculée),

Calcul des duplis (optimisation répétex) pour les x poses

Calcul montage manuel.

Flasheuse sortant des films non montés

¨   Si pas d'étape "Duplis" dans le chemin :
Calcul flashage pour tous les bouts : le calcul du flashage de chaque bout est calculé séparément.
Si étape "Montage films" dans le chemin : Calcul montage manuel pour monter tous les bouts.

¨   Si étape "Duplis" dans le chemin :
Calcul flashage pour tous les bouts de la 1ère pose (ou pour 2 poses si imposition basculée).
Calcul des duplis (optimisation répétex) pour les x poses.
Calcul du montage pour tous les bouts.

Installation

Précédent   Suivant

L’Assistant Fiery ne fait pas partie de la procédure d’installation Graphisoft. Si vous souhaitez l’installez, demandez les fichiers nécéssaire au support de Graphisoft.

Fichiers nécessaires :

·      Setup.exe

·      FieryWizard_Setup.msi

·      FieryDiscovery.zip

 

L’intallation doit être faite sur un ordinateur où Graphisoft est déjà installé.

 

Exécutez le fichier Setup.exe et choisisez le répertoire d’installation.

Décompressez le fichier FieryDiscovery.zip dans un dossier de votre choix.

Modifiez le fichier FieryWizard.exe.config (se trouvant dans le répertoire d’instalation indiqué dans le setup) en modifiant les valeurs suivantes :

·      FieryDiscover : Indiquez le chemin complet du dossier où vous avez décompressé le fichier FieryDiscovery.zip.

·      CommandLine : Indiquez la ligne de commande permettant de se connecter à la base de données Graphisoft. Si vous ne la connaissez pas, l’équipe du support vous la fournira.

Enfin, créez un raccourci du fichier FieryWizard.exe sur le bureau de Windows.

Saisie des propriétés d'un type de cahier

Pour la gamme Estiprint, les seules modifications autorisées concernent la laison avec les chemins de brochage.

Les traits pleins matérialisent une coupe et les pointillés un brochage.

Nom

Le nom est utilisé pour :

¨   Repérer les types de cahier dans l'arbre de paramétrage

¨   Au moment de la saisie des devis, si vous voulez imposer ou ignorer certains types de cahier.
Dans ce cas, c'est ce nom qui est enregistré avec le devis comme référence des types de cahier imposés ou ignorés. Si vous supprimez un type de cahier dans le paramétrage, il n'apparaîtra plus lorsque vous rappelerez un devis pour lequel vous avez imposé ou ignoré ce type de cahier.

Si vous modifiez le nom d'un type de cahier, il sera enregistré comme un nouveau type.

Nombre de fois la largeur et la hauteur du format fini

Ces deux informations définissent l'imposition du cahier par rapport à son format fini une fois plié.

Lorsque vous modifiez ces deux valeurs, le nombre de pages est automatiquement recalculé, et le cahier est représenté à la française et à l'italienne dans l'onglet "Aspect".

ATTENTION, lorsque vous modifiez ces valeurs :

¨   Modifiez le cas échéant le nom du type de cahier : par défaut, le nom du type de cahier indique le nombre de pages. La modification du nombre de pages ne modifie pas automatiquement le libellé.

¨   Vérifiez les amalgames qui utilisent ce type de cahier. Le fait de modifier le nombre de pages d'un type de cahier modifiera implicitement tous les amalgames qui l'utilisent (voir "Amalgames de cahiers").

La représentation utilise un format de rapport 21 x 29,7, mais chaque type de cahier peut être utilisé pour calculer des brochures de tous les formats.

Nombre et type de plis

Ces deux informations définissent le type de pliage nécessaire pour obtenir le cahier.

ATTENTION, il n'y a aucune vérification de cohérence entre l'imposition et le pliage.

D'autre part, vous ne pouvez définir qu'un seul type de pliage par cahier.

Le nombre et le type de plis sont utilisés conjointement avec les caractéristiques définies pour les plieuses.

Coming and Going

Cette case à cocher permet d’indiquer que le type de cahier est spécifique aux solutions en coming and going.

Les solutions en coming and going sont étudiées automatiquement pour les chemins de brochage comportant au moins une assembleuse cahiers pour laquelle vous avez coché « Coming and going ».

Chemins de brochage

L’onglet "Chemins de brochage" permet de préciser si le type de cahier est utilisable quel que soit le chemin de brochage (bouton « Tous ») ou seulement pour ceux sélectionnés dans la liste de droite (bouton « ceux sélectionnés »).

Le nombre de chemins qui peuvent être liés à un type de cahier n'est pas limité.

Réédition d'un document

Précédent   Suivant

Pour rééditer un document, sélectionnez-le dans la liste, puis cliquez sur Ouvrir dans le menu Documents ou dans le menu contextuel.

Le document est alors chargé et apparaît à l'écran tel qu'il a été enregistré.

Vous pouvez effectuer des modifications dans le document, l'imprimer et le réenregistrer.

Enregistrement des modifications

Une fois un document imprimé, les modifications éventuelles que vous ferez dans un document seront enregistrées dans le document lui-même, mais pas dans l'élément dont il est issu.

Par exemple, si vous modifiez la partie descriptif d'un document déjà imprimé de lettre de confirmation et que vous l'enregistrez, le descriptif du devis correspondant ne sera pas modifié. Lorsqu'il est enregistré, le document est délié de l'élément (devis, facture, …) à partir duquel il a été construit.

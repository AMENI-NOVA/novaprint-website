Ecran principal

Précédent   Suivant

L'écran principal de la messagerie se compose de trois parties :

¨   A gauche : la liste des dossiers (boîte de réception, éléments envoyés, brouillons, éléments supprimés et dossiers personnels).

¨   En haut : la liste des messages du dossier sélectionné.

¨   En bas : le détail du message sélectionné.

 

Liste des messages

Pour consulter un dossier, cliquez dessus dans la liste à gauche de l'écran. Par défaut, la messagerie s'ouvre sur la boîte de réception.

 

Une fois le dossier sélectionné, les messages qu'il contient sont affichés.
Les messages non lu sont affichés en gras.

 

Les colonnes affichées dans cette liste sont paramétrables (comme les listes de sociétés ou les historiques de notes). Pour cela cliquer sur l'icône  ou la commande "Options d'affichage" du menu "Affichage".

Voici les principales informations :

¨   Colonne PJ (pièce jointe):

 : Indique la présence d'un fichier lié au message

¨   Colonne @ (e-mail) :

 : Le message reçu est un e-mail, ou le message à été envoyé à quelqu'un par e-mail

 : Le message reçu est un e-mail, mais l'expéditeur n'est pas reconnu par GS CRM

¨   Colonne A. (action):

 : Une action à effectuer est liée au message.

 : L'action liée au message à été effectuée.

 : Au moins une action à été effectuée, mais pas toutes (seulement dans les éléments envoyés).

¨   Lu : Indique si le message à été lu:

 : Le message n'a pas été lu

 : Le message à été lu

 : Au moins un destinataire a lu le message, mais pas tous (seulement dans les éléments envoyés)

Pour les destinataires contactés par e-mail, le message sera toujours affiché comme étant non lu sauf si vous avez demandé et reçu une confirmation de lecture.

¨   Colonne Auteur : Nom de l'auteur du message.

¨   Colonne Destinataire : Nom du premier destinataire du message dans l'ordre alphabétique. Pour savoir s'il y a d'autre destinataire, il faut cliquer sur le message et consulter le détail du message (ligne "A :").

¨   Colonne Objet : Objet du message

¨   Colonne Action : Libellé des actions liées aux messages

¨   Colonne Date d'envoi : Date d'envoi du message.

¨   Colonne Confirmation de lecture :

 : Indique qu’une confirmation de lecture a été demandée.

¨   Colonne Preuve de dépôt :

: Indique qu’une preuve de dépôt a été demandée.

¨   Colonne Lecture seule :

 : Indique que la note est en lecture seule.

 

Il est possible de trier la liste des messages. Pour cela, cliquez sur l'entête d'une colonne pour effectuer le tri. La première fois, le tri sera ascendant. Si vous cliquez une autre fois le tri sera descendant.

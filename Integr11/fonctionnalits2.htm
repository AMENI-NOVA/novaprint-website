Exporter 

Les rapports peuvent être exportés dans plusieurs formats traditionnels tels qu'une feuille de calcul, un traitement de texte, HTML, PDF et la plupart des formats d'échange de données. Le processus d'exportation nécessite la spécification d'un format et d'une destination. Le format détermine le type de fichier et la destination détermine l'emplacement du fichier.

Expert sélection 

Suivant le type du rapport, permet de définir des critères de sélection d'enregistrements afin de limiter l’étendue du rapport.

La sélection se fait parmis les champs présents dans les rapports.

Par exemple, sur un rapport éditant la liste de tous les clients, il est possible de limiter cette liste à ceux dont le nom commence par « A ».

Dans l’expert sélection, cliquez sur « Nouveau… » pour ajouter un critère de sélection, s’affiche alors la liste des champs disponibles pour le rapport en cours. Le nom des champs est celui utilisé dans la base de données de Graphiplus (c’est à dire « nom de la table ». « nom du champ »).

Le bouton « Parcourir » permet d’afficher les valeurs du champ sélectionné contenues dans le rapport et peut vous aider à faire votre sélection.

Sélectionnez le champ sur lequel vous voulez effectuer la sélection, choisissez l’opération à effectuer et saisissez la valeur.

Vous pouvez insérer plusieurs champs de sélection en cliquant de nouveau sur « Nouveau.. » , dans ce cas le filtre sera condition A ET condition B.

Pour certains rapports, la liste des champs est vide, dans ce cas aucune sélection n’est possible.

Expert recherche 

Suivant le type du rapport, permet de définir des critères de recherche d'enregistrements à partir des champs affichés dans le rapport.

Il fonctionne comme l’expert de sélection. Il permet de se positionner sur le 1er enregistrement correspondant aux critères demandés.

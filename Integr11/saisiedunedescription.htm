Saisie d'une description

Précédent   Suivant

La description d’un fichier de tarifs papier s'effectue dans un seul écran dont les zones de saisies sont réparties dans 3 onglets :

¨   Généralités

¨   Champs

¨   Constantes

Les boutons "Enregistrer" et "Annuler" permettent respectivement de valider les modifications ou de les annuler.

 

Nom de la description

C’est lui qui est affiché dans l’écran principal. Il permet d’identifier une description.

Répondre & Répondre à tous

Précédent   Suivant

"Répondre" dans le menu "Notes" ou le menu contextuel.

 

Cette fonction permet de répondre à la note sélectionnée.

La fenêtre de saisie d'un nouveau message apparaît pré-remplie :

¨   Le nom du destinataire correspondant à l'expéditeur du message d'origine.

¨   L'objet contient "Réponse :" suivi de l'objet du message d'origine.

¨   Le texte contient la mention "Note d'origine" suivi du texte de celle-ci.

 

Si vous choisissez "Répondre à tous", tous les destinataires de la note d'origine seront aussi destinataires de cette note.

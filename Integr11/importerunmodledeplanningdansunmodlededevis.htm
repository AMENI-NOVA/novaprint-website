Importer un modèle de planning dans un modèle de devis

Précédent   Suivant

L'importation d'un modèle de planning dans un modèle de devis permet de définir un "modèle de dossier" qui associe un processus de fabrication et une planification.

Les informations de planification incluses dans le modèle de devis seront appliquées aux dossiers créés à partir de ce modèle.

Le modèle de planning est intégralement recopié dans le modèle de devis, ce qui permet ensuite de le personnaliser en modifiant la durée de chaque étape. Le modèle de planning n'est pas modifié et peut être importé dans d'autres modèles de devis.

Pour importer un modèle de planning dans un modèle de devis

1. Ouvrez la liste des modèles de devis

2. Sélectionnez un modèle de devis, puis choisissez l'option "Propriétés" du menu "Modèles de devis" ou du menu contextuel.

3. Sélectionnez l'onglet "Modèle de planning", puis cliquez sur le bouton <Importer>

4. Sélectionnez le modèle de planning à importer, puis cliquez sur <Valider>

5. Modifiez la durée des étapes si nécessaire, avant d'enregistrer le modèle de devis avec sa planification.

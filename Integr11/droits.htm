Les droits d’accès permettent de restreindre l'accès aux fonctions du logiciel selon les utilisateurs que vous aurez définis.

Les fonctionnalités de l’application sont regroupées en catégories. Pour développer une catégorie et avoir accès à ses fonctionnalités pour les autoriser ou les interdire, cliquez sur le signe ‘+’.

Autoriser des fonctionnalités

Pour autoriser une fonctionnalité, sélectionnez-la dans la liste "Fonctionnalités interdites", puis cliquez sur le bouton de flèche droite ou appuyez sur la barre d'espace.

Pour autoriser un groupe de fonctionnalités, sélectionnez-le dans la liste "Fonctionnalités interdites", puis cliquez sur le bouton de flèche droite ou appuyez sur la barre d'espace.

Pour autoriser toutes les fonctionnalités, cliquez sur le bouton "Tous". Ce bouton peut être utile pour paramétrer les droits d'accès d'un utilisateur ayant droit à beaucoup de fonctionnalités. Il suffit ensuite d'interdire celles auxquelles il n'a pas droit.

Interdire des fonctionnalités

Utilisez le bouton flèche gauche sur la liste "Fonctionnalités autorisées", et le bouton "Aucun", pour interdire des fonctionnalités.

Paramétrage minimum indispensable pour la gestion des droits d'accès

Lorsque vous paramétrez les droits d’accès, n’oubliez pas :

¨   d’interdire l’accès à ce paramétrage, aux utilisateurs n’ayant pas tous les droits d’accès. En effet, dans le cas contraire, ils pourraient aller s’autoriser eux-mêmes les fonctionnalités que vous leur avez interdites.

¨   de mettre un mot de passe aux utilisateurs qui ont des droits aux fonctionnalités auxquelles d’autres utilisateurs ne doivent pas avoir accès.

Droits et « Sociétés autorisées »

Les « sociétés autorisées » sont les sociétés qu’un utilisateur a le droit de voir et d’utiliser. En fonction des droits de l’utilisateur, cela peut s’appliquer à tout Graphisoft ou seulement certaines parties du programme.

 

Comment définir les sociétés autorisées ?

Ces sociétés sont définies en fonction du droit « Voir toutes les sociétés » et des droits « Voir les sociétés de X ».

·      L’utilisateur peut avoir accès à toutes les sociétés avec le droit « Voir toutes les sociétés ». Dans ce cas, toutes les sociétés sont visibles sans restriction dans l’ensemble de l’application.

·      Les droits « Voir les sociétés de X » permettent de définir les sociétés autorisées en fonction du commercial attachés à celles-ci. Il existe autant de droits « Voir les sociétés de X » que de commerciaux enregistrés.

 

Dans l’exemple suivant, l’utilisateur a accès aux sociétés dont le commercial est soit « Bertrand », soit « Sophie ».

Toutes les autres sociétés (sociétés appartenant à d’autres commerciaux ou sans commercial défini) ne sont pas autorisées.

 

Voici la liste des droits concernés par les « sociétés autorisées » :

·      Voir toutes les sociétés     ou Voir les sociétés de X ;

·      Voir tous les devis            ou Voir les devis des sociétés autorisées ;

·      Voir toutes les commandes ou Voir les commandes des sociétés autorisées ;

·      Voir toutes les factures     ou Voir les factures des sociétés autorisées ;

·      Voir tous les BLs              ou Voir les BLs des sociétés autorisées ;

·      Voir tous les documents    ou Voir les documents des sociétés autorisées.

 

« Sociétés autorisées » appliquées aux sociétés

Avec le droit « Voir toutes les sociétés », l’utilisateur n’a aucune restriction.

 

Avec les droits « Voir les sociétés de X », l’utilisateur n’a accès qu‘aux sociétés des commerciaux correspondants ou des sociétés où il est lui-même le commercial.

 

Attention

Lors de la création (ou la modification) d’une société, il faut bien vérifier que le commercial affecté à la société est « autorisé » (que l’utilisateur dispose du droit de voir les sociétés de ce commercial).

Si ce n’est pas le cas, après l’enregistrement de la société, celle-ci ne sera plus accessible à l’utilisateur.

Avant l’enregistrement de la société, un message averti l’utilisateur et demande confirmation.

 

Les restrictions sont appliquées sur :

¨   La liste des sociétés : Toutes les listes de sociétés du programme n’affichent que les sociétés autorisés;

¨   Saisie de devis : impossible de sélectionner une société non autorisée ;

¨   Devis simplifiés : impossible de sélectionner une société non autorisée ;

¨   Propriétés d’un devis : impossible de sélectionner une société non autorisée ;

¨   Création d’une commande : impossible de sélectionner une société non autorisée 

¨   Propriétés d’une commande : impossible de sélectionner une société non autorisée ;

¨   Modification d’une société : impossible d’ouvrir la fiche d’une société non autorisée ;

¨   Création d’une facture : impossible de sélectionner une société non autorisée 

¨   Transfert d’historique d’une société impossible vers une société non autorisée;

¨   Saisie d’une note : impossible de lier une note à une société non autorisée ;

¨   Saisie d’une note : impossible d’envoyer une note à une société non autorisée ;

¨   Historique des notes sociétés : Seules les notes liées à des sociétés autorisées sont visibles.

 

« Sociétés autorisées » appliquées aux devis

Avec le droit « Voir toutes les sociétés », l’utilisateur n’a aucune restriction.

Le droit « Voir tous les devis » donne accès à tous les devis sans restriction.

 

Avec le droit « Voir les devis des sociétés autorisées » seuls les devis des sociétés autorisées sont accessibles.

Attention, si l’utilisateur dispose des droits « Voir les devis des sociétés autorisées » + « Voir toutes les sociétés » cela équivaut au droit « Voir tous les devis ».

 

Dans tous les cas, un utilisateur voit toujours les devis dont il est le deviseur ou le commercial (commercial définit dans les propriétés du devis).

 

Les restrictions sont appliquées sur :

¨   La liste des devis : Toutes les listes de devis du programme n’affichent que les devis de sociétés autorisés ;

¨   Devis simplifié : il est impossible de changer de devis en saisissant un numéro de devis d’une société non autorisée ;

¨   Saisie d’une note : impossible de lier une note à un devis d’une société non autorisée ;

¨   Le tableau de charges ;

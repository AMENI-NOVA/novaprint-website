Réordonnancer (trier) les opérations d’un poste

Précédent   Suivant

Il est possible de modifier l’ordre des opérations d’un poste afin qu’il représente l’ordre logique et chronologique du déroulement des opérations sur ce poste.

Cet ordre étant utilisé entre autres lors du suivi d’un dossier pour afficher son avancement. Il est préférable que les opérations soient ordonnancées correctement pour que l’affichage de l’avancement soit représenté chronologiquement.

 

Pour modifier l’ordre d’une opération (on la fait remonter ou descendre dans l’arbre) :

1. Sélectionnez l’opération dans l’arbre.

2. Au choix :

·    Dans le menu, <Arbre> < Remonter / Descendre >.

·    Ou par le menu contextuel (clic droit à la souris sur l’opération), puis <Arbre> < Remonter / Descendre >.

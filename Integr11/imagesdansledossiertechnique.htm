Insertion d’une image

Les images proviennent de fichiers au format JPEG, GIF, PNG, BMP, TIF et WMF. Elles sont incorporées dans le modèle, donc leur présence est nécessaire uniquement lors de cette incorporation. Par la suite, elles n’ont plus besoin d’être présentes sur le disque.

Afin de réduire la taille des modèles, il est préférable d’utiliser des images provenant de fichiers compressés (JPEG), quitte à effectuer les conversions nécessaires au préalable.

Les images peuvent être insérées soit à une position fixe (on dit « Ancré dans la page »), soit dans le texte comme un caractère (on dit alors « Ancré dans le texte »). On peut les placer dans le corps, dans les en-têtes et pieds de page et dans les blocs de texte.

Pour insérer une image « Ancré dans le texte » :

¨   Positionnez le curseur à l’emplacement de l’insertion.

¨   Utilisez la commande du menu « Insertion » « Image » « Ancré dans le texte » ou cliquez sur le bouton  et « Ancré dans le texte ».

¨   Choisissez l’image. L’image est incorporée avec sa taille initiale, il est ensuite possible de la modifier (voir plus bas).

Pour insérer une image « Ancré dans la page » :

¨   Utilisez la commande du menu « Insertion » « Image » « Ancré dans la page » ou cliquez sur le bouton  et « Ancré dans la page ».

¨   Ensuite faites comme pour insérer un bloc de texte. L’image s’inscrira dans le rectangle dessiné.

¨   Finissez en choisissant l’image.

 

Déplacement d'une image

Une image « Ancrée dans le texte » se déplace comme un simple caractère, couper – coller.

Une image « Ancrée dans la page » se déplace comme un bloc de texte.

 

Modification de la taille d’une image :

La modification de la taille d’une image « Ancrée dans la page » se fait à la souris comme pour un bloc de texte.

Pour une image « Ancrée dans le texte » utilisez la fenêtre des propriétés de l’image (voir ci-après).

 

Propriétés d’une image

La fenêtre des propriétés d’un bloc permet :

¨   De changer sa taille et sa position.

¨   De déterminer la position et la distance du texte du corps entourant l’image.

 

Pour modifier les propriétés d’une image :

¨   Sélectionnez l’image en cliquant dedans.

¨   La commande du menu « Format » « Image ».

Modifier les propriétés

 

Remarque :

Pour une image « Ancrée dans le texte » il est seulement possible de modifier la taille de l’image, largeur et hauteur.

Afin de positionner au mieux une image « Ancrée dans la page » et placée dans un bloc de texte, les coordonnées « Gauche » et « Haut » peuvent être saisies soit par rapport au bord de la page (feuille) soit par rapport au bord du bloc.

 

Suppression d'une image

Une image se supprime comme du texte, après l’avoir sélectionnée, il suffit d’appuyer sur la touche <Suppr>

 

Modification de l’ordre des images :

Même commandes que pour les blocs de texte.

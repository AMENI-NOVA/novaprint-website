Les fonctionnalités de traitement et d’utilitaires sur les devis sont accessibles depuis la liste des devis.

Les fonctionnalités générales de la liste liées à l'affichage des informations (colonnes, filtres et tri), impression et exportation, sont communes à toutes les autres listes (sociétés, devis, commandes, factures, ...) et sont décrites dans le chapitre "Introduction".

Numérotation des devis

Les numéros des devis se composent :

¨   d’un préfixe paramétrable par base de données (3 lettres maximum)

¨   d’un numéro incrémenté automatiquement

¨   d’un numéro de variante séparé du numéro de devis par le caractère « / ». (NB : le caractère de séparation peut être redéfini dans Paramétrages généraux)

Informations de la liste des devis

¨   Client :  Donne accès aux informations de la société pour laquelle le devis a été calculé

¨   Coefficient : coefficient sur le total du devis

¨   Coefficient mille + : coefficient sur le mille +

¨   Coefficient interne : coefficient sur les coûts internes

¨   Coefficient matière : coefficient sur les coûts de matière

¨   Coefficient sous-traitance : coefficient sur les coûts de sous-traitance

¨   Commande : Coché si le devis a été transformé en commande au moins une fois

¨   Commercial : Code du commercial qui s’occupe de la société pour laquelle le devis a été fait

¨   Coûts : vous trouverez sous cette rubrique, une rubrique pour chaque centre de coût et une rubrique « Total » dans lesquels se trouvent les informations apparaissant dans le dossier de gestion :

· Coûts internes et montant de la marge sur les coûts internes

· Montant des achats de matière et montant de la marge sur ces achats

· Montant des achats de sous-traitance et montant de la marge sur ces achats

· Prix de revient

· Marge globale

· Valeur ajoutée

· Coefficient interne : coefficient sur les coûts internes

· Coefficient matière : coefficient sur les coûts de matière

· Coefficient sous-traitance : coefficient sur les coûts de sous-traitance

· Dans la rubrique « Total », les informations suivantes sont également présentes :

· Valeur ajoutée en % du prix de revient

· Marge interne en % du prix de revient

· Marge globale en % du prix de revient

¨   Date : date du devis

¨   Devis (complet) : numéro complet du devis sous la forme [Préfixe]Numéro[/Suffixe]

¨   Devis de prépresse : Coché s’il s’agit d’une devis de prépresse.

¨   Devis simplifié : Coché si le devis à été enregistré par le devis simplifié

¨   Devise : devise avec laquelle a été calculé le devis (peut être différente pour certains clients étrangers)

¨   Deviseur : Code de l’utilisateur qui a enregistré le devis

¨   En attente : devis mis en attente

¨   Lettre éditée : Coché si la lettre de confirmation du devis a été éditée

¨   Mille+ Vente (<Symbole de la devise principale>) : mille + dans la devise principale, coefficient inclut

¨   Mille+ Vente (Devise) : mille + dans la devise du client

¨   Numéro (non complet) : numéro du devis sans préfixe ni suffixe

¨   Pages : nombre total de pages du devis pour tous les types d’élément excepté les types spécifiques au hardcover  (plats, dos ou matière de couverture).  Les éléments de type « page de garde » sont comptabilisés.

¨   Pages (cahiers) : nombre total de pages du devis pour les éléments de type « cahier »

¨   Pages (cahiers, encart) : nombre total de pages du devis pour les éléments de type « cahier » ou « encart »

¨   Préfixe : préfixe du numéro de devis

¨   Prix sur barème : Coché si le prix du devis est défini par un barème

¨   Produit : nom du produit rattaché au devis

¨   Quantité : nombre d'exemplaires du devis

¨   Référence : titre de référence saisi au devis

¨   Relance: indique si la relance a été effectuée ou est à faire

¨   Relanceur prévu: Code de l’employé prévu pour effectuer la relance

¨   Relanceur réel : Code de l’employé qui a effectué la relance

¨   Relance prévue (date) : date prévue pour la relance

¨   Relance réelle (date) : date à laquelle la relance a été effectuée

¨   Tarif : tarif utilisé pour le calcul du devis (de 1 à 4) (gammes Graphiplus et Intégrale uniquement).

¨   Total Vente (<Symbole de la devise principale>) : total du devis dans la devise principale, coefficient inclut

¨   Total Vente (Devise) : total du devis dans la devise du client

¨   Total vente standard : total du devis hors coefficient

¨   Type de devis : suivant les types de devis définis par Graphisoft (Brochure, Dépliant, Affiche, Liasses, ...)

¨   Variante : n° de la variante du devis

Utilisation de plusieurs devises

Si un devis a été établi dans une devise dont le cours a changé, lorsque vous demandez une lettre de confirmation ou la transformation du devis en commande, un dialogue vous propose de :

¨   Conserver le prix en devise. Cette option modifiera les montants dans votre devise et donc les coefficients.

¨   Actualiser le prix en devise. Vous conservez alors les montants dans votre devise.

¨   Modifier les prix. La fenêtre des propriétés du devis est affichée et vous pouvez modifier les prix et coefficients manuellement.

¨   Recalculer le devis. Le devis est ouvert et prêt à être recalculé.

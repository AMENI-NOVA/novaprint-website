Créer un fichier de synchronisation

Précédent   Suivant

Cette fonction permet de synchroniser les données de deux ordinateurs n'étant pas connectés en eux. Pour cela, un fichier contenant les modifications effectuées est créé. Ensuite, il est copié sur disquette ou envoyé automatiquement par e-mail à l'autre ordinateur.

 

Seules les données suivantes sont transférées :

¨   Les nouvelles notes avec leurs pièces jointes

¨   Les nouveaux clients et les clients modifiés

¨   Les nouveaux devis et leurs lettres de confirmations

Les autres données ne sont pas transférées: par exemple, nouvelles factures, nouvelles commandes, modification d'un devis (descriptif ou commentaire), …

 

Le principe de fonctionnement est le suivant :

1. Choisissez si le fichier est envoyé par e-mail ou copié quelque part.

2. Sélectionnez le type des données que vous souhaitez transférer : Notes, devis, …

3. Sélectionnez la date de début du transfert.

Par défaut cette date correspond à la dernière fois où l'on a créé un fichier de synchronisation.

4. Si nécessaire, choisissez une macro pour les devis. Cela permet de filtrer de manière précise les devis à transférer. "< Par défaut >" signifie qu'il n'y pas de filtre.

5. Cliquez sur "Créer le fichier".

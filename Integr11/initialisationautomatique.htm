Initialisation automatique

Précédent   Suivant

L’initialisation automatique permet de « réinitialiser » une opération devis, un poste devis, un type de poste devis ou tout, avec les valeurs par défaut. C’est-à-dire que le programme mettra pour le service, le poste et l’opération de gestion de production la valeur par défaut correspondant au type de poste, au poste et à l’opération devis.

1. Selon le type d’initialisation souhaité :

·    Pour « réinitialiser » une opération, sélectionnez sa ligne dans le tableau

·    Pour « réinitialiser » un poste, sélectionnez une ligne où il apparaît dans le tableau

·    Pour « réinitialiser » un type de poste, sélectionnez où il apparaît dans le tableau

·    Pour « réinitialiser » tout, n’importe quelle ligne peut être sélectionnée dans le tableau

2. Menu, <Lien> < Initialisation automatique>, ou par le menu contextuel (clic droit à la souris) puis < Initialisation automatique>.

3. Dans le menu choisir :

·    <Opération : ...> pour « réinitialiser » une opération

·    <Poste : ...> pour « réinitialiser » un poste

·    <Type de poste : ...> pour « réinitialiser » un type de poste

·    <Tout> pour « réinitialiser » Tout

4. Dans le menu choisir :

·    <Simplifié : ...> pour une initialisation simplifiée.

·    <Détaillé : ...>  pour une initialisation détaillée.

·    L’initialisation détaillée lie une opération devis à une opération d’un poste gestion de production. Alors que l’ initialisation simplifiée regroupe certaines opérations devis dans une seule opération Gestion de production.

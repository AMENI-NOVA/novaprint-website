Paramètres d’affichage du tableau

Précédent   Suivant

Les éléments de présentation du tableau qui peuvent être modifiés sont les suivants :

¨   Colonnes : colonnes à afficher, ordre et largeurs des colonnes affichées

¨   Filtres sur certaines informations (par exemple afficher les employés au moins affectés sur un poste)

¨   Fréquence de rafraîchissement des informations

¨   Options de fusion des cellules contigües du tableau qui contiennent la même information

¨   Couleurs utilisées pour différencier l’état des employés, postes ou dossiers.

Les réglages ainsi effectués peuvent être enregistrés dans un modèle pour pouvoir être utilisés ultérieurement. Vous pouvez créer autant de modèles de présentation que vous le souhaitez.
Ceci ne concerne pas les couleurs qui sont globales à tous les modèles (voir le paragraphe « Autres fonctionnalités -  Préférences »)

Le programme propose par défaut 4 modèles, qui vous serviront de base pour personnaliser les votres : <Clients>, <Dossiers>, <Employés> et <Postes>. Ces modèles par défaut ne peuvent pas être modifiés ni supprimés, ce qui signifie que les modifications que vous effectuerez sur ces modèles de base devront être enregistrées dans de nouveaux modèles pour être conservées. Le paragraphe « Modèles » un peu plus loin dans ce chapitre, détaille l’utilisation de ceux-ci.

Pour modifier les couleurs d’affichage, sélectionnez l’option Préférences du menu Contrôle atelier.

Pour modifier les autres paramètres d’affichage du tableau, cliquez sur Propriétés dans le menu Modèles, ou cliquez sur le bouton  de la barre d’outils.

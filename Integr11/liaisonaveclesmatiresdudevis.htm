Liaison avec les matières du devis

Précédent   Suivant

Ce paramétrage vous permet d’établir des relations entre les matières utilisées dans les devis et le stock. Il n’est disponible que si vous avez l’option Gestion de production.

Ce paramétrage liste les matières du devis en les groupant selon les types de matière (voir Gestion des matières dans le paramétrage de Gestion de Production)

¨   Papier

¨   Films

¨   Plaques

¨   Plaques vernis

¨   Ecrans

¨   Encres

¨   Vernis

¨   Autre

¨   Opération de préparation

¨   Opérations de façonnage

¨   Formes de découpe

¨   Conditionnements

Pour obtenir la liste pour un type de matières, sélectionnez le type dans la liste déroulante. Les modifications faites dans la liste affichée sont enregistrées avant de changer de type.

Pour chaque matière listée, vous pouvez indiquer le code d’un stock (ou seulement le début de ce code) pour préparer le lien avec les stocks.

Cliquez sur le bouton <…> pour obtenir la liste de sélection des stocks ou tapez le code souhaité.

!! Attention : seul le code article est copié dans la liste pour permettre le choix automatique du stock en fonction de la société qui utilisera la matière. Pour imposer un possesseur, complétez ce code avec le code du possesseur.

Pour supprimer un lien, il suffit de supprimer le texte de la colonne  « Code Article » et d’enregistrer.

Le papier étant reconnu automatiquement par ses codes catégorie et papier, il n’est pas obligatoire de saisir ici de relation entre la matière du devis et celle du stock. Mais si les codes stocks et ceux de votre fichier papier sont différents ou si plusieurs papiers du devis peuvent correspondre à un même stock (fournis par différents fournisseurs) alors vous pouvez entrer ici une relation entre un papier du devis et une famille de stocks papiers.
NB : la création automatique des stocks à partir du fichier des papiers du devis utilise également ces relations et permet ainsi de créer plusieurs conditions d’achat (une par fournisseur) pour un même stock papier à partir de (papiers-fournisseurs) différents de votre fichier papier.

 

A la transformation d’un devis en comande :

¨   Si un lien vers un stock est défini pour la matière utilisée dans le devis, alors la liaison est établie pour gérer automatiquement, les réservations et commandes.

¨   Si le lien n’est que partiel (seulement le code de la famille par exemple) alors ce code est proposé et réduit la liste de choix lorsque vous établissez la liaison au stock.

¨   Le lien ne concerne pas l’entrepôt dans lequel le stock peut se trouver même si le dialogue de sélection affiche cette information.

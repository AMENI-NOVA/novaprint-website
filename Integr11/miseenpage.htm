Mise en page

Précédent   Suivant

Pour chaque modèle d’édition, vous pouvez paramétrer les informations concernant l’imprimante, le format de papier et la mise en page.

Quel que soit le type de modèle d’édition (lettre, facture, étiquettes, …), les informations à saisir se présentent toujours de la même manière.

Imprimante

Si vous sélectionnez une imprimante dans la liste, le modèle d’édition sera lié à cette imprimante. Cela signifie qu’il ne pourra être utilisé que sur les ordinateurs ayant accès à une imprimante de ce nom.

Si vous cochez "Non définie", le modèle ne sera pas lié à une imprimante, et il pourra être utilisé depuis n’importe quel ordinateur.
Dans ce cas, les choix des bacs et de la qualité d’impression  ne sont plus accessibles et les formats de papier disponibles sont : "A4", "US Letter" et "Spécial". Le format "Spécial" vous permet de définir un format de papier en cm.

Trier

Cochez cette case pour que les copies du document soient assemblées (1.2.3. 1.2.3.) ou décochez pour ne pas les trier (1.1.  2.2.  3.3.)

Imprimer dans l'ordre inverse des pages

Cochez cette option pour que l'ordre d'impression des pages soit inversé pour les documents comportant plusieurs pages. Ceci est utile si votre imprimante sort les feuilles les unes par dessus les autres avec la face imprimée sur le dessus.

A l'impression (enregistrement du document)

¨   Pas d'enregistrement : les documents construits à partir de ce modèle ne seront pas enregistrés lors de leur impression. Vous avez toutefois la possibilité d'enregistrer ponctuellement le document en utilisant les options de menu.

¨   Enregistrement automatique : les documents construits à partir de ce modèle seront systématiquement enregistrés lors de leur impression.

¨   Proposer l'enregistrement si modifié : si le document a été modifié, le programme demandera s'il doit être enregistré .

Dans la plupart des cas, il est très utile d'enregistrer tous les documents imprimés, pour en conserver la trace, particulièrement pour les documents modifiés.

La désactivation de l'enregistrement automatique présente néanmoins deux intérêts :

¨   Pour les modèles servant à éditer les traites : les traites étant éditées à partir d'informations de la facture, les modèles de traites sont en fait des modèles de factures. Dans le cas où la traite n'est pas éditée à la suite de la facture, vous devez utiliser un modèle de traite séparé du modèle de facture. En désactivant l'enregistrement automatique du document pour les modèles de traites, vous pouvez imprimer vos factures en enregistrant les documents correspondants, puis éditer une série de traites sans que les documents des factures soient écrasés par ceux correspondant aux traites.

¨   Les documents occupent un espace disque non négligeable. Si vous ne pensez pas régulièrement à purger les anciens documents, le fichier qui les contient peut devenir très volumineux.

Cette option ne concerne pas les types d'édition pour lesquels les documents ne peuvent pas être enregistrés : descriptif et étiquettes.

Modèle suivant

Cette option permet de définir un autre modèle du même type pour imprimer ou envoyer par mail un document imprimé à partir du modèle courant. Vous pouvez ainsi lier un modèle imprimant une facture avec le papier du bac 1 à un modèle identique mais utilisant le bac 2 pour construire une liasse.
Un modèle de facture peut être lié à un modèle ne contenant que la traite pour imprimer la facture sur un papier d’un bac et les traites séparées sur une autre imprimante ou un autre bac.

Remarques :

¨   Les modèles liés ne donnent pas lieu à l’enregistrement du document imprimé. Ainsi, seul le document imprimé avec le premier modèle est enregistré selon l’option choisie.

¨   Les étiquettes de livraison sont imprimées après les impressions liées du bon de livraison.

¨   La liaison n’est appliquée que pour l’impression et l’envoi par mail. L’enregistrement du document ne propose pas de charger le modèle suivant. Utilisez l’option « Changer de modèle » pour charger le document avec le modèle souhaité. Dans le cas d’un bon de livraison, si vous enregistrez le document, le modèle suivant n’est pas chargé, ce sont les étiquettes qui sont directement proposées.

¨   Le modèle suivant peut lui-aussi avoir un suivant et ainsi de suite. Si la chaine ainsi construite forme une boucle, l’impression sera infinie et vous devrez arrêter le programme pour l’interrompre.

¨   Lorsque vous supprimez un modèle défini comme suivant d'un autre modèle, la liaison est supprimée sans avertissement.

Marges

Elles permettent de définir les limites pour un bloc « pleine page » (voir les propriétés des blocs) et pour la répartition des étiquettes sur les planches.

Elles permettent aussi de définir des marges pour les documents enregistrés au format PDF pour les envois par e-mail.

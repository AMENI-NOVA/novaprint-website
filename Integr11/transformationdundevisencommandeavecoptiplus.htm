Transformation d'un devis en commande avec Optiplus

Précédent   Suivant

Le fonctionnement général de la transformation d'un devis en commande avec Optiplus est le même qu'avec Graphiplus.

Fiches de travail et de sous-traitance

Pour déterminer les postes de travail des fiches créées, le programme utilise le paramétrage des liens entre le devis et la gestion de production de l'imprimeur pour lequel le devis a été enregistré, excepté pour les opérations de prépresse et de façonnage au tarif acheteur. Dans ce cas, c'est le paramétrage des liens faits pour l'acheteur qui est utilisé.

Sous-traitance

Les fiches de sous-traitance générées ont comme sous-traitant l'imprimeur pour lequel le devis a été calculé.

Liaison des matières prévues aux articles du stock

En version Optiplus, si le client fourni le papier, on proposera un stock dont le possesseur est le client.

Si c’est l’acheteur qui fourni la matière le possesseur cherché sera l’acheteur.

Dans les autres cas, la matière est fournie par l’imprimeur, elle n’apparaît pas dans ce tableau mais dans l’écran suivant.

L’entrepôt cherché pour la matière est toujours le lieu où elle sera consommée. Une ligne supplémentaire dans le détail de l’article montre l’entrepôt cherché.

Lancement de Notify

Précédent   Suivant

Graphisoft Notify peut être lancé automatiquement, en même temps que Graphisoft. Pour cela, vous devez indiquer, dans les préférences d'utilisations de la messagerie, que vous souhaitez être averti de l'arrivée d'un nouveau message (voir "Préférences d'utilisations" plus haut dans ce chapitre).

Si vous travaillez avec d'autres logiciels, vous n'êtes pas obligé de laisser le programme Graphisoft ouvert en permanence. En effet, Graphisoft Notify peut fonctionner seul. Pour cela, utilisez le raccourci situé dans le menu « Démarrer », du menu "Graphisoft". Pour que le programme soit lancé systématiquement avec l'ordinateur, vous pouvez recopier ce raccourci dans le dossier "Démarrage" du menu "Démarrer".

S'il est lancé seul, Graphisoft Notify utilisera, pour la recherche des nouveaux messages, le nom du dernier utilisateur ayant travaillé dans Graphisoft sur cet ordinateur.

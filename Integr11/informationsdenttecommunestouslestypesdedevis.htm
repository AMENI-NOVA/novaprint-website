Client, en-cours et interlocuteur
Client

Tapez le nom (interne) d’un client ou seulement ses premières lettres. Au fur et à mesure de la saisie, la liste déroulante se positonne sur la société dont le nom commence par les lettres saisies. Vous pouvez utiliser les touches de flèches Haut et Bas pour sélectionner la société voulue.

Si la société n'existe pas dans la liste, tapez son nom dans la zone de saisie, puis appuyez sur la touche Tabulation. Une boîte de dialogue vous demande alors si vous désirez ajouter un nouveau client. Si c’est le cas, l’écran de saisie de mise à jour d’une société vous permet de saisir les informations nécessaires (voir le chapitre correspondant).

Après avoir sélectionné une société existante dans le fichier, les informations la concernant viennent s’afficher : interlocuteur par défaut, tarif et coefficient.

Si vous tapez le nom d'un client archivé, le programme vous propose de le réactiver.

Si vous n'indiquez pas de client, vous pourrez tout de même calculer le devis, mais vous ne pourrez l'enregistrer qu'en attente ou comme modèle de devis.

Coefficient

Lorsque vous sélectionnez un client, le coefficient du devis est initialisé avec son coefficient par défaut qui peut être modifié localement pour le devis.

Si l’utilisateur dispose du droit « Saisie du coefficient global », ce coefficient est le coefficient global sinon, il s’agit du coefficient interne.

Nb : Dans le paragraphe suivant ‘Coefficient affiché’ fera référence soit au coefficient global, soit au coefficient interne selon le droit dont dispose l’utilisateur.

Si vous avez chargé un devis pour lequel le coefficient du mille + est différent du coefficient affiché :

¨   Si vous ne modifiez pas le coefficient affiché : le coefficient du mille + sera conservé tel qu’il est dans le devis d’origine

¨   Si vous modifiez le coefficient affiché, le coefficient du mille + sera initialisé avec le nouveau coefficient global (si l’utilisateur ne dispose pas du droit de saisie du coefficient global, celui-ci sera recalculé depuis le coefficient interne MAIS cela uniquement lorsque le devis sera recalculé).

 

Si vous avez chargé un devis pour lequel des coefficients détaillés ont été appliqués (par nature ou par centre de coût), le coefficient global indique le coefficient global calculé à partir du total après application de tous les coefficients détaillés, par rapport au total sans coefficient.

Remarques :

¨   Vous pouvez modifier le coefficient du mille + dans le devis simplifié ou dans les propriétés du devis.

¨   Vous pouvez définir un coefficient minimum autorisé par employé (voir « Employés » dans le chapitre « Paramétrages généraux »).

En-cours

Si l’en-cours est géré pour le client et si vous disposez des droits suffisants, un bouton présente l’encours actuel. Cliquez le bouton pour accéder à l’échéancier.

L'accès aux en-cours client est impossible pour la gamme Estiprint et nécessite l'option "Vente & En-cours" pour la gamme Graphisoft.

Interlocuteur

Il s'agit de la personne à contacter chez ce client (celle à qui sera adressée la lettre de confirmation du devis). Sélectionnez l'interlocuteur dans la liste de ceux qui sont enregistrés pour la société sélectionnée. Si votre interlocuteur n'a pas encore été enregistré, tapez son nom dans la zone de saisie, puis appuyez sur la touche 'Tabulation'. Une boite de dialogue vous demande alors si vous désirez ajouter un nouvel interlocuteur pour cette société. En répondant 'Oui', le programme ouvre la fenêtre de saisie de la fiche société, positionnée sur un nouvel interlocuteur.

Description générale
Référence

Vous disposez de 200 caractères pour entrer un texte libre qui vous aidera à identifier le devis ultérieurement. La référence peut contenir plusieurs lignes.

Devis de prépresse

Cette case à cocher n'apparaît que lorsque vous chargez un devis de prépresse existant. En la décochant, vous pouvez transformer le devis de prépresse en devis d'impression. Vous pouvez alors compléter la saisie puis relancer le calcul.

Produit

Vous pouvez sélectionner un type de produit parmi ceux que vous avez paramétrés dans la liste "Produits fabriqués" (par l'entreprise) (voir . Vous pouvez ainsi classer les devis selon votre propre nomenclature.
Le type de produit est un des critéres de sélection et d'affichage de la liste des devis.

Certification

Vous pouvez sélectionner une famille de certifications qui restreindra le calcul du devis aux papiers dont la certification appartient à cette famille.

Voir « Certifications (PEFC, FSC, …) » dans le chapitre « Paramétrage de calcul des devis ».

Article

Cette zone permet de lier le devis à un article de produit fini géré en stock.

Voir « Description et devis d’un article » dans le chapitre « Articles du stock ».

Quantités supplémentaires

Ce bouton vous permet de faire apparaître 3 zones dans lesquelles vous pouvez saisir des quantités supplémentaires.

 

Le calcul des quantités supplémentaires peut être effectué de deux manières différentes :

¨   Optimisation sur chaque quantité : c'est le choix par défaut.
Dans ce cas, le devis est optimisé pour chaque quantité saisie. La solution technique retenue pourra donc être différente pour chaque quantité. Cette méthode doit être utilisée si les quantités sont très différentes.
A la fin du calcul, la solution technique retenue pour chaque quantité apparaît dans un onglet différent.

Lors de l'enregistrement du devis, chaque quantité supplémentaire est enregistrée comme une variante (avec le même numéro de devis, suivi d'un suffixe qui identifie la variante).

¨   Calcul des quantités supplémentaires à partir du mille suivant de la première quantité, sur laquelle s'effectue l'optimisation. Dans ce cas, la solution technique est la même pour chaque quantité. Cette méthode peut être utilisée si les quantités sont similaires.
A la fin du calcul, le prix pour chaque quantité supplémentaire est présenté dans un tableau.

Dans ce cas, il n'y a qu'un seul devis enregistré.

La méthode de calcul peut être choisie dans la partie avancée de la saisie en cochant ou non "Optimisation sur chaque quantité".

Remarques :

La saisie de quantités supplémentaires n'est pas possible

¨   S’il y a plusieurs versions (dans le cas où des éléments ont des quantités différentes)

¨   En mode "Optiplus".

Gâche de façonnage

Ce pourcentage est utilisé pour majorer le nombre d'exemplaires saisi pris en compte pour le calcul, de manière à simuler une gâche de façonnage.
Cette gâche est prise en compte pour toutes les opérations, même le façonnage (coupe, assemblage, brochage), mais pas pour le conditionnement ni la livraison.
Le mille suivant sera calculé à partir de la quantité saisie sans le pourcentage de gâche de façonnage.

Pour les devis de brochures, dépliants et affiches, vous pouvez aussi saisir un pourcentage de gâche différent pour chaque élément, qui viendra s’ajouter au pourcentage de gâche global (voir « Optimisation »).

NB : Vous pouvez obtenir un calcul des gâches plus précis en paramétrant des valeurs de gâches (calage et roulage) séparément pour toutes les machines de façonnage, opérations de surface et opérations de façonnage. Dans ce cas, les gâches sont calculées précisément opération par opération et prises en compte pour le calcul des opérations précédentes dans le processus de fabrication. Par exemple, si la plieuse prévue engendre une gâche de 50 feuilles, le programme de calcul prévoira d'imprimer 50 bonnes feuilles de plus  (voir le chapitre "Paramétrages de calcul des devis").

Quantité pour le prix unitaire

Si vous saisissez une quantité différente de 0 pour le prix unitaire, Graphisoft calculera le prix unitaire en fonction de cette unité, et l’affichera sur la lettre de confirmation, à la place du mille suivant.

Le prix unitaire calculé apparaît aussi à la fin du calcul dans le tableau récapitulatif.

Vous pouvez aussi saisir, modifier ou remettre à zéro cette information, pour un ancien devis, en passant par le devis simplifié ou les « Propriétés du devis ».

Poids d’un exemplaire

Par défaut, le poids moyen d’un exemplaire est calculé automatiquement mais il est possible de forcer ce poids en saisissant la nouvelle valeur dans cette zone. Si le poids affiché dans cette zone à été entré manuellement, la case  voisine (Saisie manuelle) sera cochée.

En cas de saisie manuelle, tout changement dans le devis susceptible de modifier le poids entrainera un message demandant à l’utilisateur s’il veut utiliser le nouveau poids calculé ou garder la saisie.

Prépresse
Prépresse global au devis ou par élément

La saisie du prépresse diffère suivant le type de devis :

¨   Brochures, Dépliants, Affiche/fiche

Pour ces types de devis, il y a une saisie globale et une saisie par élément :

·    Prépresse global :

Permet de saisir des opérations de prépresse qui ne seront comptées qu’une seule fois quel que soit le nombre d’éléments composants le devis ou des opérations qui seront calculées pour tous les éléments du devis :

§  Si l’opération sélectionnée ne comporte pas d’unité fixe (page, couleur, dm2 ou modèle), elle ne sera comptée qu’une seule fois pour l’ensemble du devis

§  Si l’opération sélectionnée comporte au moins une unité fixe, le nombre d’unités sera calculé en fonction de chaque élément du devis. Par exemple pour une opération par page, le nombre d’unités pour le calcul de cette opération sera la somme du nombre de pages de chaque élément.
Les coûts fixes ne sont comptés qu’une seule fois.

·    Prépresse par élément :

Permet d’indiquer le type de prépresse qui détermine le mode de calcul, et des opérations de prépresse propres à chaque élément de devis.

¨   Affiches multi-morceaux, Liasses/Carnets, Continu

Le prépresse est toujours saisi globalement pour le devis : type de prépresse et opérations.

La saisie du prépresse est la même que le prépresse soit global ou par élément : reportez-vous paragraphe « Préparation (prépresse) » de ce chapitre pour plus de détails.

Opérations de prépresse ajoutées automatiquement pour les nouveaux devis

Des opérations peuvent être automatiquement ajoutées  lors de la création d’un nouveau devis : pour cela, dans le paramétrage des opérations de préparation prépresse du devis, cocher la case « Ajouter automatiquement aux nouveaux devis ».

Paramétrage automatique

Précédent   Suivant

Cette fonction permet d'initialiser Graph'e-mail avec les paramètres les plus souvent utilisés.

Les éléments pouvant être initialisés ici sont les suivants :

¨   Types de notes : Les types sont utilisés pour trier les notes dans l'historique.

·    Administrative

·    Commerciale

·    Technique

·    Changement de prix d'un devis

¨   Actions : Les actions sont utilisées lors de l'envoi d'un message. Elles indiquent aux destinataires les actions qu'ils doivent faire.

Les actions suivantes sont paramétrées sans date ni réponse obligatoire.

·    A suivre

·    Me tenir au courant

·    Prendre rendez-vous

·    Rappeler

·    Relancer

¨   Moyens de contacts : Les moyens de contacts sont utilisés lors de la saisie d'une note liée à une société. Ils permettent de trier et filtrer les notes dans l'historique des notes sociétés.

·    Appel téléphonique donné

·    Appel téléphonique reçu

·    Courrier envoyé

·    Courrier reçu

·    E-mail envoyé

·    E-mail reçu

·    Rendez-vous

·    Réunion

·    Télécopie envoyée

·    Télécopie reçue

·    Prospection

 

Tous ces éléments sont sélectionnés par défaut. Pour les désélectionner, il faut cliquer sur leur case à cocher.

 

Le bouton "Enregistrer" ajouter à votre paramétrage tous les éléments sélectionnés.

Le bouton "Annuler" quitte la feuille sans rien enregistrer.

Purger l'historique

Précédent   Suivant

"Purger l'historique " dans le menu "Outils".

Seulement si vous avez le droit "Notes d'autres personnes / Supprimer les notes"

 

Cette fonction permet de purger l'historique des notes, soit :

¨   en supprimant entièrement les notes,

¨   en supprimant seulement leurs pièces jointes.

 

Vous pouvez, soit seulement supprimer la note actuellement sélectionnée, soit supprimer toutes les notes de la liste. Dans le second cas, le programme tient compte des filtres actuellement utilisés dans la liste.

 

Suppression entière des notes

La suppression entière d'une note va supprimer la note de manière définitive.

Pour supprimer entièrement une note, il faut que la case "Supprimer seulement les pièces jointes…" soit décochée.

Conséquences :

¨   La note et ces pièces jointes sont totalement détruites;

¨   La note disparaît de tous les historiques de notes et des boîtes (d'envois ou de réceptions) de la messagerie;

¨   La place utilisée par les pièces jointes sur le disque dur est de nouveau disponible pour de nouvelles pièces jointes.

Suppression des pièces jointes

La suppression des pièces jointes supprime seulement les pièces jointes des notes et conserve le contenu des notes.

Pour supprimer seulement les pièces jointes, il faut que la case "Supprimer seulement les pièces jointes…" soit cochée.

Vous devez ensuite indiquer la taille minimum des pièces jointes à supprimer. Par défaut, le programme vous propose du 2 Mo (soit 2048 Ko).

 

Conséquences :

¨   Seules les pièces jointes sont totalement détruites, la note existe toujours;

¨   La note est toujours dans les historiques de notes et dans les boîtes (d'envois ou de réceptions) de la messagerie;

¨   La place utilisée par les pièces jointes sur le disque dur est de nouveau disponible pour de nouvelles pièces jointes.

¨   Dans l'affichage des notes, une icône indique des pièces ont été supprimées.

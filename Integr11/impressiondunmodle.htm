Impression d’un modèle

Précédent   Suivant

Pour imprimer le modèle :

¨   Soit utilisez la commande du menu « Fichier » « Imprimer » qui fait systématiquement passer par la fenêtre suivante et permet éventuellement de changer l’imprimante de destination, le format de papier, etc.
L’imprimante sélectionnée dans la mise en page du modèle est sélectionnée par défaut

¨   Soit cliquez sur le bouton. Dans ce cas la fenêtre de choix de l’imprimante n’est affichée que si la destination (l’imprimante) de l’impression ne peut être déterminée. En effet si l’imprimante choisie dans la mise en page du modèle est définie et existe sur le poste, l’impression lui est directement envoyée.

 

Lors de l’impression le modèle le document est envoyé sur l’imprimante sélectionnée. Si le format du modèle est trop grand pour tenir sur le papier de l’imprimante de destination, il est proposé « zoomer » l’impression afin d’éviter de tronquer des parties du modèle.

On peut ainsi définir un modèle utilisant un format de papier A3 sur l’imprimante « non définie » et l’imprimer sur une imprimante ne disposant pas de ce format en appliquant un facteur de zoom.

Si les marges du modèle sont plus petites que les marges physiques de l’imprimante, l’interface propose de les corriger. Elle fait de même pour des positions d’en-tête ou pied de page trop petites.

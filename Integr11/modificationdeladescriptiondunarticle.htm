Modification de la description d’un article

Précédent   Suivant

La description d'un article peut être modifiée en ouvrant cette description depuis l'article (menu "Modifier la description").

La description de l’article est ouverte dans la saisie de devis.

Vous pouvez modifier toutes les informations de la description technique de l’article, mais les devis et commandes créées précédemment aux modifications de la description ne seront pas modifiées.

Pour pouvoir remplacer la description, vous devez calculer le devis et l’enregistrer. Un message d’avertissement vous indique que les modificatons effectuées ne seront pas appliquées aux devis ou commandes liés à cette article, déjà existantes.

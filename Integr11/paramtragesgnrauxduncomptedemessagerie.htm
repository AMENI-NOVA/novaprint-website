Paramétrages généraux d'un compte de messagerie

Précédent   Suivant

Nom du compte

Le libellé saisi ici sert à identifier le compte dans GS CRM. Deux comptes ne peuvent pas avoir le même nom.

Nom de l'expéditeur des messages

Spécifie le nom convivial associé à votre adresse de messagerie. Lorsque vous envoyez des messages, ce nom apparaît dans la partie expéditeur.

Adresse de messagerie

Correspond à l'adresse de messagerie du compte.
Vous pouvez obtenir ces informations auprès de votre fournisseur de services Internet ou de votre administrateur réseau.

Adresse de réponse

Spécifie que vous souhaitez que les réponses à vos messages sortants soient envoyées à une adresse de messagerie différente, que vous pouvez taper ici.

Par exemple, vous pouvez envoyer des messages via un compte d'e-mail, saufrere@graphisoft.fr et spécifier que vous souhaitez que les réponses soient envoyées à gsupport@graphisoft.fr.

Si vous ne sélectionnez pas cette option, les réponses à vos messages électroniques sont envoyées au compte de messagerie à partir duquel vous avez envoyé ces messages.

Enregistrement des e-mails reçus

Tous les messages de ce compte seront enregistrés dans la boîte de réception de l'employé indiqué ici.

La liste contient tous les employés définis dans Graphisoft, sauf ceux de type "Atelier".

Récupération des e-mails reçus

Précédent   Suivant

La messagerie peut envoyer des e-mails, mais elle peut aussi en recevoir. Pour cela elle consulte vos messages et copie les nouveaux e-mails dans Graphisoft. Une fois que le nouvel e-mail est copié dans la messagerie de Graphisoft, le message peut être envoyé à une ou plusieurs personnes pour les informer.

 

Méthodes pour accéder aux e-mails

Il y a 2 méthodes possibles :

¨   Utiliser un programme de messagerie déjà installé sur l'ordinateur.
GSCRM peut se lire les e-mails de Microsoft Outlook ou tout autre programme de messagerie compatible avec le protocole Mapi qui est déclaré par défaut dans Windows.

¨   Interroger directement votre compte de messagerie via le protocole POP.

Voir le paragraphe "Configuration pour l'envoi et la réception d’e-mails" pour savoir comment choisir et paramétrer chacune des deux méthodes.

 

Méthodes pour récupérer les e-mails

Il y a 3 méthodes possibles :

¨   Importer un e-mail :
Menu "Outils" de l'écran de messagerie.
Cette fonction permet d'importer un e-mail contenu dans un fichier. Le fichier doit être au format RFC822 (format standard des e-mails). La plupart des messageries permettent d'exporter les e-mails avec ce format.
Une fois le fichier sélectionné, il est affiché dans une fenêtre.
Le bouton "Enregistrer dans la boîte de réception", importe l'e-mail dans GS CRM.

¨   Récupération manuelle des e-mails :
Menu "Outils" puis "Récupération des e-mails" de l'écran de messagerie.
Cette fonction affiche les e-mails contenus dans votre compte de messagerie. Vous pouvez ensuite sélectionner les e-mails à importer dans GS CRM.
Si plusieurs comptes sont disponibles, il faut tout d'abord sélectionner celui à interroger.
Les e-mails déjà récupérés sont affichés en gris. Rien ne vous empêche de les récupérer une nouvelle fois.
Le bouton "Télécharger" permet de récupérer les e-mails sélectionnés.

¨   Récupération automatique des e-mails :
Une fois cette fonction activée, GS CRM va régulièrement rechercher et enregistrer tous les nouveaux e-mails, sans intervention de l'utilisateur.
Pour activer cette fonction voir le paragraphe "Paramétrage".

 

Destinataires du message

Une fois le nouveau message enregistré dans Graphisoft, il est envoyé à l'employé ayant la même adresse Internet que le destinataire de l'e-mail. Si l'adresse Internet ne correspond à aucun employé, le message est envoyé à l'utilisateur qui a demandé la récupération des e-mails.
Il est possible d'informer automatiquement d'autres personnes (voir le paragraphe "Paramétrage").

 

Expéditeur du message

Avec l'adresse e-mail, le programme essaye de déterminer qui est l'expéditeur du message. Si l'adresse correspond à l'adresse e-mail d'une société enregistrée dans Graphisoft, une note automatique peut être ajoutée à l'historique des notes de cette société. Voir la section "Notes automatiques" du chapitre "Notes".
De même, si vous utilisez une messagerie unifiée qui centralise les e-mails, fax et messages vocaux, GS CRM permet d'identifier l'expéditeur d'un message vocal ou d'un fax à partir de son numéro de téléphone, mobile ou fax.
Enfin, le programme recherche si le texte du message contient des numéros de devis ou de commande. S'il en trouve, le message sera lié automatiquement aux devis ou commandes correspondants.

Dans la boîte de réception, un e-mail apparaît avec l'image suivante :

 : Le programme à déterminé l'expéditeur.

 : Le programme n'a pas déterminé l'expéditeur de cet e-mail.

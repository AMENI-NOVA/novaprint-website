Produits fabriqués

Précédent   Suivant

Les produits fabriqués vous permettent de gérer votre propre nomenclature de produits.

Le paramétrage des produits se trouvent dans la rubrique Entreprise, Produits fabriqués.

Propriétés des produits
Groupe de compte associé

Vous pouvez lier un produit à un groupe de compte par défaut (voir "Comptabilité" dans le chapitre "Paramétrages de facturation et comptabilité").

Lorsque vous préparez une facture pour une commande pour laquelle un produit est indiqué, le taux de TVA est initialisé en fonction du groupe de compte associé au produit.

Le groupe de compte associé n'est visible que si le module "GS Voice" est actif.

Utilisation des produits

Les produits fabriqués par l'entreprise sont utilisés dans les cas suivants :

¨   Employés :
Vous pouvez créer des secteurs commerciaux en fonction de vos produits.
Ces secteurs peuvent être utilisés pour l'envoi automatique de messages aux commerciaux affectés à ces produits.

¨   Sociétés :
Vous pouvez enregistrer pour chaque société les produits de votre entreprise qu'elle a achetée.
Les produits peuvent être utilisés pour des statistiques sur les sociétés et peuvent être édités sur la fiche de la société.

¨   Devis :
Chaque nouveau devis peut être rattaché à un produit de cette liste, ce qui permet de les classer autrement que par les types de devis fixes (brochure, dépliant, affiche/fiche, ...).

¨   Modèles de devis :
Les produits sont aussi utilisés pour classer les modèles de devis (voir "Modèles de Devis" dans le chapitre "Devis optimisé").
Un produit pour lequel il existe des modèles de devis ne peut pas être supprimé.

¨   Commandes :
Le produit est enregistré dans la commande et peut être modifié par les propriétés de la commande.
Lorsque vous transformez un devis en commande, le produit indiqué pour le devis est transmis à la commande.

¨   Factures :
Le produit est enregistré dans chaque facture (séparément pour chaque élément d'une facture) et peut être modifié dans l'écran de détail d'un élément de facturation.
Le produit de la facture est intialisé avec celui de la commande dont est issue la facture.

¨   Editions du chiffre d'affaires à partir de la gestion de prodution :
Vous pouvez éditer le chiffre d'affaires par produit

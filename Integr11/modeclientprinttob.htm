Les particularités du devis optimisé en mode client PrintToB avec Optiplus sont les suivantes :

Sélection des imprimeurs

La sélection des imprimeurs n'est pas visible pour les clients PrintToB.

En revanche, si le devis (ou le modèle) chargé comporte une sélection d'imprimeurs, celle-ci s'appliquera sur tous les calculs faits à partir de ce devis.

Calcul des devis

¨   Les imprimeurs sur lesquels le devis est optimisé n'apparaissent pas dans la liste arborescente

¨   A la fin du calcul, le programme affiche le total de la solution trouvée pour l'imprimeur le moins cher et enregistre ce devis automatiquement. Le nom de l'imprimeur du devis retenu n'apparaît pas.

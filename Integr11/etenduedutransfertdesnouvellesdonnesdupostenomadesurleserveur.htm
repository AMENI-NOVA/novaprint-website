Données à transférer

Voici le détail des informations synchronisées pour chaque type de données :

Nouvelles notes

¨   La note elle-même,

¨   Les pièces jointes de la note,

¨   Les informations internes liées à la note,

¨   Les sociétés liées à la note (même si la case "Nouveaux clients"n'est pas cochée).

¨   Les devis liés à la note (même si la case "Nouveaux devis" n'est pas cochée).

Nouveaux clients ou clients modifiés

¨   Les informations sur le client,

¨   Les interlocuteurs du client,

¨   Les informations administratives et techniques du client.

Nouveaux devis

¨   Les informations du devis,

¨   La société liée au devis (même si la case "Nouveaux clients" n'est pas cochée),

¨   Les lettres de confirmation du devis,

 

Toutes les nouvelles données à partir du

Les données créées (ou modifiées, pour les clients) après cette date seront transférées.

Par défaut cette date correspond à la dernière fois où l'on à mis à jour la base nomade avec les données du réseau. Voir "Copier toutes les données utiles à une utilisation nomade".

Macro à utiliser pour les devis

Il est possible de sélectionner les devis à transférer de façon très précise. Le premier critère est que la date de création soit supérieure à la date saisie dans la zone "Toutes les nouvelles données à partir du". Pour ajouter d'autres critères, utilisez une macro.

La liste correspond aux macros définies dans la liste des devis de la base nomade. Si vous sélectionnez une macro, les critères de filtres de celle-ci sont ajoutés au critère sur la date de création. Il est conseillé de vérifier la validité de la macro en l'utilisant sur la liste de devis de la base nomade.

La macro "< Standard >" ne contient aucun filtre.

Numéros de devis à transférer

Tapez les numéros de devis concernés en les séparant par des points-virgules.

Indiquez des groupes de devis en insérant un trait d'union entre le premier et le dernier numéro.

Par exemple, pour transférer les devis 2, 4, 5, 6 et 8, tapez 2;4-6;8

 

Pour transférer tous les devis à partir d'un numéro : Saisissez seulement le premier numéro suivi d'un trait d'union sans indiquer le dernier numéro (ex : "257 -" = "257 - 99999999…").

Pour transférer tous les devis jusqu'à un numéro : Saisissez seulement un trait d'union suivi du dernier numéro sans indiquer le premier numéro (ex : "- 257" = "0 - 257").

 

Si rien n'est saisi, cette contrainte sur le numéro des devis est ignorée.

 

Pour qu'un devis soit transféré, il faut que :

¨   Sa date de création soit supérieure ou égale à la date saisie dans "Toutes les nouvelles données à partir du", ET qu'il corresponde aux critères de la macro sélectionnée,

¨   OU que son numéro corresponde aux numéros de "Numéros de devis à transférer".

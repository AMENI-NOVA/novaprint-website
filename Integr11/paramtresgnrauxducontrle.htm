Paramètres généraux du contrôle

Précédent   Suivant

Libellé

C’est le nom du contrôle. Il est obligatoire et doit être unique

Contrôle bloquant

Permet d’indiquer si le contrôle est bloquant pour la suite du processus.

Les effets d'un contrôle bloquant sont décrits dans le paragraphe "Contrôles bloquants" de ce chapitre.

Remarque : La validation du contrôle est nécessaire pour les contrôles bloquant.  Si l’option « Contrôle bloquant » est sélectionnée, l’option « validation du contrôle » est forcée.

Durée

Permet de définir une plage de temps (en jours ouvrés) dans laquelle le contrôle doit être fait.

La saisie de la durée n’est possible que pour les contrôles non liés aux services

Activation du contrôle : Personne à notifier

Cet ensemble d’options permet de spécifier quelle est la personne à notifier lors de l’activation du contrôle.

Les valeurs possibles sont :

¨   Ne pas notifier (pas de notification)

¨   Commercial du dossier

¨   Interlocuteur défini pour le dossier

¨   Information complémentaire du dossier. Vous devez dans ce cas sélectionner  l’information complémentaire dans la liste située en regard. Cette liste affiche les informations complémentaires de type fonction ou interlocuteur. La personne, spécifiée pour l'information complémentaire du dossier concerné par le contrôle, sera notifiée.

Remarque : Lorsqu’une option de notification est sélectionnée, l’onglet "Mise en forme des notifications" devient accessible et permet de saisir le message de notification.

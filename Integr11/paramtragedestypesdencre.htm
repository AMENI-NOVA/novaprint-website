Ajout d’un type d’encre

Pour ajouter un type d’encre, sélectionnez la rubrique « Encres et vernis », puis cliquez sur Nouveau dans la barre d'outils ou dans le menu contextuel.

Sélectionnez le type de machine lié à ce type d’encre : seules les machines de ce type pourront utiliser ce type d’encre.

Prix et consommations des encres

Les prix et consommations sont paramétrables en fonction du type de couleur :

¨   Encre noire

¨   Encre primaire

¨   Encre PMS

NB : Il est possible de définir des prix et/ou des consommations spécifiques par type d’encre, par catégorie de papier ou par papier.

Modes de calcul de l’encre

Graphisoft vous permet de choisir entre 2 méthodes pour le calcul de l’encre :

¨   Calcul de l’encre sur la base d’un seul type de couleur

¨   Calcul de l’encre cumulatif par type de couleur

Pour sélectionner la méthode de calcul de l’encre, sélectionnez la rubrique « Encres et vernis », puis cliquez sur Propriétés dans la barre d'outils ou dans le menu contextuel.

Calcul de l’encre sur la base d’un seul type de couleur

Ce mode de calcul est actif  lorsque « Calcul de l’encre cumulatif par type de couleur » est décoché.

Le type de couleur est déterminé pour un élément de devis de la manière suivante :

¨   Si l'impression est entièrement en noir, c'est le prix "Encre noire" qui est utilisé.

¨   Si l'impression comporte au moins une PMS, c'est le prix "Encre PMS" qui est utilisé.

¨   Dans les autres cas, c'est le prix "Encre primaire" qui est utilisé.

Le calcul de l’encre ne tient pas compte du nombre de couleurs, mais tient compte du nombre de côtés imprimés : pour un même travail, le coût de l’encre pour une impression recto/verso sera le double du coût de l’encre pour une impression recto seul.

Calcul de l’encre cumulatif par type de couleur

Dans ce cas, la consommation d’encre est calculée pour chaque type de couleur utilisée pour l’élément de devis et tient compte du nombre de couleurs de chaque type.

Par exemple, pour un élément imprimé en quadri + 1 PMS recto verso, Graphisoft calculera l’encre pour :

¨   Noir recto/verso

¨   3 couleurs primaires recto/verso

¨   1 PMS recto/verso

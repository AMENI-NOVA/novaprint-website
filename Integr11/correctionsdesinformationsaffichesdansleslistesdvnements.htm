Corrections des informations affichées dans les listes d’événements

Précédent   Suivant

Les corrections  sont possibles :

¨   Pour les présences : à partir de la liste des employés

¨   Pour les affectations : à partir de la liste des employés

¨   Pour les travaux :

·     pour les postes simples (les machines sur lesquelles plusieurs employés peuvent travailler en même temps), ou les travaux n’ayant pas d’employés d’indiqués, à partir de la liste par poste

·     pour les postes multiples (les postes de travail sur lesquels il n’y a qu’une seule personne) ou les travaux concernant un employé, à partir de la liste par employé

La fonction de navigation « Atteindre » permet en toutes circonstances de retourner dans une de ces listes pour y effectuer une correction.

Toute modification confirmée est automatiquement enregistrée dans les fichiers de la gestion de production et l’affichage est remis à jour avec les données des fichiers.

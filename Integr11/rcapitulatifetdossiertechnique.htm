Récapitulatif et dossier technique

Précédent   Suivant

A la fin du calcul, un récapitulatif est affiché, comprenant :

¨   un descriptif sommaire de la solution technique retenue,

¨   le taux de certification global estimé,

¨   le détail des coûts et du mille en + pour chaque centre de coût,

¨   le montant total et le mille en +

NB : lorsque le devis comporte plusieurs versions, le mille + ne peut pas être calculé car les quantités de chaque version peuvent être différentes. Dans ce cas, la colonne du mille + n’apparaît pas dans le récapitulatif.

Pour obtenir plus d'informations techniques sur le devis calculé, vous pouvez afficher le dossier technique correspondant (sans avoir à enregistrer le devis). Vous pouvez ainsi modifier votre devis pour le recalculer, ou l'enregistrer s'il vous convient.

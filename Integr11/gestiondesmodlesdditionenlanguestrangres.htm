Activation de la gestion des modèles d’édition en langues étrangères

¨   Ouvrez le paramétrage, sélectionnez « Editions » puis cliquez sur Propriétés dans le menu Paramétrages ou dans le menu contextuel.

¨   Une fenêtre apparaît dans laquelle vous devez cocher les langues utilisées par vos clients étrangers

Dupliquer un modèle existant pour une autre langue

C’est la méthode la plus rapide pour créer un modèle dans une autre langue. En dupliquant un modèle existant, vous récupérez toute la mise en forme du modèle. Il suffit ensuite de traduire les textes qui apparaissent sur le modèle.

Pour dupliquer un modèle existant pour une autre langue :

¨   Dans la liste des paramétrages, sélectionnez le modèle à dupliquer, et cliquez sur Dupliquer dans le menu Paramétrages ou dans le menu contextuel.

¨   Dans la fenêtre qui apparaît, saisissez le nom du nouveau modèle, et sélectionnez sa langue

¨   Ouvrez le nouveau modèle et traduisez les textes fixes qu’il contient.
ATTENTION : ne traduisez pas les textes entre accolades ({ et }) car ils désignent les champs qui seront remplacés par les informations provenant du devis, et qui seront automatiquement traduites dans la langue désignée pour le modèle.

Dans l’extrait de modèle présenté ci-dessus :

·    Traduisez : « Devis n° », « Orléans », « Suivi par » et « Le »

·    Ne traduisez pas : « {Numéros du regroupement} », « {Nom vendeur} » et « {date Document} »

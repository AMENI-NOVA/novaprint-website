Type de façonnage

Pour les devis de dépliant, fiche, liasses/carnets et sérigraphie, sélectionnez ici le type de façonnage à appliquer au devis (voir le paragraphe « Chemins et types de façonnage » dans le chapitre « Paramétrages de calcul des devis »).

Les types de façonnage proposés sont ceux qui sont autorisés pour le type de devis en cours de saisie.

NB : pour les devis de brochure, la saisie du type de façonnage s’effectue dans l’entête du devis, après avoir indiqué le format fini.

Assemblage des fiches

Si vous cochez cette option, un coût d’assemblage feuilles sera calculé pour toutes les fiches du devis (tous les modèles de tous les éléments du devis).

L’assemblage des fiches n’est calculé que si :

¨   il y a au moins 2 modèles dans un élément de devis, ou si le devis comporte plusieurs éléments,

¨   et que le chemin de façonnage utilisé comprend l’étape « Assemblage feuilles ».

NB : les chemins de façonnage permettent de pré-sélectionner les assembleuses qui peuvent être utilisées.

Façonnage sous-traité

Si vous cochez cette case, Graphisoft ne calculera que la refente éventuelle avant le passage en machine, le rainage ou la découpe, et la livraison. Les autres opérations de façonnage, à partir du pliage, ne sont pas calculées car elles sont considérées comme étant sous-traitées. Dans le cas d'une impression en rotatives, le pliage est tout de même calculé s'il est fait en sortie.

Il convient alors de saisir le prix des opérations sous-traitées, et le prix du mille + de façonnage. Vous devez également indiquer s'il s'agit d'un prix de vente ou d'un prix d'achat. Dans ce dernier cas, le prix saisi, ainsi que le mille en + seront majorés d'une marge, fonction du prix de façonnage (voir "Opérations particulières de façonnage : Marges" dans le chapitre "Paramétrages de calcul des devis").

Vous pouvez aussi sélectionner le sous-traitant.

Si vous imposez des types de cahiers au façonnage, la solution trouvée à l’impression sera cohérente avec vos choix, et la refente après impression sera calculée si nécessaire, en fonction des types de cahiers à l’impression et des types de cahiers imposés pour le façonnage.

Si vous rappelez un devis avec du façonnage sous-traité et que vous modifiez la quantité, Graphisoft vous propose de recalculer le montant de façonnage au prorata.

Si vous rappelez un devis avec du façonnage sous-traité saisi en prix d’achat et que la marge du fichier a changé depuis que ce devis a été calculé, Graphisoft vous propose de prendre en compte la nouvelle marge.

Le prix saisi correspond à la quantité principale du devis. Si vous avez saisi plusieurs quantités pour le devis, le coût qui sera pris en compte pour les autres quantités sera calculé à partir du prix de base et du mille + que vous avez saisi, en fonction de la différence entre la quantité à optimiser et la quantité principale.

Opérations particulières de façonnage
Opérations enregistrées dans le paramétrage

Pour ajouter une opération de préparation se trouvant dans votre paramétrage, cliquez sur le bouton « Ajouter » du cadre « Opérations supplémentaires ».

Une nouvelle fenêtre est affichée, vous permettant de pointer l’opération à ajouter au devis.

Les opérations sont regroupées par famille. Cliquez sur le ‘+’ se trouvant à gauche de chaque famille pour voir les opérations qu’elle contiennent.

Si une opération n’a pas de tarif défini (coûts de matière et de façon à 0), vous devrez saisir son tarif au moment de lancer le calcul du devis. Si ensuite vous rappelez le devis pour le recalculer, le tarif saisi précédemment sera réutilisé automatiquement. Si entre temps, un tarif est saisi dans le paramétrage pour cette opération, c'est ce tarif qui sera alors utilisé.

Les opérations sélectionnées apparaissent dans le tableau récapitulatif qui comprend 3 colonnes :

¨   Le nom de l’opération

¨   Le nombre d’unités libres saisi pour ce devis (si l’opération comporte une unité libre : voir « Opérations particulières de façonnage » dans le chapitre « Paramétrages de calcul des devis »).

¨   Le libellé de l’unité libre

Le nombre d’unités libres peut être modifié dans le tableau, après que l’opération ait été sélectionnée,ou lorsque vous rappelez un devis existant avec une opération comportant une unité libre.

Pour les devis de continu, les opérations particulières de façonnage vous permettent d'indiquer les opérations des bancs de façonnage qui seront utilisés sur les machines d’impression ou d’assemblage de continu (voir "Impression continu : Opérations additionnelles" dans le chapitre "Paramétrages de calcul des devis") ou les opérations qui seront réalisées avec les données du fichier de façonnage comme pour les autres types de devis.

Pour les devis de continu, vous ne pouvez sélectionner que 6 opérations au maximum.

Le cadre "Opération" vous permet de spécifier pour chaque opération un commentaire. Ce dernier peut être affiché dans le dossier technique.

Lors de l’ajout d’une opération, si un sous-traitant est paramétré pour cette opération, ce dernier est sélectionné par défaut mais il est possible d’en sélectionner un autre ou bien encore aucun (sélectionner ‘<Non défini>’ dans la liste).

Opération hors fichier

Le cadre "Opération" vous permet d’ajouter pour ce devis une ou plusieurs opérations, mais sans les enregistrer dans la base de données (opérations exceptionnelles). Ces opérations seront prises en compte uniquement pour ce devis.

Pour cela, cliquez sur le bouton « Ajouter » et sélectionnez la première opération (elle porte le libellé « Opération Hors Fichier »).  Le cadre de saisie des opérations Hors Fichier est alors accessible.

La marge de l’opération peut être automatiquement initialisée avec une marge par défaut paramétrable (voir « Marges globales sur les opérations de façonnage » dans le chapitre « Paramétrages de calcul des devis »).

Pour modifier une opération hors fichier déjà saisie, sélectionnez l’opération dans le tableau, ce qui rend le cadre de saisie accessible.

Le commentaire des opérations hors fichier peut être affiché dans le dossier technique.

NB : Il est possible d’ajouter plusieurs opérations de ce type.

NB : Si le devis comporte plusieurs versions, les opérations hors fichier seront propres à chaque version.

Le coût total saisi pour l’opération correspond à la quantité principale du devis. Si vous avez saisi plusieurs quantités pour le devis, le coût qui sera pris en compte pour les autres quantités sera calculé à partir du coût de base et du mille + que vous avez saisi, en fonction de la différence entre la quantité à optimiser et la quantité principale.

Tout comme pour les opérations paramétrées, il est possible de sélectionner un sous-traitant.

Fonctionnement

Précédent   Suivant

Le fonctionnement est assez simple et très rapide :

Vous n’avez qu’a sélectionner le nombre de morceaux souhaités et automatiquement ils sont affichés avec :

·      Un libellé adapté en fonction du nombre de morceaux ;

·      Un temps de calage dupliqué pour tous les morceaux. Globalement le temps de calage est multiplié par le nombre de morceaux ;

·      Un temps de production adapté au nombre de morceaux. Globalement, le temps de production ne change pas.

Si tous les morceaux doivent être identiques, vous n’avez plus qu’à cliquer sur « Ok ». Sinon, vous pouvez modifier les valeurs proposées avant de valider.

Après le découpage, il est possible de modifier les fiches créées (temps et opérations) depuis le tableau de charges.

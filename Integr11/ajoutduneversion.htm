Ajout d’une version

Précédent   Suivant

Pour ajouter une version utilisez l’option « Ajouter une version » du menu « Devis optimisé ».

2 versions apparaissent dans l’arbre : la version de base qui comprend par défaut tous les éléments du devis, et la nouvelle version.

NB : un devis comporte toujours au moins une version. Lorsqu’un devis ne comporte qu’une seule version, celle-ci n’est pas visible.

L’arbre récapitule les caractéristiques de chaque version.

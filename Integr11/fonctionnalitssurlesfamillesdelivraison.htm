Fonctionnalités sur les familles de livraison

Précédent   Suivant

¨   Créer

¨   Supprimer : si des zones de livraisons existent pour cette famille, elles sont supprimées aussi.

¨   Dupliquer : copie la famille ainsi que toutes les zones de livraison qu’elle contient.

¨   Imprimer : imprime le détail de chaque zone de livraison de la famille.

¨   Propriétés : permet de modifier le nom d’une famille et d’indiquer si l’on veut voir apparaître le nom de cette famille sur les éditions ( dossier technique, lettre )

¨   Mise à jour automatique : permet d’appliquer un coefficient sur les tarifs de toutes les zones de livraison de cette famille

NB : L’importation de fichiers de tarifs de livraison pour créer ou mettre à jour les livraisons est possible depuis le menu Graphisoft – Utilitaires – Importation des tarifs livraison.

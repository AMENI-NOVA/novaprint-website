Libellés de tête et de pieds de page

Le(s) texte(s) saisis ici viendront remplacer respectivement les champs "Libellé de tête" et "Libellé de pied".

Ces libellés peuvent être utilisés pour définir des textes différents selon la personnalité du contact auquel le courrier est adressé.

NB: Si la case "Tête/Pied de pages personnalisés" n'est pas cochée, il est plus simple de saisir directement le libellé dans le modèle en utilisant éventuellement un champ {Personnalité du contact} selon l'exemple donné ci-dessous.

Formule de politesse standard

Si la formule de politesse est standard quelle que soit la personnalité, on pourra écrire directement dans le modèle :   "Espérant que cette proposition retiendra favorablement votre attention, nous vous prions de croire {Personnalité interlocuteur} en l'assurance de nos courtoises salutations.", où {Personnalité interlocuteur} est un champ représentant la personnalité interlocuteur affichée sans abrégé.

Notez que cette méthode est plus rapide et moins lourde que l'utilisation des libellés de tête/pied de page dans le cas de formule où le genre n'intervient pas.

Formule de politesse personnalisée

On utilisera les libellés personnalisés pour traiter correctement l'accord :

¨   Pour un homme :  Cher Monsieur,

¨   Pour une femme:  Chère Madame,

¨   Pour une demoiselle:  Chère Madame,     (si on ne veut pas distinguer)

¨   Pour les autres personnalités (ex: colonel): Cher ^,    (le ^ sera remplacé automatiquement par la personnalité saisie)

Fichier à joindre aux e-mails

Vous pouvez joindre systématiquement un document, reprenant par exemple vos conditions générales de vente, aux e-mails de lettre de confirmation ou accusé-réception envoyés. Cliquez sur le bouton « … » pour sélectionner le document. NB : le document doit être accessible à tous les utilisateurs du modèle.

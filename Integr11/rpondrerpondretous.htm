Répondre & Répondre à tous

Précédent   Suivant

(Valable seulement pour les messages de la boîte de réception).

La commande "Répondre" du menu "Edition" ou du menu contextuel, permet de répondre à un message.

La fenêtre de saisie d'un nouveau message apparaît pré remplie :

¨   Le nom du destinataire correspondant à l'expéditeur du message d'origine.

¨   L'objet contient "Réponse :" suivi de l'objet du message d'origine.

¨   Le texte contient la mention "Note d'origine" suivi du texte de celui-ci.

 

Si vous choisissez "Répondre à tous", tous les destinataires du message d'origine seront aussi destinataires de ce message.

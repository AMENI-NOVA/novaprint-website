Imposition

Précédent   Suivant

Le programme de calcul étudie, pour chaque élément à optimiser (ayant un format défini), toutes les impositions possibles (1, 2, ..., n poses).

Pour chaque imposition, plusieurs dispositions sont étudiées. Par exemple, pour 4 poses, on étudiera :
          1 fois le grand côté, 4 fois le petit côté
          2 fois le grand côté, 2 fois le petit côté
          4 fois le grand côté, 1 fois le petit côté

Pour les dispositions étudiées, le nombre de fois le grand côté et le nombre de fois le petit côté, sont toujours des sous-multiples du nombre de poses de la solution en cours de calcul. De cette façon, les impositions pour lesquelles toutes les poses ne seraient pas imprimées ne sont pas envisagées.

Si toutefois, on souhaite que ces solutions soient explorées, il suffit d’imposer le nombre de poses voulu, et dans ce cas toutes les dispositions de ce nombre de poses seront envisagées.

Par exemple, en imposant 5 poses, les dispositions étudiées seront :
          1 fois le grand côté, 5 fois le petit côté
          2 fois le grand côté, 3 fois le petit côté (1 pose non imprimée)
          3 fois le grand côté, 2 fois le petit côté (1 pose non imprimée)
          4 fois le grand côté, 2 fois le petit côté (3 poses non imprimées)
          5 fois le grand côté, 1 fois le petit côté

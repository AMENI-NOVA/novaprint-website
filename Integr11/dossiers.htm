Par défaut, la messagerie dispose de quatre dossiers :

¨   Boîte de réception : ce dossier contient toutes les notes qui vous avez reçues.

¨   Eléments envoyés : ce dossier contient toutes les notes que vous avez envoyées à quelqu'un. Si vous avez saisi une note sans destinataire (par exemple, un contact avec une société), cette note ne sera pas affichée ici (pour notre exemple, dans l'historique des sociétés).

¨   Brouillons : ce dossier contient les brouillons de notes.

¨   Eléments supprimés : ici sont regroupées toutes les notes supprimées de la "Boîte de réception" ou des "Eléments envoyés".

Dossiers personnalisés

Afin d'organiser les notes, il est possible de créer des sous-dossiers dans la "Boite de réception", dans les "Eléments envoyés" et dans les "Brouillons". Pour cela cliquez avec le bouton droit de la souris sur un dossier. Les commandes suivantes sont disponibles :

¨   Ajouter : Permet de créer un nouveau dossier sous le dossier sélectionné.

¨   Renommer : Permet de renommer le dossier actuellement sélectionné.

¨   Supprimer: Supprimer le dossier sélectionné. Le dossier doit être vide.

 

Pour déplacer des notes d'un dossier à l'autre :

1. Sélectionnez les notes à déplacer,

2. Faites-les glisser dans le dossier de destination.

Le dossier de destination doit être du même "type" que le dossier source. Une note issue d'un dossier des "Eléments envoyés" ne peut être déplacée dans un dossier de la "Boîte de réception". Et vice-versa.

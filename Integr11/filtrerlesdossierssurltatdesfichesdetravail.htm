Filtrer les dossiers sur l’état des fiches de travail

Précédent   Suivant

Menu : "Affichage", "Filtrer les dossiers".

A condition que le curseur soit placé auparavent dans une colonne de type GP (Poste, Information complémentaire ou Point de contrôle), il est possible d’appliquer un filtre complémentaire sur les dossiers affichés selon l’état des fiches de travail pour chaque poste inclus dans le regroupement.

Les options possibles sont :

¨   Fiches non prêtes : afficher les dossiers dont les fiches sont « non prêtes »

¨   Fiches prêtes : afficher les dossiers dont les fiches sont « prêtes mais non commencées»

¨   Fiches commencées : afficher les dossiers dont les fiches sont « commencées »

¨   Fiches non terminées : afficher les dossiers dont l’état des fiches va de « non prêtes » à « commencées »

¨   Fiches terminées : afficher les dossiers dont les fiches sont « terminées »

¨   Fiches facturées : afficher les dossiers dont les fiches sont « facturées »

¨   Toutes les fiches : afficher tous les dossiers sans restriction sur l’état des fiches.

 

Attention :

·      Ce type de filtre n’est pas enregistré.

·      En aucun cas, ce filtre ne porte sur l’état des points de controles. Ainsi, si un regroupement ne comporte aucun poste mais uniquement des points de contrôle et/ou des informations complémentaires, appliquer un filtre autre que « toutes » n’affichera aucune ligne.

·      A contrario, si un regroupement comporte plusieurs postes, le filtre sera appliqué sur les fiches de tous les postes, donc, si vous désirez ne filtrer que sur les fiches d’un seul poste, celui-ci doit être le seul poste du regroupement.

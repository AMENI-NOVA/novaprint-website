Mise à jour d’un type de prépresse (création ou modification)

Précédent   Suivant

Pour ajouter un nouveau type de prépresse, sélectionnez dans la liste, la rubrique "Types de prépresse", puis cliquez sur Nouveau dans le menu Paramétrages ou dans le menu contextuel, ou cliquez sur le bouton correspondant de la barre d’outils.

Nom

C’est ce nom qui permet de sélectionner le type de prépresse parmi la liste proposée à la saisie de chaque élément de devis.
Chaque type de prépresse doit avoir un nom unique.

Libellé pour les éditions

Ce libellé sera utilisé dans toutes les éditions du devis et dans le dossier de fabrication généré pour la gestion de production, lors de la transformation du devis en commande.

Types d'impression

Indiquez le(s) type(s) de machines d'impression concernés par ce type de prépresse parmi : Offset feuilles, Offset rotatives, Sérigraphie, Continu, Typo ou Numérique.

Par exemple, si vous n’autorisez pour un type de prépresse que les machines Offset feuilles, seules des solutions sur des machines de ce type seront envisagées lors du calcul des éléments de devis pour lesquels ce type de prépresse a été choisi.

Il est possible de choisir plusieurs types de machines pour un même type de prépresse, sauf pour les machines typo qui ne peuvent pas être choisies avec un autre type de machine d’impression.

Eléments de départ

Les choix possibles sont :

¨   films fournis

¨   films à réaliser

¨   données numériques

¨   autre (pas de film)

A la saisie du devis, pour chaque élément pour lequel on a choisi un type de prépresse ayant "Films à réaliser" comme éléments de départ, il faudra saisir le nombre et le prix des négatifs pour les films à réaliser.

Si les éléments de départ ne sont pas "Films à réaliser", pour le calcul des duplis par négatifs, Graphisoft calculera un jeu de films supplémentaires.

Duplis si films fournis

Si les éléments de départ sont "films fournis", alors Graphisoft vous invite à choisir le type d'optimisation des duplis parmi ceux qui suivent :

¨   Duplis non fournis : le coût éventuel des duplis est calculé en fonction de l’étape "Duplis" des chemins de prépresse associés au type. Si l’impression est faite en plusieurs poses, Graphisoft considère que le montage doit être fait à partir de films identiques (sans utiliser l’original), et comptera autant de duplis qu’il y a de poses.

¨   Duplis fournis à optimiser (calculer mais ne pas compter les duplis) : le coût des duplis fournis est optimisé entre négatifs et duplis au dm². Le coût au dm² n'est pas pris en compte s'il est à 0.

¨   Duplis fournis sans optimisation (ne pas tenir compte des duplis)

Cette saisie n'est pas accessible pour la gamme Estiprint. Dans ce cas, c'est l'option "Duplis fournis sans optimisation" qui sera systématiquement utilisée.

Type de prépresse par défaut

Cochez cette case, si vous désirez que ce type de prépresse soit proposé par défaut à la saisie des nouveaux devis.

Pour changer de type de prépresse par défaut, il suffit de cocher cette case pour le nouveau type par défaut. Il est inutile d’aller décocher cette case pour l’ancien type de prépresse par défaut.

Le type de prépresse ne sera proposé par défaut que pour les types de devis pour lesquels le type d'impression correspondant est coché (voir "Types d'impression"). Par exemple, si vous n'avez pas coché "Continu" pour le type de prépresse par défaut, il ne sera pas proposé pour les nouveaux devis de continu.

Chemins du type

Les chemins sélectionnés seront successivement étudiés et leur coût respectif comparés à chaque solution calculée.

Vous pouvez également ajouter ou enlever des chemins à un type, directement depuis la liste des paramétrages :

¨   Pour ajouter un chemin à un type :

·    Ouvrez la liste des chemins de prépresse

·    Cliquez sur le chemin avec le bouton gauche de la souris, puis tout en maintenant le bouton enfoncé, faites glisser le chemin jusqu'au type de prépresse dans lequel il doit être ajouté.

·    Le curseur de la souris se transforme alors en un feu vert vous indiquant que l'opération est possible. Relâchez le bouton de la souris.

¨   Pour enlever un chemin d'un type de prépresse :

·    Ouvrez la liste des types de prépresse, puis ouvrez le type de prépresse dont vous voulez enlever un chemin.

·    Dans la liste des chemins du type de prépresse, sélectionnez celui qui doit être enlevé, cliquez sur Supprimer dans le menu Paramétrages ou dans le menu contextuel, puis confirmez la suppression.
Le chemin sélectionné est alors enlevé du type de prépresse dans lequel il se trouvait, mais il n'est pas supprimé et vous pouvez le retrouver dans la liste des chemins de prépresse.

Définition des types d’encre

Précédent   Suivant

Les types d’encre désignent les différentes technologies d’impression, tous types de machine confonfus. Par exemple : offset, UV, électrophotographie, jet d’encre, …

Les types d’encre sont paramétrables. Vous pouvez en créer autant que nécessaire.

 

Un type d’encre est lié à un seul type de machine d’impression : offset feuilles, offset rotatives, sérigraphie, continu presse numérique, numérique grand format, …

Il peut exister plusieurs types d’encre différents pour un même type de machine d’impression.
Exemples :

¨   pour les machines offset, on peut avoir le type d’encre « Traditionnel » et « UV ».

¨   pour les presses numériques, on peut avoir les types d’encre : « Electrophotographie » et « Jet d’encre »

 

Chaque machine d’impression peut être liée à un ou plusieurs types d’encre. Par exemple, une machine offset peut être liée au type d’encre « Traditionnel » et « UV », alors qu’une autre machine offset ne sera liée qu’au type d’encre « Traditionnel ».

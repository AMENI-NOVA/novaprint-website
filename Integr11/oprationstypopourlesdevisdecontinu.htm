Opérations typo pour les devis de continu

Précédent   Suivant

Le numérotage est supposé possible quel que soit le nombre de poses sur toutes les machines de continu. Il s'agit ici d'une indication qui figurera sur les dossiers technique et de fabrication. La mention numérotage est indiquée pour la dernière machine utilisée. Toutes les machines de continu sont supposées pouvoir perforer en long autant de fois que nécessaire sans surcoût. Les opérations des perforation plus particulières peuvent être gérées par les opérations particulières de façonnage réalisables sur machine en continu.

On ne peut pas saisir d’opération de typo pour les devis d’étiquettes imprimées en continu.

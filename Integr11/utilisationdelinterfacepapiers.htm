Utilisation de l'interface papiers

Précédent   Suivant

La fenêtre principale de l’interface papier affiche les descriptions connues des fichiers de tarif papetiers.

 

 

En règle générale, la description à utiliser est déjà dans la liste et vous n’avez alors plus qu’à cliquer sur le bouton <Ouvrir un tarif décrit par ...> pour initialiser l’importation d’un fichier correspondant à la description.

 

Cinq boutons permettent la gestion des descriptions :

¨   Créer une nouvelle description. Voir le chapitre « Descriptions des fichiers d'importation ». 

¨   Dupliquer la description sélectionnée : permet de créer une nouvelle description à partir d’une description existante. La nouvelle description prend le nom initial + (1) ou (2) ... et l’écran de saisie est automatiquement ouvert. On peut bien sûr modifier le nom donné par défaut.

¨   Modifier la description sélectionnée

¨   Supprimer la description sélectionnée

¨   Importer des descriptions. Graphisoft maintient à jour un ensemble de descriptions avec les papetiers. Ces descriptions sont envoyées avec chaque mise à jour du progiciel sous la forme d’un fichier. Cliquez sur « Importer des descriptions » pour mettre à jour les descriptions définies dans votre base de données avec ces descriptions standards.
La liste des descriptions présentes dans le fichier sont présentées, cochez celles que vous voulez importer puis cliquez sur <Ok>.
Si une description de même nom existe déjà, elle sera mise à jour (l’identité du fournisseur ne sera pas modifiée ni les sélections que vous avez définies), sinon elle sera créée..

 

Pour importer directement un fichier généré par l’export des tarifs dans la liste des papiers de Graphisoft, utilisez l’option « Importer un export Graphisoft ».

Bon de livraison

Précédent   Suivant

Ce type d'édition n'est pas accessible dans la gamme Estiprint sans l'option "Livraison & Facturation".

Champs possibles pour un modèle de bon de livraison
Document

¨   Date document : date du bon de livraison. Lorsque le bon de livraison est enregistré, cette date est enregistrée dans le dossier comme date du dernier bon de livraison.

¨   Nom  document

¨   Numéro de page (doit être placé dans un bloc répété sur toutes les pages)

¨   Nombre de pages (doit être placé dans un bloc répété sur toutes les pages)

¨   [Repeter / Fin] : balises de répétition pour les champs liés à une ligne de bon de livraison (un bon de livraison de sortie de stock peut comporter plusieurs lignes).

¨   Taux devise

Utilisateur

Mêmes champs que pour la lettre de confirmation.

Bon de livraison

¨   Destinataire (Nom, Adresse, Téléphone, Fax, E-mail): informations renseignées dans la fiche société du destinataire désigné du bon de livraison.

¨   Expéditeur (Nom, Adresse, Téléphone, Fax, E-mail): informations renseignées dans la fiche société de l’expéditeur désigné du bon de livraison.

¨   Consignes : consignes de livraison, initialisées à chaque nouveau bon de livraison avec les consignes de livraison renseignées pour l’adresse du destinataire (voir « Livraisons » dans le chapitre « Sociétés ») et modifiables pour le bon de livraison.

¨   Interlocuteur destinataire : interlocuteur du destinataire

¨   Libellé livraison : nom du point de livraison auquel le bon de livraison est associé

¨   Nom Transporteur

¨   Numéro du bon de livraison

¨   Quantité livrée (de chaque ligne du bon de livraison)

¨   Prix hors taxe (de chaque ligne du bon de livraison)

¨   Prix unitaire (de chaque ligne du bon de livraison) : prix à l’unité ou selon le nombre d’unités défini dans la commande.

¨   Référence document : Référence globale saisie du bon de livraison

¨   Unité de quantité (de chaque ligne du bon de livraison)

Dossier

¨   Certification. En fonction de la mise en forme, vous pouvez afficher soit son libellé, soit le logo défini dans la famille de certification.

¨   Chaine de contrôle de la certification,

¨   Commentaire de la commande figurant sur le bon de livraison

¨   Conditionnement

¨   Conditionnement détaillé (avec les informations de paquetage)

¨   Date commande

¨   Descriptif

¨   Informations complémentaires

¨   Numéro devis

¨   Numéro dossier

¨   Poids

¨   Produit

¨   Quantité commandée

¨   Quantité livrée (somme des quantités des bons marqués enlevés plus celui édité)

¨   Référence (de chaque ligne du bon de livraison) : Référence de la commande livrée ou nom de la version livrée .

¨   Référence commande : référence commande client

¨   Reste à livrer : Quantité commandée diminuée de la quantité livrée du dossier. Ne peut pas être négatif.

Imprimerie

Mêmes champs que pour la lettre de confirmation.

Société

Informations concernant la société destinataire de la livraison.

Mêmes champs que pour la lettre de confirmation.

Stocks

Informations spécifiques aux stocks livrés.

¨   Certification du stock

¨   Code article

¨   Code complet article

¨   Code famille

¨   Code Lot

¨   Code possesseur

¨   Commentaire

¨   Commentaire Stock

¨   Désignation article

¨   Désignation complète article

¨   Désignation famille

¨   Informations complémentaires de stock

¨   Nom possesseur

¨   Qté du mouvement en unité 2

¨   Référence client du stock

¨   Référence externe du mouvement de sortie

¨   Unité 2

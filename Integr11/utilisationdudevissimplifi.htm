Utilisation du devis simplifié

Précédent   Suivant

Navigation dans l’arborescence de la liste

Pour naviguer dans l’arborescence et retrouver une information, vous devez utilisez les lignes d’entête qui sont signalées par une icône  située à gauche de la ligne.

Lorsque vous positionnez le curseur sur une ligne d’entête et que vous tapez ENTREE, une nouvelle fenêtre apparaît, légèrement décalée par rapport à la précédente, et affichant le détail de l’élément que vous avez validé.

Cette fenêtre peut à son tour comporter d’autres lignes d’entête vous permettant d’accéder à encore plus de détail.

Par exemple, pour modifier le nombre de plaques de l’intérieur d’une brochure, vous devrez successivement valider "Elément Intérieur" puis "Impression". La liste affichée comportera alors le détail de toutes les opérations concernant cet élément d’impression.

Modification des informations

Lorsque vous avez positionné le curseur sur l’information que vous souhaitez modifier, des zones de saisie apparaissent en bas de la fenêtre pour saisir vos modifications.

Suivant le type d’information, vous aurez la possibilité de saisir une ou plusieurs valeurs, par exemple :

¨   Ligne "Encre" : saisie du coût de l’encre ;

¨   Ligne "Calages plaques" : saisie du nombre de calages et du prix d’un calage ;

¨   Ligne "Négatifs" : saisie de la quantité, du coût façon et du coût matière.

Dans certains cas, une liste d’aide permettant d’en sélectionner un des éléments, est disponible. C’est le cas notamment pour le papier ou les machines.

Dès que vous modifiez une valeur, deux boutons apparaissent :

¨   "Ok" : pour valider les modifications et recalculer le total du devis

¨   "Annuler" : pour annuler les modifications.

Lorsque vous avez effectué toutes les modifications voulues, vous devez enregistrer le devis modifié sous un nouveau numéro. Cliquez sur l'option Enregistrer du menu Devis simplifié, ou fermez la fenêtre de devis simplifié forçant ainsi Graphiplus à vous proposer d'enregistrer les modifications.

Remarque : Certaines modifications peuvent empêcher l’affichage du schéma d’imposition du devis. Ce sont :

¨   Le format fini global du devis,

¨   Les formats fini, à plat, de l’amalgame et du papier d’un élément du devis,

¨   Le développement ou la laize d’un élément d’impression.

Autres fonctions du devis simplifié

Les menus et la barre d’outil donnent accès aux fonctions suivantes :

¨   Imprimer : imprime la liste complète des informations accessibles depuis le devis simplifié.

¨   Enregistrer : crée un nouveau devis à partir du devis en cours, en enregistrant les modifications effectuées.

Le nouveau devis peut être une variante du devis : on enregistre alors le devis  avec les modifications effectuées en conservant le même numéro de devis mais en incrémentant le numéro de variante (le devis d’origine se voit attribuer la variante 1 et le nouveau devis la variante 2). Le nombre de variantes d’un devis est illimité.

NB : le descriptif et les commentaires enregistrés du devis d’origine sont reportés dans le devis enregistré. Vous pouvez les modifier dans le devis simplifié avant son enregistrement ou ensuite depuis la liste des devis.

¨   Modèle de devis : enregistre un nouveau modèle de devis ou remplace le modèle de devis chargé

Si le devis actuellement chargé est un modèle de devis, le programme demande si on souhaite remplacer ce modèle ou en créer un nouveau.

¨   Insérer une machine : permet d’insérer un nouvel élément d’impression qu’il est ensuite possible de compléter en saisissant chacune des informations. Cette fonction n’est accessible que lorsque vous vous trouvez dans une fenêtre affichant les informations d’un élément de devis (intérieur, couverture, ...).

¨   Supprimer une machine préalablement insérée par le devis simplifié. Cette option n’est disponible que si le curseur se trouve sur une ligne d’impression marquée : "Ajouté au devis simplifié".

¨   Commentaires : affiche et permet de modifier les commentaires du devis.

¨   Dossier technique : ouvre la fenêtre du dossier technique du devis en cours tel qu’il est enregistré. Les modifications ne sont donc prises en compte que si vous enregistrez auparavant. La fenêtre de devis simplifié n’est pas fermée et les modifications en cours restent en attente. Vous pouvez revenir au devis simplifié à tout moment pour terminer vos modifications.

¨   Recharger le devis : Recharge les informations à partir du devis enregistré.

Dossier de fabrication

Précédent   Suivant

EmaPlan permet d’afficher le dossier de fabrication d’une commande. Vous avez ainsi accès à toutes les informations de vos commandes.

Il est accessible depuis le menu contextuel d’une fiche de travail.

 

EmaPlan affiche le dernier dossier de fabrication enregistré pour cette commande.

S’il n’y en a pas, il en construit en utilisant le modèle défini dans le paramétrage

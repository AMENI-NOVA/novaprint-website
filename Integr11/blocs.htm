Blocs

Précédent   Suivant

Un bloc est un container dont l’emplacement est fixé par rapport à la page, qui peut contenir du texte et des champs qui peuvent être mélangés pour former des paragraphes ou se trouver en tableaux. Il doit toujours au moins y avoir un bloc, car les informations à éditer ne peuvent pas être disposées directement sur la page.

Un même bloc peut contenir de nombreuses lignes de texte et autant de champs que nécessaire. Par exemple, pour un modèle de lettre de confirmation, on peut avoir un bloc pour l’adresse et un autre pour le reste du document.

Pour créer un bloc, cliquez avec le bouton gauche de la souris sur le fond de page, et déplacez-la tout en maintenant le bouton enfoncé.

Pour déplacer un bloc existant, appuyez sur la touche ALT du clavier et tout en maintenant la touche enfoncée, cliquez n’importe où sur le bloc avec le bouton gauche de la souris. Vous pouvez alors déplacer le bloc ainsi que toutes les informations qu'il contient. Le redimensionnement d’un bloc s’effectue de la même manière, mais en cliquant cette fois sur un des bords du bloc, tout en maintenant le touche ALT du clavier enfoncée.

Vous pouvez aussi saisir très précisément les coordonnées et les dimensions d’un bloc en appelant la fenêtre des propriétés du bloc. Pour cela cliquez sur le bloc à n’importe quel endroit, et cliquez sur Bloc dans le menu Format ou cliquez sur le bouton Propriétés de la barre d’outils.

La fenêtre des propriétés permet également de modifier le comportement du bloc pour gérer les éditions pouvant s’étendre sur plusieurs pages :

¨   Seulement sur la 1ère page : c’est par exemple le cas pour un bloc contenant l’adresse

¨   Identique sur toutes les pages (position et contenu)

¨   Etendu sur les pages suivantes : le contenu d'une page est poursuivi sur les pages suivantes, par exemple dans le cas d’une lettre de confirmation pour un bloc contenant le corps de la lettre. Dans un modèle, un seul bloc peut être étendu sur les pages suivantes.

¨   Fin de document : le bloc est répété à l’identique sur toutes les pages, mais les informations du bloc ne sont affichées que sur la dernière page. Les blocs de ce type sont souvent utilisés pour afficher les totaux dans les modèles de facture.

A l’édition du document (à l’écran comme à l’imprimante), le contour des blocs disparaît. A l’édition à l’écran d’un document, une petite croix apparaîtra au milieu du bord inférieur d’un bloc, si celui-ci est trop petit pour contenir toutes les informations paramétrées. Vous pouvez imprimer le document tel quel, la croix n’étant pas imprimée, sachant que des informations ont été tronquées pour ce bloc.

Calcul du flashage

Précédent   Suivant

La surface d'imposition utilisée pour le calcul, ne comprend pas la prise de pince.

Pour les flasheuses ayant une sortie plaques, le nombre de flashages est égal au nombre de plaques flashées.

Le coût de flashage sera mis dans le centre de coût "PHOTOGRAVURE"

Coût de façon

¨   Fixe : Coût fixe x Nombre de flashages.

¨   Variable : Coût au dm² x Surface d'imposition x Nombre de flashages.

Coût de matière

¨   Fixe : Coût fixe x Nombre de flashages

¨   Variable :

·    Flasheuse sortie film en bobine :
Coût au dm² x Laize x Développement x Nombre de flashages,
Calcul du développement :
Plus petite valeur possible du format d'imposition, soit :

§  Largeur du format d'imposition si la longueur du format d'imposition tient dans la laize utile de la flasheuse.

§  Longueur du format d'imposition si supérieure à la laize utile de la flasheuse.

La valeur est ensuite majorée de la gâche de développement.
Si la valeur obtenue est inférieure au développement minimum de la flasheuse, c'est ce dernier qui est pris en compte.

Exemple pour une flasheuse en sortie bobine

¨   Paramétrages de la flasheuse :

·    Caractéristiques techniques :

Laize film                         : 460 mm
Laize utile                       : 445 mm
Gâche sens développement          : 100 mm
Minimum sens développement       : 300 mm
Flashage de films montés   : Oui

·    Tarifs (en fonction du niveau de complexité choisi pour le devis) :

Coût fixe matière    : 0,25
Coût matière par dm²        : 1,00
Marge matière                  : 20 %
Coût fixe façon                : 1,5
Coût façon par dm² : 0,5
Marge façon           : 10%

¨   Devis

Affiche 21x29,7 Quadri R°/V°
Impression en 4 poses basculées       Format d'impression : 44,5x60,9 cm

Les caractéristiques indiquées pour la flasheuse permettent de flasher les 4 poses en une seule fois (flashage de films montés : Oui).

Etant donné qu'on imprime 4 couleurs recto/verso en basculant, 4 flashages sont nécessaires (1 pour chaque plaque).

¨   Calcul du flashage

·    Format d'imposition utilisé pour calculer le flashage :
43,5  (44,5 – 1cm de prise de pince)  x  60,9 cm

·    Calcul du coût de la façon :

Surface à flasher : Format d'imposition x Nombre de flashages, soit : 4,35 x 6,09 x 4 = 105,966 dm²

Coût de la façon :
[ ( Coût fixe x Nombre de flashages ) + ( Surface à flasher x Prix au dm² ) ] + Marge
soit : [ ( 1,5 x 4 ) + ( 105,966 x 0,5 ) ] + 10% = 64,8813

·    Calcul du coût de la matière :

La laize utile étant de 44,5cm, c'est le côté 43,5 qui sera dans le sens laize et le côté 60,9 dans le sens développement.
Longueur de film nécessaire = 60,9 + 10 (gâche sens développement) = 70,9 cm (7,09 dm)
Le minimum dans le sens développement est de 300mm (3 dm). Le calcul sera donc fait sur 7,09 dm.
(si la longueur de film nécessaire avait été inférieure à 3 dm, le calcul aurait été fait sur 3 dm).

Surface de film nécessaire = 7,09 x 4,6 (laize film flasheuse) = 32,614 dm²
Surface de films indiquée sur le dossier technique = 32,614 x 4 (Nombre de flashages) = 130,456 dm², arrondi à 130 dm²

Coût de la matière :
( [ Coût fixe + ( Surface x Prix au dm²) ] x Nombre de flashages ) + Marge
( [ 0,25 + ( 32,614 x 1) ] x 4 ) + 20% = 157,7472

·    Coût total du flashage = Façon + Matière = 64,8813 + 157,7472 = 222,6285 arrondi à 222,63

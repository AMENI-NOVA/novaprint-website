Paramétrage des amalgames de cahiers

Précédent   Suivant

Pour y accéder, sélectionnez la rubrique « Amalgame de cahiers ... », dans l’arbre de paramétrage.

L’arbre présente la liste des amalgames de cahiers disponibles triés en ordre croissant du nombre de page.

 

Pour modifier les caractéristiques d’un amalgame de cahiers ou le renommer :

Sélectionnez-le dans la liste, puis, cliquez sur Propriétés ou Renommer dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils.

Pour ajouter un nouvel amalgame de cahiers :

Sélectionnez dans la liste, la rubrique « Amalgames de cahiers... », puis cliquez sur Nouveau dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils.

Pour supprimer un amalgame de cahiers :

Sélectionnez-le dans la liste, puis, cliquez sur Supprimer dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils.

Pour imprimer les caractéristiques d’un amalgame de cahiers :

Sélectionnez-le dans la liste, puis :

¨   cliquez sur Imprimer dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils.

ou

¨   affichez les propriétés de l'amalgame et cliquez sur le bouton imprimer de la fenêtre des propriétés d’un amalgame de cahiers.

Pour imprimer les caractéristiques de tous les amalgames de cahiers :

Sélectionnez dans la liste, la rubrique « Amalgames de cahiers... », puis cliquez sur Imprimer dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils.

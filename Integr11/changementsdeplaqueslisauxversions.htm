Changements de plaques liés aux versions

Précédent   Suivant

La saisie des changements de plaques pour un élément, peut être associée à différentes versions du devis.

Pour saisir des changements de plaques, affichez la rubrique « Impression » de l’élément du devis pour lequel il y a des changements. Dans la partie « Avancée », cliquez sur le bouton <Changements…>.

Une nouvelle fenêtre permet de saisir le nombre de changements, les couleurs, ainsi que le détail de chaque changement (nom et quantité) si la case « Activer les versions pour cet élément » est cochée.

Lorsque la fenêtre de saisie des changements est validée, le programme crée alors automatiquement une version du devis pour chaque changement de plaques. Si le devis comporte d’autres éléments de devis, ils sont ajoutés à chacune des versions.

Par exemple, si vous saisissez 2 changements de plaques, 2 versions seront ajoutée automatiquement au devis (en + de la version de base)

Le devis passe alors en mode de saisie multi-versions.

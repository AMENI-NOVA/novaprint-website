Mise à jour des informations complémentaires

Précédent   Suivant

Création

Pour créer une nouvelle information complémentaire, ouvrez la rubrique "Paramétrages généraux " de la liste des paramétrages, puis sélectionnez la rubrique "Informations complémentaires des devis et dossiers". Appuyez ensuite sur la touche "Inser", ou cliquez sur cette rubrique avec le bouton droit de la souris puis choisissez  "Nouveau", ou cliquez sur  "Nouveau" dans le menu "Paramétrages".

Vous pouvez créer autant d’informations complémentaires que vous le désirez

Saisie et modification

Type d’information

Le type d’information permet au programme d’effectuer une vérification par rapport à la saisie effectuée, ce qui limite les risques d’erreurs. Il permet aussi de filtrer les dossiers (dans la liste des commandes) ou les devis (dans la liste des devis) , pour restreindre par exemple la liste à ceux dont le contenu d’une information de type date est antérieur à une date donnée.

Liste de choix

Cette option ne concerne que les informations de type texte.
Elle permet de définir des choix par défaut qui seront proposés à l’utilisateur au moment de la saisie de cette information, de manière à accélérer la saisie et limiter les risques d’erreur.

Lorsque vous cochez l’option « Liste de choix », vous avez ensuite la possibilité de remplir la liste des choix par défaut qui apparaît sur la droite de la fenêtre.
Pour ajouter un nouveau choix par défaut, saisissez le libellé dans la zone de saisie se trouvant à gauche du bouton « Ajouter », puis cliquez sur celui-ci. Le nouveau choix vient s’ajouter dans la liste se trouvant au-dessous.
Pour supprimer un des choix par défaut, sélectionnez-le dans la liste, puis cliquez sur le bouton « Enlever ».

Pour importer la liste depuis un fichier texte, cliquez sur le bouton « Importer », sélectionnez le fichier à lire. Avant d’importer le nouvel ensemble de choix, l’application vous demandera de confirmer le traitement car les choix existants seront remplacés par les lignes lues dans le fichier, l’ordre des choix dans la liste étant celui des lignes du fichier.

Saisie libre possible

Cette option ne concerne que les informations de type texte pour lesquelles vous avez défini une liste de choix par défaut.
Elle permet d’autoriser ou non l’utilisateur à saisir pour cette information, un libellé qui ne se trouve pas dans la liste des choix par défaut. Si vous ne cochez par cette option, la saisie sera limitée à une sélection parmi les choix définis au préalable.

Informations de type Fonction

Les informations de type Fonction sont liées à une fonction d'employés : après avoir coché "Fonction", vous devez sélectionner la fonction liée à l'information. Lors de la saisie de cette information pour un devis ou un dossier, vous sélectionnerez alors un employé parmi ceux qui appartiennent à la fonction.

Etant donné qu'un employé peut appartenir à plusieurs fonctions, vous pouvez ainsi définir par l'intermédiaire des fonctions autant de groupes d'employés que nécessaire sans avoir à paramétrer des listes de choix.

D'autre part, les informations de type fonction peuvent être utilisées pour la gestion des points de contrôle, pour désigner les personnes qui doivent effectuer des contrôles ou qui doivent être informées lorsqu'un contrôle est prêt à être effectué.

Informations de type Interlocuteur

Lors de la saisie de cette information dans le devis/dossier, vous choisirez un interlocuteur parmi ceux du client concerné par le devis ou le dossier.

Les informations de type Interlocuteur peuvent être utilisées dans la gestion des points de contrôle, pour informer un interlocuteur du client qu'un contrôle est prêt à être effectué (bon à tirer par exemple).

Modification du type d'information

Vous pouvez modifier le type d’une information existante dans les conditions suivantes :

¨   Passage à une information de type ‘Texte’ depuis n’importe quel autre type : pas de problème

¨   Passage d’un type à l’autre sauf texte (par exemple transformer une information ‘Texte’ en une information ‘Date’ :

·    Enregistrement des informations d’un devis/dossier :
L’enregistrement ne sera possible que si toutes les informations correspondent au type défini dans le paramétrage. Par exemple si un texte quelconque a été saisi pour une information de type ‘Texte’ et que cette information est ensuite transformée en type ‘Date’, il faudra ressaisir une information de type ‘Date’ (ou la mettre à blanc) pour pouvoir enregistrer les propriétés, même si l’information d’origine n’est pas modifiée.

·    Liste des commandes :
Le changement de type d’une information n’empêche en aucun cas l’affichage du contenu de cette information pour chaque devis/dossier, même si le type ne correspond pas.
Pour les filtres, les informations enregistrées dans l’ancien type ne seront pas prises en compte. Par exemple, si une information ‘Texte’ est transformée en ‘Date’, les devis/dossiers pour lesquels le contenu de cette information n’est pas une date n’apparaîtront pas dans la liste si on fait un filtre sur cette information (par rapport à une date).

Vérifier la cohérence

Cette option ne concerne que les informations de type date.

La vérification de la cohérence des dates permet d’être sûr que les informations de type date, saisies pour un même dossier respectent une certaine chronologie par rapport aux numéros d’ordre.

Par exemple, si vous créez 2 informations de type date : "Bon à graver" ayant ‘1’ comme numéro d’ordre, et "Fin d’impression" ayant ‘5’ comme numéro d’ordre, la date saisie pour "Bon à graver" devra toujours être antérieure ou égale à celle saisie pour "Fin d’impression", pour le même dossier.

La vérification de la cohérence ne s’applique qu’aux dates pour lesquelles vous cochez cette case. Vous pouvez créer par ailleurs d’autres informations de type date, pour lesquelles vous ne cochez pas cette case, et qui ne seront l’objet d’aucune vérification par rapport aux autres dates saisies pour le même dossier.

Ordre des informations

L’ordre est utilisé pour :

¨   l’affichage des informations dans la liste arborescente du paramétrage et dans le tableau de saisie de ces informations pour chaque devis/dossier.

¨   la vérification  de la cohérence des dates pour lesquelles vous avez coché la case "Vérifier la cohérence".

Utilisez les boutons de flèches vers le haut ou vers le bas pour modifier l’ordre de l’information en cours de saisie.

Saisie obligatoire pour

Cochez les cases correspondantes pour être sûr que les informations seront saisies.

Apparaît sur

Cochez les cases correspondantes pour faire apparaître ou non les informations dans les éditions ou les fonctionnalités correspondantes.

¨   Dossier de fabrication :
Pour le dossier de fabrication issu du devis, vous avez en plus la possibilité d’éditer les informations même si elles sont vides, en cochant la case « Même si non renseignée ». Cela peut permettre ensuite aux utilisateurs de ne pas oublier de compléter à la main les informations manquantes.

¨   Etat détaillé en temps : les informations apparaissent dans l’entête de l’édition des détails du dossier.

¨   Planning mural : les informations seront visibles dans la fenêtre des propriétés de la fiche ou du dossier sélectionné.

Suppression

Vous pouvez supprimer n’importe quelle information, même si des valeurs ont été saisies pour cette information, pour un ou plusieurs devis/dossiers.

Les informations supprimées n’apparaissent plus dans la liste des devis/commandes, ni sur les éditions.

Informations gérées par l’interface

Précédent   Suivant

Les correspondances suivantes peuvent être définies dans une description :

Aucune relation

La colonne du fichier à importer est ignorée

Code Catégorie

Le code sur 4 caractères de la catégorie de papier (facultatif).
Remarque :  N’utilisez ces valeurs que si elles sont reliées uniquement au nom de la catégorie . C’est à dire un code et un seul pour un nom de catégorie.

Nom Catégorie

Le nom de la catégorie de papier.
Si le fichier d'importation ne comprend pas d'information de catégorie, les papiers seront importés dans une catégorie nommée arbitrairement "CATEGORIE INCONNUE".

Code Papier

Le code sur 4 caractères du papier (facultatif)
Remarque :  N’utilisez ces valeurs que si elles sont reliées uniquement au nom du papier. C’est à dire un code et un seul pour un nom de papier.

Nom Papier

Le nom du papier

Couleur

La couleur du papier

Certification

La certification du papier FSC, PEFC etc.

Unité des prix unitaires

L’unité de quantité correspondant aux prix.

(Ex : " Feuille " si les prix sont aux 1000 feuilles)

Code unité de prix

Code désignant la quantité correspondant aux prix.

(Ex : " MF " pour 1000 Feuilles)

Qté. des prix unitaires

La quantité correspondant au prix.

(Ex : " 1000 " si les prix sont pour 1000 Feuilles)

Grammage

Le grammage du papier en g/m²

Largeur

La largeur du format de papier

Unité largeur

L’unité utilisée pour exprimer la largeur ci-dessus

Longueur

La longueur du format de papier

Unité longueur

L’unité utilisée pour exprimer la longueur ci-dessus

Paquetage (fls)

L’unité de vente en nombre de feuilles

Tarif 1

Le tarif pour les quantités inférieures à la 1ère borne

Tarif 2

Le tarif pour les quantités inférieures à la 2nde borne

Tarif 3

Le tarif pour les quantités inférieures à la 3ème borne

Tarif 4

Le tarif pour les quantités inférieures à la 4ème borne

Tarif 5

Le tarif pour les quantités inférieures à la 5ème borne

Tarif 6

Le tarif pour les quantités supérieures à la 5ème borne

Borne 1

La 1ère borne de quantité achetée

Borne 2

La 2nde borne de quantité achetée

Borne 3

La 3ème borne de quantité achetée

Borne 4

La 4ème borne de quantité achetée

Borne 5

La 5ème borne de quantité achetée

Epaisseur

Si la valeur est numérique et positive, elle sera enregistrée dans l’article importé. Si l’article imprimeur mis à jour est géré à l’épaisseur (voir la catégorie du papier), cette information est déterminante, sinon elle n’est qu’informative.

Référence

Le code de référence fournisseur de l’article (20 car. Maximum) permet d’identifier de façon unique l’article pour le fournisseur. Les sélections que vous faites des articles à importer ou non sont basées sur cette information.

Remarque : Ce code permet d’initialiser automatiquement les références fournisseurs dans les papiers du devis. Il sera transmis aux conditions d’achat des articles papiers de la gestion des Stocks par la fonction de « Mise à jour par le fichier papier devis » dans Graphisoft. Il est aussi utilisé dans les bons de commandes.

Poids

Le poids de 1000 feuilles en Kg

Fournisseur

Le nom interne du fournisseur. (cette fonctionnalité est utilisée par l’import des tarifs exportés depuis la liste des papiers pour gérer un tarif multi fournisseurs.

Remarque : les informations « Nom catégorie », « Nom Papier » « Certification » et « Couleur » peuvent être attribuées à plusieurs colonnes du fichier d’entrée. La valeur obtenue résultera alors de la concaténation des différentes colonnes du fichier d’entrée. NB : si le contenu d’une colonne est déjà intégralement dans la valeur finale, il n’est pas ajouté à nouveau. Ex : Si on a déjà obtenu  « Affiche extra blanche » et qu’on doit concaténer avec une colonne contenant « Blanche », cette dernière ne sera pas ajoutée car « Blanche figure déjà complètement dans le résultat. Ceci permet d’éviter d’avoir « Affiche extra blanche Blanche ».

Etiquettes de livraison

Précédent   Suivant

Les étiquettes de livraison peuvent recevoir les mêmes champs que le bon de livraison et en plus

¨   Libellé du conditionnement

¨   N° du paquet

¨   Nombre de paquets

¨   Quantité dans le paquet

¨   Premier et dernier numéros d’exemplaires numérotés présents dans le paquet.

¨   SSCC (Serial Shipping Container Code)

¨   GS1-128 : code international GS1 composé de différentes données paramétrables

NB : l’édition d’un SSCC ou d’un GS1-128 avec l’identifiant (00)  provoque l’attribution automatique de numéros de colis suivant la séquence paramétrable dans « Paramétrage – Paramétrages généraux – Numérotation des bons et colis de livraison ». Le code SSCC de l’imprimerie doit être défini pour que le SSCC puisse être construit.

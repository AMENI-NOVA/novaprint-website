Paramètres généraux des types de façonnage

Ces paramètres sont valables pour tous les types de devis : brochure, dépliant, fiche, …

Nom

C’est ce nom qui permet de sélectionner le type de brochage parmi la liste proposée pour le choix "Façonnage" à la saisie des devis de brochure.
Chaque type de brochage doit avoir un nom unique.

Libellé pour les éditions

Ce libellé sera utilisé dans toutes les éditions du devis et dans le dossier de fabrication généré pour la gestion de production, lors de la transformation du devis en commande.

Utilisation du type de façonnage pour chaque type de devis

Un tableau vous permet d’indiquer pour chaque type de devis, si le type de façonnage :

¨   est interdit pour ce type de devis : il n’apparaîtra pas dans la la liste des types de façonnage proposés pour les devis de ce type,

¨   peut être utilisé pour ce type de devis : il apparaîtra dans la liste des types de façonnage proposés pour les devis de ce type,

¨   est le type de façonnage par défaut pour ce type de devis : il sera automatiquement sélectionné pour tous les nouveaux devis de ce type.

NB : un type de façonnage peut être utilisé avec plusieurs types de devis.

Chemins du type

Liste des chemins regroupés dans ce type de brochage.
Le programme de calcul optimisera le calcul du devis pour chaque chemin du brochage, pour déterminer quelle est la meilleure solution.

Vous pouvez également ajouter ou enlever des chemins à un brochage, directement depuis la liste des paramétrages :

¨   Pour ajouter un chemin à un brochage :

·    Ouvrez la liste des chemins de brochage

·    Cliquez sur le chemin avec le bouton gauche de la souris, puis tout en maintenant le bouton enfoncé, faites glisser le chemin jusqu'au brochage dans lequel il doit être ajouté.

·    Le curseur de la souris se transforme alors en un feu vert vous indiquant que l'opération est possible. Relâchez le bouton de la souris.

¨   Pour enlever un chemin d'un brochage :

·    Ouvrez la liste des brochages, puis ouvrez le brochage dont vous voulez enlever un chemin.

·    Dans la liste des chemins du brochage, sélectionnez celui qui doit être enlevé, cliquez sur Supprimer dans le menu Paramétrages ou dans le menu contextuel, puis confirmez la suppression.
Le chemin sélectionné est alors enlevé du brochage dans lequel il se trouvait, mais il n'est pas supprimé et vous pouvez le retrouver dans la liste des chemins de brochage.

Paramètres spécifiques aux devis de brochure

Calcul automatique de l'épaisseur du dos de la couverture

Dans la saisie des devis de brochure, le format à plat de la couverture sera ajusté automatiquement en fonction du nombre de pages de la brochure, si vous avez sélectionné un type de brochage pour lequel cette case est cochée.

Type de brochage standard

Le type de brochage standard permet d’identifier précisément un type de façonnage.

L’identification du type de façonnage s’effectue par rapport à la nomenclature définie dans la norme JDF. Le nom dans la nomenclature JDF est indiqué entre parenthèses :

¨   Dos carré collé (JDF : SoftCover)

¨   2 piqûres à cheval (JDF : SaddleStitch)

¨   Couture fil (JDF : HardCover)

L’identification d’un type de façonnage par rapport à la nomenclature JDF n’est pas obligatoire, mais les zones de saisie disponibles dans la saisie de devis dépendent du type de brochage standard associé au type de façonnage sélectionné :

¨   Les devis de livres à couverture rigide ne peuvent être saisis que si le type de façonnage utilisé est lié au type de brochage « Couverture rigide (HardCover) ».

¨   La saisie de pages de garde n’est possible que si le type de façonnage utilisé est lié au type de brochage « Couverture rigide (HardCover) » ou « Dos carré collé (SoftCover) ».

Paramètres spécifiques aux livres à couverture rigide

Lorsque le type de brochage standard est « Couverture rigide (HardCover) », le cadre « Couverture rigide » est actif et permet de saisir les informations spécifiques aux types de façonnage correspondant aux livres avec couverture rigide.

Type de dos standard

Plat

Arrondi

Plat en retrait

Arrondi en retrait

Le type de dos (tel que défini par la norme JDF) a une incidence sur le calcul automatique des dimensions de l'élément "Matière de couverture " et est utile à titre informatif (édité sur le descriptif).

Paramètres permettant de calculer automatiquement les dimensions des éléments qui composent la couverture rigide.

¨   Rembordage côté : utilisé pour calculer la largeur de la matière de couverture

¨   Rembordage haut (et bas) : utilisé pour calculer la hauteur de la matière de couverture

¨   Espacement plat/dos : utilisé pour calculer la largeur de la matière de couverture

¨   Chasse : espace entre le bord de la couverture et le bord de l’intérieur

¨   Ecart largeur plat/intérieur : différence de largeur entre le plat de couverture et l’intérieur. Utilisé pour calculer la largeur du plat de couverture à partir de la largeur de l’intérieur.
NB : cette valeur peut être négative dans le cas où le plat de la couverture est moins large que l’intérieur.

NB : Lorsque le curseur se trouve dans une des zones « rembordage » ou bien dans le zone « espacement plat/dos », l’image située à proximité change pour vous montrer la dimension concernée dans le schéma.

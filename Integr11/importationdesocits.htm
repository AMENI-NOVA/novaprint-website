Importation de sociétés

Précédent   Suivant

Cette fonction permet d'ajouter, dans Graphisoft, des sociétés enregistrées avec un autre programme (votre comptabilité par exemple).

 

Pour accéder à cette fonction, ouvrez la liste des sociétés et utilisez la commande "Importation" du menu "Utilitaires".

Pour avoir accès à cette fonction, il faut disposer du droit "Ajouter une société"

 

Le transfert se fait par l'intermédiaire d'un fichier texte. Le programme contenant les sociétés à importer, doit créer ce fichier, ce qui se fait généralement par la fonction d'exportation du logiciel. Le fichier doit répondre aux critères suivants :

¨   Il doit être au format "Texte" : c'est à dire qu'il doit être lisible depuis n'importe quel éditeur de texte. Vous pouvez ouvrir le fichier avec l'utilitaire "Bloc-notes" de Windows pour vérifier si tous les caractères sont compréhensibles.
L'extension du fichier est généralement ".txt" ou ".csv". Ceci n'est pas obligatoire, mais conseillé.

¨   Le fichier doit correspondre a l'un des deux formats suivants :

·    Format texte 'standard' :
Fichier d'extension ".txt" pouvant être généré par quasiment tous les programmes ayant une fonction d'exportation.

§  Toutes les données d'une société sont enregistrées sur la même ligne. Ces données peuvent être séparées par un caractère particulier (par exemple : point-virgule ou tabulation) ou de longueur fixe (les données sont alignées et séparées par des espaces).

§  Une seule ligne par enregistrement de société. Faites attention que vos données ne contiennent pas de caractères de fin de ligne (retour chariot). Cela peut être le cas dans les adresses.

·    Format texte 'CSV' :
Fichier d'extension ".cvs" où les données sont séparées par des points-virgules (ou des virgules) et où les adresses peuvent contenir de caractères de fin de ligne (retour chariot).
Ce format est de plus en plus utilisé car plus souple qu'un 'simple' fichier texte (exemple : Outlook, Excel).

 

Pour effectuer l'importation, le programme vous demande de :

1. Définir les caractéristiques du fichier d'importation.

2. Indiquer les informations que vous souhaitez importer.

3. Saisir d’éventuelles correspondances entres les données lues dans le fichiers et celles à enregistrer dans Graphisoft.

4. Saisir les valeurs par défauts des données non récupérées.

5. Saisir les informations concernant d'éventuelles notes.

Tous ces paramètres peuvent être enregistrés dans des macros. Ainsi, lors d'une prochaine importation, il ne sera pas nécessaire de les ressaisir.

 

La liste des informations  que l'on peut importer dans Graphisoft est détaillée plus bas dans le paragraphe "Informations à importer".

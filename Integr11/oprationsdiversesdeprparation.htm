Opérations diverses de préparation

Précédent   Suivant

Les opérations sont regroupées par familles, regroupant à votre convenance plusieurs opérations de même type ou les opérations proposées par un même fournisseur.

Pour ajouter une famille, sélectionnez la rubrique "Opérations", puis cliquez sur Nouveau dans le menu Paramétrages ou le menu contextuel, ou cliquez sur le bouton correspondant de la barre d’outils.

Le nom de la famille doit être unique.

La liste "Centre de coût" permet d'indiquer à quel centre de coût sont attachés les montants de toutes les opérations de la famille.

Les boutons "Précédente" et "Suivante", permettent de naviguer d'une famille à l'autre.

 

Pour ajouter une opération à une famille, sélectionnez la famille dans la liste, puis cliquez sur Nouveau dans le menu Paramétrages ou le menu contextuel, ou cliquez sur le bouton correspondant de la barre d’outils.

Pour dupliquer une opération existante, sélectionnez l’opération à dupliquer et cliquez sur Dupliquer. Toutes les informations de l’opération d’origine seront reprise à l’exception du nom qui est « Copie de <l’opération>".

Données d'une opération

¨   Affichage sur les courriers

Ceci permet de faire la différence entre les opérations courantes, et les opérations exceptionnelles ou demandées par le client qui doivent apparaître sur la lettre de confirmation et l'accusé-réception de commande.
Vous avez quatre possibilités :

·    Afficher le nom de l’opération ainsi que la quantité saisie

·    Afficher le nom de l’opération

·    Afficher le nom de la famille de l’opération

·    Ne rien afficher pour cette opération

¨   Famille : Le nom de la famille est rappelé ici, mais vous pouvez  le modifier pour changer l’opération de famille.

¨   Sous-traitant : vous pouvez indiquer le sous-traitant pour l’opération. Le choix se fait parmi la liste des sous-traitants enregistrés à partir de la liste des sociétés.

¨   Unité

Il s'agit d'un texte libre (ex : Photo(s), Cromalin(s)). Si vous le remplissez, à chaque fois que vous choisirez cette opération pour un devis, Graphisoft vous demandera le nombre de fois que l'on doit compter cette unité pour cette opération (ce nombre peut être différent pour chaque devis)

¨   Unités fixes : pages, dm2, modèles, couleurs, plaque, jeu de plaque

Elles influeront sur le calcul de l'opération en fonction du devis que vous saisissez, pour les éléments suivants :

·    le nombre de pages : nombre de pages imprimées
Les changements sont considérés comme des pages différentes. Par exemple pour un 4 pages avec 2 changements (soit 3 versions), le nombre de pages utilisé pour le calcul de l'opération sera : 4 x (1 + 2) = 12

·    le nombre de couleurs
Pour les liasses il s’agit du nombre d’impressions. Dans ce cas, l’unité "Page" n’est pas prise en compte, même si elle a été sélectionnée pour l’opération.

·    Dm2 :

§  Pour les opérations ayant aussi comme unité fixe « plaque » ou « jeu de plaque » :

§  Si le chemin de prépresse comporte une flasheuse avec des plaques en sortie, la surface prise en compte est le format utile des plaques de la flasheuse.

§  Dans les autres cas, la surface prise en compte est le format d’impression maximum de la machine utilisée

§  Sinon, la surface prise en compte est celle de la page

·    le nombre de modèles

·    plaque : nombre de plaques nécessaires pour l’impression.

Ce type d’opération ne peut être utilisée que dans un chemin de prépresse. Il est impossible de les sélectionner à la saisie d’un devis.

·    jeu de plaque : cette unité fonctionne de la même manière que pour le calcul du nombre d’ozalids

Ce type d’opération ne peut être utilisée que dans un chemin de prépresse. Il est impossible de les sélectionner à la saisie d’un devis.

Vous pouvez sélectionner une ou plusieurs de ces unités en plus de l'unité libre, avec les restrictions suivantes :

·    Les unités « plaque » et « jeu de plaque » ne peuvent être combinées qu’avec l’unité « dm2 »

·    Les unités « plaque » et « jeu de plaque » ne peuvent pas être combinées ensemble.

¨   Ajouter la marge dégressive à la marge fixe

Si cette case est cochée, la marge du tableau dégressif suivant les prix (voir : "Marges globales sur les opérations de prépresse") sera ajoutée à la marge pour cette opération. Le montant qui sert à déterminer la marge dégressive, est le coût façon + matière (hors marges).

Une fois la marge dégressive déterminée, celle-ci est ajoutée à la marge façon et à la marge matière pour le calcul final du coût de l’opération. Par exemple, si la marge façon est de 10%, et que la marge dégressive trouvée est de 20%, le coût façon sera calculé avec une marge de 30%.

¨   Ajouter automatiquement aux noveau devis

Si cette case est cochée, l’opération sera ajoutée automatiquement à tous les nouveaux devis.

Pour les devis de brochure, dépliants ou affiche/fiche, l’opération sera ajoutée aux opérations globales du devis.

En mode multi-sociétés, seules les opérations ayant un tarif défini pour la société utilisatrice courante seront ajoutées.

¨   Prix de l'heure

Il sera utilisé en combinaison avec le coût façon calculé, pour estimer le temps nécessaire pour l'opération, sur le dossier de fabrication et dans la gestion de production.

¨   Coût façon et matière de l'opération

Le coût façon est séparé du coût de la matière. Les deux sont calculés de la même façon en fonction des unités choisies.

Les tarifs façon peuvent être saisis en coût ou en temps passé suivant votre choix. Lorsque vous saisissez des        temps, les coûts sont calculés à partir du prix de l’heure saisi.

Si vous laissez tous les tarifs à 0, ils vous seront demandés à chaque fois que cette opération sera sélectionnée pour un devis (si les tarifs sont à 0 pour le niveau de complexité choisi pour le devis). Ceci vous permet d’enregistrer des opérations pour lesquelles les tarifs varient en fonction du travail à effectuer.

Calcul des opérations

Le calcul du coût d’une opération varie suivant que l’opération est saisie globalement pour le devis ou pour un élément du devis.

Opérations saisies globalement pour le devis

¨   Coûts fixes : ils ne sont comptés qu’une seule fois.

¨   Coût variables :

·    Si l’opération a au moins une unité fixe, le nombre d’unités servant au calcul du coût variable est la somme du nombre d’unité calculé pour chaque élément du devis. Par exemple pour une opération dont l’unité est la page, le nombre de pages pris en compte pour le calcul sera la somme du nombre de pages de chaque élément.
Le nombre d’unités calculé est ensuite muiltiplié par le nombre d’unités libres le cas échéant.

·    Si l’opération n’a pas d’unité fixe, le coût variable ne dépend pas des éléments du devis.

Affichage des barre d'outils

Précédent   Suivant

Cliquez à l'aide du bouton droit dans un menu ou une barre d'outils quelconque, puis cliquez sur le nom de celle que vous désirez masquer ou afficher dans le menu contextuel.

Conseils

Si la barre d'outils souhaitée ne figure pas dans le menu contextuel, cliquez sur Personnaliser les barres d’outils dans le menu Fenêtre, puis sur l'onglet Barres d'outils. Activez ensuite la case à cocher de la barre d'outils.

Pour masquer rapidement une barre d'outils flottante, cliquez sur le bouton Fermer <x> de la barre d'outils.

Lettre type à utiliser

Précédent   Suivant

Cette liste propose les modèles de lettres types que pouvez utiliser. Elle correspond aux modèles paramétrés dans la section "Editions" "Lettres types" de la fenêtre de paramétrage. Une lettre type se définit exactement comme une lettre de confirmation de devis ou un accusé réception de commande. Pour plus d'informations, consultez le chapitre "Paramétrage des éditions".

Sociétés de pays différents

Durant le publipostage, le modèle de lettre peut changer automatiquement en fonction de la langue définie pour chaque société. Pour cela, il faut disposer de plusieurs modèles de lettres types ayant le même nom, mais avec une langue différente. Pour chaque société, le programme recherche s'il existe un modèle avec la langue de la société et le nom du modèle choisi dans la liste "Lettre type à utiliser". S'il n'en trouve pas, il utilise le modèle sélectionné dans la liste

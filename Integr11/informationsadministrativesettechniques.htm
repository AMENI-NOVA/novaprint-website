Informations administratives et techniques

Précédent   Suivant

Les informations enregistrées pour chaque société peuvent être générales, telles que l'adresse ou le n° de téléphone (ce type d'information est valable pour toutes les entreprises), ou spécifiques à l'entreprise en fonction de son domaine d'activité. Ce paramétrage vous permet donc de personnaliser les informations que vous souhaitez enregistrer pour chaque société. A chaque information, une valeur différente pourra être saisie pour chaque société.

Les informations sont regroupées par catégorie de société, permettant ainsi par exemple de gérer des informations différentes pour les clients et les fournisseurs.

Paramétrage

Ces informations sont similaires dans leur fonctionnement et possibilités aux informations complémentaires des devis et des dossiers, le paramétrage et le fonctionnement de ces dernières  sont décrits dans le paragraphe « Informations complémentaires des devis et des dossiers ».

Pour rappel, les caractéristiques de chaque information sont les suivantes :

¨   Le nom

¨   Le type de donnée que contient l'information : caractères (texte), date, nombre, booléen (oui ou non), liste de choix, fonction ou interlocuteur (dans le cas des informations administratives et techniques, les Interlocuteurs seront à choisir dans les ceux de la société utilisatrice.

¨   Obligatoire ou non (une société ne pourra être enregistrée que si cette information est renseignée, excepté la société utilisatrice, pour laquelle aucune information n’est obligatoire).

¨   Le numéro d’ordre de l’information, toutes catégories confondues. Les informations seront affichées dans la feuille société dans cet ordre.

 

Dupliquer des informations d'une catégorie à l'autre

Il est possible de dupliquer le paramétrage d'une catégorie. Pour cela, il faut ouvrir la feuille de paramétrage, sélectionner la catégorie dans l'arbre et cliquer sur le bouton "Dupliquer"

 

Edition des informations administratives et techniques

Les informations administratives et techniques peuvent apparaître sur les éditions suivantes :

¨   Lettre de confirmation : informations concernant le prospect/client

¨   Accusé-réception de commande : informations concernant le client

¨   Bon de livraison : informations concernant le client du dossier

¨   Bon de commande : informations concernant le fournisseur ou le sous-traitant

 

Pour faire apparaître une information sur une édition :

1. Ouvrez le modèle d'édition sur lequel vous voulez faire apparaître l'information,

2. Positionnez le curseur à l'endroit où vous voulez que l'information soit éditée,

3. Cliquez sur le bouton  de la barre d'outils ou cliquez dans le menu sur Insertion puis Champ.

4. Dans la fenêtre de propriétés des champs, sélectionnez "Inf. Administrative …" ou "Inf. Technique …" suivant le type d'information que vous voulez ajouter au modèle, puis sélectionnez l'information dans la liste "Mise en forme".

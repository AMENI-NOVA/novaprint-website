Informations générales du devis

Précédent   Suivant

Dans cet onglet, vous pouvez modifier :

¨   La référence

¨   Le produit

¨   La société et l’interlocuteur (seulement si l’utilisateur à le droit « Modifier la société dans les propriétés du devis »)

NB : si le devis a un barème, vous pouvez remplacer le client par un client pour lequel le calcul de ce barème est possible (le client doit appartenir à une catégorie qui définitles opérations utilisées dans le barème). Dans ce cas, le prix du devis est ajusté pour être en accord avec le nouveau barème.

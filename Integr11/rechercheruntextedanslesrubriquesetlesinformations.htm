Rechercher un texte dans les rubriques et les informations

Précédent   Suivant

Une fonction de recherche accessible à partir de la liste par le bouton "Chercher" permet de retrouver un mot en parcourant les rubriques et les informations.

 

La fenêtre de recherche présente les options suivantes :

¨   Information à rechercher : tapez ici le texte que vous souhaitez rechercher.
Vous pouvez taper un mot, un groupe de mots ou même une phrase complète.
La liste située sous cette zone de saisie indique comment la recherche s'effectue si plusieurs mots sont saisis

¨   "Au moins un mot" : Les informations ou rubriques contenant au moins un des mots saisis sont affiché.

¨   "Tous les mots" : Le programme recherche les informations ou les rubriques contenant TOUS les mots saisis, quel que soit l'ordre des mots dans le texte.

¨   "Expression exacte" : Le programme recherche les informations ou les rubriques contenant EXACTEMENT le texte saisi.
Les espaces tapés seront pris en compte pour la recherche. Par exemple si vous tapez "calcul " (avec un espace derrière le mot "calcul", les rubriques ou informations comportant le mot "calculateur" ne seront pas pris en compte, à cause de l’espace tapé à la suite du mot recherché.

¨   Respecter la casse : cette option permet de déterminer si la recherche doit tenir compte des minuscules/majuscules.
Si cette case est cochée, la recherche distinguera les minuscules des majuscules. Dans ce cas, si vous recherchez par exemple le mot "Calcul", les rubriques ou informations comportant le mot "calcul" ne seront pas pris en compte.

¨   Rechercher l’information dans : indiquez dans quels éléments le texte doit être recherché.

¨   Rubriques à parcourir : indiquez ici l’étendue de la recherche. La rubrique sélectionnée est la rubrique sur lequel se trouve le curseur dans la liste des rubriques et des informations. Si le curseur se trouve sur une information et que vous spécifiez une recherche par rapport a la rubrique sélectionnée, c’est la rubrique dans laquelle se trouve l’information dans la liste, qui sera pris en compte.
Si vous spécifiez une recherche pour la rubrique sélectionnée uniquement, la recherche s’effectuera dans la rubrique ainsi que dans les informations qui y sont rattachées.

¨   Modifiée depuis le : cette option permet de rechercher toutes les informations modifiées depuis la date spécifiée.

Une fois les options de la recherche définies, cliquez sur le bouton "Rechercher" pour lancer la recherche.

Les rubriques parcourues lors de la recherche défilent dans une zone d’affichage en bas de la fenêtre.

La recherche en cours peut être interrompue en cliquant sur le bouton "Arrêter" qui apparaît pendant le traitement.

Résultat de la recherche

Les rubriques et informations trouvés comportant le texte recherché sont affichés dans la liste. Les éléments en gras correspondent aux critères de recherche.

Pour annuler la recherche et revenir à un affichage normal, cliquez sur "Afficher tout" ou de nouveau sur "Rechercher".

Lire un message

Précédent   Suivant

Pour lire un message, il faut le sélectionner avec les touches [Flèche haut] et [Flèche bas]. Il s'affiche alors dans la partie basse de l'écran. Pour indiquer que le message a été lu, on peut appuyer sur [Entrée]. Si le message reste affiché plus de quelques secondes, il est automatiquement marqué comme lu. Ce délai est paramétrable, voir plus bas, le paragraphe "Paramétrage".

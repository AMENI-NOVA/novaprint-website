Création d’une description pour un article de produit

Précédent   Suivant

Pour créer une description d’un article à partir de la saisie de devis optimisé, vous pouvez ouvrir un nouveau devis ou un devis existant, à condition que celui-ci ne soit pas déjà lié à un article.

Sélectionnez le client de l’article.

Pour lier le devis à un article, saisissez le code de l’article dans la zone de saisie se trouvant entre le produit et la quantité.

Le bouton <…> permet d’ouvrir une fenêtre permettant de sélectionner l’article dans une liste. La liste présente les articles actifs du client sélectionné pour le devis.

NB : les articles ayant déjà une description n’apparaissent pas dans la liste.

Si l’article n’a pas été créé dans la gestion de stocks, vous pouvez le faire à ce moment-là en cliquant sur « Nouveau ».

Lorsque l’article est sélectionné et le devis complètement saisi, calculez le devis et enregistrez-le.

L’enregistrement effectue les deux opérations suivantes :

¨   L’enregistrement d’un nouveau devis lié à l’article. Ce devis pourra ensuite générer une commande qui sera elle aussi liée à l’article

¨   L’enregistrement d’une description attachée à l’article, qui pourra ensuite être utilisée pour créer de nouveaux devis pour cet article. Un message vous en informe.

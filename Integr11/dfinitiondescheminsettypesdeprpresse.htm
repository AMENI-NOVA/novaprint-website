Chemins de prépresse

Un chemin de prépresse permet de décrire la succession des opérations à effectuer avant l’impression.
Vous pouvez créer plusieurs chemins, dont vous définissez et ordonnez vous-même toutes les étapes, parmi des opérations proposées par défaut par Graphisoft, telles que Montage films, Duplis, Plaques, etc..., ou des opérations de prépresse que vous avez paramétrées.
L'enchaînement de ces étapes, ainsi que leur ordonnancement, sera prise en compte lors du calcul du devis.
Exemple de chemin : Duplis, Montage films, Copie plaques, Matière plaques, Ozalids

Types de prépresse

Un type de prépresse permet de regrouper un ou plusieurs chemins (de 1 à 5) pour optimiser le coût du prépresse d’un devis, entre les différents chemins du type. Tous les chemins d’un type doivent avoir des éléments de départ identiques (films à réaliser, films fournis, photogravure typographique, etc...).

S’il n’y a pas d’optimisation du calcul du prépresse entre plusieurs chemins, créez alors un type de prépresse pour chaque chemin.

A la saisie du devis (pour chaque élément), la liste des types de prépresse est proposée. Le calcul de l’élément sera optimisé en fonction de chaque chemin du type sélectionné.

Remarque : les chemins et types de prépresse ne sont utilisés que pour les devis d'impression. Pour les devis de prépresse, on peut indiquer un type de prépresse, mais les chemins du type sélectionné ne sont pas pris en compte dans le calcul. Le devis de prépresse n'est qu'un devis "caddie" permettant de sélectionner des opérations de prépresse.

Vous pouvez créer autant de types de prépresse que vous le souhaitez.

La liste arborescente des paramétrages vous permet de voir les types de prépresse existants, ainsi que les chemins de chaque type.

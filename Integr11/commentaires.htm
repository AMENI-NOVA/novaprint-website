Commentaires

Précédent   Suivant

Des lignes de commentaires peuvent être ajoutées au devis en cours de saisie.

Les lignes de commentaires peuvent être de 3 types différents :

¨   Internes : n’apparaissent que sur les documents internes à l’entreprise tel que le dossier technique,

¨   Externes : n’apparaissent que sur les documents destinés à sortir de l’entreprise telle que la lettre de confirmation,

¨   Internes/externes : apparaissent sur tous les documents

¨   Facture : apparaitront sur la facture

Les commentaires peuvent être ajoutés ou modifiés à tout moment, même après le calcul du devis ; contrairement aux autres saisies, ils peuvent être enregistrés sans qu’il soit besoin de recalculer le devis.

Lorsque vous chargez un devis existant pour le modifier et le recalculer, les commentaires éventuels sont aussi chargés. Si vous ne modifiez pas les commentaires existants et que vous calculez puis enregistrez le devis,  Graphisoft vous demande si les commentaires du devis rappelé doivent être enregistrés pour le nouveau devis.

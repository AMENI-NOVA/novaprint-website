Pour mettre en place l'interface JDF, vous devez faut passer par les étapes suivantes :

1. Créer les connexions JDF correspondant aux différents partenaires JDF

2. Si vous utilisez l'interface avec le prépresse, vous devez faire correspondre les types de cahiers de Graphiplus avec ceux du JDF (dans le paramétrage des types de cahiers).

3. Vous devez indiquer quelles sont les machines interfacées en JDF (depuis le paramétrage des postes de gestion de production).

Lettre de confirmation

Précédent   Suivant

La lettre de confirmation est une édition de type formulaire (voir le paragraphe correspondant).

En général, un modèle de lettre est constitué de deux blocs :

¨   Un bloc pour l’adresse

¨   Un bloc pour le corps du document. Ce bloc doit être paramétré pour s’étendre sur les pages suivantes.

Voir l’exemple présenté dans la partie "Blocs" du paragraphe "Formulaire".

Si le modèle est destiné à être utilisé avec des liaisons de devis, il faut placer les champs qui dépendent du devis entre des champs de début et de fin de répétition. Les champs se trouvant entre ces deux marques seront répétés pour chaque devis de la liaison. Un tel modèle peut aussi très bien convenir pour un devis simple.

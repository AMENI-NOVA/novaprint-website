Accès aux fonctions principales

Précédent   Suivant

La fenêtre du programme englobe des menus, une barre d’outils et une barre d’état. Les fenêtres que vous serez amené à ouvrir seront englobées dans cette fenêtre principale.

La barre d’outils présente un accès rapide à la plupart des fonctions du programme.

La fenêtre « Menu principal » présente une autre possibilité d’accéder aux mêmes fonctionnalités regroupées par modules.

Toutes les fonctions sont accessibles par les menus :

¨   Graphisoft :

·    Gestion relation client : Accès aux fonctions de CRM

·    Gestion commerciale : Accès aux fonctions des devis, commandes, factures

·    Gestion de production : Accès aux fonctions de gestion de production

·    Gestion de stock : Accès aux fonctions de gestion des stock

·    Messagerie : Accès direct aux fonctions liées à la messagerie interne ou externe.

·    Notes : Gestion des notes

·    Edition de rapports : Accès aux rapports Crystal Report

·    Paramétrages : Accès aux paramétrages

·    Utilitaires : Accès aux fonctions de sauvegarde, restauration, interfaces etc...

·    Changer d’utilisateur : Permet de changer d’utilisateur sans quitter Graphiplus.

·    Lancer Netviewer : Démarrer l’application de support à distance (à n’effectuer que sur demande d’un technicien du service après vente Rhapso).

·    Quitter : Quitte le programme.

¨   Fenêtre :

·    En cascade : Les fenêtres sont disposées les unes sur les autres et décalées

·    Mosaïque horizontale : Les fenêtres sont réparties en bandes horizontales dans l’écran

·    Mosaïque verticale : Les fenêtres sont réparties en bandes verticales dans l’écran

·    Réorganiser les icônes : Les fenêtres réduites sont alignées au bas de la fenêtre principale.

·    Barre d’outil standard : Affiche ou masque la barre d’outils standard.

·    Barre d’état : Affiche ou masque la barre d’état au bas de la fenêtre

·    Menu principal : Affiche ou masque la fenêtre de menus permettant d’accéder aux fonctions du programme.

·    Personaliser les barres d’outils : Vous permet de modifier la barre d’outils standard ou de définir de nouvelles barres personalisées.

 

¨   ?: 

·    Sommaire de l’aide : Affiche le sommaire de l’aide en ligne.

·    Aide sur … :  Affiche l’aide en ligne

·    Dernière modifications : Affiche un document récapitulant les modifications apportées au programme.

·    Télécharger NetViewer : Télécharger la dernière version de l‘application de support à distance (à n’effectuer que sur demande d’un technicien du service après vente Rhapso).

·    Lancer Netviewer : Démarrer l’application de support à distance (à n’effectuer que sur demande d’un technicien du service après vente Rhapso).

·    Site Web : lancer une connexion sur le site web de Rhapso.

·    Utilisateurs dans Graphisoft : lister les utilisateurs présents dans l’application et dans la base de données.

·    Journal des actions utilisateurs : Affiche la liste des actions effectuées par les utilisateurs dans Graphisoft. Seulement disponible pour l’administrateur.

·    A propose de la base de données : Affiche des informations sur le moteur d’accès aux données utilisés par le programme.

·    A propos de Graphisoft : Affiche le numéro de la version du programme, les modules dont vous disposez et le nombre de droits d’utilisation.

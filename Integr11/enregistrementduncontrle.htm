Enregistrement d'un contrôle

Précédent   Suivant

Pour que le contrôle soit enregistré, il est impératif  de respecter l'unicité du nom du contrôle. Cependant, si vous saisissez un nom de contrôle qui a déjà été archivé (voir "Suppression d'un contrôle") un message vous demandera si vous voulez enlever l'archivage de ce contrôle, en faisant cette action vous pourrez alors réutiliser ce  contrôle.

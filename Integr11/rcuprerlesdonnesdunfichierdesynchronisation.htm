Récupérer les données d'un fichier de synchronisation

Précédent   Suivant

Cette fonction permet de synchroniser vos données avec celles d'un fichier de synchronisation.

 

La zone "Nom du fichier" indique l'emplacement et le nom du fichier de synchronisation. Le bouton "…" permet de sélectionner le fichier.

 

Pour lancer la synchronisation, il faut cliquer sur "Récupérer".

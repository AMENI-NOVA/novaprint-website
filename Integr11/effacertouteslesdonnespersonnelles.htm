Effacer toutes les données personnelles

Précédent   Suivant

Cette fonction est accessible depuis le menu "Utilitaires" de la liste des sociétés, si vous êtes identifiés en tant qu’administrateur de Graphisoft.

Cette fonction supprime toutes les données personnelles de la base : prénom, nom, numéros de téléphone et e-mails des contacts et utilisateurs.

Si vous êtes amenés à fournir au support technique d’EFI une copie de votre base de données, vous pouvez utiliser cette fonction pour supprimer toutes les données personnelles contenues dans votre base.

ATTENTION : vous ne devez utiliser cette fonction que sur une copie de votre base de données, car l’effacement est irréversible.

Procédure à suivre pour effectuer une sauvegarde de votre base ne contenant aucune donnée personnelle :

1.  Effectuer une sauvegarde de votre base de données

2.  Dans SQL Server, créer une nouvelle base de données dans laquelle vous restaurez cette sauvegarde

3.  Créez un nouveau raccourci de démarrage sur cette base de données.

4.  IMPORTANT : assurez-vous que ce raccourci démarre bien Graphisoft sur la copie de votre base

5.  Démarrez Graphisoft à partir de ce raccourci en tant qu’administrateur

6.  Ouvrez la liste des sociétés et cliquez sur « Effacer toutes les données personnelles » à partir du menu « Utilitaires ».
NB : vous devez confirmer 2 messages successifs pour procéder à l’effacement

7.  Effectuer une sauvegarde de votre base de données

Paramétrage des modèles d’éditions pour les connexions Print2B/Web2Print Pro

Précédent   Suivant

Dans la barre d’outils, cliquez sur l’icône  ou dans la barre de menu Fichier, Paramétrages.

Dans la fenêtre des paramétrages :

¨   Double-cliquez sur « Editions »,

¨   Double-cliquez sur « Lettre de confirmation ».

 

Il faut lier chaque connexion Print2B/ Web2Print Pro pour un client à un modèle d’édition, le même modèle d’édition pouvant être utilisé pour plusieurs connexions, même s’il s’agit de sociétés différentes.

En revanche, les modèles auxquels on peut lier une connexion Print2B doivent répondre à deux contraintes :

¨   Ils ne doivent pas être liés à une imprimante

¨   Ils doivent correspondre à la langue indiquée pour la société à laquelle se rapporte la connexion Print to B.

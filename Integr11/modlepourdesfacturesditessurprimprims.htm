Modèle pour des factures éditées sur pré-imprimés

Précédent   Suivant

Les pré-imprimés les plus couramment utilisés sont constitués de plusieurs zones distinctes qui correspondront chacune à un bloc dans le modèle :

¨   L’adresse du client

¨   Les informations générales au document

¨   Le détail des éléments de la facturation : ils seront inclus dans un tableau

¨   Le récapitulatif des taux de TVA : dans un autre tableau

¨   Le récapitulatif total

Pour gérer le fait qu’un document puisse regrouper plusieurs éléments de facture, il faut inclure les informations entre des marques de répétition (avant "Ref. cmde" et après "Montant HT". Les informations situées entre ces deux marques seront répétées autant de fois qu’il y a de factures regroupées dans le document.

De même pour que le récapitulatif des taux de TVA fasse apparaître les différents taux utilisés dans le document, il faut utiliser d’autres marques de répétition.

Vous constaterez que dans cet exemple, le modèle ne comporte aucun texte, mais seulement des champs qui seront remplacés à l’édition par les informations correspondantes, car ce modèle est prévu pour être utilisé avec des documents pré-imprimés. Mais il est tout à fait possible de rajouter des textes dans le modèle pour éditer une facture avec la même présentation, mais sur du papier à entête sans pré-impression. Par exemple, on peut facilement rajouter un tableau pour imprimer les entêtes de colonnes.

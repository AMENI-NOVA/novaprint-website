Suivi des messages internes

Précédent   Suivant

Pour suivre les messages vous disposez des informations suivantes pour chaque destinataire interne :

¨   Date de notification : Date et heure à laquelle le message à été notifié au destinataire.

¨   Date de lecture : Date et heure à laquelle le destinataire a lu le message.

¨   Action : Indique si une réponse ou une action est souhaitée pour le message. Si la réponse est faite, la date de celle-ci est affichée. Si l'action est effectuée, la date de celle-ci est affichée.

 

Ces informations sont affichées quand on sélectionne un message dans la boîte d'envoi. De plus, l'icône du message récapitule l'état du message (voir « Ecran principal » plus haut dans ce chapitre).

 

Pour diminuer le nombre de messages affichés dans les boîtes d'envoi et de réception, deux commandes existent :

¨   "Masquer les messages lus" : Masque les messages lus. Ainsi seuls les messages non lus restent affichés.

¨   "Masquer les actions faites" : Masque les messages où toutes les actions sont faites. Ainsi, il ne reste à l'écran que les messages où les actions qui ne sont pas encore effectués.

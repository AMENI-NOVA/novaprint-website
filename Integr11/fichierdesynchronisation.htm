Précédent   Suivant

Le protéger par un mot de passe

Cette option verrouille le fichier de synchronisation avec un mot de passe. Toutes les personnes qui voudront utiliser le fichier de synchronisation devront saisir le mot de passe.

L'envoyer par mail …

Si vous choisissez cette option, le fichier sera envoyé par e-mail à l'adresse saisie.

Le copier …

Le fichier de synchronisation sera copié à l'emplacement indiqué. Cela peut être une disquette (A:\) ou un répertoire sur votre disque dur. Le bouton "…" permet de sélectionner l'emplacement du fichier.

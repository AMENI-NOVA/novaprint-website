Ce service est utilisé avec le JDF.

 

Ce service permet d'intégrer les requêtes reçues (par GSTraiteHttp) dans Graphiplus. Il vérifie régulièrement si une requête est présente dans le répertorie de stockage, va l'analyser et intégrer ces informations dans Graphiplus.

Il n'est pas obligatoire que ce programme soit lancé sur le même ordinateur que GSServeurHttp.exe. Mais, évidement, il faut que ces deux services utilisent le même répertoire.

Trier les dossiers

Précédent   Suivant

Menu : "Affichage", "Trier les dossiers".

 

Il existe trois possibilités de tri pour organiser les dossiers :

¨   Via le paramétrage des colonnes de la même manière que dans la liste des devis ou des commandes : « Affichage » et « Options ».

Cette technique ne permet pas de trier sur les colonnes de poste mais il est en revanche possible d’utiliser plusieurs colonnes simultanément pour le tri : par exemple, trier sur la date de commande par ordre croissant, puis le nom du client par ordre décroissant et enfin le numéro de dossier par ordre croissant.

¨   Cliquer sur la colonne désirée. Le premier clic entraîne un tri par ordre croissant et le second, un ordre décroissant. Les derniers tris effectués sont ajoutés au menu « Affichage » et «Trier les dossiers». Ils peuvent ainsi être réutilisés sans avoir d’abord à placer le curseur sur la colonne à trier.

Cette technique ne permet pas de trier sur les colonnes de poste.

¨   Trier le tableau en fonction du contenu d’une colonne quelconque.  Pour cela, il faut être positionné sur la colonne devant servir au tri et cliquer sur Trier les dossiers dans le menu Affichage. Une option de menu correspondant à la colonne sur laquelle se trouve le curseur (par exemple Sur GTO ) vous permet ensuite d’accéder à un autre sous-menu pour trier dans l’ordre croissant, décroissant ou suivant l'ordre dans le planning mural.

Les derniers tris effectués sont ajoutés à ce menu et peuvent ainsi être réutilisés sans avoir d’abord à placer le curseur sur la colonne à trier.

Chaque colonne sur laquelle un tri est appliqué est distinguée par une image à gauche du nom dans l'entête.

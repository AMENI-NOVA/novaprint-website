Exemple : Opération  "Prise de vue":

Précédent   Suivant

Données de l’opération :

¨   Marge à appliquer (façon) = 10%

¨   Marge matière = 5%

¨   Coût fixe = 50 F.

¨   Coût de l'opération façon= 100 F.

¨   Coût matière = 30 F.

¨   Unités: Photo(s) et par couleur, par dm2 et par modèle

Le devis indique :

¨   Format fini de l'élément = 21 x 29.7 cm

¨   3 couleurs Recto, 2 couleurs Verso, 1 modèle

Vous sélectionnez l'opération "Prise de vue" et Graphisoft vous demande d'indiquer le nombre de photo(s) : vous répondez 2 par exemple.

Calcul du coût :
          Nombre d'unités : 5 couleurs x 1 modèle x 2 photo(s) x (21x29,7/100)dm2 = 62,37

Coût de l'opération : (50 x 10%) + (62,37 x 100 x 10%) + (62,37 x 30 x 5%) = 8880,36 F.

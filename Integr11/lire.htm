Pour afficher un message, il faut le sélectionner dans la liste. Il apparaît ensuite dans la partie inférieure de l'écran de messagerie. Si la zone d'affichage est trop petite pour le message, vous pouvez l'agrandir en "tirant" sa bordure vers le haut.

La commande "Afficher" du menu contextuel de la liste des messages (ou double-cliquez sur le message dans la liste) permet d'afficher le message dans une fenêtre séparée.

Auteur

Indique l'auteur du message. Le nom de l'utilisateur de GS CRM est affiché s'il s'agit d'un message interne.
Pour un e-mail reçu, le nom de l'expéditeur et son adresse e-mail sont affichés. Si l'auteur n'est pas reconnu par le programme (adresse e-mail non enregistrée dans les fichiers), un message l'indique à la suite de l'adresse. Dans ce cas, vous pouvez utiliser la fonction "Définir l'expéditeur" accessible depuis la liste des messages.

Enregistrée le

Indique à qu'elle date la note à été enregistrée dans GS CRM.
Cette date d'enregistrement est très souvent identique à la date indiquée dans la note (champs Date), mais pas toujours. Dans le cas d'un e-mail, la date indiquée dans l'e-mail est sa date de création, si l'e-mail récupéré dans GS CRM que le lendemain, la date d'enregistrement sera différente de la date de l'e-mail.

Modifiée par :

Si la note à été modifiée après sa création, le nom de l'utilisateur ayant fait la modification d'affiche ici.

Modifiée le :

Si la note à été modifiée, la date de modification est affichée ici

Etat de la note

En haut à droite, est affichée une série d’icones qui détaille l’état de la note.

 

Une info bulle apparait en passant la souris sur ces icônes. Elle décrit à quoi celle-ci correspond.

 : Une confirmation de lecture à été demandée.

 : Une preuve de dépôt à été demandée.

 : Cette note est en lecture seule.

 : Cette note est un e-mail reçu. L’info-bulle indique qui à téléchargé l’e-mail et avec quel compte de messagerie. De plus, si vous cliquez sur cette icône, le code source du message est affiché.

A, Cc, Cci

Ces trois lignes affichent les destinataires du message.

La ligne "A", affiche les destinataires normaux.

La ligne "Cc", affiche les destinataires de type "Copie conforme".

La ligne "Cci", affiche les destinataires de type "Copie conforme invisible". Cette ligne n'est visible que :

¨   Si vous êtes l'auteur de la note. Dans ce cas, tous les destinataires Cci sont affichés.

¨   Si vous êtes un destinataire Cci. Dans ce cas vous ne voyez pas les autres destinataires Cci.

 

S'il y a plus d'un destinataire, leur nombre est indiqué entre parenthèses. Le nom et prénom de tous les destinataires s'affichent les uns après les autres sur une seule ligne.

S'il y a trop de destinataires, les derniers noms ne sont pas visibles. Le bouton  permet d'étendre l'affichage des destinataires. Ainsi, tous les destinataires sont visibles, soit directement, soit en utilisant l'ascenseur.

De plus, la vue étendue indique les informations suivantes:

¨   Destinataire interne :
Destinataire du message , ou destinataire "redistribué"

·    Date de lecture de la note (si la note a été lue,  est affichée),

·    Date de notification de la note,

·    L'action liée à la note pour ce destinataire.

¨   Destinataire e-mail :

·    L'adresse e-mail utilisé pour l'envoi de la note.

·    Date de lecture de l’e-mail ( est affichée). Seulement si une confirmation de lecture à été demandée et si celle-ci à été envoyée par le destinataire.

Liée à

Cette ligne affiche les éléments liés à la note. Elle fonctionne comme la ligne des destinataires.
Le nom de tous les éléments sont affichés les uns après les autres.

Une fois étendue, les éléments sont affichés les uns en dessous des autres. Les sociétés sont affichées sur deux lignes, afin d'indiquer l'adresse.

Lorsque la liste des liaisons est ouverte, il est possible de cliquer sur une liaison pour l'ouvrir. Par exemple, si l'on clique sur une société, la liste des sociétés s'ouvre positionnée sur la société correspondante.

Pièce(s) jointe(s)

Lorsque des fichiers sont liés à une note, la ligne "Pièce(s) jointe(s)" est affichée. Elle indique le nom des fichiers ainsi que leur taille.

Les pièces jointes incorporée dans les notes ont l'icône correspondant à leur type de fichier.
Les pièces jointes liées à la note ont l'icône  (elles ne sont pas enregistrées dans la note, mais seulement leur chemin d'accès).

Un clic avec le bouton droit de la souris sur une pièce jointe, fait apparaître le menu suivant:

Ouvrir, Ouvrir avec

La commande "Ouvrir" ouvre le fichier avec l'application qui lui est associée. Si ce type de fichier n'est pas connu par Windows, la fenêtre "Ouvrir avec" s'ouvre automatiquement.

La fenêtre "Ouvrir avec" permet de sélectionner l'application à utiliser pour ouvrir le fichier. Même si une application est déjà associée à ce type de fichier.

 

Avant d'ouvrir une pièce jointe, celle-ci est d'abord enregistrée dans le répertoire temporaire de Windows avec l'attribut "Lecture seule". Cela signifie que vous ne pourrez pas la modifier directement

Enregistrer sous

Cette fonction permet d'enregistrer une pièce jointe sur votre disque dur. Une fenêtre vous demande de sélectionner le dossier de destination. Il est possible de changer le nom du fichier.

Enregistrer tout

Cette fenêtre permet d'enregistrer plusieurs pièces jointes en même temps.

Pour cela,

1. Sélectionnez les pièces jointes à enregistrer. Elles le sont toutes par défaut.

2. Sélectionnez le répertoire de destination, soit en le saisissant dans "Enregistrer vers", soit en cliquant sur "Parcourir".

3. Cliquez sur enregistrer.

Date

Date et heure de la création du message

Type

Type défini lors de la saisie du message dans GS CRM. Le type sert à classer les messages dans les différents historiques (voir "Historique des notes" dans le chapitre "Notes")

Note d'origine

La ligne "Note d'origine" apparaît si la note affichée résulte des commandes "Répondre" ou "Faire suivre" depuis GS CRM. Elle indique le nom de la note de "départ".

Si l'on clique sur cette ligne, la note d'origine est affichée.

Objet

Cette ligne indique l'objet de la note.

Corps du message

Quand un message est affiché, son contenu est analysé pour voir s'il fait référence à des éléments. Ces éléments sont les devis, les commandes/dossiers et les factures. Les textes suivants seront mis en évidence dans le message:

¨   Devis n°xxx

¨   Devis xxx

¨   Commande xxx

¨   Dossier xxx

¨   Facture numéro xxx

¨   Facture xxx

Les textes trouvés sont affichés en bleu et souligné. Si vous cliquez dessus, la liste correspondant à l'élément est ouverte et positionnée sur celui-ci. Attention, la liste peut-être filtrée, il est possible que l'élément souhaité ne soit pas visible.

De plus, les noms de fichiers et les adresses Internet apparaissent comme des liens. Vous pouvez cliquer dessus pour les ouvrir.

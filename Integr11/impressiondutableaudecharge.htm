Impression du tableau de charge

Précédent   Suivant

La fonction Imprimer… du menu Tableau de charge permet d'effectuer une édition sur imprimante du tableau de charge.

Pour utiliser cette fonction, vous devez posséder le droit "Edition du planning en cours" ou "Edition du planning en prévisionnel", selon le type de tableau à imprimer.

Si le nombre de colonnes est trop important pour la largeur de la feuille, le programme imprimera une seconde feuille pour les colonnes suivantes et ainsi de suite.

Il est possible d'agir sur l'édition en modifiant les paramètres suivants :

¨   Titre de l'édition : Titre qui apparaîtra dans l'entête de chaque page.

¨   Afficher les filtres dans le pied de page : Affiche les filtres en cours au moment de l'édition dans le pied de chaque page. Cette information est utile pour connaître la portée de l'édition.

¨   Imprimer tous les dossiers... : Permet de ne pas tenir compte des filtres en cours et d'imprimer tous les dossiers du tableau de charge.

¨   Imprimer le tableau en couleurs : Si vous disposez d'une imprimante couleur ou niveaux de gris, cette option est intéressante car elle permet de distinguer les lignes paires et impaires.

¨   Indiquer l'état d'avancement des fiches : Cette option n'est disponible que si l'on imprime le tableau en couleurs. Ceci permet d'imprimer la couleur d'avancement des fiches de travail. Cette couleur est identique à celle affichée à l'écran.

¨   Imprimer avec le paramétrage actuel ou avec le modèle... : Choisissez "avec le paramétrage actuel" si vous souhaitez imprimer le tableau avec les mêmes regroupements de postes que ceux utilisés pour l’affichage à l’écran. Sinon, vous pouvez imprimer le tableau de charge avec le modèle de regroupements de postes de votre choix.

¨   Afficher cette fenêtre avant chaque édition : Si vous ne souhaitez pas modifier ces paramètres a chaque édition, décochez cette case. Vous pourrez de toute façon modifier ultérieurement ces informations en cliquant sur Préférences… dans le menu Tableau de charge.

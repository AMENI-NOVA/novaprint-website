Les modèles d’éditions

Précédent   Suivant

Les modèles d’édition permettent d’enregistrer les paramètres choisis pour un type d’édition (Etat des stocks, historique etc.). Les modèles sont gérés à partir du menu des éditions.

Toutes les sélections effectuées sont enregistrées dans les modèles.

Création d’un modèle

Pour créer un modèle, il faut indiquer son nom, saisir les paramètres propres à l’édition et cliquer sur « Enregistrer ». Le modèle ainsi créé peut être édité immédiatement en appuyant sur le bouton «Visualiser » ou à partir du menu des éditions auquel il est ajouté.

Suppression d’un modèle

Pour supprimer un modèle, sélectionnez le dans le menu et demandez « Supprimer » puis confirmez.

Modification d’un modèle

Pour modifier un modèle, vous pouvez soit demander « Propriétés », soit double cliquer sur le nom du modèle. L’écran de modification est le même que celui de la création sauf que les boutons « Précédent » et « Suivant » peuvent être actifs. Ces boutons permettent de passer d’un modèle à un autre pour un même type d’édition.

Duplication d’un modèle

Pour dupliquer un modèle, sélectionnez le dans le menu et demandez « Dupliquer ». Vous devez saisir un nom pour le nouveau modèle.

Edition d’un modèle

Les modèles peuvent être édités à partir des écrans de modification et de création mais aussi à partir du menu des éditions. Lorsqu’on demande une édition de modèle à partir du menu, la saisie des paramètres n’est pas proposée de nouveau et l’édition se fait directement.

Modèles par défaut

La procédure d'installation de Graphiplus initialise le fichier des modèles avec des modèles par défaut vous présentant certaines des possibilités offertes. Vous pourrez ensuite modifier ces modèles pour les adapter à vos besoins, et supprimer ceux qui ne vous conviennent pas.

Les modèles d'édition de la gestion des stocks sont enregistrés dans un fichier nommé MacrosGS.dat. Lors de l'installation, si un fichier de ce nom est déjà présent, le fichier des modèles par défaut n'est pas installé. Pour installer les modèles par défaut, vous devrez alors copier le fichier MacrosGS.dat se trouvant sur le CD d'installation de Graphiplus, dans le répertoire dans lequel sont stockés les fichiers de données de Graphiplus (ce répertoire se trouve sur un serveur dans le cas où vous travaillez en réseau). Attention, dans ce cas, les modèles d'édition que vous aurez créés seront alors écrasés. Cette manipulation doit impérativement être faite par une personne maîtrisant le système d'exploitation utilisé.

Charger un modèle existant

Précédent   Suivant

Pour charger les paramètres d’affichage d’un modèle existant, vous pouvez :

¨   Sélectionner le modèle voulu dans la liste déroulante de la barre d’outils ou du menu.

¨   Cliquez sur le nom du modèle dans la barre d’outils, si celui-ci s’y trouve (voir plus loin le paragraphe « Ajouter et supprimer des raccourcis de modèle dans la barre d’outils »).

¨   Ouvrir la fenêtre des propriétés du modèle courant, choisir un modèle dans la liste déroulante, puis cliquer sur « Appliquer »

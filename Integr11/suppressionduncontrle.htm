Suppression d'un contrôle

Précédent   Suivant

Pour supprimer un contrôle, sélectionnez le contrôle, cliquez droit dessus, puis cliquez sur supprimer.

La suppression est possible que si le contrôle n'est pas utilisé par un dossier, dans ce cas le contrôle sera supprimé définitivement.

Si vous essayez de supprimer un contrôle dans la liste du paramétrage alors qu'il est utilisé dans la planification d'un dossier, le logiciel vous proposera de l'archiver. Cette action entrainera la suppression du contrôle dans les modèles de regroupements ainsi que dans les modèles de planning. Cependant si un point de contrôle est planifié dans un dossier, il ne sera pas supprimé.

Note : Si un contrôle est archivé il n'apparait plus dans le tableau des charges.

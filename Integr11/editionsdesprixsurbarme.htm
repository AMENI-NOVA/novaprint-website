Editions des prix sur barème

Précédent   Suivant

Dans le paramétrage des éditions de la lettre de confirmation, de l’accusé-réception de commande et de la facture, un champ « Descriptif du prix sur barème» est disponible pour faire afficher la description des opérations composant le prix de vente.

Ce nouveau champ présente un format qui permet de limiter les informations affichées, définies dans chaque opération par le paramétrage du barème.

Vous avec le choix parmi : 

¨   « Sans prix » : seules la famille, le nom de l’opération et le nombre d’opérations sont affichés.

¨   « Prix Totaux » : la famille, l’opération, le nombre et le total sont affichés.

¨   « Prix détaillés » : le détail pour obtenir le prix total pour la quantité demandée est affiché.

 

Ce champ renvoie la totalité du texte décrivant le prix sur barème. Les différentes valeurs sont séparées par des tabulations.

NB : Pour aligner correctement les valeurs, vous devriez placer le champ dans une ligne avec :

-   3 tabulations fer à droite à 7cm ; 8,5cm ; 10,3cm.

-   Puis 2 tabulations fer à gauche à 10,5cm ; 12,5cm

-   Enfin 2 tabulations fer à droite à 15,7cm et 18cm

Bien sûr, ces valeurs sont données à titre d’exemple et vous devrez les ajuster selon le niveau de détail et les textes des opérations, en particulier le libellé de l’opération et de l’unité.

 

Vous pouvez ainsi afficher à la fois ou sélectivement une description du produit réalisé et le détail des éléments qui composent le prix de vente avec plus ou moins de détail.

Remarque : si vous changez dans le paramétrage la façon dont s’affichent les opérations, la modification ne sera effective que pour les prix sur barèmes enregistrés ensuite ; pas pour les prix déjà enregistrés. Pour l’appliquer à un prix enregistré, ouvrez le prix sur barème et cliquez « Ok ».

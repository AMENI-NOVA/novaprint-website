Modèles et versions

Précédent   Suivant

Lorsque vous faites un devis comportant des modèles en amalgames détaillés, Graphisoft crée en interne autant de versions qu’il y a de modèles.

Par défaut, ces versions ne sont pas visibles pour ne pas surcharger la saisie de devis.

Dans certains cas, vous pouvez avoir besoin d’accéder aux versions créées automatiquement, par exemple pour regrouper plusieurs modèles dans une même version, ou pour saisir des opérations de façonnage qui s’applique à certains modèles mais pas aux autres.

Pour afficher les versions créées automatiquement, cliquer sur l’icône  de la barre d’outils, ou cliquez sur « Versions » dans le menu « Devis optimisé ».

NB : une fois les versions affichées, il n’est plus possible de revenir en arrière et de les masquer.

Par défaut les versions ont le même nom que les modèles.

Lorsque les versions sont visibles pour le devis, la quantité des modèles n’est plus modifiable dans la saisie détaillée des modèles. Les modifications de quantité doivent être faites au niveau des versions.

Lorsque vous supprimez un modèle qui n’est présent que dans une seule version (visible ou non), cette dernière est supprimée du devis.

Interface devis au format XML

Précédent   Suivant

Graphisoft dispose d’une interface au format XML pour les devis :

¨   Depuis la liste des devis, la fonction « Ouvrir un fichier XML » permet d’initialiser la saisie d’un nouveau devis à partir d’informations contenues dans un fichier XML

¨   Le composant métier Devis de Graphisoft dispose d’une interface permettant de calculer un devis dont le détail se trouve dans une structure XML.

En retour, la structure XML est complétée avec des informations techniques concernant la solution retenue par le moteur de calcul.

La structure XML utilisée par ces fonctionnalités pour décrire un devis est décrite dans le guide de mise en route du Service Web de calcul de devis.

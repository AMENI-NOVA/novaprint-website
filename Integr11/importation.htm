Importation

Précédent   Suivant

Le programme va lire indépendamment chaque ligne du fichier d'importation pour la traiter.

Pour chaque information de Graphisoft, la donnée correspondante est lue dans le fichier d'importation. Si celle-ci n'est pas récupérable ou si elle est vide, le programme prend la valeur par défaut.

 

Si une société avec le même nom interne existe déjà dans le fichier des sociétés de Graphisoft, un choix vous est proposé :

¨   Ne pas importer : Cette ligne du fichier d'importation ne sera pas traitée, et le traitement passera à la ligne suivante.

¨   Mettre à jour : La société existante sera mise à jour avec les informations du fichier d'importation.

¨   Nouvelle société : Une nouvelle société sera créée avec un nom différent. Par exemple : "Graphisoft" deviendra "Graphisoft 2", "Graphisoft 3", … La société existante ne sera pas modifiée.

¨   Arrêter : Ce choix arrête l'importation. Cette action n'annule pas l'importation des sociétés déjà traitées.

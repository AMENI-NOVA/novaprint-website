But de l'interface d’importation des papiers

Précédent   Suivant

L’interface papier est chargée d’importer un fichier au format texte dans les tarifs utilisés par Graphisoft pour calculer les devis.

Si le papier importé existe déjà dans votre base, il sera mis à jour avec les tarifs du fichier importé. Par contre, le paquetage de calcul défini dans votre base ne sera pas modifié. Il définit en effet le minimum de facturation à vos clients que vous avez défini.
La correspondance entre un papier importé et un papier existant se fait seulement en utilisant le nom du papier et le nom du fournisseur. Vous pouvez donc changer la catégorie ou le code du papier après l’avoir importé : La prochaine importation les retrouvera tant que vous ne changez pas le nom du papier ou du fournisseur.

Si vous supprimez des papiers d’une catégorie importée, la prochaine importation les créera à nouveau s’ils sont sélectionnés pour l’importation. Il se passera la même chose pour les grammages ou formats supprimés après import dans votre fichier : La prochaine importation recréera les grammages et formats manquants. Pour exclure des articles de l’importation, utilisez les coches de l’arbre de sélection. Ces choix sont enregistrés et restaurés lors des importations ultérieures.

Si des articles sont supprimés du catalogue fournisseur, une liste vous permettra de sélectionner ceux à supprimer également de votre base de données Graphisoft. La suppression ne se fera que si vous cochez les articles correspondants dans la liste présentée.

Remarque pour les utilisateurs de GS Buyer (anciennement appelé Optiplus) et du module multi-société GS Group : Seuls les tarifs de la société utilisatrice sont concernés par l’interface papier. Par contre, la nomenclature des papiers peut-être modifiée par la création et éventuellement la suppression de papiers ou d’articles.

 

Pour effectuer l’importation d’un fichier de tarif, il faut utiliser une description de ce fichier. Si elle n’est pas dans la liste présentée au lancement de l’interface, il vous faudra la créer ou l’obtenir de nos services.

L’option « Importer un export Graphisoft » permet d’importer un fichier exporté depuis la liste des papiers Graphisoft. Cette option utilise une description prédéfinie conforme à l’export créé par Graphisoft.

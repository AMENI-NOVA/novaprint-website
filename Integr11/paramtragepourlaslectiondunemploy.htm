Paramétrage pour la sélection d'un employé

Précédent   Suivant

Il est possible de paramétrer l'affichage du nom des employés et la durée d'affichage d'un employé en début de liste (employé habituel du terminal).

 

Pour atteindre l'écran de paramétrage, il faut cliquer sur le bouton "Options" dans la barre des actions ou appuyer sur [Ctrl]+[F12]:

 

Ces paramètres sont enregistrés séparément pour chaque terminal.

 

Nom des employés :

Ce paramétrage permet de définir de quelle manière seront affichés les employés dans la liste (un seul choix possible):

¨   Code de l'employé :              seul le code de l'employé est affiché.

¨   Prénom + nom de l'employé :  le nom et le prénom sont affichés dans la liste.

¨   Code + prénom + nom de l'employé : cette option cumule les deux précédentes.

Recherche :

Si la case « Seulement sur le CODE de l’employé » est cochée, et que l’on tape des lettres au clavier, seul le code de l’employé est utilisé pour de la recherche.

Durée d'affichage :

Ce paramétrage permet de définir combien de temps un employé sera affiché en début de liste (employé habituel du terminal).

Par défaut cette valeur est fixée à 7 jours.

Si au bout de cette durée l'employé n'a pas utilisé le terminal, il apparaîtra en grisé dans la seconde partie de la liste.

Retour automatique à la liste des employés

Ce paramétrage défini la durée d’inactivité (en minutes) après laquelle la liste des employés est affichée automatiquement.

Cette  fonction n’est active que si un employé utilisant le terminal est paramétré avec la saisie systématique de son mot de passe. Voir le chapitre « 3. Paramétrage généraux », la section « Employés » puis le paragraphe « Terminal Atelier : Saisie du mot de passe systématique ».

Bouton Appliquer

Applique les paramètres sur la liste des employés sans fermer la fenêtre. Cela permet de voir immédiatement l'incidence d'une modification.

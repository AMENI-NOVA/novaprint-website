Démarrage « standard »

Précédent   Suivant

Tous les services Graphisoft peuvent être lancés comme un programme normal. C'est-à-dire avec une interface utilisateur qui permet d’interagir avec lui et aussi de voir ce qui se passe.

Pour lancer un service dans ce mode, il faut utiliser le gestionnaire de service Graphisoft (GSServcies.exe), sélectionner le service et cliquer sur « Démarrage standard ».

Le service est alors démarré avec une interface utilisateur, en utilisant les paramètres de ligne de commandes définis lors de son installation.

 

Attention, le compte utilisateur peut être différent entre le « mode service » et le « mode standard ».

En mode service, le compte utilisateur utilisé est celui défini lors de l’installation du service.

En mode standard, le compte utilisateur est celui de la session courante.

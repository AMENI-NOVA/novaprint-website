Opérations sur barème

Précédent   Suivant

Entrez le libellé de l’opération. Vous pouvez ici déplacer l’opération dans une autre famille en changeant sa famille.

Attribuez un centre de coût pour l’opération afin que la facturation puisse répartir les prix facturés.

Indiquez sur quelle valeur le tableau de tarifs de l’opération sera basé :

¨   Quantité devisée,

¨   Nombre d’opérations,

¨   Quantité devisée x Nombre d’opérations.

Si le calcul est basé sur le nombre d’opérations ou la quantité devisée x le nombre d’opérations vous pouvez spécifier un libellé pour l’unité des opérations.

 

S’il y a lieu, sélectionnez le groupe de comptes qui doit être imposé lors de la facturation. En choisissant (non défini), le groupe n’est pas imposé par le barème.

Lors de la facturation d’une commande avec un prix sur barème, le groupe de comptes appliqué sera celui imposé par les opérations du prix sur barème, à défaut le groupe de comptes défini par le client ou à défaut le produit qui est utilisé.

Si plusieurs groupes de comptes apparaissent dans le prix sur barème, autant de lignes de facture que de groupes différents seront créées et les opérations du prix sur barème seront réparties sur l’élément avec le groupe de comptes leur correspondant.

 

Vous pouvez choisir le mode de calcul de l’opération sur barème entre les méthodes « Prix de vente et 1000 en + » ou « Prix aux » et définir une unité de prix (à l’unité, aux 100, aux 1000 etc.).

 

Les cases à cocher permettent de paramétrer l’affichage de cette opération dans le descriptif sur les courriers.

 

Durant la saisie d’une opération sur barème, l’affichage en bas de la fenêtre présente un exemple d’affichage du descriptif du prix sur barème tel qu’il pourra être affiché sur les courriers (lettre d’offre, accusé-réception, facture).

 

NB : Si vous renommez une opération sur barème, un message vous demande si vous voulez mettre à jour cette opération dans les prix sur barèmes définis dans les modèles de devis. Répondez Oui si vous voulez que les modèles de devis conservent leur lien vers cette opération, non dans le cas contraire. Dans le second cas, il vous faudra modifier manuellement les modèles de devis concernés ou renommer une copie de l’opération avec le nom d’origine. En effet, le lien entre les devis sur barème et le paramétrage s’effectue par le libellé de la famille et de l’opération.

Cas où le schéma d'imposition ne peut être affiché

Précédent   Suivant

Les schémas d'imposition ne peuvent pas être édités pour les devis en attente ni les devis de prépresse.

Remarques

¨   Cette édition ne pourra être obtenue que pour les devis enregistrés avec la version 5.0 (ou plus récente) de Graphiplus. Si vous voulez l'édition des schémas pour un devis fait avec une version antérieure, vous devrez le recalculer par le devis optimisé (et non par le devis simplifié).

¨   Certaines modifications au devis simplifié peuvent empêcher l’affichage du schéma d’imposition du devis.
Ce sont :

·    Le format fini global du devis,

·    Les format fini, format à plat, format de l’amalgame et format du papier d’un élément du devis,

·    Le développement ou la laize d’un élément d’impression rotative ou continu.

¨   L'édition du schéma est impossible si le format d'imposition a été forcé à la saisie du devis (voir "Elément de dépliant" dans le chapitre "Devis optimisé").

¨   Pour les devis enregistrés avec une version 11.5 ou antérieure, le schéma est affiché suivant l’ancienne présentation. Pour avoir le schéma dans sa nouvelle présentation, vous devez recalculer le devis.

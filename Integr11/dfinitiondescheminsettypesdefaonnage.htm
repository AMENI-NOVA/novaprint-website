Chemins de façonnage

Un chemin de façonnage permet de décrire la succession des machines à utiliser et des opérations à effectuer pour un type de façonnage.

Les chemins de façonnage sont utilisés pour décrire les opérations à calculer pour la finition. Ils s’appliquent à tous les types de devis, sauf les devis de continu.

Vous pouvez créer ou modifier les chemins en définissant et ordonnant vous-même les étapes, parmi des opérations par défaut proposées par Graphisoft, des types de machines de façonnage ou des opérations particulières de façonnage que vous avez paramétrées.

Par exemple pour la couture fil, le chemin par défaut est constitué des étapes suivantes :

¨   Pliage,

¨   Assemblage cahiers,

¨   Couture fil,

¨   Collage à chaud,

¨   Coupe 3 faces.

Types de façonnage

Pour pouvoir être utilisés lors du calcul d’un devis, les chemins doivent se trouver dans des types de façonnage.
Un type de façonnage désigne un type de finition, par exemple "2 piqûres à cheval", ou "Dos carré collé".
Un type de façonnage peut comporter de 1 à 5 chemins différents. Par exemple, le façonnage "2 piqûres à cheval" est constitué par défaut d’un chemin utilisant une encarteuse piqueuse, et d’un chemin utilisant une assembleuse piqueuse pour des 4 pages.
Un même chemin peut appartenir à plusieurs type de façonnage différents.
C’est la liste des types de façonnage qui est proposée à la saisie des devis pour le choix "Façonnage". Le calcul du devis sera optimisé pour chaque chemin du type de façonnage sélectionné.
Vous pouvez modifier les types de façonnage existants (changer leur nom ou le libellé apparaissant à l’édition, ou y intégrer de nouveaux chemins), ou créer de nouveaux types de façonnage en utilisant des chemins existants ou de nouveaux chemins.
Vous pouvez créer autant de types de façonnage que vous voulez.

La liste arborescente des paramétrages vous permet de voir les types de façonnage existants, ainsi que les chemins de chaque type en ouvrant le type correspondant.

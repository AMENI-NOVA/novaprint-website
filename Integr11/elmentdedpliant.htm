Nombre de pages

Le nombre de pages détermine les possibilités de saisie pour le pliage (voir "Pliage").

Pour indiquer que l'élément n'est pas plié, saisissez un nombre de pages égal à 2.

Formats

C’est le format à plat qui est pris en compte pour optimisé l’impression.

Si vous avez paramétré des formats standards, le bouton <...> vous permet de sélectionner un format dans la liste.

Pliage

Vous devez indiquer le nombre et le type des plis. Vous avez la possibilité de mixer des types de plis différents, par exemple : 2 plis roulés + 1 pli croisé. Dans ce cas, Graphisoft considèrera pour le calcul, qu'il s'agit de plis croisés (ici 3 plis croisés), mais sur la lettre de confirmation, le libellé aura la même précision que lors de la saisie du devis.

Pour saisir les plis, indiquez le nombre de plis du premier type, puis le type de ces plis. S’il y a plusieurs types de plis différents, utilisez les zones suivantes. Il est possible de mixer 3 types de plis différents.

Les possibilités de saisie pour le pliage dépendent du nombre de pages indiqué plus haut. Par exemple, pour pouvoir saisir plusieurs types de plis, le nombre de pages doit au moins être égal à 8. D'autre part, si vous avez saisi 2 pour le nombre de pages, l'élément n'est pas plié et la saisie du pliage est donc impossible.

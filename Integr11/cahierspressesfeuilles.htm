Types de cahiers autorisés

Vous pouvez sélectionner les types de cahier autorisés pour la machine.

¨   Si aucun type de cahier n'est sélectionné (dans la liste "Types de cahier autorisés"), tous les types de cahier et amalgames pourront être utilisés avec cette machine lors de l'optimisation des devis.

¨   Si au moins un type de cahier est sélectionné, seuls les types de cahier présents dans la liste "Types de cahier autorisés" seront utilisés avec cette machine lors de l'optimisation des devis. D'autre part, tous les amalgames seront ignorés avec cette machine, même ceux ne contenant que des types de cahier sélectionnés.

Les sélections effectuées ne s'appliquent que pour l'optimisation des cahiers de brochure. Pour les encarts et les couvertures, ou les autres types de devis, les machines sont toujours prises en compte, mêmes si des cahiers ont été sélectionnés.

Pour sélectionner ou déseléctionner des types de cahier, utilisez les boutons  et , ou double-cliquez sur les types de cahier pour les faire passer d'une liste à l'autre.

Commande de production d’un produit fini

Précédent   Suivant

Vous pouvez créer directement une commande à partir d’un stock de produit fini pour lequel une description a été enregistrée.

Ouvrez la liste des stocks, sélectionner le stock, et depuis le menu « Stocks » ou le menu contextuel, sélectionnez l’option « Commande de production… ».

NB : Ce choix n’est disponible que pour les articles ayant une description enregistrée. L’information « Description » de la liste des stocks permet de voir les articles pour lesquels une description a été enregistrée.

Vous pouvez alors saisir la quantité souhaitée et indiquer le point et l’adresse de livraison.

En cliquant sur suivant, le devis est recalculé avec la quantité et la livraison demandée puis le prix de vente calculé est présenté.

·      Si le stock défini son prix de vente, le prix de vente de la commande est le prix de vente du stock augmenté du montant de la  livraison calculé (avec le coefficient défini dans la description).

·      Si le stock ne définit pas son prix de vente alors le prix de vente de la commande est le total vente du devis calculé qui reprend le coefficient de la description attachée à l’article.

En cliquant « créer le dossier », vous enregistrerez le nouveau devis avec le total vente imposé comme décrit ci-dessus et la commande y fera référence.

Familles d’opérations de pelliculage vernis

Précédent   Suivant

Les opérations de pelliculage vernis sont groupées en familles d’opérations. Une opération appartient toujours à une famille.

Pour créer une nouvelle famille d’opérations, sélectionnez la rubrique "Pelliculage vernis", puis cliquez sur Nouveau dans le menu Paramétrages ou dans le menu contextuel, ou cliquez sur le bouton correspondant de la barre d’outils.

Ce type de vernis peut être fait sur un groupe d'impression

Cette option sert à identifier les vernis qui peuvent être passés en utilisant un des groupe d'impression de la machine, comme par exemple les vernis offset. Dans ce cas, n'importe quel machine d'impression pourra être utilisée pour faire des vernis de cette famille, même celles qui n'ont pas de groupe vernis spécifique.

NB : cette possibilité ne concerne que les presses offset feuilles.

Si la machine n'a pas de groupe spécifique pour ce type d'opération, on considère que le vernis ne peut pas être fait au même passage que l'impression.

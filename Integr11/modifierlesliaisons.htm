Modifier les liaisons

Précédent   Suivant

"Modifier les liaisons" dans le menu "Notes" ou le menu contextuel.

 

Cette fonction permet de modifier les liaisons de la note pointée. Attention si vous supprimez toutes les liaisons, la note ne sera plus visible dans les historiques.

Pour être autorisé à l'utiliser, il faut disposer du droit "Modifier les liaisons" du groupe "Notes" "Notes d'autres personnes".

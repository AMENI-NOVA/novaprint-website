Prévisualisation de l’impression.

Précédent   Suivant

Pour prévisualiser l’impression d’un document :

¨   Cliquez sur le bouton « Aperçu »  pour faire apparaître la fenêtre de prévisualisation :

¨   Utilisez les outils de la barre d’outils pour parcourir le document, zoomer, etc. Ils sont aussi accessibles dans un menu contextuel que l’on obtient par un click droit sur la fenêtre.

¨   Cliquez sur l’outils « Fermer » ou appuyez sur la touche <Echap> pour fermer la fenêtre.

 

Commandes

Description

Fermer

Ferme la fenêtre de pré visualisation.

Imprimer

Imprime la feuille courante

Première page

Affiche la première page.

Page précédente

Affiche la page précédente.

Page suivante

Affiche la page suivante.

Dernière page

Affiche la dernière page.

Zoom avant

Agrandi le zoom.

Zoom arrière

Diminue le zoom.

Zoom

Permet de choisir le zoom dans la liste ou saisir directement sa valeur.

Largeur page

Une page occupe la largeur de la fenêtre.

Page entière

Une page occupe toute la fenêtre.

Deux pages

Deux pages l’une à coté de l’autre occupent toute la fenêtre.

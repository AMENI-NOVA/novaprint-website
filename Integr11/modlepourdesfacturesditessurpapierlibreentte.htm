Modèle pour des factures éditées sur papier libre à entête

Précédent   Suivant

Créez un bloc pour l’adresse et un autre pour les informations générales au document (date, n° de facture).

Le reste des informations peuvent être mises dans un grand bloc constituant le corps du document. Ce bloc doit pouvoir s’étendre sur les pages suivantes si nécessaire.

Pour gérer les regroupements de plusieurs factures dans un même document, vous devez inclure les champs se rapportant aux éléments de factures entre deux champs de répétition. La marque de début de répétition à utiliser dans ce cas est celle qui concerne les éléments du document, comme c’est indiqué dans la partie droite de la fenêtre de choix du champ à insérer.

Les montants sont alignés sur la droite en utilisant une tabulation décimale.

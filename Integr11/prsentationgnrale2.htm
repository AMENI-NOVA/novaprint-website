Les notes ne sont disponibles qu'avec le module GS CRM.

 

Le but des notes est de pouvoir saisir des commentaires sur divers éléments de Graphisoft: une société, un devis, une commande et un poste de gestion de production.
Ces notes sont ensuite disponibles sous la forme d'un historique, mais elles s'affichent aussi quand on travaille sur l'élément concerné.

De plus, il est possible de les envoyer à d'autres personnes par l'intermédiaire de la messagerie.

Pour plus d'informations sur la création des notes, voir le paragraphe " Création d'une note ".

 

Toutes les notes d'un élément sont regroupées dans une liste que l'on appelle "Historique des notes".
Il est possible d'utiliser les notes pour saisir les contacts que l'on a avec ses clients, ce qui permet de disposer par clients, d'une liste chronologique des contacts.

Pour plus d'informations, voir le paragraphe " Historique des notes ".

 

Les notes peuvent êtres regroupées dans des rubriques des informations internes. Cela permet d'organiser par thèmes et de partager les notes avec d'autres utilisateurs.

Pour plus d'informations, voir le paragraphe " Enregistrer la note dans les rubriques des informations interne ".

 

La création de notes peut-être accélérer en utilisant des modèles de notes. Un modèle pouvant contenir toutes les informations d'une note, il ne reste plus qu'à le compléter pour enregistrer une note.

Pour plus d'informations, voir le paragraphe " Modèles de notes ".

 

Les notes sont saisies par l'utilisateur, mais Graphisoft peut générer des "notes automatiques". Par exemple, si un utilisateur modifie le prix d'un devis dans le dossier technique, une note peut être créée par le programme. Ainsi, dans l'historique du devis, mais aussi, dans celui du client, une note indiquera de combien le devis à été modifié et qui à fait la modification.
Pour plus d'informations, voir le paragraphe " Notes automatiques ".

 

Il est possible de faire afficher les notes d'un élément à chaque fois que l'on travaille avec. C'est le "Rappel des notes". Au moment de faire un devis ou une facture, si des notes existent pour ce client, celles-ci peuvent s'afficher automatiquement à l'écran. Ainsi, les informations importantes ne peuvent plus être oubliées.

Pour plus d'informations, voir le paragraphe " Rappel des notes ".

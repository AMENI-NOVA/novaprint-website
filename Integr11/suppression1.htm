Suppression des messages reçus

Si vous choisissez "Manuelle", les messages reçus resteront dans votre boîte de réception jusqu'au moment où vous les supprimerez.
" Automatique : les messages traités depuis plus de …" signifie que le programme supprimera automatiquement les messages reçus x jours après leur "traitement".

Suppression des messages envoyés

Si vous choisissez "Manuelle", les messages envoyés resteront dans votre boîte d'envoi jusqu'au moment où vous les supprimerez.
"Automatique : les messages traités depuis plus de …" signifie que le programme supprimera automatiquement les messages envoyés x jours après leur "traitement".

Purge de la corbeille

Si vous choisissez "Manuelle", les messages supprimés resteront dans votre corbeille jusqu'au moment où vous les supprimerez.
"Automatique : Purge immédiate" signifie que les messages reçus ou envoyés que vous supprimez sont immédiatement purgés. Il n'y a donc aucun moyen de les récupérer en cas d'erreur.
"Automatique : les messages traités depuis plus de …" signifie que le programme supprimera automatiquement les messages envoyés x jours après leur "traitement" et pas leur suppression.

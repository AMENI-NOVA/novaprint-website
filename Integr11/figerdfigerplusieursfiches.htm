Figer / Défiger plusieurs fiches

Précédent   Suivant

Il est possible de figer / défiger toutes les fiches du début du planning  jusqu’à un certain temps via l’échelle des temps. Pour figer ou défiger toutes les fiches jusqu'à un certain temps, il faut cliquer sur la date souhaitée dans le l’échelle des temps.

 

Une fenêtre s'ouvre et propose de figer ou défiger toutes les fiches antérieures à la date définie.

La date peut être modifiée en cliquant sur le calendrier.

Le lien « Spécifier les postes... » permet de sélectionner les postes sur lesquels l'action prendra effet.

Par défaut, les postes sélectionnés sont les postes sélectionnés dans les préférences de l'utilisateur.

 

Cliquer sur « Ok » pour valider.

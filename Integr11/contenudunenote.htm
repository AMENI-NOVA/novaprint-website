Contenu d'une note

L'objet

La saisie de l'objet est importante. C'est toujours l'objet qui est affiché en premier.
Dans les boîtes aux lettres (de réception ou d'envoi), c'est l'objet qui apparaît et non le texte du message. De même, dans l'historique des notes, seul l'objet peut être affiché dans les colonnes.

Si l'objet d'une note est vide, le programme enregistre automatiquement les 60 premiers caractères du texte dans l'objet.

Le type

Le type de la note sert à trier et à filtrer les notes dans les historiques de notes.

Pour ne pas "typer" une note, il faut choisir "(non défini)".

Les types présents dans la liste sont paramétrables. Pour cela, aller dans "Messagerie / Notes" puis "Type de note" dans l'écran de paramétrage.

Si vous saisissez un type qui n'est pas présent dans la liste et que vous disposez du droit "Paramétrage des types de notes", le nouveau est ajouté à la liste. Sinon, le programme refuse votre saisie.

La date

La date permet un classement chronologique des notes.

A la création d'une nouvelle note, cette zone contient la date et l'heure courante. Il est possible de modifier la date proposée. Afin d'éviter les erreurs de saisies, il est interdit de saisir une date postérieure à la date courante.

Lorsque vous modifiez une note, au moment de l'enregistrer, le programme vous demande si vous souhaitez conserver la date d'origine (précédemment saisie) ou si la note doit être enregistrée avec la date courante.

Le texte

La saisie du texte de la note est facultative.

Le texte de la note peut être mis en forme (gras, italique, taille de la police, etc.). Pour mettre une partie du texte en forme, il faut procéder exactement comme dans un traitement de texte classique.

Pour modifier la couleur du texte et celle du fond, il faut utiliser les commandes "Couleur du texte" et "Couleur de fond" du menu "Outils".

Lecture seule / Modifications autorisées

En bas à gauche de l'écran de saisie de la note un bouton indique si la note sera modifiable par d'autres utilisateurs que l'auteur.

              Tous les utilisateurs peuvent modifier cette note.

            Seul l'auteur peut modifier cette note.

Pour passer d'un mode à l'autre, il suffit de cliquer sur le bouton.

 

Par défaut, les notes créées depuis les informations internes sont modifiables par tout le monde. Dans tous les autres cas, la note est en lecture seule.

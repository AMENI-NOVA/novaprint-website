Opérations de pelliculage et de vernis (traitement de surface)

Précédent   Suivant

Types d'opération

La liste déroulante vous présente les opérations quelle que soit leur famille.

Si l'opération sélectionnée appartient à une famille liée à une ou plusieurs machines d'impression (c'est-à-dire dans le cas où une machine d'impression a un groupe vernis permettant de réaliser les opérations de cette famille : voir "Offset feuilles : Opérations additionnelles"), le programme de calcul optimisera le coût de l'opération entre la machine d'impression et le tarif spécifique (généralement celui de votre sous-traitant).

Recto ou Recto/Verso

Après avoir sélectionné une opération, vous devez indiquez s'il s'agit d'une opération recto seul ou recto/verso.

Couverture

Si le pourcentage de couverture est :

¨   inférieur à 100%, il s’agit alors d’un vernis avec réserve qui nécessite une ou deux plaques vernis.

¨   supérieur ou égal à 100%, il n’y a pas de réserve. Le pourcentage peut être supérieur à 100% pour tenir compte d’une conmmation de matière exceptionnelle.

En option

Vous avez ensuite la possibilité de décider si cette opération est une option du devis. Dans ce cas, sur la lettre de confirmation (et l'accusé réception de commande), le total du devis sera édité sans le montant de cette opération, qui sera inclus dans une 2ème ligne de total (et mille en +).

Impression typo (suivant le type de prépresse)

Il est possible de choisir une opération de vernis offset qui sera comptée sur la machine typo utilisée pour l’impression. Dans ce cas, un calage et un lavage seront comptés au coût d’impression, ainsi que des tours supplémentaires. La matière est comptée comme le vernis sur machine offset. Il n’y a pas de réserve possible ; on peut saisir une valeur inférieure à 100%, mais aucun coût de plaque n’est compté.

Chargement d'un devis calculé avec une opération n'existant plus

Si vous chargez un ancien devis, calculé avec une opération de surface qui n'existe plus dans vos paramétrages, le devis est chargé sans l'opération.

Saisie de + de 2 opérations

Lorsque vous sélectionnez une 2ème opération, un bouton <+ d’opérations> affiche une fenêtre dans laquelle un tableau vous permet de saisir autant d’opérations que vous le souhaitez.

Ce tableau permet également d’ordonner les opérations entre elles.

Lorsque vous validez la fenêtre, les 2 premières opérations apparaissent dans les zones de saisie habituelles et les opérations suivantes sont mentionnées au-dessous.

La liste arborescente sur la gauche, indique les 3 premières opérations et le nombre d’opérations supplémentaires.

Ordre des opérations

S'il y a au moins 2 opérations, l'ordre de la saisie sera utilisé pour l'optimisation du devis. Cela implique que la 2ème opération et les suivantes ne pourront jamais être calculées au même passage que l'impression. Par exemple, si vous saisissez Pelliculage + Vernis offset, l'opération de vernis offset sera forcément calculée avec un passage en machine séparé de l'impression.

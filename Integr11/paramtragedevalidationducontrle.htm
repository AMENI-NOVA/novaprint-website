Paramétrage de validation du contrôle

Précédent   Suivant

Indique si la validation du contrôle est requise. Si la validation n’est pas requise, le contrôle sera uniquement destiné à notifier une personne. Si cette option est activée, vous devez alors sélectionner la personne pouvant effectuer le contrôle dans l’onglet "Validation du contrôle".

Les choix possibles sont :

¨   Par une personne désignée.  Cette option active le cadre «Personne désignée »  Vous devez sélectionner parmi les options proposées les personnes pouvant effectuer le contrôle.

¨   Par le poste. La personne qui valide le contrôle est l'employé qui traite la fiche de travail dans l'atelier

Personne désignée

Cet ensemble d’options permet de spécifier quelles sont les personnes pouvant effectuer le contrôle.

Les options disponibles sont:

¨   Commercial du dossier

¨   Information complémentaire du dossier.

Vous devez dans ce cas sélectionner  l’information complémentaire dans la liste située en regard. Cette liste affiche les informations complémentaires de type fonction. La personne, spécifiée pour l'information complémentaire du dossier concerné par le contrôle, pourra effectuer le contrôle.

¨   Employé désigné : sélectionnez une personne dans la liste. Dans ce cas, c'est toujours cette personne qui est chargée de valider ce contrôle.

Attention : Si la personne désignée dépend du dossier (commercial ou information complémentaire), vous devez vous assurer que les commerciaux, ou les employés appartenant à la fonction liée à l'information complémentaire sélectionnée, fassent bien partie des utilisateurs de type "classique" ou "Graph'email", car dans le cas contraire la personne désignée n'aura pas la possibilité de valider le contrôle.
Si la personne désignée est un utilisateur de type "Graph'email", elle ne pourra valider le contrôle qu'en effectuant l'action liée au message de notification envoyé lors de l'activation de ce dernier. Cela implique que le contrôle doit être paramétré pour qu'une notification soit envoyée à la personne désignée pour la validation.

 

Dans le cas où la personne désignée ci-dessus serait indisponible, 3 cases à cocher supplémentaires permettent d'indiquer qui peut aussi valider ce contrôle :

¨   De la même fonction : validation possible par une personne de la même fonction que la personne désignée

¨   De la fonction : validation possible par une personne de la fonction à choisir dans la liste.

¨   Désignée : validation possible par une personne à choisir dans la liste.

Notifier la personne devant valider

Indique s’il faut notifier la personne devant effectuer le contrôle. Cette option n’est disponible que si la validation du contrôle est activée et l’option validation par une personne désignée est sélectionnée

Remarque :

Si l’option "Contrôle bloquant" est sélectionnée, l’option "Validation du contrôle" est forcée et ne peut être désactivée

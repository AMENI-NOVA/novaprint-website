Edition sur imprimante

Précédent   Suivant

¨   Depuis l’arbre de paramétrage , sélectionnez l’élément à éditer dans la liste arborescente :

·    "Papiers" : pour éditer l’ensemble de votre fichier papier

·    Une catégorie : pour éditer les informations concernant cette catégorie ainsi que tous les papiers qui la composent

·    Un papier : pour éditer les informations concernant ce papier ainsi que tous ses tarifs avec les grammages (ou épaisseurs) et formats (ou laizes).

¨   Cliquez sur Imprimer dans le menu Paramétrages ou dans le menu contextuel, ou cliquez sur le bouton correspondant de la barre d’outils.

¨   Choisissez l’imprimante à utiliser pour l’édition.

¨   Les informations à éditer sont affichées dans une fenêtre dans laquelle vous pouvez procéder à des modifications.

¨   Pour imprimer, cliquez sur Imprimer dans le menu Fichier.

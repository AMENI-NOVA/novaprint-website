Imposition non basculée

Coût de répétition  =  ( Programmation 
                             +  Nombre de couleurs total
                                     x  (   [Nombre de bouts par pose x Calage d’un film]
                                        + Calage plaque + [Nombre de poses x Coût par pose]    )     )
                               x   Nombre de cahiers ou de formes  x  Marge                           

Imposition basculée

¨   Si Format de 2 poses (R°+V°) plus petit que le Format maxi des films de la machine à répéter :
Répétition du R°+V° (une seule opération)
Coût de répétition  =  Programmation  +  ( Nombre de couleurs  x  Nombre de poses  /  2 x Coût par pose)
Remarque : Nombre de couleurs  =  Couleurs recto + Couleurs verso - Couleurs basculables
                       +  (Changements  x  (Changements recto + Changements verso - Changements basculés)

¨   Si Format de 2 poses (R°+V°) plus grand que le Format maxi des films de la machine à répéter :
Répétition du recto puis du verso (deux opérations)
Coût de répétition  =  (Programmation  x  2)
            +  (Nombre de couleurs basculées  x  Nombre de poses  x Coût  par pose)
            +  (Nombre de couleurs non basculées  x  Nombre de poses  /  2  x  Coût par pose)

Coût total de répétition  =   (Coût de répétition (suivant un des deux cas précédents)
                                         +  (Nombre de bouts par pose  x  Calage d’un film  x  Nombre de couleurs total)
                               +  (Nombre de plaques  x  Calage plaque)  )
                           x  Nombre de cahiers ou de formes  x  Marge

Remarques :

¨   Nombre de couleurs total =

·    Devis autres que liasses et continu :  Couleurs recto + Couleurs verso + Changements

·    Liasses et continu : Nombre de plaques

¨   Nombre de bouts par pose :  Nombre de modèles amalgamés dans une pose, ou Nombre de pages / 2 si couverture ou encart ou dépliant et choix de la surface de film en fonction du format fini (voir « Informations générales sur les films »).

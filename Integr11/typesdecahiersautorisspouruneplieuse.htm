Types de cahiers autorisés pour une plieuse

Précédent   Suivant

Le paramétrage des plieuses permet d’indiquer les types de cahier que la plieuse peut plier.

Lorsque la plieuse est paramétrée pour les brochures, l’onglet « Cahiers » permet d’indiquer les cahiers possibles.

Lorsque la case « Tous les types de cahiers » est cochée, tous les types de cahiers peuvent être pris en compte dans la limite du nombre et du type de plis, et du grammage correspondant, définis dans l’onglet « Caractéristiques ».

Lorsque la case « Tous les types de cahiers » n’est pas cochée, la liste des types de cahier est visible et vous permet d’indiquer ceux qui sont autorisés pour la plieuse, avec pour chacun le grammage maximum. Dans ce cas, le nombre de plis et les grammages maximums définis dans l’onglet « Caractéristiques » sont ignorés pour les cahiers de brochure.

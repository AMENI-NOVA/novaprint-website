Afficher les points de contrôle

Précédent   Suivant

La fonction Affichage des points de contrôle… du menu Traitement permet de faire apparaître la liste des instances de point de contrôle pour le dossier et le contrôle (ou le regroupement) sur lequel se trouve le curseur.

Cette fonction est accessible directement par la touche F2.

¨   S’il existe une seule instance du point de contrôle et que son état peut être modifié ( « prêt » ou « non validé ») alors le programme affichera automatiquement la fenêtre de validation du contrôle.

¨   S’il existe plusieures instance du point de contrôle (ou une seule mais dont l’état n’est pas modifiable ( « non prêt » ou « validé »)), la liste de ces instances est affichée. Si une instance dont l’état est modifiable est sélectionnée (la ligne est en fond bleu), le bouton « Sélectionner » est dégrisé et affiche (en cliquant dessus) la fenêtre de validation.

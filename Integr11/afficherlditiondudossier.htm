Afficher l’édition du dossier

Précédent   Suivant

Pour afficher l’édition du dossier de la fiche en cours, il faut appuyer sur [F10].

 

Cette fonction permet d'afficher soit « l'état détaillé du dossier en temps », soit « le dossier de fabrication ».

L’édition affichée dépend d’un paramétrage défini dans Graphiplus.

 

Pour plus d'informations sur « l’état détaillé du dossier en temps », voir le chapitre "Editions et statistiques de la gestion de production".

Pour plus d’informations sur « le dossier de fabrication », voir la section « Dossier de fabrication » du chapitre « Commandes ».

 

Pour se déplacer dans l'édition, il faut utiliser les touches [Page haut] ou [Page bas].

 

Par défaut, le dossier de fabrication s’affiche dans la fenêtre principale du Terminal Atelier. Pour cela, il utilise Acrobat Reader, qui doit donc être installé sur l’ordinateur.

S’il n’est pas possible d’installer Acrobat Reader, le dossier de fabrication peut d’afficher mais cette fois-ci dans une fenêtre séparée. Dans ce cas l’utilisateur doit utiliser la souris pour fermer le dossier de fabrication et revenir au Terminal Atelier.

Le mode d’affichage est paramétrable dans les options du Terminal Atelier.

 

Paramétrage de l’édition utilisée :

Pour effectuer ce paramétrage, ouvrez l’arbre de paramétrage de Graphiplus, puis sélectionnez « Gestion de production », « Paramétrages généraux » et enfin « Atelier : Edition du dossier ».

 

 

Edition utilisée par défaut…

Choisissez dans la liste déroulante l’édition que vous souhaité vous apparaître dans l’atelier.

… sauf dans les cas suivants

Il est possible de définir des cas particuliers en fonction du service, du poste ou du type de devis concerné.

Par exemple, vous pouvez choisir un modèle de dossier de fabrication différent entre le service prépresse et l’impression. De plus, un dossier de fabrication peut être différent entre une brochure et une liasse.

 

Pour ajouter un cas particulier, positionnez-vous dans la ligne commençant par une * et sélectionnez le service ou le poste, puis le type de devis et enfin l’édition.

Pour modifier un cas particulier, sélectionnez la ligne correspondante et effectuez la modification.

Pour supprimer un cas particulier, cliquez dans la colonne grisée à gauche afin de faire apparaître la flèche sur la ligne souhaitée. Ensuite appuyer sur la touche [Suppr.].

 

Le programme recherche l’édition, dans l’ordre suivant, en fonction:

1.  du poste et du type de devis ;

2.  du poste sans type de devis ;

3.  du service et du type de devis ;

4.  du service sans type de devis ;

5.  du type de devis sans service ni poste ;

Si la commande ne correspond pas aux critères définis, le programme utilise l’édition par défaut.

 

Dossier de fabrication enregistré

Cette option permet d’indiquer si l’on souhait afficher systématiquement le dossier de fabrication déjà enregistré.

Ainsi, si vous avez enregistrés des annotations dans le dossier de fabrication, celles-ci seront visibles dans l’atelier.

 

Attention si vous avez enregistrés plusieurs dossiers de fabrication pour le même dossier, seul le dernier sera affiché.

 

Si aucun dossier de fabrication n’est trouvé, alors celui-ci est créé en fonction du modèle paramétré ci-dessus.

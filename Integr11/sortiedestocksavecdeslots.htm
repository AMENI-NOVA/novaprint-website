Sortie de stocks avec des lots

Précédent   Suivant

A la sortie d’un stock, si la gestion des lots est activée pour ce stock, vous devrez spécifier le ou les lots qui sont sortis et en quelle quantité.

Dans la pièce de sortie :

¨   Si une réservation existe, sélectionnez la,
sinon sélectionnez l’article livré puis le possesseur,
ou saisissez le code d’un lot connu dans la colonne « Code Article »,
ou tapez un code lot fournisseur reconnu (Code IFRA) dans la colonne « Code Article ».
Remarque : Selon la saisie, plus ou moins d’informations seront remplies dans la ligne d’écriture. Voir plus loin la remarque « Sortir un lot inexistant ».

¨   Dans la colonne quantité, entrez ou corrigez la quantité totale puis tapez « Entrée » ou cliquez sur le bouton « ... » pour accéder au détail des lots.


La colonne « Sélection » permet d’indiquer les lots qui interviennent dans le mouvement, les colonnes quantité et unité permettent de ne sortir que partiellement un lot. Le paquetage, la position, l’état et la quantité actuelle du lot sont affichés pour information et ne sont pas modifiables.
Le cumul des quantités des lots est affiché ainsi que le total de l’écriture pour aider au choix des lots.
Remarque : Si vous avez saisi une réservation à compenser, les lots (dont l’état est « Ok ») nécessaires pour satisfaire la réservation sont sélectionnés par défaut. Eventuellement un lot n’a été pris que partiellement si son paquetage est positif.

¨   Confirmez les saisies en cliquant le bouton « Ok » (« Ctrl+S ») ou bien annulez avec le bouton « Annuler » (« Ctrl+Z »). La fenêtre des lots est fermée et la ligne d’écriture en cours de saisie est mise à jour avec le total de la quantité sélectionnée.

¨   Entrez d’autres lignes d’écriture pour d’autres stocks qu’ils soient ou non gérés par lot.

¨   Confirmez la pièce avec le bouton « Ok »

Sortir un lot déjà totalement sorti ou un lot inconnu de la Gestion des Stocks

A la mise en place de la gestion par lot, il se peut que certains lots n’aient pas encore été pris en compte dans l’inventaire. Plus tard, on aura peut-être oublié de retourner un lot ou tout simplement on voudra sortir une matière alors que la Gestion des Stocks déclare qu’il n’y en a plus. On utilise normalement une écriture de régularisation pour rétablir la réalité des faits dans la Gestion des Stocks, la sortie donne toutefois un raccourci pour résoudre ce genre de cas

Dans l’écran de sélection des lots :

¨   Entrez une quantité à sortir et, s’il y a lieu, le code du lot dans la colonne « Code Lot ». Il peut s’agir du code d’un lot connu ou d’un code reconnu (IFRA).

¨   La ligne est sélectionnée automatiquement. Entrez ainsi tous les lots sortis avec leurs quantités.

¨   Lorsque vous confirmez les saisies, un dialogue vous propose pour chaque lot de :

·    Retourner la quantité nécessaire si le lot est connu et a été sorti précédemment.

·    Régulariser à la quantité que vous déclarez sortir si le lot est connu mais présente une quantité nulle sans sortie retournable.

·    Initialiser le lot à la quantité nécessaire si le lot est inconnu.

¨   Si vous confirmez, les lots seront créés et les écritures de retour et de régularisation validées automatiquement et immédiatement dans les stocks.

¨   Si vous ne confirmez pas :

·    Si le lot n’existe pas, la pièce ne pourra pas être validée

·    Si le lot existe mais la quantité est nulle ou insuffisante, vous pourrez enregistrer la pièce et les lots pourront présenter suite à sa validation des quantités négatives. Il faudra alors appliquer une régularisation ou un retour pour rétablir la quantité réelle.

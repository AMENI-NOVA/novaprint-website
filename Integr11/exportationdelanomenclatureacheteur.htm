Exportation de la nomenclature "Acheteur"

Précédent   Suivant

Accessible depuis la feuille de paramétrage : "Calcul des devis, Optiplus, Exporter sa nomenclature".

 

L'exportation de la nomenclature consiste à créer un fichier qui sera utilisé ensuite par un imprimeur pour exporter son paramétrage.

 

La fenêtre suivante présente un tableau hiérarchique contenant tous les éléments de la nomenclature.

Vous pouvez sélectionner les éléments que vous souhaitez exporter. Lors de la première utilisation, tout est sélectionné. Les fois suivantes, le programme se souvient des choix précédents.

 

Pour sélectionner (ou désélectionner) un élément, il faut cocher (ou décocher) la case se trouvant à gauche de son nom.

Pour sélectionner un groupe d'éléments en une seule fois, il suffit de sélectionner l'élément "parent".

Un élément grisé indique que certains de ces "enfants" sont sélectionnés, mais pas tous.

Une fois les sélections effectuées, vous devez indiquer le chemin et le nom du fichier de destination. Le bouton <Parcourir…> fait apparaître une fenêtre pour choisir un fichier existant.

Le bouton <Exporter> crée le fichier contenant la nomenclature. Vous devez ensuite le transmettre aux imprimeurs concernés. Il n'est pas obligatoire que tous les imprimeurs reçoivent le même fichier de nomenclature.

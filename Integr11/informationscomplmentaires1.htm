Informations complémentaires

Précédent   Suivant

Vous pouvez saisir ici des valeurs pour les informations complémentaires du devis que vous avez paramétrées (voir "Informations complémentaires des devis et des dossiers" dans le chapitre "Paramétrages généraux").

Le contenu de l’information peut être soit saisi directement dans le tableau, soit choisi dans une liste déroulante.

Lorsque vous transformez un devis en commande, les informations complémentaires du devis sont proposées par défaut pour la nouvelle commande.

Les informations complémentaires peuvent aussi être saisies pour les modèles de devis. Lorsque vous faites un nouveau devis à partir d'un modèle, les informations complémentaires du modèle sont automatiquement recopiées dans le nouveau devis. De même, lorsque vous créez une nouvelle commande à partir d'un modèle de devis, les informations complémentaires du modèle sont proposées par défaut pour la nouvelle commande.

Lorsque vous ouvrez un devis existant contenant des informations complémentaires, et que vous l'enregistrez, soit sous un nouveau numéro, comme variante ou comme modèle de devis, les informations complémentaires du devis d'origine sont automatiquement recopiées dans le nouveau devis ou modèle.

Les informations complémentaires sont également recopiées lorsque vous enregistrez un devis depuis le devis simplifié.

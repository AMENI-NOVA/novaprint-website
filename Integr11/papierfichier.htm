Papier fichier

Précédent   Suivant

Vous pouvez sélectionner une catégorie de papier pour laisser Graphisoft optimiser entre tous les papiers de cette catégorie, ou imposer un papier en particulier.

Le choix d’une catégorie ou d’un papier peut s’effectuer de deux manières différentes :

¨   Saisie du code désignant la catégorie ou le papier. Cette méthode est la plus rapide, à condition de connaître le code.

¨   Choix dans la liste : cliquez sur le bouton <…> se trouvant à droite de la zone de saisie pour faire apparaître la liste des catégories. Choisissez une catégorie ou un papier, puis cliquez sur le bouton <Sélectionner>.
Remarque : Si un papier existe dans différentes couleurs, la ligne du papier ne fait pas apparaître son code et une arborescence supplémentaire (+) permet de choisir la couleur.

Après avoir indiqué le type de papier, Graphisoft affiche son libellé et éventuellement sa couleur puis affiche dans une liste déroulante tous les grammages disponibles pour ce papier, parmi lesquels vous pouvez choisir.

Couleur du papier

Si vous avez sélectionné une catégorie comportant des papiers existant dans différentes couleurs, une zone déroulante vous permet de sélectionner la couleur souhaitée parmi celles existantes pour le grammage que vous avez indiqué. Si une couleur n’est pas définie pour un papier, elle n’est pas listée.

S'il y a plusieurs couleurs possibles, le choix "Quelconque" est ajouté à la liste pour permettre de ne pas tenir compte de la couleur du papier; dans ce cas le programme de calcul choisira le papier le moins cher dans le format adéquat.

Si vous sélectionnez une des couleurs proposées, seuls les papiers de cette couleur seront pris en compte lors de l'optimisation.

Pour les devis de liasses, la zone de saisie de couleur du papier apparaît systématiquement, même si la catégorie sélectionnée contient des papiers pour lesquels la couleur n'est pas définie. Vous pouvez alors indiquer le nom de la couleur du papier. Cette information n'a alors aucun impact sur l'optimisation.

Papier du fichier avec prix d’achat imposé

En cochant « Prix d’achat », vous pouvez imposer le prix du papier pour ce devis, tout en calculant celui-ci à partir des formats de papiers existants dans le fichier.

Prix d’achat imposé pour une catégorie de papier

Si vous avez choisi une catégorie de papier et que vous imposez le prix d’achat, Graphisoft vous propose la liste des papiers de cette catégorie dans une liste déroulante se trouvant à droite de l’unité du prix d’achat. Vous devrez alors sélectionner celui sur lequel le devis sera calculé. En effet, si vous imposez un prix du papier pour une catégorie, le programme de calcul n’a plus aucun moyen de choisir entre les différents papiers de cette catégorie.

Si le papier n’est pas sur fabrication, la liste des papiers de la catégorie est réduite à ceux pour lesquels des prix sont enregistrés pour le grammage sélectionné. Si le papier est sur fabrication, tous les papiers de la catégorie sont présents dans la liste.

Prix d’achat imposé pour un papier avec un prix à la feuille ou au paquet

Si vous saisissez un prix d’achat de papier dont l’unité est la feuille ou le paquet, une autre liste déroulante vous permet de choisir le format sur lequel sera calculé le devis, parmi ceux existant dans le fichier pour le papier sélectionné. En effet, dans ce cas le prix ne peut pas être le même pour chaque format.
Si vous utilisez Graphistock, vous pourrez ainsi profiter au moment de la transformation du devis en commande de la liaison automatique entre le papier sélectionné au devis et un article du stock, tout en imposant votre prix. Dans la précédente version, la seule manière d’imposer un prix de papier était de passer par un papier hors fichier, ce qui rendait toute liaison automatique impossible avec le stock.

Unité de prix de papier proposée par défaut

Si vous cochez « Prix d’achat » pour un nouveau devis ou un devis chargé mais pas encore calculé, c’est l’unité de prix de papier définie dans les préférences de saisie qui sera proposée par défaut.

SI le devis vient d’être calculé, c’est l’unité de prix de papier du devis calculé qui sera proposée par défaut.

Imposer un format ou une laize

Le bouton <Formats> affiche dans une fenêtre les formats et les laizes existants en fonction de la sélection effectuée : catégorie ou papier, grammage et couleur.

Vous pouvez alors décocher des formats des laizes ou des paquetages pour en imposer certains.

Lorsque vous avez imposé certains formats ou laizes et que vous modifiez ensuite la catégorie, le papier, le grammage ou la couleur, le programme vous demande si vous souhaitez conserver vos sélections. Si Oui, les formats et laizes pris en compte dans l’optimisation seront ceux qui ont été imposés et qui sont disponibles suivant la nouvelle saisie. Si non, tous les formats et laizes en fonction de la nouvelle saisie seront pris en compte pour le calcul du devis.

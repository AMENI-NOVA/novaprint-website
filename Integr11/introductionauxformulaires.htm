Le paramétrage d'un formulaire consiste à disposer sur le fond de page, un ou plusieurs blocs dans lesquels on peut ensuite insérer du texte fixe ou des champs qui seront remplacés par les informations à éditer.

Enregistrement des modèles

Vous pouvez enregistrer le modèle avec le même nom ou utiliser l’option « Enregistrer sous » pour enregistrer une copie sous un autre nom.

Impression des modèles

Vous pouvez imprimer le modèle avec ou sans les cadres délimitant les zones de texte pour vérifier leur positions sur du papier pré-imprimé.

Exportation, Importation des modèles

La version enregistrée du modèle courant peut être exportée au format « Xml » pour être importée à la place d’un modèle du même type. Remarque : Les images utilisées dans le modèle ne sont pas enregistrée dans le fichier d’export, il faut les enregistrer séparément si vous souhaitez les joindre.

La saisie du prépresse dépend du type de devis (voir « Informations d’entête communes à tous les types de devis »).
Il est global au devis pour les feuillets de liasses/carnets, les liasses en continu, et les affiches multi-morceaux. Dans les autres cas, le prépresse est spécifique à chaque élément du devis.

Type de prépresse

La liste déroulante vous permet de sélectionner le type de prépresse à partir duquel le devis sera calculé, parmi ceux enregistrés dans votre paramétrage. Le calcul de l’élément de devis sera optimisé parmi les chemins que contient le type choisi.

Dans le paramétrage des types de prépresse, vous avez la possibilité d’en sélectionner un qui sera celui proposé par défaut à chaque nouveau devis.

Pour les devis de prépresse, cette saisie est surtout indicative. Il n'y a aucun calcul ni optimisation entre les chemins du type de prépresse sélectionné. Les opérations de prépresse des différents chemins ne sont pas non plus prises en compte. Seules les opérations de prépresse saisies avec le devis sont calculées.

Complexité

Indiquez le niveau de complexité pour les tarifs de flashage et d'opérations de prépresse.

Cette saisie n'apparaît pas pour la gamme Estiprint, ou si vous avez choisi de n'utiliser qu'un seul niveau de complexité.

Par défaut, les niveaux de complexité sont numérotés de 1 à 10, mais vous pouvez leur donner des noms qui apparaîtront à la saisie des devis et dans le dossier technique (voir "Niveaux de complexité" dans le chapitre "Paramétrages de calcul des devis").

Films

Si le type de prépresse sélectionné a été paramétré avec "Films à réaliser" comme "Eléments de départ", des zones de saisie supplémentaires vous permettent d’indiquer le nombre et le prix des films.

Ces informations sont remplies automatiquement par Graphisoft de la manière suivante :

¨   1 ou 2 films (suivant le paramétrage : voir le paragraphe "Films" dans le chapitre "Paramétrages de calcul des devis") par page et par couleur.

¨   Le tarif affiché provient du fichier des films négatifs. Il s'agit de celui du film le moins cher, correspondant au format. Il comprend les prix façon et matière auxquels s'additionnent les marges correspondantes.

Pour un nouveau devis, le calcul est fait lors de l’affichage de l’onglet de préparation ou sinon juste avant le calcul du devis.

Pour un devis existant, les films ne sont pas recalculés lors du chargement du devis.

Dans tous les cas, si une donnée intervenant dans le calcul des films (format de l’élément, nombre de pages, nombre de modèles, nombre de couleurs, nombre de changements) est modifiée, Graphisoft demande juste avant le calcul du devis si les films doivent être recalculés. Cette question n’est pas posée si le nombre ou le prix des négatifs est modifié après avoir modifié la donnée.

Le calcul des films peut être refait à tout moment an appuyant sur le bouton "Valeurs par défaut"

Lorsque vous modifiez le nombre et le prix des films, si vous diminuez le prix, la différence s'appliquera d'abord sur le montant de façon, puis le reste sur le montant de matière. Attention, si vous diminuez le prix d'un montant équivalent ou supérieur au montant de la façon enregistré dans le fichier, il n'apparaîtra pas de temps sur le dossier de fabrication, ni au planning dans la gestion de Production.

Opérations de préparation
Opérations enregistrées dans le paramétrage

Pour ajouter une opération de préparation se trouvant dans votre paramétrage, cliquez sur le bouton « Ajouter » du cadre « Opérations supplémentaires ».

Une nouvelle fenêtre est affichée, vous permettant de pointer l’opération à ajouter au devis.

Les opérations sont regroupées par famille. Cliquez sur le ‘+’ se trouvant à gauche de chaque famille pour voir les opérations qu’elle contiennent.

Lorsque vous rappelez un devis comportant des opérations de prépresse, le coût de celles-ci est recalculé en fonction du tarif existant dans votre paramétrage, pour tenir compte des éventuelles modifications.

Si les tarifs de l'opération sélectionnée sont tous à 0 (suivant le niveau de complexité indiqué pour l'élément de devis), Graphisoft vous demandera de saisir les tarifs à appliquer pour cette opération sur ce devis. Cette saisie vous est automatiquement proposée dès que vous  sélectionnez un autre élément du devis en cours, ou avant de lancer le calcul.

Les opérations sélectionnées apparaissent dans le tableau récapitulatif qui comprend 3 colonnes :

¨   Le nom de l’opération

¨   Le nombre d’unités libres saisi pour ce devis (si l’opération comporte une unité libre : voir « Opérations » dans le chapitre « Paramétrages de calcul des devis »).

¨   Le libellé de l’unité libre

Le nombre d’unités libres peut être modifié dans le tableau, après que l’opération ait été sélectionnée,ou lorsque vous rappelez un devis existant avec une opération comportant une unité libre.

Opération hors fichier

Pour ajouter une opération de prépresse non référencée dans votre paramétrage, saisissez un libellé dans la zone « Opération hors fichier » et un coût dans « Coût total (hors marge) ».

Cette opération sera prise en compte uniquement pour ce devis et ne sera pas ajoutée au paramétrage.

La marge globale des opérations de prépresse de votre paramétrage, viendra s’ajouter automatiquement en fonction du montant saisi (voir "Préparation (opérations diverses) : Marges" dans le chapitre "Paramétrages de calcul des devis").
Lorsque vous rappelez un devis pour le recalculer, la marge globale utilisée précédemment est conservée, même si elle n’est plus la même dans le paramétrage. En revanche, si vous changez le devis de tarif ou si vous modifiez le libellé ou le montant de l'opération, Graphisoft ira recharger la marge globale se trouvant dans votre paramétrage.

Lorsque le devis comporte plusieurs versions, l’opération hors fichier est rattachée à la 1ère version.

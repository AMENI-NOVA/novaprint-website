Coûts de tirage

Précédent   Suivant

¨  Nombre de tours : nombre de fois qu'une feuille passe en machine (1 seule fois si impression recto/verso avec dispositif pour faire repasser la feuille pour l'impression verso => cadence double)

¨  Nombre de passages par feuille :

·   2 si impression recto/verso et Impression recto/verso est à Non pour la machine : l'opérateur doit recharger la machine avec les feuilles imprimées au recto

·   1 si impression recto/verso et Impression recto/verso est à Oui pour la machine :

§  Soit la machine a des groupes d'impression recto/verso

§  Soit la machine a un dispositif pour faire automatiquement repasser les feuilles par les groupes d'impression pour l'impression du verso => cadence double => prix des tours doublé

Port IP à écouter

Cette zone permet d'indiquer le port où seront envoyées les requêtes http.

Le port doit contenir une valeur numérique supérieure à 0. Deux programmes sur le même ordinateur ne peuvent écouter le même port.

Si un autre programme écoute déjà ce port, une erreur est affichée, vous devrez alors choisir une autre valeur.

Répertoire temporaire de stockage

Cette zone permet de définir le répertoire de stockage des requêtes http reçues le temps qu'elles soient traitées par le programme GSTraiteHttp.

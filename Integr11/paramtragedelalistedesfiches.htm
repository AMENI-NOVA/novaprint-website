Paramétrage de la liste des fiches

Précédent   Suivant

Pour atteindre l'écran de paramétrage, il faut cliquer sur le bouton "Options" dans la barre des actions ou appuyer sur [Ctrl]+[F12]:

 

Cet écran permet de définir les colonnes qui seront affichées dans la liste. Il permet également d'indiquer jusqu'à quelle date les fiches planifiées doivent apparaître.

 

Ces paramétrages sont enregistrés dans le fichier Atelier.ini sur le serveur. Ils sont donc communs à tous les terminaux utilisant le module Atelier.

 

Bouton Appliquer:    Applique les paramètres sur la liste des fiches sans fermer la fenêtre. Cela permet de voir immédiatement l'incidence d'une modification.

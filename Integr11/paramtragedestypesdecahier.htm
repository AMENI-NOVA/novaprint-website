Paramétrage des types de cahier

Précédent   Suivant

Pour y accéder, sélectionnez la rubrique « Types de cahier... », dans l’arbre de paramétrage.

L’arbre présente la liste des types de cahier disponibles et les chemins de brochage pour lesquels ils sont utilisables.

La liste est triée dans l’ordre croisant du nombre de pages.

Si la liste des chemins de brochage d’un type de cahier est vide, cela signifie que tous les chemins de brochage sont utilisables pour réaliser ce type de cahier.

 

Pour modifier les caractéristiques d’un type de cahier ou le renommer :

Sélectionnez-le dans la liste, puis cliquez sur Propriétés ou Renommer dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils.

Pour ajouter un nouveau type de cahier :

Sélectionnez dans la liste, la rubrique « Type de cahier », puis cliquez sur Nouveau dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils.

Pour supprimer un type de cahier :

Sélectionnez-le dans la liste, puis, cliquez sur Supprimer dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils.

Il n’est pas possible de supprimer un type de cahier utilisé par un amalgame de cahier. Il faut supprimer l’amalgame auparavant.

Pour imprimer les caractéristiques d’un type de cahier :

Sélectionnez-le dans la liste, puis :

¨   cliquez sur Imprimer dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils

ou

¨   affichez les propriétés du type de cahier, puis cliquez sur le bouton imprimer.

Pour imprimer les caractéristiques de tous les types de cahier :

Sélectionnez dans la liste, la rubrique « Type de cahier », puis cliquez sur Imprimer dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils.

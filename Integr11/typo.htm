Ce type de machine est prévu pour les opérations de :

¨   Rainage

¨   Découpe

¨   Numérotage, s'il n'est pas possible au même passage que l'impression sur la machine offset, ni sur l'assembleuse feuilles (devis de liasses, carnets).

¨   Perforation

¨   Gaufrage (voir Elément de brochure, Opérations typo, dans le chapitre "Devis optimisé).

Vous avez aussi la possibilité d’ajouter jusqu’à 5 opérations paramétrables, qui doivent être créées au préalable dans la rubrique « Calcul des devis », « Opérations typo paramétrables ». Ces opérations sont communes à toutes les machines typo, et vous pouvez pour chaque machine quelles opérations sont prises en comptes avec ses coûts associés.

Impression sur typo

La machine typo peut également être prise en compte pour l'impression, sauf si la case "Impression" est décochée. Vous devez aussi sélectionner un type d’encre pour la machine (onglet « Types d’encre »).

Pour cela, à la saisie du devis, vous devez indiquer "Photogravure typographique".

Dans ce cas, l'impression sera toujours envisagée, soit en 1 pose ou 2 poses basculées (pas de duplis). La prise de pince est fixée par défaut à 1 cm.

Prise de pince

La prise de pince est prise en compte quelle que soit l’opération effectuée sur la machine : impression, numérotage, perforation, …

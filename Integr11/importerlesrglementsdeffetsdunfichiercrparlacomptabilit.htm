Importer les règlements d’effets d’un fichier créé par la comptabilité

Précédent   Suivant

Si votre interface comptable le permet, vous pouvez importer un fichier exporté par votre comptabilité pour importer directement dans Graphisoft les règlements des effets. Le fichier et le paramétrage de l’import dépendent de votre comptabilité. Voir Paramétrages généraux – Encours clients.

Le fichier issu de la comptabilité doit contenir des écritures lettrées des comptes clients avec des informations indiquant la facture concernée, le code journal, la date de règlement etc. Les factures et avoirs trouvés sont marqués réglés. Si une information de règlement concerne une facture dans un relevé, le relevé est éclaté en tous ses éléments pour qu’ils soient traités individuellement.

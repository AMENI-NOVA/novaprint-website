Fonctionnement en mode « standard »

Précédent   Suivant

La partie supérieure de l'écran affiche les requêtes non traitées.

La partie inférieure indique les requêtes non intégrées dans Graphiplus car elles comportent des erreurs. L'erreur détectée est affichée sous la requête.

 

Si l'on double-clique sur une requête elle s'affiche dans une fenêtre séparée.

 

Le bouton <Démarrer / Arrêter> permet de démarrer (ou d'arrêter) le traitement des requêtes reçues.

Le bouton <Actualiser> affiche les requêtes reçues (actif seulement si le traitement est arrêté).

Le bouton <Historique> affiche la liste des requêtes déjà traitées (actif seulement si le traitement est arrêté). Si on double-clique sur une requête elle s'affiche dans une fenêtre.

 

Messages en erreurs :

Le bouton <Réessayer> relance le traitement sur la requête en erreur sélectionnée.

Le bouton <Réessayer tout> relance le traitement sur toutes les requêtes en erreur.

Le bouton <Purger> supprime la requête en erreur sélectionnée.

Le bouton <Purger tous> supprime toutes les requêtes en erreurs.

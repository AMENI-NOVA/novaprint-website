Format des sauvegardes

Précédent   Suivant

En fonction de la version de SQL Server, le fichier de sauvegarde n’a pas la même extension :

·      Jusqu’à SQL Server 2008 (10.00), l’extension de la sauvegarde est « .svg ». La sauvegarde est réalisée par SQL Serveur, puis le fichier est compressé par Graphisoft.

·      A partir de SQL Server 2008 R2 (10.50), l’extension est « .bak ». La sauvegarde est réalisée et compressée par SQL Server. De plus, celle-ci peut aussi être restaurée en utilisant les utilitaires de SQL Serveur comme SQL Server Management Studio.

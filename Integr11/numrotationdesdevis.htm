Numérotation des devis

Précédent   Suivant

Les devis sont numérotés par un compteur simple : 1, 2, 3, … précédé d’un préfixe composé de 3 lettres maximum. Le préfixe peut être vide. Il sert à différencier les devis provenant de bases différentes, par exemple des devis venant d’un poste portable d’un commercial et des devis faits dans l’entreprise.

Ce paramétrage permet de forcer le numéro qui sera utilisé pour le prochain devis enregistré.

Le nouveau numéro devra être supérieur au dernier numéro utilisé pour le même préfixe.

Si vous modifiez le préfixe, le prochain numéro utilisé sera celui qui existe pour ce préfixe augmenté de un.

Attention, lorsque le numéro du prochain devis à utiliser est modifié, il est impossible de revenir en arrière.

Poids moyen par paquet

Le poids moyen que vous indiquerez pour chaque type de paquetage sera celui pris en compte automatiquement dans le devis, si vous mettez 0 dans le nombre d'exemplaires par paquet (voir le paragraphe "Conditionnement" dans le chapitre "Devis optimisé"). Dans ce cas, le nombre d'exemplaires par paquet sera calculé à partir du poids d'un exemplaire et de ce poids moyen.

Calcul du poids d'un exemplaire

Pour les brochures, les dépliants et les liasses, le poids de l'exemplaire est constitué du total de tous les éléments (si on a un dépliant en 2 modèles, le poids est celui des 2 modèles réunis).

Méthodes de calcul du poids de l'exemplaire :

¨   Brochures

Le poids d'un exemplaire est la somme du poids des éléments :

format fini x grammage x nombre de pages

¨   Dépliants

Le poids d'un exemplaire est la somme du poids des éléments :

format à plat x grammage x nombre de modèles

¨   Affiches/Fiches (pour chaque élément)

Le poids dépend de l'assemblage. Si les fiches sont assemblées, le poids est celui de tous les modèles réunis. Dans le cas contraire, il s'agit du poids d'un seul modèle (s'il y a plusieurs éléments différents, le poids d'un modèle change suivant l'élément; Graphisoft en tiendra compte pour calculer le conditionnement, mais dans ce cas le poids de l'exemplaire ne sera pas affiché).

¨   Liasses/Carnets

Le poids d'un exemplaire est la somme du poids des feuillets :

format fini x grammage

Le poids d'un exemplaire comprend aussi le poids de l'encre.

Tarif

Coût de façon pour un paquet.

Marge en %

S'applique sur le coût de façon calculé à partir du prix indiqué à la ligne "Tarif"

Tarification minimum

Tarif forfaitaire pris en compte lorsque le coût calculé (façon + matière) est inférieure à la tarification minimum.

Lorsqu'il y a plusieurs points de livraisons ou plusieurs versions, la tarification minimum pour un type de conditionnement est appliquée par rapport au total de ce conditionnement pour toutes les livraisons et les versions qui l’utilisent. La différence entre la tarification minimum et le total du conditionnement est reportée dans le coût de façon du conditionnement de la 1ère version de la 1ère livraison pour laquelle ce conditionnement est utilisé.

Exemple de calcul

Devis :
10 000 exemplaires de 109 grammes chacun environ (brochure 21x29,7 fini de 32 pages intérieur sur du 90g/m2 + couverture 4 pages 29,7x42 sur du 150 g/m2).
Conditionnement sous film rétractable puis en palettes
Nombre d'exemplaires par paquet : calcul automatique en fonction du poids moyen

Paramétrage des conditionnements :

¨   Sous film rétractable :
Poids moyen : 2,5 kg
Tarif : 0,40 + 10% de marge
Matière pour 100 paquets : 0,76 + 20% de marge

¨   Palettes :
Poids moyen : 500 kg
Tarif : 7,62 + 10% de marge
Matière pour 100 paquets : 228,67 + 20% de marge

Calcul :

¨   Film rétractable :
Paquets de 2,5 kg maximum. Poids d'1 exemplaire 109g. Nombre d'exemplaires par paquet : 22 arrondi à 20.
Nombre de paquets sous film rétractable : 10 000 / 20 = 500
Coût façon : ( 500 x 0,4 ) + 10% = 220
Coût matière : ( 500 x 0,76 / 100 ) + 20% = 4,56

¨   Palettes :
Palettes de 500 kg maximum. Poids d'1 paquet sous film : 20 x 109g = 2,18 kg. Nombre de paquets par palette : 500 / 2,18 = 229 arrondi à 200.
Nombre de palettes : 500 / 200 = 2,5 soit 3 palettes
Coût façon  : ( 3 x 7,62 ) + 10% = 25,15
Coût matière : ( 3 x 228,67 / 100 ) + 20% = 8,23

Coût du conditionnement = 220 + 4,56 + 25,15 + 8,23 = 257,94
Mille en + :

Pour le calcul du mille en + le coût est calculé par rapport à 2,5 palettes :
Façon palettes :   ( 2,5 x 7,62 ) + 10% = 20,955
Matière palettes : ( 2,5 x 228,67 / 100 ) + 20% = 6,86
Pour les paquets sous film, le calcul est le même.
Mille + total = ( 220 + 4,56 + 20,96 + 6,86 ) / 10 = 25,24

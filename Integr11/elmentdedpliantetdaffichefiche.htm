Format

La 1ère valeur saisie représente la largeur et la 2ème la hauteur du document.

Fond perdu

¨   Pas de fond perdu : les poses seront mises côte à côte sans double coupe mais avec la prise de pince, le blanc tournant (sur 3 côtés), et la gamme Bruner.

¨   Fond perdu : même imposition que "Pas de fond perdu", mais avec en plus la prise en compte de la double coupe entre deux poses.

¨   Blanc tournant : 

·    simple coupe entre les poses,

·    pas de gamme Bruner

·    le blanc tournant sur 4 côtés peut se trouver en dehors du format maxi d’impression de la machine

·    la prise de pince peut être incluse dans le blanc tournant.

¨   Repiquage (ou format direct) : impression par une seule pose sans blanc tournant, sans prise de pince ni gamme Bruner (par exemple  pour les enveloppes).

Remarques :

·    Le format fini ou à plat indiqué est comparé au format papier maximum de la machine d’impression (au lieu du format d’impression maximum dans les autres cas). Vous devez vous assurer que le format d’impression effectif du repiquage est bien compatible avec le format maximum d’impression de la machine utilisée.

·    Si pour le papier sélectionné, il existe un format correspondant exactement au format à plat du document, les autres formats de papier sont ignorés.

·    Les formats de papier qui sont plus grands que le format papier maximum de la machine sont ignorés (aucune refente n’est calculée dans le cas d’un repiquage).

Le blanc tournant, le fond perdu et la gamme Brunner, sont paramétrables dans les données techniques (voir "Données techniques : Calcul des formats" dans le chapitre "Paramétrages de calcul des devis").

Nombre de modèles (amalgames)

Le nombre de modèles correspond au nombre de modèles différents d’un élément.

Graphisoft dispose de 2 modes de saisie pour les modèles en amalgames :

¨   Le mode simplifié, pour lequel tous les modèles ont le même format et la même quantité

¨   Le mode détaillé, pour lequel les modèles peuvent avoir des formats et quantités différentes. Ce mode détaillé correspond à l’option « GS Gang » qui n’est pas disponible en standard. Le fonctionnement de ce module est décrit dans le paragraphe « Amalgames de modèles de formats et quantité différentes » de ce chapitre.

Remarque : cette saisie vous permet aussi de forcer le nombre de poses et le format d'imposition en sélectionnant "Poses" à la place de "Modèles" dans la zone de liste déroulante (voir le paragraphe "Nombre de poses et format d'imposition" plus bas dans ce chapitre).

Lorsque vous saisissez un nombre de modèles supérieur à 1, une case à cocher apparaît  et vous permet de définir l’amalgame :

¨   si les modèles ont tous le même format, vous pouvez laissez cette option décochée, car Graphisoft est prévu pour calculer des amalgames de formes de même format. N’oubliez pas que si vous saisissez un devis de 1000 exemplaires de 2 modèles, le calcul portera sur 2000 modèles.

¨   Si les modèles ont des formats différents, vous devez cocher cette case, ce qui vous donne accès à la saisie du format de l’amalgame. Le format que vous indiquez doit correspondre à l’amalgame de tous les modèles différents. Par exemple, si vous saisissez 1000 exemplaires de 4 modèles avec un format d’amalgame de 60/80, Graphisoft optimisera l’impression de 1000 feuilles imprimées en 60/80.

Modèles des éléments « Dos de couverture » et « Page de garde ».

¨   Page de garde : Par défaut, les éléments de ce type comportent deux modèles (un modèle « Avant » et un modèle « Arrière »), permettant ainsi de ne définir qu’un seul élément lorsque ces deux modèles sont « semblables » (format, quantité, impression, etc) . Dans le cas contraire, il faudra créer deux éléments de ce type mais n’ayant chacun qu’un seul modèle. Un devis pourra donc comporter 0 éléments (de ce type), 1 élément (de ce type) avec les deux modèles Avant/Arrière ou bien 2 éléments (de ce type) mais n’ayant qu’un modèle (différent) chacun.

¨  

¨   Plat de couverture : Par défaut, les éléments de ce type comportent deux modèles (un modèle « Avant » et un modèle « Arrière »), permettant ainsi de ne définir qu’un seul élément lorsque ces deux modèles sont « semblables » (format, quantité, impression, etc) . Dans le cas contraire, il faudra créer deux éléments de ce type mais n’ayant chacun qu’un seul modèle. Un devis pourra donc comporter 0 éléments (de ce type), 1 élément (de ce type) avec les deux modèles Avant/Arrière ou bien 2 éléments (de ce type) mais n’ayant qu’un modèle (différent) chacun. Par ailleurs, si le dos de couverture est fait à partir de la même matière que le plat de couverture, il est possible de le rajouter en tant que modèle à ce dernier (en précisant ses dimensions), un devis ne pouvant avoir simultanément un élément dos de couverture et un modèle « dos de couverture » pour le plat.

¨  

¨   Ces types de modèles ne donnent pas lieu à la création de versions.

Nombre de poses et format d'imposition

La saisie du nombre de modèles permet aussi de forcer le nombre de poses et le format d'imposition.

Lorsque vous sélectionnez "Poses" dans la liste déroulante, vous pouvez alors forcer le format d'imposition, pour tenir compte par exemple d'éventuelles imbrications. Dans ce cas, les solutions étudiées pour l’imposition sont des multiples du nombre de poses forcé.
Lorsque le format d'imposition est forcé :

¨   Les opérations typo seront calculées sur des multiples du nombre de poses forcé. Par exemple, si vous avez forcé 10 poses et que l’impression est calculée en 30 poses, les opérations typo pourront être calculées en 30 poses ou en 10 poses selon les caractéristiques des machines typo disponibles.

¨   Pour les dépliants, le pliage est systématiquement calculé en 1 pose.

¨   Les solutions avec basculage ne sont pas étudiées

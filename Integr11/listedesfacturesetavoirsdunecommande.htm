Liste des factures et avoirs d’une commande

Précédent   Suivant

La facturation d’une commande provoque l’affichage de la liste récapitulative de tous les éléments de factures et d’avoirs déjà préparés pour cette commande.

Si aucune facture n’a encore été préparée pour cette commande, la liste est vide. Cliquez alors sur le bouton <Nouvelle facture> pour facturer la commande.

Si la commande appartient à une liaison, un message propose de charger toutes les commandes de la liaison. Cliquez sur <Non> pour ne charger que la commande sélectionnée.

Si la commande a déjà été facturée, partiellement ou en totalité, cliquez sur les boutons :

¨ <Nouvel avoir> pour préparer un nouvel avoir pour la commande.

¨ <Nouvelle facture> pour préparer une nouvelle facture pour la commande.

¨ <Ouvrir> pour charger la pièce à laquelle appartient l’élément sélectionné dans la liste. Vous pouvez aussi double cliquer sur un élément de la facture à ouvrir.

Remarques :

1. Lorsque vous choisissez « Nouvelle facture » pour une commande qui fait partie d’une liaison de commandes, un dialogue vous permet de charger l’ensemble de la liaison dans la facture.

2. Si vous souhaitez facturer séparément les différentes versions d’une commande, vous devrez saisir vous-même la répartition du prix de vente sur les versions ; Par défaut, toutes les versions sont considérées équivalentes en prix de vente.

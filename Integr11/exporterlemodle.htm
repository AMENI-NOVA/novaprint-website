Exporter le modèle :

Précédent   Suivant

Il est possible d’exporter un modèle ou une édition dans les formats suivants :

¨   Texte ANSI (.txt)

¨   Page Web (.htm, .html)

¨   Rich text format (.rtf)

¨   Texte Unicode (.txt)

¨   Microsoft word (.doc)

¨   Adobe PDF (.pdf)

 

Pour exporter un modèle ou une édition :

¨   Utilisez la commande du menu « Fichier » « Exporter ».

¨   Saisir un nom de fichier et choisir le format ou le type d’exportation parmi les suivants.

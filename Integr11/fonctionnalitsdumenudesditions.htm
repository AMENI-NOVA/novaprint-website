Fonctionnalités du menu des éditions

Précédent   Suivant

Le menu affiche un arbre composé de plusieurs niveaux :

¨   Niveau 1 : les titres qui servent à classer les éditions, précédés d’aucune icône ou de l’icône : .

¨   Niveau 2 : les éditions, précédées de l’icône :

¨   Niveau 3 : les modèles qui représentent l’enregistrement du paramétrage d’une édition, précédées de l’icône :

Toutes les fonctionnalités sont accessibles par la barre d’outils   ou par le menu contextuel obtenu en appuyant sur le bouton droit de la souris.

Commandes

Description

Fermer

Ferme le menu des éditions.

Nouveau modèle

Crée un nouveau modèle pour l’édition pointée.

Dupliquer le modèle

Copie le modèle sous un autre nom.

Supprimer le modèle

Supprime le modèle pointé.

Visualiser l’édition

Edite (concerne soit une édition , soit un modèle).

Propriétés

Permet de modifier un modèle.

 

Les fonctions disponibles changent suivant l’élément sélectionné dans le menu :

Elément sélectionné

Fonctions autorisées

Titre (niveau 1)

Aucune

Edition (niveau 2)

Visualiser , Nouveau modèle

Modèle (niveau 3)

Visualiser , Dupliquer, Supprimer un modèle, Propriétés

 

Actions possibles avec la souris :

Action

Description

Double click gauche sur une édition n’ayant pas de modèle

Ouvre la fenêtre de paramétrage de l’édition.

Double click gauche sur un modèle

Modification du modèle.

Click droit

Affiche le menu contextuel.

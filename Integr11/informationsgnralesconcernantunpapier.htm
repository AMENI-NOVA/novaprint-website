Descriptif

¨   Nom du papier : Il peut être dupliqué à condition que la couleur ou le nom du fournisseur soit différent.

¨   Couleur du papier : Pour un même nom de papier et fournisseur, il ne peut y avoir deux fois la même couleur.

¨   Code papier : Il comprend au maximum 8 caractères et doit être unique. Un même code ne peut être utilisé pour un papier et une catégorie.

¨   Catégorie : par défaut la catégorie à partir de laquelle le papier a été créé, mais il est possible de changer un papier de catégorie. La liste des catégories est remplie avec des catégories de même "type", c'est-à-dire ayant le même paramétrage pour la gestion à l'épaisseur.

¨   Fournisseur : A choisir parmi la liste des fournisseurs ou « non défini ».

¨   Certification : à choisir parmi la liste des certifications paramétrées.

Type de conditionnement

Permet de différencier les papiers en feuilles des bobines. Pour les feuilles vous devrez saisir pour chaque article, le format et le paquetage, tandis que pour les bobines, c'est la laize, la longueur de la bobine et la dimension du mandrin qui seront demandés.

Attention : Cette information ne peut être saisie qu’à la création d’un nouveau papier.

Prix stock
Saisie des prix

Indiquez si les prix sont par grammage ou différents pour chaque format. Dans le cas où les prix sont par grammage, l'unité des prix stock ne peut être que le Kg ou le m2.Une catégorie peut contenir des papiers avec des prix par grammage, et d'autres avec des prix par format.

Attention :  Cette information ne peut être saisie qu’à la création d’un nouveau papier.

Unité de prix

Indiquez l'unité utilisé pour les prix stock. Vous devez rentrer un nombre et une unité parmi les 5 proposées.Par exemple : 100 et Kg pour indiquer que les prix stock pour ce papier seront aux 100 kg.
Respectez les règles suivantes :

¨   L'unité mètre n'est disponible que pour les papiers en bobines.

¨   L'unité feuille n'est pas autorisée pour les papiers en bobines.

¨   Une catégorie peut contenir des papiers avec des unités de prix différentes.

Tranches de prix

Les valeurs saisies serviront de bornes pour les colonnes des tarifs sur stock.

Le nombre de tranches est paramétrable pour l'ensemble du fichier papier (voir "Paramétrages généraux des papiers").
Si vous avez choisi de n'utiliser qu'une seule tranche, le tableau de saisie n'apparaît pas.
La valeur saisie pour la dernière tranche utilisée est automatiquement recopiée pour les tranches non visibles. Ainsi, si vous décidez plus tard de travailler avec plus de tranches, les nouvelles tranches seront déjà initialisées avec des valeurs permettant de calculer vos devis de la même manière qu'avant.

Prix sur fabrication
Unité de prix

Même principe que pour les prix stock, mais ici l'unité est soit le Kg soit le m².

Tranches de prix

Bornes pour les colonnes de tarifs sur fabrication. Le paramétrage du nombre de tranches de prix s'applique aussi aux prix sur fabrication.

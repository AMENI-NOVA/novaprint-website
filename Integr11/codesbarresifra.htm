Codes à barres IFRA

Précédent   Suivant

Le code IFRA est une codification utilisée par les fabricants pour identifier les bobines de papier journal. Le code est formé de 16 chiffres. Le 14ème chiffre code les caractéristiques du papier, les chiffres 13, 15 et 16 définissent le fournisseur, les chiffres 9 à 12 donnent le poids en Kg de la bobine.

Graphisoft peut reconnaître ces codes pour initialiser les informations liées au lot comme le code article, la quantité et le fournisseur.

Pour paramétrer la liaison entre les codes IFRA et les stocks, vous devez d’abord créer :

¨   Les références des stocks de papiers journal dans la branche « Articles » du paramétrage

¨   Les fournisseurs de papier journal dans la branche « Noms – Fournisseurs » du paramétrage.

Ensuite, ouvrez « Paramétrage des codes barre » dans la branche Gestion des Stocks du paramétrage.

Le type de codage s’affiche en haut à gauche. Pour l’instant seul le codage IFRA est géré.

Les différentes informations qui peuvent être reliées sont listées dans la liste de déroulante à droite.

Les valeurs du code et leur correspondance dans les stocks sont listées dans le tableau principal.

Dans la colonne de gauche figurent les valeurs des caractères du code aux positions indiquées dans l’entête de colonne. Entrez ici les valeurs sans séparateur.
Exemple : Si les caractères 13, 15 et 16 du code IFRA sont 1,4 et 2 ou 2, 4 et 2 alors le fournisseur est Chapelle Darblay en France. Entrez 142 dans la colonne de gauche et sélectionnez Chapelle Darblay dans votre liste de fournisseurs. Faites de même dans une nouvelle ligne avec le code 242.

La colonne de droite permet de taper ou sélectionner le stock ou le fournisseur correspondant. Le bouton « ... » ou le bouton pour dérouler la liste donne accès aux éléments déjà paramétrés dans Graphistock.

Pour relier une valeur du code à un groupe de stocks (le chiffre 14 du code IFRA ne donne que le grammage et la qualité du papier), utilisez le caractère « * ».
Exemple : La valeur 1 du code correspond à du 40g/m² journal standard sans préciser la laize. Reliez cette valeur à « JOURSTD...40* » (où les points marquent les espaces) pour englober toutes les laizes en 40g du papier « JOURSTD ».

Remarques :

¨   Il n’est pas nécessaire de définir une relation entre le code et le poids car celui-ci est écrit en clair dans le code et Graphistock le reconnaît automatiquement.

¨   Le lien entre le code IFRA et le stock ne précise pas l’entrepôt même si le dialogue de sélection affiche cette information.

¨   Le code IFRA inscrit sur une bobine par le fabricant est reconnu sans modification par Graphistock. Mais Graphistock lui adjoint le préfixe « @1 » pour l’identifier comme tel dans les affichages internes. Les codes internes, décrits ci-après, utilisent le préfixe « @0 ».

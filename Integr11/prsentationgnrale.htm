La messagerie n’est disponible qu’avec le module GS CRM.

Cas particulier : Il est néanmoins possible d’accéder aux « Eléments envoyés » sans le module GS CRM si vous configurez les comptes de messagerie (Smtp) dans Graphisoft.

 

La messagerie permet d'envoyer des messages à toutes les personnes référencées dans le logiciel : les employés bien sûr, mais aussi les clients (et leurs interlocuteurs), les fournisseurs, les sous-traitants et les possesseurs de la gestion de stock.

 

Les employés peuvent être contactés par des messages "internes" au logiciel, mais aussi par un e-mail. L'e-mail est utile pour les commerciaux itinérants ou pour les employés travaillant sur un site déporté. Les e-mails sont utilisés pour les autres destinataires comme les sociétés et leurs interlocuteurs.

 

La messagerie peut envoyer des e-mails, mais elle peut aussi en recevoir. Pour cela, elle peut consulter votre programme de messagerie Internet déjà installé ou interroger directement votre compte Pop. Ensuite, elle intègre les nouveaux e-mails dans Graphisoft.

Pour plus d'informations, voir le paragraphe "Récupération des e-mails reçus".

 

L'utilisation des e-mails requiert une connexion internet.

Pour plus d'informations, voir le paragraphe "Configuration pour l'envoi et la réception  d'e-mails".

 

Les messages internes sont plus complets que les e-mails. En effet, on dispose d'un suivi complet du message : on sait quand le message à été notifié au destinataire, et quand il l'a réellement lu.

Pour plus d'informations, voir le paragraphe "Suivi des messages".

 

A chaque destinataire "interne" peut être liée une action à faire. L'expéditeur peut savoir, si l'action a été faite et quand celle-ci a été réalisée.

Pour plus d'informations, voir le paragraphe "Actions liées aux messages".

 

Des liens "hypertexte"sont automatiquement créés avec les devis, dossiers et factures. Si dans un message apparaît le texte suivant "le devis n°y" ou "la facture numéro x", vous pouvez cliquer sur ce texte pour aller directement sur l'élément concerné.

Pour plus d'informations, voir le paragraphe "Lire les messages" dans la section "Traiter les messages".

 

Le destinataire est averti immédiatement de l'arrivée d'un message, même s'il ne travaille pas actuellement dans Graphisoft.

Pour plus d'informations, voir le paragraphe "Notification des nouveaux messages : Graphisoft Notify".

 

La messagerie est totalement intégrée au programme Atelier. La communication entre les bureaux et l'atelier est ainsi améliorée.

Pour plus d'informations, voir le paragraphe "Messagerie dans l'atelier".

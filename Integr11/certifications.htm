Certifications

Précédent   Suivant

Le bouton <Certifications> vous permet d’imposer une ou plusieurs certifications pour cet élément de devis. Si vous imposez au moins une certification, seuls les papiers appartenant aux certifications sélectionnées seront pris en compte dans le calcul du devis. Voir le paragraphe « Certifications » dans le chapitre « Paramétrages de calcul de devis ».

Si vous avez sélectionné une famille de certification pour le devis, les certifications de cette famille sont sélectionnées par défaut.

Dans le cas d’un papier hors fichier, cette fenêtre vous permet de préciser la certification du papier. Dans ce cas, vous ne pouvez sélectionner qu’une seule certification.

La certification du papier (choisi dans le fichier ou indiquée pour un papier hors fichier) peut être affichée dans le descriptif du devis.

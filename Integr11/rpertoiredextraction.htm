Répertoire d'extraction

Précédent   Suivant

Toutes les pièces jointes seront extraites dans le répertoire saisie dans "Répertoire où extraire les pièces jointes".

Vous pouvez saisir manuellement ce répertoire ou le sélectionner en cliquant sur "Parcourir".

 

Si durant l'extraction deux pièces jointes ont le même nom, la seconde pièce jointes est renommée en {Nom du fichier}_2.

 

Attention : Assurez-vous que ce répertoire soit accessible à tous les utilisateurs. Après l'extraction, seul le chemin du fichier est enregistré dans la base. Si l'utilisateur n'a pas les droits pour accéder au répertoire d'extraction, il ne pourra pas voir le fichier.

 

A noter : C'est le chemin UNC du répertoire qui est enregistré dans la base de données Ex : "\\MonServeur\MonRepertoire"

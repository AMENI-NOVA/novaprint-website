Informations d’entête spécifiques aux devis de brochures

Format fini

Il est à entrer en cm. L’ordre des valeurs permet de déduire l’orientation de la brochure : la 2ème valeur indique le côté sur lequel est effectué le brochage (29,7 x 21 correspond à une brochure à l’italienne).

Si vous avez paramétré des formats standards, le bouton <...> vous permet de choisir un format dans la liste (voir "Formats standards" dans le chapitre "Paramétrage de calcul des devis").

Le format proposé par défaut pour les nouveaux devis est le format standard défini par défaut, ou 21 x 29,7 si aucun format standard n'a été paramétré.

Orientation

Indique si la brochure est à la française ou à l'italienne. Cette information est déduite de l’ordre des valeurs saisies pour le format fini.

Brochage

Choisissez un type de façonnage parmi ceux que vous avez paramétrés (voir "Chemins et types de façonnage" dans le chapitre "Paramétrages de calcul des devis").

Les types de façonnage proposés sont ceux qui sont autorisés pour les devis de brochure.

Nombre d’exemplaires imposés

Vous pouvez imposer le nombre d’exemplaires pour le brochage.

Cette informations est utilisée dans  les cas suivants :

¨   Chemin de brochage comportant une encarteuse-piqueuse : vous pouvez choisir d’imposer le piquage par 1 ou 2 exemplaires.

¨   Chemin de brochage comportant une assembleuse-cahiers : vous pouvez chosir d’imposer le brochage par 1 exemplaire ou par 2 exemplaires en coming and going.

NB : les solutions en coming and going nécessitent le paramétrage de types de cahier spécifiques.

NB : si vous imposez le brochage par 2 exemplaires alors qu’aucun chemin de brochage n’utilise d’encarteuse-piqueuse par 2 exemplaire ou d’assembleuse cahier en coming and going, le calcul du devis sera impossible.

Cette fenêtre regroupe différents paramétrages concernant la création et l’utilisation des fiches de travail

Pour accéder au paramétrage des fiches de travail, ouvrez les rubriques "Gestion de production", puis "Paramétrages généraux".

Affichage des messages de confirmation dans l'atelier

Message de confirmation pour chaque donnée saisie dans le traitement atelier

Cette option, si elle est cochée, entraîne l’ajout d’un message de confirmation après la saisie de chaque quantité d’opération réalisée, lors du traitement des fiches de travail par l’intermédiaire des terminaux de l’atelier.

Confirmation de la quantité saisie au-delà d’un certain pourcentage par rapport à la quantité prévue

Cette option permet d’introduire un message de confirmation supplémentaire dans le traitement des fiches de travail par les terminaux de l’atelier, après chaque saisie d’une quantité d’opération réalisée, dans le cas où la quantité saisie dépasse d’un certain pourcentage, la quantité initialement prévue.

Exemple :
Le paramétrage indique : "10 est le pourcentage au-delà duquel …".
Un employé indique par l’intermédiaire d’un terminal de l’atelier, la fin d’une opération de tirage pour laquelle il était prévu 20 000 tours. Il saisit 23 000 comme quantité réalisée. Le message "La saisie cumulée dépasse de plus de 10% la quantité prévue. Confirmez-vous la saisie (O/N) ?" lui demande de confirmer la quantité réalisée saisie. Ce message n’apparaîtra dans ce cas, que si la quantité réalisée cumulée pour l’opération est supérieure à 22 000.

Fin d’une fiche de travail

¨   Réponse par défaut du message de confirmation : permet de choisir la réponse par défaut proposée pour le message de confirmation de la fin d’une fiche de travail, dans le traitement des fiches par les terminaux de l’atelier.
Ce message "La fiche est-elle complètement terminée (O/N) ?" apparaît lorsqu’un employé termine l’opération en cours sur une fiche de travail, sans en démarrer une autre sur cette même fiche.

¨   Message de reconfirmation si une des quantités saisies est inférieure à la quantité prévue correspondante :
Dans le traitement des fiches de travail par les terminaux de l’atelier, lorsqu’une fiche est déclarée comme étant terminée, si cette option est cochée, un message de confirmation apparaîtra pour chaque opération de la fiche pour laquelle la quantité réelle est inférieure à la quantité prévue.

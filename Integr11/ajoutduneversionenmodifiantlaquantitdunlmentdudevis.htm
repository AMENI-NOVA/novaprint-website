Ajout d’une version en modifiant la quantité d’un élément du devis

Précédent   Suivant

Lorsque le devis ne comporte qu’une seule version, la quantité à imprimer d’un élément peut être modifiée, ce qui a pour effet de créer automatiquement une nouvelle version.

Le devis passe alors en mode de saisie multi-version en intialisant les 2 versions en fonction du type de devis :

¨   Brochure :

·    La version de base contient tous les éléments du devis avec la quantité minimale commune à tous les éléments

·    La nouvelle version est intialisée en fonction de la quantité modifiée :

§  Si la quantité saisie est inférieure à la quantité nominale, la nouvelle version contient tous les éléments hormis celui dont on a modifié la quantité

§  Si la quantité saisie est supérieure à la quantité nominale, la nouvelle version contient uniquement l’ élément dont on a modifié la quantité

¨   Dépliant ou Fiche : une version est créée pour chaque élément séparément. Le nom de la version est initialisé avec la référence de l’élément si elle a été saisie.

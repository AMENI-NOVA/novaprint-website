Plaques

Chaque type d’impression utilisé dans les chargeurs a une entrée ici, sauf numérique évidemment, pour que vous puissiez définir le prix de l’heure et les coûts de copie, d’ozalid, de plaque, écran, manchon ou cylindre et les marges spécifique par type d’impression. Les types inutilisés sont affichés sur fond grisés.

Mise en route

Ce temps est compté une fois.

Temps de base passage papier

C'est le temps mis pour passer la première nappe à chaque passage en rotative.

Temps par nappe supplémentaire

Pris pour chaque nappe à partir de la deuxième à chaque passage en rotative.

Changement de bobine

Ce temps est compté à chaque changement de bobine en entrée de machine.

Calage retiration

Ce temps est compté une fois sur la machine si on imprime au recto et au verso au même passage.

Calage sortie coupe

Ce temps est compté une fois sur la machine si la recette s’effectue en feuilles.

Calage délamination

Ce temps est compté une fois sur la machine si une impression sur la masse adhésive a lieu.

Recherche de teinte, calage d’un groupe, lavage d’un groupe

Ces temps doivent être définis par type d’impression. Pour le numérique, seul le « Calage tons directs » est à sasir.

Vitesse

¨   Vitesse max. si délamination (m/mn) : C'est la vitesse maximum en  m/mn de défilement du papier si on doit imprimer sur la masse adhésive (étiquettes). Si elle est nulle, la délamination n’est pas possible sur la machine.

¨   Vitesse de roulage (m/mn) : C'est la vitesse de défilement du papier en machine sans tenir compte du type d’impression. Si elle est nulle, c’est la vitesse de frappe qui définira la vitesse de roulage et si la machine produit des paravents, la vitesse en paravents peut définir également la vitesse de travail. Sinon la machine ne pourra pas être retenue dans le calcul.

¨   Vitesse pour paravents de 12 pouces (plis/mn) : C'est la vitesse de base du poste de pliage accordéon. Si les paravents sont plus petits, cette vitesse est diminuée linéairement avec la hauteur au pli. Cette vitesse sert à limiter la vitesse de tirage dans le cas de recette en paravents.

Vitesses liées aux types d’impression

¨   Vitesse de roulage/impression (m/mn) : C'est la vitesse maximum de roulage si le type d’impression est utilisé. Si elle est nulle, c’est la vitesse de frappe qui définira la vitesse de roulage et si la machine produit des paravents, la vitesse en paravents pourra limiter également la vitesse de travail. Si la vitesse résultante est nulle, la machine ne pourra pas être retenue dans le calcul.

¨   Vitesse de frappe (frappes/mn) : Permet de définir une vitesse liée à la cadence de frappe. Si elle est nulle, cette vitesse est ignorée. Une frappe correspond à l’impression d’un développement.

Autres coûts

¨   Coût aux 1000 clics par couleur : Attention, la saisie s’effectue pour 1000 clics et n’est possible que pour le type numérique.

Minorations de la vitesse

¨   Minoration de la vitesse au delà de 12 pouces (plis/mn) : Si les paravents sont plus grands que 12 pouces, on diminue la vitesse en plis de cette quantité pour chaque pouce au dessus de 12.

¨   Minoration par groupe utilisé (%) : Elle est appliquée sur la vitesse de roulage en % par groupe utilisé. Cette minoration est cumulée avec la "Minoration si Crimplock utilisé (%)" dans le cas où le Crimplock est réalisé sur la rotative.

¨   Minoration par opération à façonnage multiple (%) : chaque opération de façonnage spécial diminue la vitesse de rembobinage de ce taux.

¨   Minoration si sortie en coupe (%) : Elle est appliquée au dernier passage en machine sur la vitesse de rembobinage dans le cas de coupe en sortie de rotative.

Opérations particulières

¨   Temps de calage : compté si l'opération est réalisée sur la machine.

¨   Vitesse maxi de tirage (m/mn) : limite la vitesse de tirage lors du dernier passage si l'opération est réalisée sur la machine. Si cette valeur est nulle, l'opération ne limite pas la vitesse de tirage.

Gâches & Encre

¨   Gâche de base (nombre de paravents) : c'est le nombre de feuilles (si sortie coupe), paravents (si sortie plis) ou développements gâchés pour la préparation de la machine.

Par type d’impression

¨   Gâche de recherche de teinte/Calage tons direct : c'est le nombre de feuilles (si sortie coupe), paravents (si sortie plis) ou développements gâchés pour chaque couleur non primaire imprimée sur la machine.

¨   Gâche de calage par plaque calée : c'est le nombre de c'est le nombre de feuilles (si sortie coupe), paravents (si sortie plis) ou développements gâchés pour caler les groupes d'impressions.
La gâche de calage par plaque est toujours calculée sur les derniers feuillets (pour les couleurs d'impression identiques avec les feuillets précédents), car en général, le papier utilisé pour le dernier feuillet est moins cher que les autres.

Par exemple :

  Feuillet 1 : Noir (1) + Bleu (2) : gâche de calage comptée pour la plaque de Bleu uniquement

  Feuillet 2 : Noir (1) + Rouge (3) : aucune gâche de calage comptée, car elle est comptée sur le feuillet 3

  Feuillet 3 : Noir (1) + Rouge (3) + Vert (4) : gâche de calage comptée pour les 3 couleurs

Importation

Précédent   Suivant

Quand vous cliquez sur <Importer>, la fenêtre d’importation apparaît.

 

Avant la création d’une catégorie, une fenêtre vous permet d’identifier la catégorie importée avec une catégorie existant déjà dans votre fichier papier.

Mettre à jour la sélection : la catégorie ne sera pas importée et les papiers de cette catégorie seront éventuellement importés dans votre catégorie.

Créer : La catégorie sera ajoutée aux catégories dans votre fichier.

Toujours créer la catégorie : Toutes les nouvelles catégories seront ajoutées aux catégories de votre fichier sans que cette fenêtre apparaisse.

 

Avant la création d’un papier dont le nom existe déjà dans votre fichier mais pour un autre nom de fournisseur, une fenêtre apparaît vous permettant d’identifier le papier importé avec un papier existant déjà dans votre fichier papier mais avec un nom de fournisseur différent. (ici "RUYSSHER" au lieu de "DE RUYSSHER")

Mettre à jour la sélection : Le papier ne sera pas créé, mais le nom du fournisseur sera modifié. Les articles de ce papier seront éventuellement ajoutés et les tarifs des articles existants mis à jour.

Créer : Le papier sera créé avec ses articles.

Toujours créer le papier : Les nouveaux papiers seront importés sans que cette fenêtre apparaisse.

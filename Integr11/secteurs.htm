Créer des secteurs

Pour créer des secteurs, sélectionnez le pays concerné dans la liste, puis cliquez sur Nouveau dans le menu Paramétrages ou le menu contextuel, ou utilisez le bouton correspondant de la barre d’outils supérieure.

Modifier un secteur

Pour modifier le nom d’un secteur, pointez-le dans la liste, puis cliquez sur Renommer dans le menu Paramétrages ou le menu contextuel, ou utilisez le bouton correspondant de la barre d’outils supérieure.

Fusionner deux secteurs

Cette fonction permet de réorganiser les secteurs ou d’éliminer les doublons.

Ø Pour fusionner deux secteurs :

1. Sélectionner dans la liste, le secteur (source) qui doit être fusionné dans un secteur existant (destination).

2. Cliquez sur Fusionner dans le menu Paramétrages ou le menu contextuel, ou utilisez le bouton correspondant de la barre d'’utils supérieure.

3. Dans la fenêtre qui apparaît, sélectionnez le secteur (destination) dans lequel doit être fusionné le secteur (source) précédemment choisi.

La fusion effectue les opérations suivantes :

¨   Les employés ayant des droits ou étant affectés au secteur ‘source’ sont affectés au secteur ‘destination’.

¨   Les sociétés appartenant au secteur ‘source’ sont transférées dans le secteur ‘destination’.

¨   Le secteur ‘source’ est supprimé.

Afficher les fiches

Précédent   Suivant

La fonction Afficher les fiches… du menu Traitement permet de faire apparaître la liste des travaux pour le dossier et le poste (ou le regroupement) sur lequel se trouve le curseur.

Si toutefois le curseur se trouve sur la ligne des activités secondaires, cette fonction affichera toutes les fiches d'activités secondaires non terminées.

La colonne "En cours" indique si la fiche de travail est actuellement en cours de traitement dans l’atelier.

Cette fonction est accessible directement par la touche F6.

Liste des mouvements

Précédent   Suivant

Cette liste permet d’avoir une vue d’ensemble des mouvements de stock validés répondant à certains critères tout en pouvant choisir quelles informations vous voulez voir affichées.

Chaque type de pièce est identifié par un code couleur commun dans l’application : le même code est utilisée à dans la saisie des pièces, dans les éditions d’historiques de la Gestion de Stocks, dans l’historique des mouvements de la liste des stocks et dans cette liste.

Cette liste est aussi accessible depuis la liste des commandes (cliquez sur Mouvements de stocks dans le menu Commande de cette liste ou par le menu contextuel) : dans ce cas, elle est chargée avec la macro courante puis un filtre express est appliqué pour restreindre sur les mouvements du dossier sélectionné.

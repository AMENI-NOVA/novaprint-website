Créer une commande à partir d'un modèle de devis

Précédent   Suivant

Si le modèle de devis a été calculé avant d'être enregistré, vous pouvez le transformer en commande. Les modèles de devis calculés sont affichés dans la liste avec l’information « En Attente » non cochée.

Sélectionnez-le dans la liste, et choisissez l'option Commande du menu. Sélectionnez ensuite le type d'opération de commande à effectuer : Nouvelle, Remplacement ou Prévisionnelle.

Lorsque vous créez une commande à partir d'un modèle de devis, un nouveau devis est créé à partir du modèle et c'est celui-ci qui est transformé en commande.

Si le modèle a un descriptif modifié ou des commentaires, le devis créé automatiquement reprend les descriptifs et commentaires.

Avant de démarrer le processus de création de la commande, vous avez la possibilité de modifier le devis qui sera créé à partir du modèle que vous avez sélectionné. C'est la fonction de la fenêtre qui apparaît.

Informations affichées
Modèle de devis

Indique la référence du modèle de devis qui a été sélectionné.

Le bouton <Ouvrir…> permet d'ouvrir le modèle de devis dans la saisie du devis optimisé.

Devis

Affiche les informations du modèle de devis sélectionné ou du dernier devis enregistré si le modèle de devis a été ouvert dans la saisie de devis optimisé.

Vous pouvez modifier certaines informations directement dans cette fenêtre sans passer par le devis optimisé :

¨   Client

¨   Commercial

¨   Référence

La quantité n'est pas modifiable dans cette fenêtre car elle implique un recalcul du devis. Si vous souhaitez la modifier pour le devis, ouvrez le modèle de devis dans la saisie de devis optimisé. La quantité du devis proposée par défaut pour la nouvelle commande peut de toute façon être modifiée dans la fenêtre de création de celle-ci.

La zone "Descriptif" vous permet de consulter le descriptif du modèle de devis ou du dernier devis enregistré. Le descriptif est affiché en fonction du modèle de descriptif par défaut s'il y en a un, ou sinon à partir du premier modèle trouvé par ordre alphabétique. Le descriptif ne peut pas être modifié.

Modifier et recalculer le devis

Le bouton <Ouvrir…> vous permet d'ouvrir le modèle de devis dans la fenêtre de saisie de devis optimisé.

Vous pouvez alors modifier le devis, le calculer, l'enregistrer et consulter son dossier technique.

Vous pouvez enregistrer plusieurs devis à la suite. C'est le dernier devis enregistré qui sera transformé en commande en cliquant sur le bouton <Créer la commande…>.

Si vous modifiez ensuite le client, le commercial ou la référence, ces modifications seront appliquées au dernier devis enregistré au moment de la création de la commande.

Créer la commande

Le bouton <Créer la commande…> ouvre la fenêtre de création de commande.

¨   Si vous n'avez pas enregistré de devis, un nouveau devis est créé automatiquement à partir du modèle de devis, en reprenant les informations éventuellement modifiées : client, commercial, référence. C'est ce devis qui est transformé en commande.
Si vous annulez la création de la commande, le devis créé automatiquement est supprimé.

¨   Si vous avez enregistré un devis en ouvrant le modèle dans la saisie de devis optimisé, la zone "Numéro" indique le numéro du devis qui sera transformé en commande. C'est le dernier devis qui a été enregistré depuis le devis optimisé.

Le processus de création de commande est ensuite le même que lorsque vous transformez un devis en commande (voir "Transformation d'un devis en commande" dans le chapitre "Traitement des devis et utilitaires").

Types d’articles

Précédent   Suivant

Les types d’articles servent à classer les familles d’articles. Vous devez créer des types d’articles, avant de créer des familles et des articles.

Création d’un type d’articles

Dans la liste arborescente, sélectionnez la rubrique « Articles » dans « Gestion des stocks », puis cliquez sur Nouveau dans le menu Paramétrages ou dans le menu contextuel.

Les informations à saisir pour un type d’article sont :

¨   Le nom du type.

¨   Un code d’un caractère qui doit être unique.

¨   Et si vous diposez du module pont vers la comptabilté :

¨   Numéro de compte de stock par défaut pour les articles de ce type

¨   Numéro de compte de variation de stock par défaut pour les articles de ce type

¨   Articles produit : permet d’indiquer que les articles des familles de ce type correspondent à des produits finis qui pourront être liés aux devis et commande.

Modification d’un type d’articles

La fonction Propriétés vous permet de modifier le nom d’un type d’articles et les numéros de compte.

Suppression d’un type d’articles

Vous pouvez supprimer un type d’articles, dès l’instant qu’il ne contient plus aucune famille. S’il en contient, un message vous proposera de supprimer les familles avec tous leurs articles et stocks avant de supprimer le type d’article lui-même.

Si un des stocks de la famille n’est pas soldé, un message le signale :

Vous pouvez alors cliquer « Oui » et le stock est supprimé quand même. (Voir « Suppression des stocks » dans "Création des articles" ci-dessus).

Si répondez non, le stock est conservé, l’article et la famille également et le type d’article ne sera pas supprimé.

Pour faire la même réponse pour tous les autres stocks non soldés, cochez la case « Faire de même pour tous les stocks suivants » avant de cliquer « Oui » ou « Non ».

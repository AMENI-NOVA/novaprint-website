Filtrer les dossiers

Précédent   Suivant

Les fonctionnalités générales de la liste liées à l'affichage des informations (colonnes (hors postes), filtres et tri), sont communes à toutes les autres listes (sociétés, devis, commandes, factures,...) et sont décrites dans le chapitre "Introduction".

 

Menu : "Affichage", "Options", puis onglet « Filtres »

 

Vous pouvez restreindre les dossiers affichés dans le tableau, suivant un ou plusieurs critères.

Une fois défini, le filtre saisi peut être enregistré dans une macro. Vous pouvez créer autant de macros que vous le souhaitez. Une macro peut être activée depuis la fenêtre de saisie des critères de filtre ou directement depuis le tableau de charge, par l’intermédiaire d’une zone blanche se trouvant dans la partie inférieure gauche de la fenêtre. Cliquez dans cette zone pour dérouler les macros.

Ici aussi, les macros peuvent être organisées dans des groupes.

 

Les filtres actifs sont indiqués dans la partie centrale de la barre d’état de la fenêtre.

 

Rappel : Le droit "Voir toutes les commandes" permet de voir tous les dossiers. Si vous n'avez pas ce droit, seuls les dossiers des clients dont vous êtes le commercial sont visibles.

Filtres express

Le cadre "Filtres express" situé en bas de la fenêtre, vous permet de filtrer plus rapidement les dossiers par rapport à une société et/ou un commercial.

¨   Filtre sur la société : Il faut saisir soit le nom complet de la société souhaité, soit seulement les premières lettres suivi du caractère % (par exemple GR%). Dans le second cas, plusieurs sociétés peuvent correspondre au critère saisi.
Le raccourci clavier ALT + S permet d’amener immédiatement le curseur dans la zone de saisie.

¨   Filtre sur le commercial : Le choix du commercial s'effectue par l'intermédiaire d'une liste déroulante.
Le raccourci clavier ALT + C permet de dérouler immédiatement la liste.

Lorsque vous avez saisi le filtre (société ou commercial), cliquez sur le bouton "Appliquer" ou validez par ENTREE pour que le tableau soit mis à jour.

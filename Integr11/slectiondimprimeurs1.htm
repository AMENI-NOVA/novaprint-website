Sélection d'imprimeurs

Précédent   Suivant

Vous pouvez sélectionner les imprimeurs pour lesquels vous souhaitez calculer le devis.

Pour sélectionner des imprimeurs, cliquez sur "Imprimeurs sélectionnés" dans la liste arborescente.

Cette sélection peut s'effectuer suivant la catégorie, le pays ou le secteur. Il est possible de combiner différents critères de sélection.

Par exemple, si vous choisissez comme critères : France, Angleterre, Imprimeurs luxe (catégorie), l'optimisation se fera sur les imprimeurs fançais ou anglais, appartenant à la catégorie "Imprimeurs luxe".

Vous pouvez aussi sélectionner des imprimeurs qui viendront s'ajouter à ceux de la sélection effectuée par la catégorie, le pays ou le secteur.

Lors de la sélection, si un conditionnement précédement utilisé n’est plus disponible pour les imprimeurs sélectionnés, un message vous en informera.

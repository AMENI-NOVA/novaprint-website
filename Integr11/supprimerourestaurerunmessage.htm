Supprimer ou restaurer un message

Précédent   Suivant

Après avoir sélectionné un message, appuyez sur [Suppr] ou utilisez la commande "Supprimer" du menu "Edition".

Attention : un message supprimé votre boîte de réception, n'est pas supprimé de la boîte d'envoi de l'expéditeur (et vice versa).

Les messages supprimés sont stockés dans la corbeille. Cela permet en cas d'erreur de les restaurer. Pour cela, il faut ouvrir la corbeille, sélectionner le message et cliquer sur "Restaurer" dans le menu "Edition" ou dans le menu contextuel. Le message reprend sa place dans la boîte d'envoi ou de réception.

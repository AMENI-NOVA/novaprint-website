Création de vos propres rapports Crystal

Précédent   Suivant

Vous pouvez créer vos propres rapports, ils apparaîtront dans le logiciel. Pour cela, il faut renseigner la propriété « Commentaires du rapport », c’est cette propriété qui est affichée dans la liste des rapports.

Liaison d’une édition à la liste des commandes :

Pour créer un rapport lié à une commande, il faut que dans la propriété «Mots Clés »  de l’état, vous indiquiez le mot « commande ». et également vous ajoutiez un champ de formule nommé « Paramètre » qui sera rempli automatiquement par l’identifiant  du dossier.

Attention : ces éditions doivent impérativement se trouver dans le répertoire de l’application.

Autres formules ou paramètres :

Pour afficher le nom de la base de données courante, ajouter une formule nommée « PARAM_NOMBDD » que vous poserez ensuite sur l’état.

Formules ou paramètres liés à la notion de société utilisatrice :

Pour limiter les données à celle-ci, ajoutez une formule nommée « PARAM_IDSOCUTIL » (formule que vous devrez ensuite utiliser dans la requête SQL pour restreindre les données).

Pour afficher la raison sociale de cette dernière, ajoutez une formule nommée « PARAM_RSSOCUTIL » que vous poserez ensuite sur l’état.

Pour afficher (comme critères de sélection par exemple) les employés de celle-ci, ajoutez un paramètre nommé « Employes » de type chaîne, statique, avec description uniquement, sans valeurs personnalisées ni de plage, autorisant plusieurs valeurs et les valeurs discrètes. Les employés y seront indiqués sous la forme « nom »espace »prénom ». Valeurs qui devront ensuite être utilisés comme sélection.

Vous pouvez prendre exemple sur l’édition « Présences des employés » (fichier EmployesPresences.rpt) pour l’utilisation de ce paramètre.

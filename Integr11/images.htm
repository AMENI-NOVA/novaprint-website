Images

Précédent   Suivant

Vous pouvez insérer des images dans un bloc comme vous le feriez pour un champ :

1. Placez le curseur à l'endroit où vous voulez insérer l'image

2. Choisissez l'option de menu "Insérer - Image"

3. Dans le dialogue d'ouverture de fichier, sélectionnez le fichier contenant l'image et cliquez <Ok>

 

Les formats d'image reconnus sont :

¨   BMP

¨   TIF

¨   WMF

¨   GIF

¨   JPG

 

La taille de l'image est adaptée au format du bloc dans laquelle elle est créée, mais elle ne sera pas redimensionnée si vous redimensionnez le bloc la contenant.

Si vous travaillez en réseau, placez le fichier contenant l'image dans un répertoire accessible à tous avec le même chemin. Par exemple, placez-le dans le répertoire de travail de Graphiplus et ouvrez ce fichier en passant par voisinage réseau (ou favori réseau) puis en ouvrant successivement le serveur puis tous les répertoires conduisant à ce fichier. Le chemin ainsi obtenu est absolu et sera reconnu sur tous les ordinateurs du réseau.

Si un ordinateur ouvre un document ou un modèle comportant une image, alors qu'il n'a pas accès au fichier la contenant, l’image sera cherchée automatiquement dans le sous-répertoire « Images » du répertoire de travail. Si elle n’y est pas non plus, seul le chemin du fichier image sera affiché.

Cette recherche permet aux postes nomades (voir le chapitre sur la synchronisation) d’utiliser les modèles habituels sans manipulation, car le sous dossier « Images » est transféré automatiquement par la synchronisation.

Vous pouvez aussi modifier la source de l’image en double-cliquant l’image du modèle ou de l’édition en cours. Un dialogue apparaît alors pour sélectionner le fichier source à utiliser.

L'image se comporte dans le document comme un simple caractère. On peut donc la déplacer en tapant des espaces devant, en l'alignant sur une tabulation ou la centrer en utilisant le bouton de centrage de paragraphe etc.

 

Lors de l'envoi par e-mail, l'image est enregistrée intégralement dans le fichier joint au mail.

Vos clients n'ont donc pas besoin du fichier image pour visionner votre logo.

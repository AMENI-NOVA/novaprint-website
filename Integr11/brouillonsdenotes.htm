Brouillons de notes

Précédent   Suivant

Un brouillon est une note en cours de rédaction que vous souhaitez enregistrer afin d'y revenir ultérieurement.

 

Tant qu'une note est considérée comme un brouillon, elle n'est pas envoyée aux destinataires et elle n'est pas visible dans les différents historiques de GS CRM .

 

Pour enregistrer un brouillon de votre note, utilisez le menu "Fichier", puis "Enregistrer le brouillon".

 

Tous les brouillons sont enregistrés dans la boite "Brouillons" de la messagerie.

 

Pour modifier un brouillon, il faut le sélectionner dans la messagerie et demander "Modifier" (ou double-cliquer dessus). La note s'ouvre de nouveau dans l'état où vous l'avez enregistrée. Vous pouvez maintenant l'envoyer, la modifier comme une note normale.

Principes de saisie

La saisie s'effectue en mode graphique à l'aide de la souris.

La saisie des propriétés d'un amalgame consiste à disposer les différents cahiers les uns par rapport aux autres.

Lorsque l'amalgame est enregistré, c'est la position des cahiers les uns par rapport aux autres qui est enregistrée.

Les amalgames sont enregistrés indépendamment du format et du type de brochage, qui peuvent toutefois être saisis pour visualiser la représentation de l'amalgame dans une configuration donnée (voir "Options d'affichage").

 

Aspect de la fenêtre de propriétés

La fenêtre se compose des éléments suivants :

¨   Un menu et la barre d'outils correspondante dans la partie supérieure, permettant d'accéder aux différentes fonctionnalités

¨   La liste des types de cahiers, dans la partie gauche, sous l'onglet "Types de cahier", permettant d'ajouter des types de cahier à l'amalgame

¨   Les options d'affichage, dans la partie gauche, sous l'onglet "Options d'affichage", permettant de modifier la représentation de l'amalgame en fonction du chemin de brochage, du format fini et de l'orientation

¨   Le nom de l'amalgame dans la partie supérieure droite

¨   L’orientation autorisée pour l’amalgame : à la française, à l’italienne ou les deux.

¨   La représentation de l'amalgame dans la partie droite.

·    Chaque type de cahier composant l'amalgame est matérialisé par un rectangle avec une bordure noir.

·    Un seul cahier de 4 pages est affiché avec la rogne, le grecquage et le dépassant pour chaque type de cahier. Les autres 4 pages sont séparés par des traits vert pleins (coupe) et le trait pointillé matérialise l'emplacement du brochage.

·    Si le type de cahier n'est pas composé de cahiers de 4 pages, les pages le composant sont matérialisées avec un trait rouge les séparant.

¨   La barre d'état dans la partie inférieure, indiquant :

·    Le format de l'amalgame en fonction des options d'affichage courantes

·    Le nombre de pages de l'amalgame

·    Le nombre de pages et le format du type de cahier sélectionné, en fonction des options d'affichage courantes

 

Saisie des types de cahiers constituant un amalgame
Pour ajouter un type de cahier à l’amalgame :

1. Sélectionnez le type de cahier à ajouter dans l’onglet "Type de cahier".

2. Cliquez sur le bouton  "Ajouter le type de cahier".

3. Le type de cahier est ajouté en bas à droite de l’amalgame. Il suffit de le déplacer pour le mettre à sa place.

On peut aussi double-cliquer sur le type de cahier dans l’onglet pour l’ajouter.

Remarque : Un amalgame ne peut pas contenir plus de 20 cahiers.

Pour déplacer un type de cahier dans l’amalgame :

1. Dans la représentation de l'amalgame, sélectionnez le type de cahier à déplacer en cliquant dessus avec la souris. Il est alors entouré par des carrés bleus.

2. Tout en maintenant le bouton gauche de la souris enfoncé, déplacez le curseur pour amener le cahier à sa nouvelle position.
Pendant son déplacement le cahier est matérialisé par un rectangle gris épais avec une flèche en haut à gauche qui suit les mouvements de la souris.

3. L’endroit exact ou il se positionnera est aussi matérialisé par un rectangle épais (rose). Ce rectangle se déplace en se collant au bord des autres types de cahier (effet d’aimant).
Relâchez le bouton de la souris lorsque le rectangle rose correspond à l'endroit où vous voulez déplacer le cahier.

On peut annuler le déplacement du cahier en cours en appuyant sur la touche « Echap » ou en le lâchant sur le fond de l’amalgame (pas sur un autre type de cahier).

Conséquences du déplacement des cahiers

Si au moment où vous relâchez le bouton de la souris :

¨   la flèche du type de cahier déplacé se trouve sur un autre type de cahier, il prendra sa place et le poussera sur la droite.

¨   la flèche du type de cahier déplacé se trouve sur le bord droit d’un autre type de cahier, le type de cahier déplacé se mettra sur sa droite et poussera sur la droite ceux qui y étaient auparavant.

¨   la flèche du type de cahier déplacé se trouve sur le bord inférieur d’un autre type de cahier, le type de cahier déplacé se mettra sur son bord inférieur et poussera sur la droite ceux qui étaient auparavant sur son bord inférieur.

Chevauchement des cahiers

Le fond de l’amalgame apparaît en rose au lieu de gris si des cahiers se chevauchent dans l’amalgame.

Cela ne veut pas systématiquement dire que l’amalgame est incorrect. Il est incorrect pour le format de page et le chemin de brochage choisi. Si c’est ceux-ci qui sont choisis pour calculer le devis, cet amalgame ne sera évidemment pas retenu… Pour d’autres formats de page et d’autres chemins de brochage, l’amalgame peut très bien être correct et utilisable pour le calcul du devis.

 

Options d'affichage

Les options d'affichage permettent de visualiser l'amalgame et ses dimensions exactes, et de vérifier sa cohérence (chevauchement des cahiers) par rapport à un format donné ou à un chemin de brochage.

Chemin de brochage

Lorsque vous choisissez un chemin de brochage, l'amalgame est représenté en tenant compte des différentes marges paramétrées pour ce chemin : rogne, grecquage, dépassant.

Dans l’exemple ci-dessus, le format de papier est exagérément petit pour mettre en évidence la rogne et  le grecquage.

 

Liste de référence des fonctionnalités
Menu "Amalgame"

¨   Précédent   : charger l’amalgame précédent.

¨   Suivant      : charger l’amalgame suivant.

¨   Nouveau      : créer un nouvel amalgame de cahier.

¨   Enregistrer   : enregistrer les modifications

¨   Imprimer      : imprimer les caractéristiques et le dessin de l’amalgame

¨   Supprimer    : supprimer l'amalgame ou annuler sa création.

¨   Fermer       : fermer la fenêtre.

Menu "Type de cahier"

¨   Ajouter       : ajouter le type de cahier sélectionné dans l’onglet « type de cahier » à l’amalgame

¨   Supprimer   : supprimer le type de cahier sélectionné (celui entouré de petits carrés bleus)

¨   Pivoter       : pivoter / basculer le type de cahier sélectionné de 90°

Menu "Options"

¨   Zoom Auto  : afficher l’amalgame de cahier en entier

¨   Zoom Plus   : agrandir et afficher plus de détail

¨   Zoom Moins : réduire et afficher moins de détail.

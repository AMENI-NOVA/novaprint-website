Présentation de la messagerie

Précédent   Suivant

Le module d'atelier intègre les principales fonctionnalités de la messagerie Graphisoft. Pour avoir de plus amples détails et connaître toutes les fonctionnalités de la messagerie, consultez le chapitre "Messagerie".

 

Il est possible d'accéder à la messagerie en cliquant sur le bouton "Messages" ou en appuyant sur [Ctrl]+[F10].

 

Les options d'envoi sont limitées pour un terminal atelier :

¨   Les destinataires sont restreints aux employés contactés par un message interne.

¨   Impossible de lier une action à un message.

¨   Impossible de demander une réponse.

 

Un employé qui reçoit un message est informé par un bandeau en bas de l'écran qui lui indique le nombre de messages et d'actions liées aux messages. Ce bandeau s'efface dès que les messages sont lus et les actions effectuées.

 

La liste des messages de l'employé est en haut de l'écran. Le contenu du message sélectionné est affiché en dessous.

Les messages non lu sont affichés en gras. De plus, une image indique l'état du message :

 : Nouveau message non lu.

 : Message lu.

 : Message non lu avec une action à effectuer.

 : Message lu avec une action à effectuer.

 : Message lu et action effectuée.

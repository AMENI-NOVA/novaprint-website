Modèle par défaut

Précédent   Suivant

Le bouton "Défaut" de la liste de choix d’un modèle, permet de spécifier le modèle à utiliser par défaut pour un type d’édition. Ceci peut être utile si vous avez accès à plusieurs modèles d’édition depuis votre poste de travail, mais que vous utilisez toujours le même. Dans ce cas, sélectionnez ce modèle dans la liste et cliquez sur le bouton "Défaut".  Lorsque vous éditerez un autre document de ce type, le modèle par défaut sera utilisé systématiquement, sans avoir à le sélectionner dans la liste.

Si vous voulez faire une édition pour un élément (un devis par exemple) pour lequel un document a déjà été enregistré, la liste des modèles est quand même affichée, même si vous avez défini un modèle d’édition par défaut. Cela vous permet de voir les documents déjà enregistrés pour cet élément et éventuellement de les rappeler.

Si vous souhaitez ponctuellement  ne pas utiliser le modèle par défaut, utilisez l’option Changer de modèle… du menu Document, puis cliquez sur « Ok » pour reconstruire l’édition à partir du modèle sélectionné.

Pour désactiver le modèle par défaut, et avoir à nouveau la possibilité de choisir le modèle à utiliser dans la liste avant que l’édition ne soit chargée, utilisez l’option Changer de modèle… du menu Document pour faire afficher la liste des modèles, sélectionnez le modèle par défaut, et cliquez sur le bouton « Défaut ».

Détails tiers

Précédent   Suivant

 

Les fournisseurs, sous-traitants et possesseurs de stocks sont gérés dans les sociétés sous la dénomination Tiers. Les autres tiers, ateliers, bureau et divers utilisés par la gestion des stocks sont gérés dans le paramétrage de la gestion des stocks car il ne s’agit pas de sociétés.

Les fournisseurs et les sous-traitants sont utilisés dans les cas suivants :

¨   chaque fiche matière ou de sous-traitance pourra être liée à un fournisseur ou un sous-traitant

¨   gestion des bons de commande par fournisseur/sous-traitant

¨   interlocuteurs pour les mouvements de la gestion des stocks

¨   transporteurs pour les livraisons

La fonction d'initialisation des postes et opérations par le devis, crée en même temps les fournisseurs de papier, et les sous-traitants enregistrés pour les opérations de prépresse, à partir des fichiers de Graphiplus.

Type de tiers :

Les tiers gérés ici sont : fournisseur, sous-traitant et possesseurs de stock

Sous-type :

Les fournisseurs sont classés par type de matière :

¨   Papiers

¨   Plaques

¨   Encres

¨   Films

¨   Autres

Les Sous-traitants par type de prestation :

¨   Prépresse

¨   Impression

¨   Opérations de surface

¨   Façonnage

¨   Transport : utilisé dans les livraisons pour ne proposer que les transporteurs

¨   Autre

Code

Chaque tiers est identifié par un code de 4 caractères. La saisie de ce code est obligatoire pour pouvoir créer un nouveau tiers. Une fois créé, ce code ne peut plus être modifié. Il est utilisé par Graphistock et dans l’export des factures d’achat en comptabilité.

Numéro comptable

Entrez ici le numéro comptable du tiers en comptabilité. Cette information est utilisée dans l’export des factures d’achat en comptabilité.

Création des commandes proposée par défaut à la création des nouveaux dossiers (informations spécifiques pour les fournisseurs)

Lors de la transformation d’un devis en commande, la colonne "Bon de commande" sera cochée par défaut pour les fiches matières concernant cet interlocuteur.

En gestion des stocks, une commande fournisseur sera créée automatiquement pour chaque commande concernant les fournisseurs pour lesquels vous avez coché cette option. La commande fournisseur est créée au moment de la validation du tampon.

Création automatique des bons de commande pour les nouveaux dossiers (informations spécifiques pour les fournisseurs et sous-traitants)

Cette option permet de lier automatiquement chaque nouvelle commande concernant le fournisseur ou le sous-traitant, à un bon de commande.

Si l’option "Regroupement de commandes de dossiers différents" est cochée, la nouvelle commande sera rattachée à un bon de commande existant non édité et non validé, concernant le même fournisseur ou sous-traitant. Ce bon de commande pourra contenir des commandes concernant des dossiers différents.
Si cette option n’est pas cochée ou si aucun bon de commande n’existe pour l’interlocuteur, un nouveau bon de commande sera créé automatiquement.

Exemples :

¨   Vous souhaitez avoir un bon de commande différent pour chaque dossier :

·    Cochez "Création automatique des bons de commande pour les nouveaux dossiers"

·    Décochez "Regroupement de commandes de dossiers différents"

¨   Vous souhaitez regrouper des commandes de dossiers différents dans le même bon de commande :

·    Cochez "Création automatique des bons de commande pour les nouveaux dossiers"

·    Cochez "Regroupement de commandes de dossiers différents"

Informations spécifiques aux possesseurs
Saisie obligatoire des factures d’entrée

Pour les possesseurs, indiquez si vous devez saisir les factures fournisseurs. Si c’est le cas, chaque entrée devra être suivie d’une facture ou d’un renvoi au fournisseur. L’entrée sera marquée ‘à payer‘ tant qu’elle n’aura pas été totalement facturée et ne pourra pas être archivée.
Dans le cas contraire, l’entrée est marquée ‘automatiquement facturée’. Vous pouvez encore saisir une facture pour cette entrée si le prix se révèle différent de celui prévu à la commande, mais ce n’est plus une obligation.

NB : Si l’export comptable est activé alors les factures pour les stocks du possesseur « La société utilisatrice » sont obligatoires quel que soit votre choix ici.

Information de facturation
Groupe de comptes par défaut

Vous pouvez indiquer le groupe de comptes d’achat qui devra être utilisé par défaut pour les factures d’achat à ce fournisseur. Cf. paramétrages de la facturation.

Schéma de taxes par défaut

Vous pouvez indiquer le schéma de taxes applicables sur les achats qui devra être utilisé par défaut pour les factures d’achat à ce fournisseur. Cf. paramétrages de la facturation.

Règlement

Vous pouvez définir le mode de règlement pour le paiement des factures au tiers. Ces informations sont utilisées dans l’export des factures d’achat en comptabilité.

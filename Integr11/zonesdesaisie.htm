Zones de saisie

Précédent   Suivant

La partie droite de la fenêtre est réservée aux zones de saisie.

L’affichage varie en fonction de la rubrique sélectionnée dans la liste de gauche et les informations à saisir peuvent être différentes suivant le type de devis.

Les zones de saisie sont regroupées en 3 écrans :

¨   Entête :

¨   Eléments (ou feuillets) composant le devis :

¨   Pied :

Chaque écran présente les différentes zones de saisie regroupées par rubrique séparées par une ligne horizontale, par exemple : client, prépresse, papier, impression ou façonnage.

Certaines rubriques peuvent se trouver dans des écrans différents suivant le type de devis saisi : par exemple, la rubrique prépresse se trouve dans l’écran des éléments pour les brochures, mais dans celui du pied pour les devis de liasses.

Lorsque vous effectuez une saisie dans une zone, la rubrique contenant cette zone est affichée dans une couleur de fond différente. Vous pouvez modifier cette couleur dans les préférences de saisie.

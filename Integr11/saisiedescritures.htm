Saisie des écritures

Précédent   Suivant

Dans la saisie des écritures, vous travaillez sur les pièces d’un tampon. Vous devez donc, avant toute chose, choisir le tampon de travail.

Sélectionnez un code de tampon existant dans la liste déroulante ou tapez le code d’un nouveau tampon (3 caractères maximum). Cliquez ensuite le bouton Ok pour valider votre choix, la barre de titre de la fenêtre affiche le code du tampon courant.

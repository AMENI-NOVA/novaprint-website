Données envoyées par Graphiplus

Précédent   Suivant

Voici les données que Graphiplus est capable d'envoyer avec l'interface JDF. Ces informations sont ensuite exploitées, ou non, en fonction de système JDF receveur.

Ces données sont réparties en trois :

¨   Une partie commune : ce sont les informations globales (administratives) concernant le travail. Ces informations sont envoyées systématiquement au flux prépresse comme aux presses.

¨   Une partie prépresse : ce sont les informations concernant exclusivement le flux de prépresse.

¨   Une partie impression : ce sont les informations concernant exclusivement les presses.

Paramétrage des chemins de façonnage

Précédent   Suivant

Pour ajouter un nouveau chemin de façonnage utilisez une des deux méthodes ci-dessous :

¨   sélectionnez dans la liste, la rubrique "Chemins de façonnage", puis cliquez sur Nouveau dans le menu Paramétrages ou dans le menu contextuel, ou cliquez sur le bouton correspondant de la barre d’outils.

¨   sélectionnez dans la liste des façonnage, celui dans lequel devra se trouver le nouveau chemin, puis cliquez sur Nouveau dans le menu Paramétrages ou dans le menu contextuel, ou cliquez sur le bouton correspondant de la barre d’outils. Dans ce cas, le nouveau chemin sera automatiquement intégré au façonnage sélectionné au départ.

Nom du chemin

Chaque chemin doit avoir un nom unique. Si vous tentez d’utiliser pour un nouveau chemin, un nom déjà existant, Graphisoft vous invite à changer de nom.

Dimensions de rognes, dépassant, grecquage et gamme Bruner

Vous pouvez indiquer pour chaque chemin des valeurs différentes pour les rognes, le fond perdu inclus dans les rognes, le dépassant, le grecquage et la gamme Bruner.

Pour le dépassant, indiquez s’il s’agit d’un dépassant avant ou arrière.

Le fond perdu inclus aux rognes est utilisé dans le cas où vous avez autorisé la rogne à inclure la prise de pince, la gamme Bruner ou les bandes de freinage. Il définit la valeur qui ne doit pas être utilisée pour le chevauchement. Par exemple, pour une rogne de 5mm et un fond perdu de 2mm, le chevauchement maximum sur la rogne sera de 3mm.

Ces informations ne sont utilisées que pour le calcul des brochures.

Calcul du format des types de cahier des brochures

Les formats des différents types de cahiers sont calculés à partir du format d'un cahier de 4 pages. Pour calculer le format d'un cahier de 4 pages, on part du format fini, soient :
          PC : petit côté du format fini
          GC : grand côté du format fini 
          ROGT : rogne tête/pied de la brochure en haut et en bas des pages
         ROGB : rogne barbe de la brochure à gauche et à droite des pages
          DEP : dépassant si 2 piqûres à cheval
          GRE : grecquage au pli si dos carré collé

Format du 4 pages :

¨   A la française :
2 piqûres à cheval : (PC x 2) + (ROGB x 2) + DEP  /  GC + (ROGT x 2)
Dos carré collé : (PC x 2) + (ROGB x 2) + GRE  /  GC + (ROGT x 2)
Autres façonnages : (PC x 2) + (ROGB x 2)  /  GC + (ROGT x 2)

¨   A l'italienne :
2 piqûres à cheval : PC + (ROGT x 2)  /  (GC x 2) + (ROGB x 2) + DEP
Dos carré collé : PC + (ROGT x 2)  /  (GC x 2) + (ROGB x 2) + GRE
Autres façonnages : PC + (ROGT x 2)  /  (GC x 2) + (ROGB x 2)

Le format des autres types de cahier se déduisent du 4 pages.

La rogne barbe est prise en compte sur 1 seule côté pour une brochure imprimée en feuilles (2 pages).

Couvertures et encarts

Les rognes et le dépassant sont ajoutés au format de la couverture et des encarts brochés pour l’optimisation. Ils ne sont pas ajoutés pour les encarts jetés.

Etapes du chemin

Les étapes du chemin sont les opérations à calculer pour ce chemin.

Selon le type de brochage standard du type de façonnage du chemin, l’application proposera une ou plusieurs listes d’étapes pour le chemin :

¨   Une liste ‘étapes du chemin’ si le type de brochage standard est différent de ‘Couverture rigide’

¨   Trois listes (‘Etapes de l’intérieur’, ‘Etapes de la couverture rigide’ et ‘Etapes du livre’) si le type de brochage est ‘Couverture rigide’.

Etapes possibles

Les étapes possibles peuvent être de plusieurs types :

¨   Type de machine de façonnage ou sélection de machines :
Concerne le Pliage, l’Assemblage cahier, l’Encartage piquage et l’Assemblage feuilles.
Vous pouvez sélectionner le type de machine ou seulement certaines machines.
Dans les deux cas, le programme de calcul fera pour chacune de ces étapes du chemin une optimisation entre les machines de ce type correspondant à la sélection.

·    Sélection du type de machine
L’optimisation sera faite sur toutes les machines du type présentes dans votre paramétrage. Si des nouvelles machines de ce type sont ensuite ajoutées, elles seront aussi prises en compte sans qu’il soit nécessaire de modifier le chemin.

·    Sélection d’une ou plusieurs machines
L’optimisation sera faite entre les machines sélectionnées uniquement. Si des nouvelles machines de ce type sont ensuite ajoutées, elles ne seront pas prises en compte.

Lorsqu’un type de machine est sélectionné, la liste des étapes du chemin fait apparaître ce type de machine, suivi de la mention « toutes les machines ». Pour restreindre l’optimisation à certaines machines de ce type, vous devez d’abord enlever le type de machines de la liste des étapes du chemin, ce qui le fait réapparaître dans la liste des étapes possibles avec le détail de toutes les machines concernées. Vous pouvez ensuite sélectionner les machines désirées.

¨   Opération de façonnage définie par le logiciel
Collage à chaud, Collage à frois, Collage sur rotative, Couture fil, Coupe 3 faces, Coupe 4 faces.
Dans ce cas, le coût correspondant à l'opération sera ajouté aux coûts du chemin.

¨   Opération de façonnage définie par l'utilisateur
Il peut s'agir de n'importe laquelle des opérations particulières de façonnage.
On peut avoir jusqu'à 95 opérations particulières de façonnage pour un chemin.

On ne peut pas ajouter dans la liste des étapes du chemin, une étape qui y est déjà présente. Par exemple, si la liste des étapes du chemin contient déjà "Couture fil", il n'est pas possible de rajouter cette étape de nouveau dans la liste.

Particularités suivant les étapes choisies pour le chemin

¨   Encartage piquage :

·    Il est impossible d'avoir en plus dans le même chemin : Assemblage feuilles, Collage à chaud, Assemblage cahiers

·    L’optimisation avec piquage par 2 exemplaire sera faite suivant les possibilités des encarteuses du fichier

·    Le pliage de la couverture (4 pages) ne sera pas compté sur une plieuse si l'encarteuse choisie permet de le faire.

¨   Assemblage feuilles :
Les chemins créés avec cette étape concernent les deux possibilités prises en compte par Graphisoft pour l'assemblage des brochures (voir aussi le paragraphe "Utilisation des assembleuses feuilles pour le calcul des devis" plus haut dans ce chapitre) :

·    Assemblage, pliage et piquage de cahiers de 4 pages :

§  L'assembleuse plie des cahiers de 4 pages (imposés au façonnage par le programme de calcul) et les pique. Il est inutile de mettre une étape "Pliage" dans ce chemin.

§  Coupe après impression comptée suivant le coût de la refente, dans tous les cas, même si impression d'un 4 pages en 1 pose (coupe de préparation avant le passage sur l'assembleuse).

§  Le nombre de postes pris en compte (comparé au nombre de postes maxi de l'assembleuse) est le nombre total de pages de la brochure, divisé par 4.

§  L'optimisation sur assembleuse feuilles n'est pas possible si un des éléments de la brochure est une couverture (ou un encart) avec rabat (nombre de pages différent de 4).

§  Si des cahiers sont imposés au façonnage, le calcul sur assembleuse feuilles n'est envisagé que si on n'a pas imposé des cahiers autres que 4 pages et qu'on n'a pas ignoré les cahiers de 4 pages.

§  Le chemin de façonnage doit appartenir à un type de façonnage pour lequel le nombre de pages du plus petit cahier est 4 (voir le paragraphe "Paramétrage des types de façonnage", plus loin dans ce chapitre).

·    Assemblage en feuilles :

§  Cette possibilité permet notamment de calculer des devis de brochures imprimées sur des copieurs.

§  Le chemin de façonnage correspondant doit appartenir à un type de façonnage pour lequel le nombre de pages du plus petit cahier est 2 (voir le paragraphe "Paramétrage des types de façonnage", plus loin dans ce chapitre).

§  Vous devrez probablement dans ce cas, ajouter au chemin une opération de façonnage en fonction du brochage souhaité (collage, agrafage, …).

·    Il est impossible d'avoir en plus dans le même chemin les étapes suivantes : Encartage piquage, Collage à chaud, Assemblage cahiers.

¨   Collage à chaud :

·    Il est impossible d'avoir en plus dans le même chemin : Encartage piquage, Assemblage feuilles,

·    Le pliage de la couverture n'est pas calculé s'il n'y a qu'un seul pli.

·    La couverture n'est pas comptée dans le nombre de postes à l'assemblage.

¨   Assemblage cahiers :

·    Il est impossible d'avoir en plus dans le même chemin : Encartage piquage, Assemblage feuilles.

¨   Collage sur rotatives :

Les chemins comportant cette étape ne sont étudiés que si le devis ne comporte qu'un seul élément.

Seules les solutions d’impression en rotatives offset sont étudiées, même si le type d’impression « Offset feuilles » est coché dans le chemin.

Seules les solutions pour lesquelles l’impression de toutes les pages est faite en un seul passage sont possibles. Les solutions en multi-bandes sont possibles.

Le programme considère que le collage au pli est toujours fait parallèlement au sens de défilement de la bande.

¨   Coupe 3 faces :

·    Sera comptée si l'étape fait partie du procédé et si aucune des machines choisies pour le procédé n'a de massicot trilame en sortie. Attention, pour que la coupe 3 faces ne soit pas calculée si la machine d’assemblage a un massicot trilame en sortie, cette étape doit se trouver après l’étape d’assemblage ou d’encartage, sinon la coupe est calculée pour chaque élément.

·    Ne peut être choisi en même temps que Coupe 4 faces.

Ordre des étapes du chemin

Vous pouvez modifier l'ordre des étapes du chemin en utilisant les boutons  et  se trouvant à droite de la liste des étapes sélectionnées.

Cet ordre sera utilisé :

¨   lors de l'optimisation du devis :

·    pour répercuter les éventuelles gâches des machines ou opérations liées aux différentes étapes. Par exemple, la gâche calculée pour une opération de façonnage du chemin sera répercutée dans le calcul des étapes précédentes.

·    Pour déterminer si les coûts doivent être calculés élément par élément ou version par version.

Les étapes se trouvant avant l’étape d’assemblage ou d’encartage sont calculées séparément pour chaque élément du devis car les éléments du devis ne sont pas encore assemblés.

A partir de l’étape d’assemblage ou d’encartage, les coûts sont calculés pour l’ensemble du devis ou version par version le cas échéant (il y a plusieurs versions s’il y a des quantités différentes par élément).

¨   Au moment de la transformation du devis en commande, pour gérer l'ordre des fiches de travail générées à partir du devis. Ceci permet de rendre visible une fiche de travail sur les terminaux de l'atelier lorsque les travaux précédents de cette fiche ont été au moins commencés. Cette possibilité ne concerne que les utilisateurs de l'Intégrale.

L'ordre ne s’applique pas :

¨   à l'étape de pliage qui vient toujours avant les autres,

¨   aux devis de fiche, dépliants et liasses/carnets, sauf pour l’ordre des opérations de façonnage les unes par rapport aux autres.

NB : pour les devis de liasses/carnets, l’ordre des étapes est toujours le suivant : collage à froid, coupe 4 faces (si assemblage par 1 pose), assemblage, coupe 4 faces (si assemblage par plusieurs poses), opérations de façonnage (dans l’ordre défini dans le chemin de façonnage).

Particularités des chemins de brochage du type de brochage « couverture rigide »

Lorsqu’un chemin de façonnage appartient à un type de façonnage correspondant à des livres à couverture rigide, il y a 3 listes d’étapes de façonnage au lieu d’une seule :

¨   Intérieur : étapes de façonnage nécessaires à la réalisation de l’intérieur du livre.

¨   Couverture rigide : étapes de façonnage nécessaires à la réalisation de la couverture rigide.

¨   Livre : étapes de façonnage nécessaires à la finition du livre (assemblage de l’intérieur et de la couverture).

NB : lorsqu’un chemin existant est ajouté à un type de façonnage « hard cover », la liste des étapes déjà existantes est présentée dans les étapes "Livre".

Etapes spécifiques aux livres à couverture rigide

Le processus de fabrication d’un livre à couverture rigide comprend des opérations incontournables qu’il est nécessaire d’identifier précisément :

¨   La fabrication de la couverture (Case making)

¨   L’emboîtage (assemblage de l’intérieur et de la couverture ou Casing In)

Ces opérations doivent être paramétrées avec des opérations particulières de façonnage et c’est le nouveau type JDF des opérations qui permet de les identifier (voir « Opérations de façonnage »).

NB : l’opération de type emboîtage est facultative dans le cas où une assembleuse cahier peut réaliser cette opération en ligne (voir « Assembleuses cahiers »).

Un chemin de façonnage pour les livres à couverture rigide doit comporter :

¨   Une étape d’assemblage cahier, soit dans les étapes intérieur, soit dans les étapes livre dans le cas où une assembleuse peut réaliser l’opération d’emboîtage.

¨   Une étape de fabrication de la couverture dans les étapes « Couverture rigide ».

¨   Une étape emboîtage ou assemblage cahiers dans les étapes « Livre ».

 

 

 

Les contraintes suivantes impliquent que pour certaines étapes, les flèches permettant d’affecter une étape à une liste seront grisées pour une ou plusieurs des listes.

¨   Les étapes « Encartage-piquage », « Collage sur rotative » et « Collage à froid » ne sont pas disponibles.

¨   La liste des étapes « intérieur » peut contenir toutes les types d’étapes à l’exception des types « fabrication de couverture rigide » et « Emboîtage ».

¨   La liste des étapes « de couverture » ne peut contenir que les opérations de façonnage non typées ou d’ « emboîtage ».

¨   La liste des étapes « de couverture » ne peut contenir que les assembleuses cahier, les opérations de façonnage non typées ou de type « emboîtage », « collage page de garde », « coupe 3 faces », « collage à chaud » et « couture fil ».

¨   Si l’assemblage cahier est fait dans les étapes « intérieur », l’étape « emboîtage » est obligatoire et les étapes « livre » ne peuvent contenir ce type d’étape.

¨   Si l’assemblage cahier est fait dans les étapes « livre », alors, ces étapes ne peuvent contenir d’étape « emboîtage ».

Type d'impression

Sert à déterminer le type de machines d'impression sur lequel doit être faite l'optimisation pour le chemin.

Machines d’impression

Il est possible d’indiquer une sélection de machines d’impression en fonction de laquelle l’optimisation devra être faite lorsque le chemin de façonnage est utilisé.

Par défaut l’optimisation est faite sur toutes les machines d’impression. La liste des machines sélectionnées comporte alors la mention « Toutes les machines ». Dans ce cas, les nouvelles machines d’impression seront prises en compte sans qu’il soit nécessaire de modifier le chemin.

Les machines sélectionnées peuvent être :

¨   Imposées : seules les machines d’impression sélectionnées seront utilisées avec ce chemin.

¨   Exclues : les machines d’impression sélectionnées ne seront jamais utilisées avec ce chemin.

Machines d’impression numérique

Si vous imposez au moins une presse numérique, les machines numériques grand format seront automatiquement exclues du chemin. A l’inverse, si vous imposez au moins une macchine numérique grand format, les presses numériques seront automatiquement exclues du chemin.

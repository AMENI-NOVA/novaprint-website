Informations d’entête spécifiques aux devis de liasses et carnets

Format fini

Le programme conserve les valeurs dans l’ordre dans lequel elles ont été saisies (ex : 29,7 x 21), ce qui permet de préciser l’orientation.

Le format fini peut être modifié indépendamment pour chaque feuillet composant la liasse.

Carnet de liasses

Pour saisir un devis de carnet de liasses, cochez "Carnets de xx liasses", et indiquez le nombre de liasses de chaque carnet. Dans ce cas, la quantité saisie correspond à la quantité de carnets, chaque liasse comprenant le nombre de feuillets que vous aurez saisis. Par exemple, 1000 exemplaires de carnets de 50 liasses correspondent à l’impression de 50 000 liasses.

D’autre part, vous devrez ajouter l'opération correspondante de façonnage avec les "opérations particulières de façonnage", ainsi qu’une autre opération pour le collage à froid, si c'est nécessaire, le collage n'étant compté automatiquement que pour les liasses indépendantes.

Encrage

Le pourcentage d'encrage est unique pour tout le devis. le calcul de l'encre est fait feuillet par feuillet suivant l'impression (recto, recto et verso, ou pas d'impression).

Par défaut, cette zone est initialisée avec une valeur standard provenant du paramétrage (voir "Paramétrages généraux des papiers " dans le chapitre "Paramétrages de calcul des devis»), qui dépend du type de devis et du nombre de couleurs saisi. Lorsque vous modifiez le nombre de couleurs, le pourcentage d'encrage est automatiquement ajusté en fonction de la nouvelle valeur, sauf si vous avez modifié directement le pourcentage d'encrage ou si le devis a été rappelé.
Le nombre de couleurs pris en compte est le plus grand trouvé parmi tous les feuillets du devis (en distinguant le recto du verso).

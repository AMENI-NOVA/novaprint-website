Préférences de saisie

Précédent   Suivant

Les préférences de saisie permettent de définir :

¨   les valeurs par défaut utilisées pour les nouveaux devis, comme par exemple la quantité, la gâche de façonnage, l'unité de prix de papier, les couleurs d'impression, etc...

¨   si les zones de saisie se trouvant dans la partie avancée doivent être systématiquement affichées

¨   la couleur de fond utilisée pour la partie en cours de saisie

Pour modifier les préférences de saisie des devis, ouvrez un n'importe quel devis (nouveau ou existant), puis sélectionnez l'option Préférences dans le menu Devis optimisé.

Pour y accéder vous devez disposer du droit "Préférences de saisie" se trouvant dans "Graphisoft" -> "Devis" -> "Saisie de devis" (voir le paragraphe "Droits" du chapitre "Paramétrages généraux").

 

Les valeurs par défaut s'appliquent pour tous les types de devis sauf dans les cas suivants :

¨   Général

·    Prix unitaire par : Ne s’applique pas aux devis de continu.

·    Fond perdu, blanc tournant, repiquage : ne concerne que les devis de dépliant, fiches ou liasses/carnets.

¨   Papier

·    Unité de prix de papier : l'unité de prix de papier par défaut est utilisée lorsque vous saisissez le prix du papier dans le devis, c'est-à-dire pour les papiers hors fichier ou si vous cochez "Prix d'achat".

NB : si le devis vient d’être calculé, c’est l’unité de prix de papier du devis calculé qui sera affichée.

¨   Impression : ne s'applique pas aux devis de liasses (impression en feuilles ou continu) et d'étiquettes en continu.

¨   Continu : Ces préférences ne s’appliquent qu’aux devis de continu

 

Les préférences de saisie sont les mêmes pour tous les utilisateurs de l'application.

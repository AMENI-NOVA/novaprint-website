Paramétrage des types de cahiers

Précédent   Suivant

Utile seulement si vous utilisez l'interface avec le prépresse.

 

Tout comme Graphiplus, le JDF dispose d'une liste de types de cahiers appelée "Fold catalog". Ce "Fold catalog" référence une centaine de types de cahiers avec la façon dont ils sont pliés (type et ordre des plis).

Pour que l'interface avec le prépresse fonctionne correctement, il faut faire correspondre les types de cahiers de Graphiplus avec ceux du JDF. Pour cela, utilisez l'écran de saisie de type de cahiers dans Graphiplus et indiquez la référence JDF pour chaque type de cahier.

 

 

La liste "Référence dans le "Fold catalog" du JDF" contient les références possibles en fonction de la description du type de cahiers (nombre de pages, nombre et sens des plis). Pour connaître quelle référence choisir reportez-vous au tableau ci-dessous (extrait de la spécification JDF). La référence à indiquer se situe en haut à gauche de chaque type de cahier JDF

 

Si pour un type de cahier dans Graphiplus vous avez besoin de plusieurs références JDF, vous pouvez dupliquer le type de cahier et indiquer l'autre référence JDF.
Pour que le calcul du devis sache quel type de cahier utiliser, vous pouvez soit le lier à une machine dans un chemin de brochage ou l'indiquer lors de la saisie du devis.

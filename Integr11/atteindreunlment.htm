Atteindre un élément

Précédent   Suivant

La liste déroulante "Atteindre le poste" permet de se positionner directement sur le poste sélectionné. Cette liste affiche tous les regroupements ainsi que leurs éléments.

Si l'on choisit un élément situé dans un regroupement fermé, ce dernier sera automatiquement ouvert.

Dans la liste déroulante les couleurs ont la signification suivante:

¨   Regroupement : bleu foncé avec une image de dossier ouvert.

¨   Elément regroupé : bleu clair.

¨   Elément non regroupé : blanc.

Le raccourci clavier ALT + P permet de dérouler immédiatement la liste des éléments pour en choisir un.

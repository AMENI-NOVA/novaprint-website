Machine à répéter

Précédent   Suivant

Si vous avez une machine à répéter, les caractéristiques et les coûts que vous saisissez ici seront utilisés pour calculer le coût des duplis pour les chemins de prépresse pour lesquels vous avez inséré l’étape "Duplis : Optimisation Duplis négatif / Répétex".

Le coût des duplis sera optimisé entre :

¨   le coût avec des négatifs (formats et/ou prix maxi par couleur et dm2)

¨   le coût sur machine à répéter

Ce paramétrage n'est pas accessible pour la gamme Estiprint.

Mise à jour des papiers

Précédent   Suivant

Depuis l’arbre de paramétrage, pour voir les papiers d’une catégorie, développez-la en cliquant dans la liste arborescente, sur le signe ‘+’ se trouvant à gauche du nom de la catégorie.

Ajout d’un papier à une catégorie

Pour ajouter un papier à une catégorie, sélectionnez la catégorie, puis cliquez sur Nouveau dans le menu Paramétrages ou dans le menu contextuel, ou cliquez sur le bouton correspondant de la barre d’outils.

Suppression d’un papier

Attention, cette fonction supprime du fichier tous les grammages, formats et tarifs de ce papier.

Duplication d’un papier

Pour dupliquer un papier, sélectionnez le papier, puis cliquez sur Dupliquer dans le menu Paramétrages ou dans le menu contextuel ou cliquez sur le bouton correspondant de la barre d’outils.

Ensuite saisissez le nom et le code pour ce nouveau papier et cliquer sur le bouton 'Enregistrer'.

Le nouveau papier est créé par défaut dans la même catégorie que le papier d'origine.

Changer la catégorie d’un papier

Pour changer la catégorie d’un papier, il suffit de le sélectionner avec la souris et de le faire glisser vers la nouvelle catégorie. Il est aussi possible de changer la catégorie depuis l'écran de mise à jour des papiers (voir ci-dessous).

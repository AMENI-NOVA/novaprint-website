Les informations de cet onglet décrivent les colonnes composant le fichier à importer.

Les informations du fichier peuvent ici être reliées aux informations qui définissent les papiers et leurs tarifs dans les progiciels de GRAPHISOFT. Dans le cas d’un fichier avec des informations de largeurs fixes, on indiquera ici les largeurs des informations dans chaque ligne. Un tableau permet de visualiser un extrait du fichier à importer. Il est modifié en fonction des indications saisies. En cliquant sur un entête de colonne de ce tableau, la ligne correspondante est affichée dans la table de correspondance.

Remarque : Voir « Informations gérées par l’interface » pour une description des informations gérées par Graphisoft.

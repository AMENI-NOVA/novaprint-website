Paramétrage de la langue pour les clients étrangers

Précédent   Suivant

Dans la barre d’outils, cliquez sur l’icône , ou dans la barre de menu Fichier, Graphiplus, Sociétés

 

Pour afficher les textes de Graphiplus dans une langue différente de celle de Windows sur le poste (indiquée dans les paramètres régionaux de Windows), il faut, dans la fiche société, modifier la langue utilisée par le client.

Le fait de relier une connexion à une société, impose comme langue d’utilisation, la langue indiquée pour la société.

Le paramétrage de la langue se trouve dans l’onglet « Détails » de la fiche.

Les connexions pour des clients étrangers devront être liées à un modèle d’édition de la langue correspondante (voir ci-dessous, Paramétrage des éditions). Les libellés enregistrés dans les fichiers dans la langue de l’imprimeur pourront être traduits par l’intermédiaire du dictionnaire.

Les options sont accessibles en cliquant sur le bouton « Options >> » se trouvant dans la fenêtre de saisie d’une relance.

Proposer la relance

Avec cette option, vous pouvez définir à quel moment le programme doit proposer de relancer le devis.
Trois choix sont possibles :

1.  Ne pas proposer : le programme ne proposera jamais de relancer le devis. Pour définir une relance, vous devrez alors passer par la liste des devis, puis choisir l’option Définir la prochaine relance du menu Traitements.

2.  A l’enregistrement de devis : après chaque enregistrement de devis, le programme affichera automatiquement l’écran de saisie de relance.

3.  A l’impression de la lettre : Une fois la lettre de confirmation imprimée, le programme affichera automatiquement l’écran de saisie de relance.

Message de rappel

Cette option ne concerne que les utilisateurs de l’Intégrale ou de Graph'e-mail.

Si cette option est cochée, un message de rappel sera envoyé depuis la messagerie Graphisoft à la personne concernée. La personne sera notifiée du message à partir du jour de la relance.

Le message contient les informations suivantes :

Objet      Rappel : Relance du devis "93820 ARGUMENTAIRE                  "Sacoche vélo""

 

Il faut relancer le client Graphisoft à propos du devis n°93820.

 

Type de devis         : Affiche

Date                         : 21/03/1997

Référence               : ARGUMENTAIRE                  "Sacoche vélo"

Exemplaires             : 5 500

Montant total            : 6 107,72 F

Ce message est juste un rappel. Le fait de l’avoir lu, ne va pas indiquer le devis comme relancé.

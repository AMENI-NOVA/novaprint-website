Famille d’opérations sur barème

Précédent   Suivant

Entrez un libellé pour regrouper un ensemble d’opérations.

NB : Si vous renommez une famille d’opérations sur barème, un message vous demande si vous voulez mettre à jour cette famille dans les prix sur barèmes définis dans les modèles de devis. Répondez Oui si vous voulez que les modèles de devis conservent leur lien vers cette famille, non dans le cas contraire. Dans le second cas, il vous faudra modifier manuellement les modèles de devis concernés ou renommer une copie de la famille avec le nom d’origine. En effet, le lien entre les devis sur barème et le paramétrage s’effectue par le libellé de la famille et de l’opération. (Voir « Suivi des grilles tarifaires » ci-après)

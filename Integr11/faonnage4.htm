Opérations de façonnage

On retrouve pour les opérations de façonnage les mêmes options que pour les opérations de prépresse, avec un fonctionnement identique :

¨   Apparaît à la saisie du devis

¨   Tarif acheteur ou imprimeur

Suppression d’une opération de façonnage

Lorsque vous supprimez une opération depuis le paramétrage d’un imprimeur, seul le tarif de l’imprimeur pour cette opération est supprimée.

Pour supprimer définitivement une opération, vous devez effectuer la suppression depuis votre paramétrage (acheteur). S’il existe au moins un tarif imprimeur pour cette opération, le programme vous demande alors si vous désirez supprimer l’opération entièrement avec tous ses tarifs, ou uniquement votre tarif pour cette opération.

 

Pelliculage Vernis
Dupliquer une opération de pelliculage vernis

Lorsque vous dupliquer une opération de pelliculage vernis d’un imprimeur, les données de l’opération sont  seulement dupliquer pour l’imprimeur en question.

Seules les opérations de pelliculage vernis possédants des tarifs peuvent être dupliquées.

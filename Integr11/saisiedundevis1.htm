Saisie d’un devis

Précédent   Suivant

Dans la saisie d’un devis optimisé, le menu « Devis optimisé » présente l’option « Prix sur barème » si le client du devis appartient à une catégorie pour laquelle un barème est défini, si un barème est défini pour le client ou si le devis a un prix sur barème défini.

Remarque : le fait de changer le client du devis peut rendre le devis incalculable du fait du barème si le nouveau client n’est pas de la même catégorie.

Le calcul du devis optimisé est réalisé de la même façon avec ou sans barème ; La différence se situe dans le total : Si un barème est défini, le prix de vente est imposé d’après l’application du barème et le coefficient est modifié en conséquence. Le coefficient saisi dans l’entête n’est donc pas conservé. La modification des opérations composant le prix sur barème obligent à recalculer le devis avant enregistrement.

Si vous supprimez le prix sur barème en enlevant toutes les opérations, le coefficient sur le prix de vente est réinitialisé à la valeur par défaut pour le client, lu dans sa fiche.

Remarque : Si un prix sur barème est défini alors il ne peut pas y avoir d’unité de quantité et donc pas de prix unitaire.

Planification des dossiers

Précédent   Suivant

Ce paramétrage permet de définir la méthode de mise au planning des dossiers :

Planifier automatiquement tous les dossiers dès leur création :

Tous les dossiers, sans exception, sont planifiés dès leur création.

Si cette méthode est sélectionnée, tous les autres choix de planification sont désactivés.

Planifier automatiquement les dossiers cumulant les critères suivants :

Seuls les dossiers répondant aux critères sélectionnés ci-dessous sont planifié.
Les critères sont les suivants :

·              Les dossiers ayant une date de réception d’éléments ou ayant reçu les éléments ;

·              Les dossiers commencés (ayant une date de début de production).

 

Dès qu’un dossier correspond aux critères sélectionnés, il est planifié dans EmaPlan.

Si un dossier déjà dans EmaPlan ne répond plus aux critères, il est automatiquement supprimé d’EmaPlan (déplanifié).

 

Si aucun critère n’est coché, alors seuls les dossiers planifiés manuellement seront dans EmaPlan.

Autoriser la planification manuelle des dossiers :

Lorsque cette option est cochée, un menu apparait dans la liste des commandes et le tableau de charge. Il permet mettre au planning les dossiers ou de les déplanifier.

Il n’est pas possible de planifier dans EmaPlan un dossier cloturé ou annulé.

Seuls les dossiers planifiés manuellement peuvent être déplanifier manuellement.

 

La liste des commandes (comme le tableau de charge) a une colonne (EmaPlan : Envoi manuel) indiquant si un dossier à été envoyé manuellement à EmaPlan.

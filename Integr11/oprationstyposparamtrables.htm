Opérations typos paramétrables

Précédent   Suivant

Ce paramétrage permet d’ajouter jusqu’à 5 opérations typo avec un libellé paramétrable, aux opérations fixes prévues par défaut : rainage, découpe, numérotage, perfo, gaufrage.

Pour ajouter une opération paramétrable, sélectionnez « Opérations typo paramétrables », puis choisissez « Nouveau » dans le menu contextuel.

Saisissez ensuite le nom de l’opération, puis validez.

Les opérations paramétrées ici apparaissent dans le paramétrage de chaque machine typo. Pour chacune d’elle, vous pouvez indiquer si l’opération est possible sur la machine, avec ses coûts associés.

Envoyer plusieurs éditions dans le même e-mail

Précédent   Suivant

Lorsque vous cliquez sur « Envoyer par mail » (une fois une édition affichée) si un mail est en cours de saisi (dans Graphisoft), il est possible de le compléter avec l’édition.

Fonctionnement

Avant de créer un nouveau mail pour envoyer une édition, le programme va regarder si une note est en cours de saisie dans Graphisoft (celle-ci peut-être en arrière plan ou réduite en icône).

Le programme vous demande alors si vous souhaitez ajouter l’édition à la note en cours de saisie.

Si vous répondez oui, la note est  complétée avec le destinataire de l’édition et l’édition en pièce jointe. L’objet et le texte de la note  ne seront pas modifiés.

Vous pouvez répéter l’opération avec autant d’éditions que vous le souhaitez.

Rappel

Depuis la version 12.3, il est aussi possible d’exporter les éditions dans des fichiers PDF. Vous pouvez ensuite ajouter manuellement ces fichiers lors de la saisie de vos mails (depuis Graphisoft ou toute autre messagerie).

 

NB : Pour envoyer plusieurs lettres de confirmation dans le même e-mail, reportez-vous au paragraphe « Envoyer plusieurs lettres de confirmation dans le même e-mail » du chapitre « Traitement des devis ».

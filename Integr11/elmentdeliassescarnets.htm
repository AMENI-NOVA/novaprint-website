Type d’élément

Un devis de carnets de liasses peut être composé d’éléments de type :

¨   Feuillet

¨   Couverture

¨   Plat dessus

¨   Plat dessous

Feuillets de liasses/carnets
Format fini

Par défaut le format fini est initialisé à partir du format fini saisi dans l’entête du devis, mais il peut être modifié feuillet par feuillet.

Couverture

Les informations à saisir sont les mêmes que pour une couverture de brochure.

Le calcul avec une couverture est possible pour un devis de carnets de liasses ou de liasses indépendantes.

Il peut y avoir une couverture avec un plat dessus et/ou un plat dessous dans le même devis.

Format à plat

Le format à plat est initialisé en fonction des éléments présents dans le devis et en tenant compte de l’épaisseur du dos : nombre de feuillets et grammage, nombre de liasses par carnet. Lorsqu’un de ces éléments est modifié, le programme propose de recalculer le format à plat de la couverture.

Plat dessus ou dessous

Les informations à saisir sont les mêmes que pour une fiche.

Il peut y avoir un plat dessus et un plat dessous dans le même devis.

Le calcul avec un plat dessus ou dessous n’est possible que pour un carnet de liasses.

Format à plat

Le format à plat est initialisé par défaut avec le format fini global du devis. Si celui-ci est modifié, le programme propose de modifer aussi le format à plat des plats dessus ou dessous.

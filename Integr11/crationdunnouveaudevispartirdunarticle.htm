Création d’un nouveau devis à partir d’un article

Précédent   Suivant

Vous pouvez créer des nouveaux devis à partir des articles pour lesquels une description a été enregistrée.

Ouvrez la liste des stocks, sélectionner l’article, et depuis le menu « Stocks » ou le menu contextuel, sélectionnez l’option « Nouveau devis ».

NB : Ce choix n’est disponible que pour les articles ayant une description enregistrée. L’information « Description » de la liste des stocks permet de voir les articles pour lesquels une description a été enregistrée.

Lorsqu'un devis est initialisé à partir d'une description d'article, les seules informations modifiables sont :

¨   l'interlocuteur, le tarif et le coefficient

¨   la référence, les quantités, le prix unitaire

¨   les données d'optimisation des éléments,

¨   les conditionnements,

¨   les livraisons.

Les informations décrivant le produit ne sont pas modifiables : format, papier, impression, opérations additionnelles.

La quantité du nouveau devis est à 0.

NB : vous pouvez aussi créer un devis à partir d’un article en ouvrant un devis existant lié à cet article.

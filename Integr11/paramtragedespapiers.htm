Paramétrage des papiers

Précédent   Suivant

Le type de papier est une information importante dans le JDF. On peut le définir dans chaque catégorie de papier.

 

Les valeurs du type de papier sont définies par la norme ISO 12647-2 :

·      Couché brillant ;

·      Couché mat ;

·      Couché mince ou couché recyclé ;

·      Non couché blanc ;

·      Non couché avec bois.

 

Si « Non défini » est sélectionné, l’information du type de papier ne sera pas envoyée dans le JDF. Ce choix n’est pas recommandé.

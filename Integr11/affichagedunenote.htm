Affichage d'une note

Précédent   Suivant

"Afficher" dans le menu "Notes" ou le menu contextuel.

 

Cette fonction permet d'afficher dans une autre fenêtre la note sélectionnée dans l'historique.

Cette fenêtre dispose de boutons "Précédente" et "Suivante" qui permettent de se déplacer d'une note à l'autre.

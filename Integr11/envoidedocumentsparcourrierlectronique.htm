Envoi de documents par courrier électronique

Précédent   Suivant

Configuration requise

Pour pouvoir envoyer des documents de Graphiplus par courrier électronique :

¨   Vous devez avoir installé sur votre ordinateur un utilitaire de messagerie comme « Outlook express », « Netscape messenger », « Eudora » ou autre

¨   Votre accès à une boite aux lettre électronique doit être correctement paramétré. A titre de vérification, vous devez pouvoir exécuter l’utilitaire de messagerie et vous envoyer un message à vous même.

¨   Dans les paramètres de votre utilitaire de messagerie vous devez avoir indiqué qu’il est un client MAPI. Ceci permet à d’autres applications, comme Graphiplus, de l’utiliser pour l’envoi de courrier électronique.

¨   Pour les postes fonctionnant sous Windows 95, il peut être nécessaire d’installer le « Gestionnaire de messagerie » du groupe « Communication » de l’installation de Windows 95. Contactez votre installateur si besoin pour procéder à cette installation.

Documents concernés

Les documents qui peuvent être envoyés par courrier électronique sont :

¨   Lettre de confirmation

¨   Accusé de réception

¨   Bon de livraison

¨   Facture / Avoir

¨   Relevé mensuel de facturation

¨   Pochette de fabrication

¨   Bon de commande fournisseur / Sous-traitant

Le document est une pièce jointe attachée au message électronique. La pièce jointe est par défaut au format « Adobe PDF », mais vous pouvez forcer l’envoi au format « Texte enrichi » ou au format « Word ».

Utilisation

Pour envoyer un document par courrier électronique :

¨   Ouvrez le document à partir de la liste des documents.

¨   Cliquez sur Envoyer  par e-mail … dans le menu Document
Remarque : Pour choisir le format utilisé pour la pièce jointe, sélectionnez l’option voulue dans le sous-menu.

¨   Cela provoque l’enregistrement du document (comme pour l’impression)

¨   Si aucune référence du destinataire ne peut être définie, un dialogue apparaît pour introduire une adresse de destinataire. Par défaut, cette adresse est initialisée avec :

·    L’adresse électronique de l’interlocuteur de la société à qui est adressé le document

·    A défaut : L’adresse électronique de la société à qui est adressé le document

·    A défaut : Le nom interne de la société à qui est adressé le document

¨   Dans la liste des documents de Graphiplus, le document est marqué « Edité » comme lorsque vous l’imprimez.

 

Un document envoyé par e-mail est sauvegardé sur le disque dur. Cela est nécessaire pour certains programmes de messagerie Internet. De plus, si vous utilisez Graph'e-mail, le document est accessible depuis l'historique des sociétés en cliquant sur le lien hypertexte.

Le répertoire d'enregistrement des documents est paramétrable dans les préférences de la messagerie. Les documents sont conservés 30 jours dans ce répertoire. Ensuite, ils sont supprimés.

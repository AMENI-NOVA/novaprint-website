Utilisation de l'écran de rappel

Précédent   Suivant

Pour agrandir l'écran de rappel, il faut déplacer le curseur de la souris dessus. Tant que la souris est dans l'écran, celui-ci ne se réduit pas.

S'il y a plusieurs notes dans le rappel, la liste "Notes" les affichent. Dessous, la zone "Contenu" affiche un extrait de la note sélectionnée dans la liste. Pour avoir plus de détails sur la note, il faut utiliser le bouton "Propriété" ou double-cliquer sur la note dans la liste "Notes".

La fenêtre de visualisation d'une note affiche le détail complet d'une note. Pour passer d'une note à l'autre, il faut utiliser les boutons "Précédente" et "Suivante". Le bouton "Imprimer" permet d'éditer la note affichée sur papier.

 

Le bouton "Nouvelle note" de l'écran de rappel, permet de saisir une note directement liée à élément sur lequel vous travaillez. Si vous êtes dans le dossier technique d'un devis, ce bouton vous permet de créer une nouvelle note pour le devis en question ou la société du devis.

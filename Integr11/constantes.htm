Constantes

Précédent   Suivant

Les informations constantes qui permettront d’interpréter correctement les informations contenues dans le fichier de tarifs.

Tranches de tarification.

Si le fichier importé comprend plusieurs colonnes de tarifs mais pas de colonne borne permettant de déterminer quelle tarif utiliser, vous devrez indiquer ici les valeurs constantes de ces bornes ainsi que l’unité dans laquelle elles sont exprimées.

Unités de longueurs utilisées.

Cette information permet d’interpréter correctement les colonnes de largeur et longueurs des feuilles de papier, qu’elles soient exprimées en mm ou en cm.

Unités des épaisseurs.

Cette information permet d’appliquer un multiplicateur pour interpréter correctement les colonnes épaisseur des feuilles de papier. Par exemple, si dans le fichier importé les épaisseurs sont en mm, entrez 1000 (1mm= 1000 µm)

Unité de tarification.

Vous pouvez ainsi définir à quelle quantité les tarifs du fichier importé correspondent. Par exemple " 1000 feuilles ".

Unité monétaire utilisée

La devise et l’unité utilisée pour les tarifs. Vous indiquez à l’aide de ces informations si les prix sont en exprimés Francs, en Euros, en centimes etc.  Exemple : Entrez 0,01 France € (EUR) si les prix sont en centimes d’euros.

Codes utilisés dans le fichier.

Si le fichier contient une des colonnes : Code unité de prix, Unité de prix unitaire, Unité de largeur, Unité de longueur, certification alors ces colonnes contiennent en général des codes que cette table permet d’interpréter.
Ex : Code = "CM unité de longueur",  Valeur = "10 mm"

Pour les certifications, il sera nécessaire de créer auparavant vos certifications dans « Paramétrage – Calcul des devis – Papiers – Certifications » pour pouvoir attacher les libellés des certifications utilisés par le papetier aux vôtres.

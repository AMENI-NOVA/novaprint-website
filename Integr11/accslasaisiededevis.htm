Accès à la saisie de devis

Précédent   Suivant

La saisie d’un devis est accessible à partir de la liste des devis.

Par défaut, cette liste n’affiche que les devis de l’utilisateur en cours (ceux dont il est le commercial ou le deviseur). Pour voir tous les devis, l'utilisateur doit avoir le droit de ‘Voir tous les devis’ (voir le paragraphe "Droits (d'un employé) " dans le chapitre "Paramétrages généraux").

Vous pouvez ensuite :

¨   Saisir un nouveau devis : choisissez entre « Devis d'impression » et « Devis de prépresse », puis son type (Brochure, Dépliant, Affiche, …) dans le menu « Devis optimisé », ou par le menu contextuel par l’option « Nouveau ».

¨   Charger un devis existant : sélectionnez-le dans la liste et choisissez l’option « Ouvrir » du menu « Devis optimisé » ou du menu contextuel. Vous pouvez aussi ouvrir un devis en double-cliquant dessus dans la liste.

¨   Charger un modèle de devis : ouvrez l'option « Modèles de devis » se trouvant dans le menu « Devis optimisé » ou dans le menu contextuel, puis sélectionnez le modèle de devis à ouvrir (les modèles de devis sont regroupés par produit).

¨   Charger un nouveau devis à partir d’un fichier XML : choisissez l’option « Ouvrir un fichier XML », puis sélectionner le fichier XML du devis à charger.

La structure du fichier XML est décrite dans le paragraphe « Interface devis au format XML » de ce chapitre.

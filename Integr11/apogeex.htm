Toutes les valeurs suivantes dépendent des jeux de paramètres (ou Parameter Set) défini dans Apogee.

Voici comment nos clients français sont généralement configurés :

·      Epreuves de pages, tâche « Image » : Pages.

·      Epreuves de pages, tâche « Output » : BAT.

·      Epreuves d’imposition, tâche « Image » : Impos.

·      Epreuves d’imposition, tâche « Ourput » : Ozalid.

·      Dépôt des fichiers, local : JT_Local.

·      Dépôt des fichiers, FTP : JT_FTP

 

·      Version d’Apogee : < 7.1

·      Modèle de traitement des couleurs si Noir + ton direct : DeviceGray

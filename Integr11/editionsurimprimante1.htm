Edition sur imprimante

Précédent   Suivant

Cliquez sur "Imprimez" dans le menu "Document". La lettre est envoyée vers l’imprimante définie dans le modèle d’édition utilisé.

L’édition sera enregistrée dans la liste des documents en fonction de l’option que vous avez choisi pour le modèle utilisé (voir « Modèles d’éditions » dans le chapitre « Paramétrage des éditions ». En cas d’enregistrement, vous devez confirmer ou modifier le nom proposé pour l’édition.

Si vous avez choisi de saisir les informations de relance du devis au moment de l’impression de la lettre, la fenêtre de saisie correspondante apparaît (voir « Définir la prochaine relance » dans le paragraphe « Relance des devis »).

Si vous utilisez Graph’e-mail et les notes automatiques, une note peut-être automatiquement créée et enregistrée dans l’historique du devis. (voir « Notes automatiques » dans le chapitre « Notes »

Fichier d'importation

Précédent   Suivant

Cette première étape permet de définir le fichier d'importation.

Une fois le fichier sélectionné, un aperçu de celui-ci est affiché en bas de l'écran. Cet aperçu tient compte de tous les paramètres que vous avez saisis. L'aperçu est limité aux 100 premières lignes du fichier.

Chemin et nom du fichier d'importation

Cette information correspond au nom du fichier d'importation. Vous pouvez, soit taper son nom, soit le rechercher sur votre disque dur ou sur votre réseau local en utilisant le bouton <…>.

Si vous sélectionnez un fichier ayant l'extension cvs, le programme sélectionne automatiquement les options adéquates pour ce type de fichier :

¨   Type des données : Délimitée

¨   La première ligne contient le nom des colonnes : Cochée

¨   Séparateur : ;

¨   Identificateur de texte : "

¨   Commencer l'importation à la ligne : 2

Commencer l'importation à la ligne

Par défaut, l'importation commence à la première ligne du fichier. Mais dans certains fichiers la première ligne est réservée pour décrire les différentes colonnes.

Ce paramètre permet de définir, à quelle ligne doit commencer l'importation. Les lignes précédentes seront ignorées.

Origine du fichier

Lorsque le fichier d'importation est créé par un programme MS-DOS, certains caractères ne sont pas lus correctement sous Windows. C'est le cas de tous les caractères accentués.

Si votre fichier à été créé sous Ms-Dos, choisissez "Ms-Dos (Pc-8)", sinon laissez "Windows (Ansi)".

Type des données

Cette information est très importante. Elle définit comment les données sont organisées dans le fichier d'importation. Deux choix possibles:

¨   Largeur fixe : Les données sont alignées et séparées par des espaces

Par exemple :
DUPONT              ORLEANS
SOCIETE DURAND      PARIS

Dans cet exemple, le nom de la société occupe toujours le même nombre de caractères.

¨   Délimité : des caractères tels que des virgules ou des tabulations séparent les données.

Par exemple :
DUPONT;ORLEANS
SOCIETE DURAND;PARIS

Dans cet exemple, le nombre de caractères occupés par le nom de la société est variable, et les différents champs sont séparés par un point-virgule.

La première ligne contient le nom des colonnes : Cochez cette case si la première ligne du fichier représente le nom des colonnes. Cela indique au programme de ne pas importer cette ligne et affiche dans l'aperçu le nom des colonnes.

 

Si vous ne savez pas quel choix faire, essayez les deux et contrôlez avec l'aperçu.

Séparateur

Indique quel est le séparateur des données quand les données sont délimitées. Vous pouvez choisir dans la liste proposée, mais aussi saisir le séparateur si celui-ci n'est pas dans la liste.

Identificateur de texte

Ce paramètre est utile que très rarement.

Parfois, les données sont encadrées par un caractère (souvent ") qui indique s'il s'agit de texte ou de chiffres.

Si vous choisissez "Aucun", les données seront importées avec éventuellement les caractères d'identification de texte. Si vous définissez le caractère d'identification de texte, celui-ci ne sera pas importé.

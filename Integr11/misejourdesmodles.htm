Mise à jour des modèles

Précédent   Suivant

Un modèle contient tous les postes déclarés dans la gestion de production. Ces postes sont organisés en regroupements. Les modèles étant indépendants, un poste peut appartenir à des regroupements différents en fonction du modèle.

Par défaut, il existe 3 modèles :

¨   « Tableau de charge – Ecran » : modèle proposé par défaut pour l’affichage du tableau de charge.

¨   « Tableau de charge – Impression » : modèle proposé par défaut pour l’impression du tableau de charge.

¨   « Planning mural » : modèle proposé par défaut pour le planning mural.

 

A partir de la liste des modèles vous pouvez :

¨   Créer des nouveaux modèles

¨   Supprimer les modèles

¨   Mettre à jour les modèles

¨   Editer les modèles sur imprimantes

Création d’un nouveau modèle

Dans la liste arborescente sélectionnez « Regroupements des postes et informations » et cliquez sur « Nouveau ».

Nom

Nom du modèle. Ce libellé apparaîtra a chaque fois qu’il y aura un choix de modèle à effectuer. Ce nom doit être unique.

Affichage détaillé

Cette option concerne uniquement le tableau de charge.

Si cette option est cochée, le détail complet des fiches de travail est affiché. Sinon, seul l'état d'avancement des fiches est indiqué, les temps et les dates sont masqués.

Basé sur le modèle

Un modèle est toujours créé à partir d’un autre modèle. C’est à dire que le nouveau modèle possédera les mêmes regroupements que le modèle de base. Le lien entre le nouveau modèle et le modèle de base n’existe qu’a la création. Les changements dans le nouveau modèle ne seront pas répercutés dans le modèle de base.

Mise à jour d’un modèle

Dans la liste arborescente sélectionnez le modèle à modifier et cliquez sur « Propriétés ».

La mise à jour d’un modèle est similaire à un ajout. Néanmoins, la zone « Basé sur le modèle » n’est plus visible.

Suppression d’un modèle

Dans la liste arborescente sélectionnez le modèle et cliquez sur « Supprimer »

Il doir rester au moins un modèle (le dernier ne peut pas être supprimé).

Edition des modèles

Pour imprimer un modèle particulier, sélectionnez le modèle dans la liste et cliquez sur « Imprimer ».

Pour imprimer tous les modèle en une seule fois, sélectionnez « Regroupements de postes », et cliquez sur « Imprimer »

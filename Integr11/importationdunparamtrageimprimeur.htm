Importation d'un paramétrage "Imprimeur"

Précédent   Suivant

Pour importer le paramétrage d'un imprimeur, ouvrez les rubriques Calcul des devis, Optiplus, puis double-cliquer sur Initialiser un tarif "Imprimeur".

1. Sélectionnez l'imprimeur dont vous importez le tarif. L'imprimeur doit avoir été créé au préalable.

2. Sélectionnez le tarif à initialiser

3. Cochez l'option "A partir d'un fichier de paramétrage"

4. Saisissez le nom du fichier d'importation ou sélectionnez-le en cliquant sur <…>.

Si le fichier sélectionné n'est pas dans votre devise monétaire le programme vous proposera de convertir tous les coûts dans votre devise.

5. Cliquez sur <Initialiser> pour créer le tarif pour cet imprimeur.

 

Attention : Le paramétrage existant de cet imprimeur pour ce tarif est définitivement effacé et remplacé par le nouveau.

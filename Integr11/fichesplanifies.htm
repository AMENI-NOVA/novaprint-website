Fiches planifiées

Précédent   Suivant

Ce paramétrage permet d'indiquer si l'on veut voir toutes les fiches qui sont planifiées ou seulement celles planifiées pour les x prochains jours.

Ce paramétrage est intéressant pour ceux qui utilisent le planning mural pour ordonnancer des travaux longtemps à l'avance. En effet, il est inutile de voir dans la liste des fiches, des travaux prévus pour le mois prochain.

Pour voir toutes les fiches planifiées il faut cliquer sur "Toutes les fiches".

Sinon , il faut cliquer sur "Jusqu'à" et définir le nombre de jours pendant lesquels les fiches seront visibles.

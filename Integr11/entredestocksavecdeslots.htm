Entrée de stocks avec des lots

Précédent   Suivant

Lorsqu’on procède à l’entrée d’un stock, si la gestion des lots est activée pour ce stock, des lots seront créés pour prendre en compte les matières réceptionnées.

Dans la pièce d’Entrée :

¨   Si une commande existe, sélectionnez la commande qui est livrée,
sinon sélectionnez l’article livré puis le possesseur
ou tapez un code lot fournisseur reconnu (Code IFRA) dans la colonne « Article ».
Remarque : si vous avez entré un code IFRA qui ne défini pas complètement le stock, vous devrez préciser de quel stock il s’agit avant de poursuivre.

¨   Dans la colonne quantité, entrez ou corrigez la quantité totale puis tapez « Entrée » ou cliquez sur le bouton « ... » pour accéder au détail des lots.

¨   Si la matière livrée correspond à un type de code fournisseur reconnu (codage IFRA) entrez les codes des lots dans les lignes du tableau. Si le code défini une quantité, elle viendra s’inscrire dans sa colonne, sinon il vous faudra la saisir.
Si aucun code lot reconnu n’est fourni, entrez seulement les quantités des différents lots dans le tableau. Les codes lots internes seront attribués automatiquement à la validation de l’écriture.

¨   Entrez éventuellement le paquetage de la matière livrée en unité du lot., la position de stockage et l’état du lot.
Au fur et à mesure que vous entrez des lots, le total des quantités est affiché et comparé à la quantité saisie dans l’écriture et l’écart est affiché.

¨   Confirmez les saisies en cliquant le bouton « Ok » (« Ctrl+S ») ou bien annulez avec le bouton « Annuler » (« Ctrl+Z »). La fenêtre des lots est fermée et la ligne d’écriture en cours de saisie est mise à jour.

¨   Entrez d’autres lignes d’écriture pour d’autres stocks qu’ils soient ou non gérés par lot.
Remarque : Si le total des quantités saisies pour les lots ne redonne pas la quantité indiquée dans l’écriture, le message « IMPOSSIBLE : Le détail des lots ne justifie pas la quantité » sera affiché en rouge au bas de la pièce et il sera impossible de quitter la ligne en cours de saisie ni d’enregistrer la pièce. Ouvrez le détail des lots, vérifiez les quantités et enregistrez pour régler le problème.

¨   Confirmez la pièce avec le bouton « Ok »

Notez que les stocks ne seront modifiés et les lots créés que lorsque vous aurez validé la pièce dans les stocks.

Remarque : un lot ne peut être entré qu’une seule fois. Vous ne pouvez pas saisir une entrée pour un code de lot existant déjà dans la Gestion des Stocks.

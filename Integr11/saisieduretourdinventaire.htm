Saisie du retour d’inventaire

Précédent   Suivant

L'écran de retour d'inventaire présente la liste d'inventaire dans un tableau similaire à celui de la fenêtre de préparation.

Vous pouvez effectuer les saisies suivantes :

¨   Les quantités comptées dans la colonne "Quantité".
Un stock non compté a sa colonne "Quantité" vide. Zéro constitue un comptage. Si vous voulez annuler un comptage vous devez mettre la zone à vide en effaçant le nombre précédemment saisi.

¨   Modifier l'unité dans laquelle a été effectué le comptage d'un article, si ce n'est pas l'unité qui se trouvait dans la liste d'inventaire.

¨   Les références positions des stocks. Ces informations seront enregistrées pour les articles pour lesquels des écritures de régularisation doivent être passées.

¨   Avec des lots, entrez une quantité pour un lot non référencé dans la ligne générique du stock. En quittant la ligne, un dialogue vous proposera d’initialiser un nouveau lot. Vous pouvez à cette occasion imprimer une étiquette pour le lot créé.
Si vous avez un code pour ce lot, entrez le dans la colonne « Code lot » puis entrez la quantité constatée.

Ø Vous pouvez changer rapidement l’unité en double cliquant ou en tapant la barre espace dans la case.

Vous pouvez quitter à tout moment la saisie de retour d'inventaire en cliquant sur "Quitter" pour y revenir plus tard. Les saisies effectuées étant enregistrées à tout moment, vous n’avez pas à vous en soucier.

Remarque : En cas de saisie sur plusieurs terminaux en même temps, si votre curseur de saisie dans la liste passe sur une ligne qui a été modifiée par un autre utilisateur, la ligne est affichée en rouge pour vous le signaler. Elle redevient normale dès que vous la quittez.

 

Le bouton "Calculer les écarts" vous permet à tout moment de générer un rapport d'inventaire en comparant les quantités saisies avec les soldes théoriques contenus dans les fiches de stocks. (Voir le paragraphe suivant)

Le rapport ne présente pas les lignes génériques des stocks gérés par lot puisqu’aucune opération ne peut être réalisée sur ceux-ci.

Vous pouvez demander ce rapport aussi souvent que vous le désirez. Vous pouvez choisir de confirmer ce rapport après chaque groupe de stocks ou seulement en fin de comptage pour générer les régularisations.

Les lignes traitées disparaissent alors de l’inventaire.

Rapport d’inventaire

Quand vous cliquez sur le bouton "Calculer les écarts", les colonnes du tableau sont modifiées et présentent en plus de l'écran précédent, les informations suivantes :

¨   Ecart : la différence de quantité entre le comptage et la fiche de stock

¨   Opération : opération à réaliser sur ce stock. Vous pouvez modifier le choix proposé.

·    Régulariser. (par défaut pour les stocks comptés) Si l’écart est nul le stock sera retiré de la liste d'inventaire et sa date d'inventaire sera mise à jour. Si l’écart n’est pas nul une écriture de régularisation sera créée. (voir « Saisie des écritures » dans le chapitre « Opérations sur les stocks »)

·    Compter. (par défaut pour les stocks non comptés) Le comptage n'a pas été saisi ou il faut le refaire. Exemple: comptage improbable.

·    Annuler. Ne pas faire d’opération pour ce stock. L'écart est insignifiant ou la régularisation est complexe et sera faite manuellement dans la saisie des opérations. Le stock sera supprimé de la liste d’inventaire.

Ø Vous pouvez changer rapidement l’opération en double cliquant ou en tapant la barre espace dans la case.

Le bouton "Confirmer" permet de confirmer le rapport d'inventaire :

¨   Un dialogue permet d’indiquer dans quel tampon seront générées les régularisations

¨   Les régularisations demandées y sont créées

¨   Les stocks régularisés ou annulés sont retirés de la liste d'inventaire.

Le bouton "Saisir quantités" permet de revenir en saisie des comptages sans effectuer les opérations.

Le bouton "Imprimer" permet d’imprimer le rapport d’inventaire. La liste est imprimée en se limitant aux stocks pour lesquels une quantité a été saisie et un tableau récapitulatif la complète.

Vous n'êtes pas obligés de confirmer systématiquement le rapport d'inventaire. Vous pouvez revenir à la saisie pour la compléter ou faire des modifications et générer ensuite un nouveau rapport.

Remarque : En cas de saisie sur plusieurs terminaux en même temps, il est préférable de suspendre les saisies et de recharger la liste à l’écran avant d’imprimer le rapport d’inventaire puis de le confirmer. En effet, la confirmation d’un rapport d’inventaire se limite aux opérations affichées à l’écran mais recharge toutes les informations pour les stocks sur lesquels une régularisation va être créée. Si une modification est détectée entre les informations affichées et celles enregistrées, l’opération est suspendue.

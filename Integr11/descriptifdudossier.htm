Descriptif du dossier

Précédent   Suivant

Le descriptif d’un dossier correspond à la description du devis dont a été issue la commande.

Il est généré soit :

¨ à partir des informations techniques du devis

¨ à partir du descriptif du devis si celui-ci a été modifié. Les modifications effectuées dans le texte descriptif d’un devis sont donc transmises à la commande correspondante.

Le descriptif d’un dossier peut être modifié depuis différentes fonctionnalités de l’application :

¨ Transformation du devis en commande : à la fin du traitement en cliquant sur le bouton "Descriptif…"

¨ Liste des commandes : option Descriptif du menu Commandes.

¨ Edition de l’accusé-réception de commande, de la pochette de fabrication, du bon de livraison et de la facture : le descriptif du dossier est une zone repérée dont les modifications seront sauvegardées séparément du reste du document. Le descriptif n’apparaît sur le document que si le modèle utilisé comprend le champ "{Descriptif}".

Lorsque le descriptif d’un dossier est modifié dans n’importe laquelle de ces fonctions, c’est la dernière version modifiée qui est ensuite affichée dans les éditions.

Quand un document comprenant le descriptif est enregistré et imprimé, le descriptif est alors "figé" dans le document. Cela a deux conséquences :

¨ si vous modifiez le descriptif (depuis la fonction "Descriptif" de la liste des commandes), les documents imprimés concernant ce dossier ne seront pas modifiés.

¨ si vous ouvrez un document imprimé, et que vous en modifiez le descriptif, le descriptif du dossier ne sera pas modifié.

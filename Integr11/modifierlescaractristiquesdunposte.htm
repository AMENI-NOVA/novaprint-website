Modifier les caractéristiques d’un poste

Précédent   Suivant

Le menu contextuel dans l’arbre permet de modifier les caractéristiques d’un poste.

Les modifications se font en sélectionnant les caractéristiques dans les boites de listes.

On peut ainsi spécifier si le poste est :

¨   Interne ou sous-traité

¨   Simple ou multiple

¨   Avec ou sans détail des opérations

¨   Et choisir son centre de coût

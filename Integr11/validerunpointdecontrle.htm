Valider un point de contrôle

Précédent   Suivant

Pour valider un point de contrôle, il faut se positionner sur la ligne avec [Flèche haut] et [Flèche bas] et appuyer sur [Entrée].

 

Rappel : les points de contrôle pouvant être validés sont indiqués par les images suivantes :  et .

Sélection de la personne autorisée à valider le contrôle

Le programme propose de sélectionner la personne souhaitant valider le point de contrôle.

¨   Si le point de contrôle doit être validé par "le poste" :

·    Si le poste actuel est un poste "simple" : Tous les employés actuellement affectés au poste sont affichés dans la liste.

·    Si le poste actuel est un poste "multiple" : C'est l'employé travaillant actuellement sur la fiche qui validera le point contrôle. Pas de sélection de personne dans ce cas.

¨   Si le point de contrôle doit être validé par des personnes désignées :
Toutes les personnes autorisées à valider le point de contrôle sont affichées dans la liste de sélection. La personne prévue pour valider le contrôle est sélectionnée par défaut.
Les personnes sont affichées, même si elles ne sont pas affectées actuellement au poste et même si elles ne font pas partie de l'atelier (paramétrage défini dans la fiche employé).
Si la personne sélectionnée a un mot de passe, celui-ci est demandé.

Validation et saisie du commentaire

Après la sélection de la personne qui valide, le programme affiche une fenêtre pour indiquer si le point de contrôle est validé ou non. De plus, il est possible de saisir un commentaire.

Pour indiquer que le point de contrôle est :

¨   Effectué et validé, il faut appuyer sur [F10], ou cliquer sur "Validé [F10]".

¨   Effectué et non-validé, il faut appuyer sur [F12] ou cliquer sur "Non-Validé [F12]".

Pour annuler cette saisie, il faut appuyer sur [Echap.] ou cliquer sur "Annuler".

 

Pour plus d'informations sur les points de contrôles, voir le chapitre "Contrôles".

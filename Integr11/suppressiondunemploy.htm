Suppression d'un employé

Précédent   Suivant

La suppression d'un employé est impossible si :

¨   un temps de présence a été enregistré pour cet employé (vous pouvez le vérifier par la fonction de contrôles et corrections pour les présences et les occupations).

¨   des fiches de travail ont été préparées pour cet employé

Dans ce cas, vous ne pourrez supprimer l'employé que lorsque toutes les opérations le concernant auront été archivées.

Si l’employé à supprimer était enregistré comme commercial, vous avez la possibilité de le remplacer par un autre, dans  toutes les fiches clients pour lesquelles il était enregistré comme commercial. Pour cela, il faut qu’il y ait un autre employé enregistré comme commercial. Le remplacement n’est pas obligatoire. Si vous ne le faites pas, les fiches clients conservent comme commercial celui qui était enregistré, le remplacement pouvant être fait plus tard, au coup par coup.
Dans tous les cas, le commercial supprimé n’apparaît plus dans la liste.

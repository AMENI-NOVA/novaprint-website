Modification

Précédent   Suivant

Le menu Propriétés  ouvre la même fenêtre que celle utilisée pour la création, à la seule différence que celle-ci est remplie avec toutes les informations relatives à ce nom. Vous pouvez vous déplacer de zone en zone en utilisant la tabulation et corriger les informations.

Note : Le code d’un nom ne peut pas être changé une fois l’interlocuteur créé.

Les commandes Précédent et Suivant vous permettent de faire défiler tous les interlocuteurs du même type de façon à modifier les informations sur l’interlocuteur de votre choix. 

Faites Enregistrer pour valider vos modifications, Fermer pour annuler ou fermer la fenêtre.

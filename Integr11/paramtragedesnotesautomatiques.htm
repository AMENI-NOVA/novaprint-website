Paramétrage des notes automatiques

Précédent   Suivant

Chaque type de note automatique est paramétrable. Pour cela, il faut ouvrir la feuille de paramétrage, choisir "Messagerie / Note" puis "Notes automatiques".

 

L'écran de paramétrage vous propose la liste des actions pouvant générer une note automatique. Sous cette liste s'affiche le détail de l'action sélectionnée.

 

 

Une fois l'action sélectionnée, on indique si l'on souhaite ou non la création d'une note.

Ensuite, pour cette action on choisit le type de la note. La liste proposée correspond aux types déjà paramétrés. Pour ajouter de nouveaux types, il faut aller dans la feuille de paramétrage, ouvrir "Messagerie / Note", "Types de notes" et cliquer sur le bouton "Nouveau".
Le type de note est très important, car il permet de trier et de filtrer les notes dans l'historique des notes.

 

La coche "Lier la note au devis" permet de définir si l'on veut que la note soit liée au devis correspondant à l'action. Cela revient à dire si cette action doit apparaître dans l'historique du devis.

 

La coche "Lier la note à la société" permet de définir si l'on veut que la note soit liée à la société concernée par l'action. Cela revient à dire si cette action doit apparaître dans l'historique de cette société.
Pour les sociétés, il est possible d'indiquer le moyen de contact. Il peut être utile pour filtrer l'historique des sociétés. Par exemple, pour l'action "Envoi d'un mail à une société", vous pouvez choisir un moyen de contact se nommant "Envoi d'un e-mail".

 

La coche "Lier la note à la commande" permet de définir si l'on veut que la note soit liée à la commande correspondant à l'action. Cela revient à dire si cette action doit apparaître dans l'historique de la commande.

 

En fonction de l'action sélectionnée, certaines coches "Lier la note à ..." sont grisées. Cela signifie que l'on ne peut pas les décocher. La note de modification du prix d'un devis sera toujours liée au devis concerné.

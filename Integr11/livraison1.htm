Livraison

Précédent   Suivant

La saisie des livraisons s’effectue toujours à partir des versions, même si elles ne sont pas visibles pour le devis.

Le bouton « Ajouter » permet d’afficher une liste des versions « non livrées ». Il est alors possible de sélectionner une ou plusieurs versions à ajouter (une ligne sera crée par version pour le point de livraison s’il y en a plus d’une qui est sélectionnée).

Dans la liste des versions à livrer, positionner le curseur dans la colonne « nom » provoque l’affichage d’un bouton. Cliquer sur ce bouton affichera alors la liste des versions « non livrées » mais attention, dans ce cas, il n’est possible de sélectionner qu’une seule version. La version choisie viendra alors remplacer la version ou se trouve le curseur.

Définition et contraintes sur les regroupements de devis

Précédent   Suivant

¨   Un regroupement est une liste ordonnée de devis liés entre eux.

¨   Un regroupement peut contenir autant de devis que l’on veut.

¨   Les devis d'un regroupement doivent être pour le même client et la même devise.

¨   Un devis ne peut appartenir qu'à un seul regroupement à la fois.

Þ  Pour regrouper plusieurs devis :

1. Dans la liste des devis, sélectionnez le premier devis du regroupement.

2. Cliquez sur Regroupement dans le menu Traitements.

3. La fenêtre de saisie d’un regroupement est affichée.

La liste récapitule les devis regroupés. Elle contient le devis précédemment sélectionné.

4. Pour ajouter un devis cliquez sur Ajouter un devis dans le menu Devis. Une fenêtre contenant les devis du client et calculés dans la même devise sont affichés. Insérer les devis que vous voulez ajouter au regroupement en cliqant sur « Sélectionner » ou en double-cliquant dessus puis fermer la fenêtre de sélection de devis.

5. Les devis regroupés seront édités sur la lettre de confirmation, dans l’ordre de la liste. Pour modifier l’ordre des devis regroupés, sélectionnez le devis dont vous souhaitez modifier l’ordre au sein du regroupement, puis cliquez sur Monter ou Descendre dans le menu Devis.

6. Pour enlever un devis d’un regroupement, sélectionnez-le dans la liste en cliquant sur son numéro, puis cliquez sur Supprimez un devis dans le menu Devis.

7. Lorsque le regroupement est complet, cliquez sur Enregistrer dans le menu Fichier.

¨   Remarque : Si vous ajoutez ou supprimez un devis du regroupement une fois la lettre éditée, si le document n’a pas été imprimé, il sera supprimé. En effet, un document non imprimé conserve un lien avec le ou les objets d’origine.

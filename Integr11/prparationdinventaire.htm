Préparation d'inventaire

Précédent   Suivant

Le but de la préparation est d'éditer une liste des stocks à compter pour qu'il n’y ait plus qu'à la compléter.

Une fenêtre est affichée, comportant  un tableau contenant la liste des stocks à inventorier.

Si une liste d'inventaire existe déjà, le tableau est rempli avec les articles déjà sélectionnés, sinon la fenêtre de sélection des stocks à inventorier est affichée.

Les cellules à fond gris ne sont pas modifiables, celles à fond blanc permettent la saisie.

Remarque : La préparation d’inventaire est une opération exclusive : Aucun autre utilisateur ne peut effectuer d’opération sur l’inventaire pendant qu’une préparation est en cours.

Avec des lots

Si parmi les stocks listés, l’un deux gère les lots, une colonne « Code Lot » est ajoutée entre le possesseur et la position. Dans ce cas, une ligne générique pour le stock est systématiquement affichée plus une ligne par lot actif, c’est à dire présentant une quantité non nulle. (Dans l’exemple ci-dessus, la dernière ligne correspond à un stock qui ne gère pas les lots. La 1ère ligne est la ligne générique d’un stock qui gère les lots et les lignes en dessous présentent les lots actifs de ce stock.

Sélection des stocks à inventorier

Dans la préparation de l'inventaire, la première étape consiste à sélectionner les stocks à inventorier.

Utilisez l'une ou l'autre des fonctions suivantes :

¨   "Nouveau" : pour faire une nouvelle sélection de stocks à inventorier. La liste existante sera écrasée par la nouvelle sélection.

¨   "Compléter" : pour ajouter ou enlever des articles à la liste existante.

Dans les deux cas, une nouvelle fenêtre vous permet de saisir les critères de sélection des stocks à ajouter ou enlever à la liste des stocks à inventorier.

La sélection des stocks pour l’ajout ou la suppression dans l’inventaire se fait sur:

¨   La date de dernier inventaire : pour limiter la sélection aux articles inventoriés avant la date saisie. Laissez la date du jour proposée par défaut pour ne pas mettre de contrainte sur cette date.

¨   La date du dernier mouvement : Pour ne traiter que les articles mouvementés depuis une date donnée. Laissez la date proposée (01/01/1601) pour ne pas appliquer ce critère.

·    Le type des articles : sélectionnez un type particulier dans la liste ou « Tous les types » pour ne pas faire de sélection. Vous pouvez également taper le code du  type.

 

¨   Le code famille :  vous pouvez sélectionner une famille dans la liste ou taper son code en totalité ou en utilisant des caractères jokers '?' et '*' de la manière suivante :

·    Le point d'interrogation remplace un caractère quelconque,

·    L'astérisque remplace une série de caractères.

¨   Le code article : même principe que pour le code famille. La liste des articles correspond à la famille sélectionnée.

¨   Le possesseur parmi les possesseurs connus. Vous pouvez saisir le code du possesseur en totalité ou une partie en utilisant les caractères jokers.

¨   Les erreurs détectées pendant la validation des mouvements de stock. Lorsque les stocks atteignent des soldes négatifs, un compteur (Nombre d’erreurs) est incrémenté. Ce compteur est remis à 0 par les régularisations qui mettent à jour la date d’inventaire du stock..
En cochant cette option, seuls les stocks ayant présentés des soldes négatifs seront listés.

¨   La quantité en stock : Seuls les articles présents dans l’entrepôt seront listés si cette case est cochée.

Exemples

¨   Pour inventorier tous les stocks de la famille Magnoprint (code ‘CMB7MAPM’), sélectionnez la famille correspondante dans la liste ou tapez directement ‘CMB7MAPM’, et laissez la liste 'Article' à 'Tous les articles'.

¨   Pour inventorier les stocks de Graphisoft, sélectionnez ce nom dans la liste des possesseurs.

¨   Pour sélectionner les stocks 'CMB7MAPM' et 'CMB7MAMM', utilisez 'CMB7MA?M*'. Le point d'interrogation remplacera aussi bien le 'P' que le 'M' ou tout autre caractère.

Cliquez sur "Ajouter" pour que les stocks sélectionnés soient ajoutés à votre liste d'inventaire. Si un stock déjà dans le fichier d'inventaire répond aux critères indiqués, il ne sera pas ajouté une deuxième fois.

Cliquez sur "Enlever" pour que les stocks sélectionnés soient supprimés de votre liste d'inventaire.

Supprimer des stocks de la liste d'inventaire

Pour enlever certains stocks de la liste d'inventaire, utilisez l'une ou l'autre des méthodes suivantes :

¨   Si les stocks à enlever peuvent être sélectionnés, soit à partir de leur code famille, soit en utilisant les caractères jokers '?' ou '*', cliquez sur "Compléter", sélectionnez les stocks à enlever, puis cliquez sur "Enlever".

¨   Pour enlever des stocks qui ne peuvent pas être regroupés dans une sélection, cochez la case de la colonne "A supprimer" dans la liste d'inventaire pour chacun des stocks à enlever, puis cliquez sur "Supprimer". NB : La case "A supprimer" peut être cochée en la cliquant ou en tapant la barre d’espace.

Note : Pour un stock géré par lots, vous pouvez sélectionner les lots à supprimer de la même façon. La ligne générique du stock n’a pas à être supprimée. Elle n’est là que pour vous permettre de prendre en compte des lots qui ne figurent pas automatiquement dans la liste.

Tri des stocks à inventorier

La liste d'inventaire peut être triée par :

¨   Les codes articles (et les codes lots): c'est le choix par défaut

¨   La famille de l’article

¨   Les possesseurs

¨   Les références position dans l’entrepôt

Exemple: Si les références position sont construites dans un ordre logique de déplacement dans l’entrepôt, un tri sur ce critère ordonnera les stocks dans la liste dans le même ordre que celui où on les trouvera.

On peut trier soit en sélectionnant dans le menu « Trier sur… » soit en cliquant sur l’entête de la colonne sur laquelle on désire trier.

Atteindre un stock de l’inventaire

Cliquez le bouton « Atteindre code article ou lot » et entrez le code du stock à atteindre (avec éventuellement les caractères jokers * et ?) puis cliquez Ok.

La liste se place sur le premier stock vérifiant ce code.

Vous pouvez entrer un code lot ici. Il sera interprété pour définir le code article correspondant. La recherche s’effectuera alors à la fois sur le code article complet et sur le code du lot.

Ajout d’un lot à l’inventaire

Pour ajouter une ligne pour un lot non listé, entrez son code lot dans la cellule vide de la ligne générique : Une autre ligne générique sera créée pour remplacer celle affectée au lot.

Faites cette manipulation en entrant la quantité dans la saisie du retour d’inventaire s’il s’agit d’un lot pour lequel vous n’avez pas encore de code.

Modification de la liste d'inventaire

Dans la liste d'inventaire à l'écran vous pouvez modifier :

¨   la référence position des stocks (ou des lots), en la saisissant dans le tableau dans la case correspondante

¨   l'unité utilisée pour chaque stock, en cliquant dans la case correspondante, puis en choisissant une autre unité parmi celles proposées dans la liste déroulante ou en double cliquant dans la case ou enfin en tapant la barre espace.

Impression de la liste d'inventaire

Une fois la liste complète et dans l'ordre souhaité, demandez son édition sur imprimante en utilisant le bouton "Imprimer".

Cette liste va maintenant pouvoir être complétée dans l'entrepôt des quantités réellement en stock et des corrections de références positions.

Note : vous pouvez redimensionner les colonnes pour permettre une impression adaptée à vos besoins. Les largeurs des colonnes seront conservées pour le prochain chargement de la liste sur le poste.

Enregistrement de la liste d'inventaire

Les modifications faites dans la liste d’inventaire sont immédiatement enregistrées. Vous n’avez donc pas à vous soucier d’enregistrer la liste.

Remplacer le nom d’une opération gestion de production par un autre

Précédent   Suivant

Cette fonction permet de remplacer de manière globale le nom d’une opération de gestion de production par un autre.

Exemple

¨   L’initialisation automatique des liens a créé l’opération « Mise en route » sur tous les postes de type « OFFSET FEUILLES »

¨   On préférerait qu’elle s’appelle « Avant roulage » pour « OFFSET FEUILLES » et reste « Mise en route » pour les autres types de poste.

¨   Dans notre exemple, il faudrait au préalable créer l’opération « Avant roulage » dans n’importe quel poste « OFFSET FEUILLES » pour qu’il puisse être dans la liste .............

¨   Après le remplacement, il faut penser aussi à réordonnancer l’opération « Avant roulage » qui a été créée dans les postes

La liste « Remplacer l’opération GP» permet de sélectionner le nom de l’opération à remplacer. Elle contient les opérations Gestion de production qui sont dans les lignes correspondant à la liste « Pour le type de poste ou le poste Devis ».

La liste « Par l’opération GP » permet de choisir le nom de remplacement. Elle contient tous les noms d’opérations affichés dans l’arbre.

La liste « Pour le type de poste ou le poste Devis» permet de sélectionner :

¨   Soit : « Tous les postes » devis.

¨   Soit : un type de poste devis.

¨   Soit : un poste devis.

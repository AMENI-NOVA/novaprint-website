Largeurs de colonnes par « défaut ».

Précédent   Suivant

Certaines éditions sont accessibles depuis des listes (telle que la liste des commandes) mais dont les colonnes auront systématiquement une largeur par défaut et doit éventuellement être modifiée à chaque édition.

Pour éviter cela, il  est possible de définir une macro par défaut qui aura pour objet de fournir les largeurs de colonnes à chaque édition ne provenant pas d’une macro.

Pour chaque type d’édition pour laquelle le besoin existe :

¨   Créer une macro dont le nom est #### (les critères ne sont pas importants, seules les largeurs de colonnes seront ultérieurement utilisées) et l’enregistrer.

¨   Lancer l’édition

¨   Modifier les largeurs des colonnes désirées

¨   Fermer l’édition

A chaque fois qu’une édition de ce type sera lancée sans macro, les colonnes auront pour largeur celles de la macro ‘####’.

Pour ne plus utiliser de macro par défaut pour ce type d’édition, il faut supprimer ou renommer la macro ‘####’.

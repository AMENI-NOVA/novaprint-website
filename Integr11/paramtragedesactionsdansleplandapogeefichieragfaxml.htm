Paramétrage des actions dans le plan d’Apogee (fichier Agfa.xml)

Précédent   Suivant

Lors de la création du travail dans Apogee (via le JDF) il est possible d’ajouter automatiquement dans actions dans le plan.

Pour cela, on utilise un fichier AGFA.XML qui est enregistré dans le répertoire de l’application. Ce fichier contient les actions à mettre dans le plan d’Apogee. Le fichier est inséré dans le JDF lors de l’envoi d’une commande.

Ce fichier est définit par un technicien d’AGFA en fonction des souhaits de l’opérateur d’Apogee..

 

Exemple de fichier AGFA.XML :

<?xml version="1.0"?>

<agfa xmlns:agfa="http://www.agfa.com/w3c/gs/jdf/agfaextensions.xsd">

  <agfa:Actions>

     <agfa:Action Flow="Auto"         Location="normalize"    Type="keepresult"/>

     <agfa:Action Flow="Auto"         Location="runlist"  Type="keepresult"/>

     <agfa:Action Flow="Auto"         Location="impose"   Type="keepresult"/>

     <agfa:Action Flow="MainOutput"   Location="render"   Type="keepresult"/>

     <agfa:Action Flow="Auto"         Location="runlist"  Type="archiveresult"/>

     <agfa:Action Flow="Auto"         Location="impose"   Type="archiveresult"/>

     <agfa:Action Flow="ImpositionProof"                  Location="trap"  Type="prooffilter_all"/>

     <agfa:Action Flow="PageProof"    Location="trap"     Type="prooffilter_all"/>

  </agfa:Actions>

</agfa>

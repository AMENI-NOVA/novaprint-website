Envoyer une note à quelqu'un

Précédent   Suivant

Lorsque l'on envoie une note à une autre personne on ne parle plus de note, mais de message.

Pour envoyer un message à une autre personne vous devez :

1. Saisir l'objet et le texte du message

2. Définir les destinataires.

Pour cela utilisez la commande "Destinataires…" du menu "Outils" ou saisissez directement le nom des destinataires dans la zone de saisie "Destinataires". Voir le paragraphe "Choix des destinataires".

3. Choisir la méthode de notification.
Ce choix s'effectue depuis le menu "Destinataire" puis "Notification des destinataires"

·    Ne pas notifier : Le destinataire ne sera pas informé de l'arrivée d'un nouveau message. Seule l'icône du programme de notification indiquera qu'il y a un message.

·    Notifier (Choix par défaut): Un message sonore suivi de l'affichage de l'écran de notification indiquera qu'il y a un nouveau message. Seuls le nom de l'expéditeur, l'objet et la date du message apparaîtront. De plus, l'écran de notification disparaît au bout d'un certain temps (paramétrable).

·    Notifier et afficher la note: Comme le mode précédent, mais le message est affiché et il ne s'efface que si le destinataire le demande. Avec ce mode, vous êtes sûr que le destinataire lit le message dès qu'il est notifié.
Pour plus d'informations sur la notification, consultez le chapitre "Notification des nouveaux messages : Graphisoft Notify".

 

Il est possible d'envoyer des messages à tous les utilisateurs référencés dans le logiciel, mais aussi à toutes autres personnes :

¨   Les employés                       :        message interne et e-mail.

¨   Les sociétés et interlocuteurs           :        e-mail seulement

¨   Personnes non référencées              :        e-mail seulement

 

Avant d'envoyer le message, le programme va vérifier que tous les destinataires contactés par e-mail ont une adresse Internet valide. Si ce n'est pas le cas, il vous sera demandé de la ressaisir. Pour être valide, une adresse e-mail doit au moins contenir les caractères "." et "@".

 

Cas où GS CRM utilise un programme de messagerie pour envoyer les e-mails.
Voir les préférences de messagerie.

Si l'adresse du destinataire n'est pas renseignée, le programme va interroger le carnet d'adresses de votre messagerie Internet. Si le destinataire est trouvé dans ce carnet d'adresses, la messagerie Graphisoft va utiliser son adresse et vous demandera s'il faut l'enregistrer dans les fichiers de Graphisoft.
Si vous avez l'habitude de contacter vos clients ou fournisseurs par e-mail, depuis votre messagerie Internet, vous n'êtes donc pas obligé de ressaisir leur adresse dans Graphisoft : elles seront récupérées automatiquement.

 

Type des destinataires

Il existe trois types de destinataires :

¨   Normal : correspond au(x) destinataire(s) concernée directement par la note. C'est le type par défaut.

¨   Copie conforme (ou Cc) : permet de mettre en copie des personnes non concernées directement par la note et que vous souhaitez mettre au courant du contenu de la note ou bien du fait même d'avoir envoyé la note au(x) destinataire(s).

¨   Copie conforme invisible (ou Cci): permet de mettre en copie des personnes sans que quiconque des destinataires ou bien des destinataires cachés ne voient qu'ils sont en copie. Il est généralement recommandé lors de l'envoi de mail à de nombreuses personnes de les mettre en Copie Conforme Invisible afin d'éviter qu'un des destinataires ne réponde à tout le monde ou bien ne constitue une liste d'adresses.

Choix des destinataires

Il existe trois méthodes pour choisir les destinataires d'une note.

¨   La saisie par groupe : vous informez des groupes de personnes : utilisateurs administratifs, utilisateurs atelier, ... . Cette méthode  ne vous permet pas de définir tous les paramètres d'un destinataire (actions liées, adresse e-mail, …).

¨   La saisie "express" : vous tapez quelques lettres et le programme vous propose tous les destinataires commençant par celles-ci. Cette méthode est très rapide d'utilisation, mais elle ne vous permet pas de définir tous les paramètres d'un destinataire (actions liées, adresse e-mail, …).

¨   La saisie "classique" : vous choisissez les destinataires dans une liste. Pour les e-mails, vous pouvez modifier l'adresse du destinataire. Pour les messages internes, vous pouvez lier une action ou demander une réponse obligatoire.

Ces méthodes sont complémentaires. Vous choisissez les destinataires avec la saisie "express" et vous pouvez modifier les paramètres de certains destinataires avec la saisie "classique".

 

Choix des destinataires : Saisie par groupes

Cette saisie vous permet d'informer des groupes de personnes sans avoir à les sélectionner nominativement ni même savoir exactement qui elles sont.

Attention, tous les destinataires sélectionnés par cette méthode sont exclusivement de type "Normal".

Les groupes sont les suivants :

¨   Utilisateurs (les destinataires de ce groupe seront informés par une note interne) :

·    Tous les utilisateurs : Tous les utilisateurs de Graphisoft seront sélectionnés pour être destinataires du message.

·    Tous les administratifs : Seuls les utilisateurs de type "Classique" seront sélectionnés.

·    Tous les commerciaux : Seuls les utilisateurs de type "Classique" et "Commercial ou vendeur" seront sélectionnés.

·    Tous les utilisateurs atelier : Seuls les utilisateurs ayant le droit d'accéder au programme atelier seront sélectionnés.

¨   Sociétés liées à la note

·    Les sociétés liées à la note : toutes les sociétés liées à la note seront destinataires de la note par e-mail.

·    Les commerciaux des sociétés : Les commerciaux des sociétés liées à la note seront destinataires et informés par une note interne.

·    Les commerciaux du secteur : Les commerciaux affectés aux secteurs des sociétés liées à la note seront destinataires et informés par une note interne.

·    Les droits du secteur : Les commerciaux ayant des droits sur les secteurs des sociétés liées à la note seront destinataires et informés par une note interne.

Les secteurs concernés sont les suivants :

·    Le pays et le secteur géographique auxquels appartiennent les sociétés liées à la note

·    Le secteur d'activité des sociétés

·    Les produits proposés par les sociétés

Toutes les personnes ayant droit ou étant affectées sur au moins un des secteurs à laquelle appartient la société concernée, seront destinataires de la note. Voir  "Paramétrage de GS CRM" pour plus d'informations.

 

Pour choisir ces groupes vous pouvez utiliser le menu "Destinataires" et choisir le ou les groupes. La "Saisie classique" permet aussi de choisir les groupes de destinataires.

 

Choix des destinataires : Saisie express

Cette saisie permet de sélectionner très rapidement les destinataires du message. Vous commencez à saisir les premières lettres du destinataire et le programme vous propose ceux qui correspondent à ces lettres.

 

Pour cela, positionnez-vous dans la zone "Destinataires", "Cc" ou "Cci". Si les zones "Cc" ou "Cci" ne sont pas affichées, utilisez la commande "Afficher Cc" ou Afficher Cci" du menu "Destinataires".

 

Dès la première lettre saisie une liste vous propose les employés correspondants. A partir de 3 lettres, le programme va aussi rechercher les sociétés, les interlocuteurs (des sociétés), les fournisseurs, les sous-traitants et les possesseurs correspondant aux lettres saisies. Ensuite, il affiche le résultat de la recherche dans une liste sous la zone de saisie.

Pour choisir un destinataire dans cette liste il faut utiliser les touches [Flèche haut] et [Flèche bas] puis [Entrée].

 

La liste de sélection des destinataires contient deux colonnes. Elles sont remplies de la façon suivante :

+ "(employé)"

Nom et/ou prénom l'employé

Les employés qui ont une adresse e-mail apparaissent deux fois : une fois pour les messages internes, une autre fois pour les e-mails.

 

Si l'adresse e-mail d'un destinataire n'est pas renseignée, le signe @ apparaît. Il n'est pas nécessaire d'utiliser la saisie "classique" des destinataires pour mettre à jour cette adresse. En effet, lors de l'envoi du message, le programme vous demande de la saisir ou essaye de la récupérer dans le carnet d'adresses de votre messagerie Internet.

 

Si le texte saisi ne correspond à rien, une seule ligne s'affiche avec "Hors fichier". Dans ce cas, le programme considère qu'il s'agit d'une adresse e-mail. Dans la "saisie classique" ce destinataire est affiché dans l'onglet "Autre".

 

Pour saisir plusieurs destinataires, vous devez taper un point virgule entre chaque nom.

 

Choix des destinataires : Saisie classique

Cette saisie est accessible depuis la commande "Destinataires" du menu "Outils" ou depuis le bouton suivant : .

La partie "Sélection automatique" permet de choisir des groupes de destinataires. Voir le paragraphe " Choix des destinataires : Saisie par groupe".

 

La partie "Sélection manuelle" regroupe les type de destinataires (employés, clients, fournisseurs, …) par onglet. Le nombre de destinataires sélectionnés par onglet est indiqué dans le titre de celui-ci.

Si vous avez utilisé auparavant la saisie express, la liste des destinataires contient votre sélection de destinataires.

Employés

¨   Pour sélectionner un employé, il faut le cocher en cliquant dessus. Par défaut, le message envoyé à cet employé sera un message interne, sans réponse ni action à effectuer.

¨   Pour se déplacer rapidement dans la liste, vous pouvez taper les premières lettres du nom.

¨   Pour retirer un employé de la liste des destinataires, il suffit de le décocher.

¨   La partie "Détails pour XXX" permet de définir le type de destinataire et la méthode d'envoi du message pour l'employé sélectionné : soit un message interne, soit un e-mail, soit les deux à la fois.

·    Type
Indiquez ici le type de destinataire souhaité : Normal, Cc ou Cci.
Cf. "Type des destinataires".

·    Message interne
Le destinataire recevra un message via Graphisoft.

§  Action à effectuer :
Vous pouvez lier une action au message en la choisissant dans la liste "Action". Les actions proposées sont paramétrables (voir "Actions" dans "Paramétrages de GS CRM "). Le choix "(pas d'action)" signifie qu'aucune action ne sera liée au message.
Vous n'êtes pas limité par la liste : vous pouvez saisir une action n'existant pas. Celle-ci ne sera prise en compte que pour ce message. Si vous souhaitez créer une action qui sera utilisée régulièrement, il faut la définir dans le paramétrage (voir "Actions" dans "Paramétrages de GS CRM").

§  "Pas de date" / "Pour le" :
Date souhaitée pour l'exécution de l'action. Certaines actions sont paramétrées avec la saisie obligatoire d'une date. Dans ce cas, le choix "Pas de date" n'est pas disponible.

§  Notifier à partir du :
Si une date est saisie, le destinataire voit le message à partir de cette date. Cela évite d'encombrer sa boîte de réception avec des messages, pour l'instant, inutiles.
Si aucune date n'est indiquée, le message est visible immédiatement après l'envoi.

§  Réponse souhaitée :
Si vous cochez cette case, le destinataire est informé que vous souhaitez une réponse. Tant qu'il ne vous aura pas répondu, le message apparaîtra dans sa liste de messages actifs.

·    E-mail :

Le message sera envoyé par e-mail. La zone "Adresse e-mail" permet de saisir l'adresse du destinataire. Elle s'affiche par défaut avec l'adresse enregistrée dans les fichiers. Il est possible de la modifier. Au moment de l'envoi du message, le programme vous demandera si vous voulez mettre à jour les fichiers avec la nouvelle adresse.

Sociétés

Pour sélectionner une société, il suffit de taper les premières lettres de celle-ci. Le programme complète automatiquement la saisie avec la première société commençant par ces lettres. Il est aussi possible de choisir la société dans une liste en utilisant le bouton "…" à droite de la zone de saisie.

L'interlocuteur privilégié de la société s'affiche dans la liste "Interlocuteur". Vous pouvez choisir un autre interlocuteur en le sélectionnant dans cette liste. Si vous choisissez "(Non défini)", le message sera envoyé à la société et non à un interlocuteur de celle-ci.

Pour ajouter la société ou l'interlocuteur dans la liste des destinataires, cliquez sur :

·      "Ajouter", pour ajouter cette société dans la liste "Normal",

·      "Cc", pour ajouter la société dans la liste "Copie conforme",

·      "Cci", pour ajouter la société dans la liste "Copie conforme invisible".

Pour le retirer, il faut le sélectionner dans la liste et cliquer sur "Supprimer".

Le bouton "E-mail" permet de modifier l'adresse e-mail.

 

Autres

Correspond à un destinataire qui ne figure par dans les fichiers Graphiplus. Vous devrez saisir le nom de celui-ci ainsi que son adresse e-mail.

Pour ajouter le destinataire, cliquez sur :

·      "Ajouter", pour ajouter ce destinataire dans la liste "Normal"

·      "Cc", pour ajouter ce destinataire dans la liste "Copie conforme",

·      "Cci", pour ajouter ce destinataire dans la liste "Copie conforme invisible".

Pour le retirer, il faut le sélectionner dans la liste et cliquer sur "Supprimer".

Le bouton "E-mail" permet de modifier l'adresse e-mail.

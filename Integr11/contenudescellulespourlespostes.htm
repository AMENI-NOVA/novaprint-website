Contenu des cellules pour les postes

Précédent   Suivant

Le contenu de cellules dépend du paramétrage choisi dans le modèle de regroupements de postes. Deux modes sont possibles :

¨   Simplifié : seul l'état d'avancement des travaux est affiché

¨   Détaillé : les temps restant sont affichés pour les fiches internes ainsi que les dates des fiches sous-traitées

Pour changer de mode, il faut choisir un modèle de regroupement de postes n'ayant pas le même mode d'affichage ("Modèle de regroupement de postes" dans le menu "Postes")

 

Pour plus d'informations sur le paramétrage des modèles de regroupements de postes voir la section "Regroupement de postes et d’informations" et chapitre "Paramétrage de gestion de production".

Mode de rappel d'une note

Précédent   Suivant

Le mode de rappel se définit quand on lie une note à un élément. Les modes sont les suivants :

¨   Pas de rappel : La note n'est pas prise en compte par la fonction de rappel.

¨   Discret : Seul le message de rappel clignote pour indiquer la note.

¨   Une fois : (Mode par défaut) La note s'affiche complètement une fois par utilisateur. Une fois lue, seul le message de rappel indique la note. Cela permet l'auteur d'être sûr que la note est lue au moins une fois.

¨   Jusqu'à archivage : La note est systématiquement affichée, même si l'utilisateur l'a déjà lue. Pour ne plus afficher la note dans le rappel, il faut cliquer sur le bouton "Archiver la note".
Si vous archivez la note, plus personne ne verra la note dans le rappel.

¨   Toujours : La note est systématiquement affichée. Ce mode permet d'afficher les informations très importantes qui doivent être rappelées à chaque fois.

 

Si une note est liée à plusieurs éléments (par exemple : un devis et la société du devis), il est possible de choisir un mode de rappel différent pour chaque élément.

 

De plus, il est possible de choisir de n’afficher le rappel d’une note que dans l’écran de facturation. Pour cela, il faut lors de la saisie du rappel cocher la case « Rappel seulement à la facturation ».

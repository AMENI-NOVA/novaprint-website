Utilisation des assembleuses feuilles pour le calcul des devis

Précédent   Suivant

Ce type de machine peut être utilisé pour calculer :

¨   l'assemblage des cahiers ou des feuilles d'une brochure,

¨   l'assemblage des fiches et amalgame,

¨   l'assemblage des liasses.

Utilisation des assembleuses feuilles pour les brochures

Pour que Graphisoft envisage le calcul de l'assemblage des brochures sur une assembleuse feuilles, vous devez créer un chemin de façonnage dans lequel vous intégrez l'étape "Assemblage feuilles", soit pour toutes les assembleuses feuilles de votre fichier, ou pour une sélection d'assembleuses (voir le paragraphe "Chemins et types de façonnage", plus loin dans ce chapitre).

Graphisoft peut calculer l'assemblage des brochures sur assembleuse feuilles, suivant deux techniques :

¨   Assemblage, pliage et piquage de cahiers de 4 pages.
Cette solution ne pourra être envisagée que pour les assembleuses pour lesquelles vous aurez coché "Pliage + piquage" (voir "Caractéristiques des assembleuses feuilles", un peu plus loin).
D'autre part, le chemin de façonnage correspondant devra appartenir à un type de façonnage pour lequel le nombre de pages du plus petit cahier est 4 (voir le paragraphe "Paramétrage des types de façonnage", plus loin dans ce chapitre).

¨   Assemblage en feuilles :
Cette solution sera envisagée si le chemin de façonnage correspondant appartient à un type de façonnage pour lequel le nombre de pages du plus petit cahier est 2 (voir le paragraphe "Paramétrage des types de façonnage", plus loin dans ce chapitre).

Utilisation des assembleuses feuilles pour les devis d'affiches/fiches

L'assemblage des fiches est calculé pour tous les modèles de tous les éléments du devis lorsque « Assemblage des fiches » est coché dans l’entête du devis.

Quantités différentes par élément

Si les quantités sont différentes par élément, les versions sont automatiquement déduites par le programme et l’assemblage est calculé séparément pour chaque version à assembler qui comporte au moins 2 éléments.

NB : dans la saisie de la quantité d’un élément du devis, il est possible de distinguer la quantité à imprimer de la quantité à assembler. Cette distinction permet de déduire les versions pour lesquelles l’assemblage doit être calculé.

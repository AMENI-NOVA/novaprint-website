Relancer une édition.

Précédent   Suivant

Cette fonctionnalité agit sur l'édition affichée dans le classeur (la feuille active). Elle permet :

¨   d'afficher les paramètres utilisés pour effectuer cette édition.

¨   d'utiliser les mêmes paramètres pour refaire cette édition.

¨   d'afficher les paramètres utilisés, les modifier pour faire une nouvelle édition dérivée de l’édition courante.

 

Pour relancer une édition :

¨   Cliquez sur le bouton " Relancer "   :

¨   Cochez ou pas la case à cocher 'Modifier les paramètres' pour afficher ou modifier ceux-ci.

¨   Validez la relance, bouton Ok ou Visualiser si on est passé pas la feuille de saisie des paramètres.

 

Si vous ne demandez pas à modifier les paramètres, l’'édition relancée et l’onglet courant est remplacé, sinon un nouvel onglet est ajouté à la fin du classeur.

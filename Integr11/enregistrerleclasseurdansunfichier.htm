Enregistrer le classeur dans un fichier.

Précédent   Suivant

Pour enregistrer le classeur (toutes les feuilles) dans un fichier lisible par Graphisoft, un tableur, un traitement de texte ou un explorateur Internet :

¨   Cliquez sur le bouton « Enregistrer »  pour faire apparaître la boîte de dialogue d’enregistrement :

¨   Eventuellement, changez le dossier et le nom du fichier de destination.

¨   Choisissez le type / le format du fichier.

Type

Description

Classeur Graphisoft (*.GCL)

Format Graphisoft. Toutes les feuilles du classeur sont sauvées dans le fichier.

C’est le seul format de fichier qui peut être relu (rechargé) par Graphiplus.

Classeur Excel 5 or 7 (*.XLS)

Format Microsoft Excel 5 ou 7. Toutes les feuilles du classeur sont sauvées dans un fichier classeur Excel.

Excel 4 (*.XLS)

Format Microsoft Excel 4. Seule la feuille courante est sauvée dans le fichier.

Texte avec tabulations (*.TXT)

Format texte. Seule la feuille courante est sauvée dans le fichier. Les cellules sont séparées par des tabulations (code ASCII 9) et les lignes par des retours chariot (code ASCII 13 + 10).

Texte avec tabulations – données seules (*.TXT)

Idem Texte avec tabulations.

Internet HTML (*.HTM)

Format HTML, c’est à dire lisible par un explorateur Internet (Internet explorer, Netscape Navigator, etc.). Seule la feuille courante est sauvée dans le fichier.

Internet HTML – données seules (*.HTM)

Idem Internet HTLM, mais les attributs de mise en forme des cellules ne sont pas sauvés (gras, souligné, alignement, etc.).

 

¨   Cliquez sur le bouton « Enregistrer » pour enregistrer ou « Annuler » pour annuler l’enregistrement.

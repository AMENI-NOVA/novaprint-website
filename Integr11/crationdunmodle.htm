Création d'un modèle

Précédent   Suivant

Pour ajouter un modèle, sélectionnez le type d’édition du modèle dans l’arbre ou un modèle de même type, cliquez sur le bouton  et concevez-le dans la fenêtre de paramétrage du modèle (voir ci-après).

Création d'un modèle à partir d’un modèle de référence

Les modèle de références sont des modèles crées par Graphisoft et non modifiables. Ils servent de base de départ pour créer vos propres modèles.

Pour créer un modèle à partir d’un modèle de référence :

¨   Utilisez la commande du menu « Fichier » « Ouvrir » ou cliquez sur le bouton

¨   La fenêtre présentant les modèles déjà enregistrées s'affiche.

¨   Cliquez sur le bouton « Afficher les modèles de références ».

Le fond de l’arbre s’affiche en gris pour marquer la différence avec les modèles déjà existants.

¨   Sectionnez un modèle de référence et cliquez sur le bouton « OK ». La sélection peut aussi s’effectuer en double cliquant sur le nom du modèle.

¨   Modifiez éventuellement le contenu et enregistrez-le.

SSL

Indique si le serveur requière une connexion sécurisée SSL.

Pour les messages sortants, plusieurs choix sont possibles :

·      Non : Pas de connexion sécurisée

·      Auto : Lors de l’envoi de chaque message c’est GSCRM qui détermine le meilleur choix mais cela prend quelques secondes supplémentaires. Il essai SSL, puis TLS.

·      SSL : La connexion sécurisée est de type SSL

·      TLS : La connexion sécurisée et de type TLS

 

 

Attention, en général, le port utilisé par le serveur n’est pas le même avec et sans SSL.

Habituellement, les ports utilisés sont les suivants, mais cela n’a rien d’obligatoire :

Sans SSL : Pop = 110 et Smtp = 25

Avec SSL : Pop = 995 et Smtp = 465

Gestion du serveur
Délai minimum entre deux récupérations automatiques

Spécifie un délai minimum entre deux récupérations automatiques de compte.

La récupération des e-mails peut-être activée sur plusieurs ordinateurs en même temps. Ce qui provoque beaucoup travail pour le serveur de messagerie. Certains supportent mal d'être trop sollicités et peuvent vous refuser l'accès.

Un délai de 5 minutes semble raisonnable. En dessous de 2 minutes, le serveur risque de vous prendre pour un "virus" et vous refuser l'accès.

Conserver une copie des messages sur le serveur

Spécifie si une copie de tous les messages reçus doit être stockée sur votre serveur.

Si vous désactivez cette case à cocher, les messages entrants sont supprimés de votre serveur une fois que vous les avez téléchargés dans GS CRM.

Supprimer du serveur après

Spécifie si les messages électroniques restants doivent être supprimés du serveur après le nombre de jours indiqué.

Certains fournisseurs de services Internet limitent le nombre de messages que vous pouvez stocker sur le serveur ou n'autorisent pas le stockage des messages sur le serveur.

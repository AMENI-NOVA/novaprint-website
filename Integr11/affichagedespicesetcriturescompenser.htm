Affichage des pièces et écritures à compenser

Précédent   Suivant

Le titre de la fenêtre indique s’il s’agit d’une sélection multiple ou si on sélectionne une compensation pour une écriture déjà existante auquel cas une seule écriture compensée est possible.

Les pièces s’affichent dans un tableau. Une case à cocher permet de sélectionner les pièces. Un signe ‘+’ permet de développer la pièce pour afficher les écritures qui la composent, le signe ‘-‘ permet de réduire la pièce à une seule ligne.

Les lignes de pièce affichent :

¨   La référence de la pièce (Type, numéro et date)

¨   Le libellé de la pièce

Les lignes d’écriture affichent :

¨   Le code du stock concerné

¨   La quantité

¨   Le dossier concerné

Validez la sélection en cliquant le bouton « Ok » ou annulez avec « Annuler »

Un menu contextuel (cliquez bouton droit dans la liste) permet :

¨   D’atteindre la première écriture sélectionnée.

¨   De développer toutes les pièces.

¨   De réduire toutes les pièces.

¨   De désélectionner toutes les écritures.

Macros : enregistrement des options

Précédent   Suivant

Il est possible d'enregistrer les options d'un publipostage dans une macro. Les macros sont affichées dans la liste des sociétés et permettent un accès rapide à l'édition d'un publipostage (ou l'envoi d'un e-mailing).

 

Les macros sont enregistrées dans la base de registre de l'ordinateur.

Cela signifie, que si un utilisateur enregistre une macro sur un ordinateur, cette macro n'est pas accessible depuis un autre ordinateur.

 

A chaque fois qu'un publipostage est lancé, le programme crée systématiquement une macro du nom "(Publipostage précédent)" afin de ne pas perdre les options choisies.

Charger une macro

Ouvrez la liste "Macros", sélectionnez le nom de la macro souhaitée et cliquez sur le bouton .

L'écran est mis à jour avec les paramètres enregistrés dans la macro.

Enregistrer une macro

Saisissez (ou sélectionnez) le nom de la macro dans la liste "Macros" et cliquez sur le bouton .

Si le nom saisi correspond à une macro existante, celle-ci est modifiée. Si le nom saisi n'existe pas, la macro est ajoutée.

Supprimer une macro

Sélectionnez le nom de la macro dans la liste "Macros" et cliquez sur le bouton .

Notifications à l’imprimeur

Précédent   Suivant

Un message peut être envoyé à l’imprimeur dans les cas suivants :

¨   Enregistrement d’un devis en attente (devis non calculé)

¨   Demande de commande

Le message peut être envoyé par email ou par une note interne (Print2B Windows uniquement).

Par e-mail

Pour recevoir une notification par e-mail, cochez la case "Par e-mail".

NB : la notification par email fonctionne avec Web2Print Pro ou Print2B.

Adresses e-mails

Indiquez les adresses e-mails à utiliser lors d'une demande de commandes ou de l'enregistrement d'un devis en attente. Les deux adresses peuvent être identiques.

Compte de messagerie

Spécifiez le compte de messagerie contenant l’adresse du serveur de messagerie à utiliser pour l’envoi des emails. (voir « Paramétrages des comptes de messagerie (Pop & Smtp) » dans le chapitre « Messagerie »).

Remarques :

¨   vous pouvez utiliser un compte de messagerie existant

¨   l’accès au paramétrage des comptes de messagerie ne nécessite pas le module « GS CRM ».

¨   l’adresse mail du compte de messagerie doit impérativement être remplie.

Par une note interne ...

Pour être averti par une note interne, cochez la case "Par une note interne ...".

Par type de notification, vous devez indiquer le nom de l'utilisateur que sera averti.

NB : la notification par note interne n’est possible que depuis Print2B.

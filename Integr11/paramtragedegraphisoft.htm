Paramétrage de Graphisoft

Précédent   Suivant

EmaPlan étant un module indépendant, il utilise sa propre base de données.

Ce chapitre va décrire comment paramétrer Graphisoft pour communiquer avec EmaPlan.

L’installation d’EmaPlan et de ces différents composants (EmaPlan, MySQL, PHP, WebService) est décrite dans un document spécifique.

 

Le paramétrage d’EmaPlan est accessible depuis l’arbre de paramétrage dans « Gestion de production », puis « Paramétrage généraux » ou directement dans la fenêtre EmaPlan (menu « Graphisoft », « Gestion de production » puis « EmaPlan ».

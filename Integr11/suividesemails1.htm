Suivi des e-mails

Précédent   Suivant

Le paramétrage du suivi des e-mails permet de définir le comportement par défaut des options de suivi (confirmation de lecture et preuve de dépôt) lors de la création d’un e-mail.

 

Les choix sont toujours les mêmes quelque soit l’opération :

¨   Toujours : Effectuer systématiquement l’opération ;

¨   Jamais : Ignorer systématiquement cette opération ;

¨   Demander : Demander à l’utilisateur ce qu’il souhaite faire.

 

Dans tous les cas, les choix faits ici sont des choix par défauts. Lors de la création d’un e-mail, il est toujours possible de faire un choix différent. Pour cela, on utilise le menu « Suivi des e-mails » dans l’écran de création d’une note.

Dans ce menu, les choix « Par défaut » reviennent à utiliser ce qui est défini dans le paramétrage.

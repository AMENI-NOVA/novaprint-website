Créer un service, un poste, une opération côté gestion de production

Précédent   Suivant

Pour créer un service, il faut se positionner sur le service <Aucun>.

Pour créer un poste, il faut se positionner sur le service qui doit contenir ce poste.

Pour créer une opération, il faut se positionner sur le poste qui doit contenir cette opération, ou sur une des opérations du poste.

 

1. Positionnez-vous au bon endroit (comme décrit ci-dessus).

2. Choisissez l’action souhaitée :

·    Dans le menu, <Arbre> < Nouveau service / Nouveau poste / Nouvelle opération>.

·    Ou par le menu contextuel (clic droit à la souris) sur le service ou le poste ou l’opération < Nouveau service / Nouveau poste / Nouvelle opération>.

3. Saisissez le nouveau nom dans la boite de dialogue.

4. Confirmez la création.

Renvoyer

Précédent   Suivant

"Renvoyer" dans le menu "Notes" ou le menu contextuel.

 

Cette fonction permet de renvoyer la note sélectionnée (seulement si la note n'est pas un e-mail reçu).

La fenêtre de saisie d'un nouveau message apparaît pré-remplie :

¨   Le nom du destinataire correspondant au message d'origine.

¨   L'objet contient "Renvoyé :" suivi de l'objet du message d'origine.

¨   Le texte contient la mention "Note d'origine" suivi du texte de celle-ci.

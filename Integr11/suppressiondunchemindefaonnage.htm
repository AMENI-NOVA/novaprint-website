Suppression d'un chemin de façonnage

Précédent   Suivant

Pour supprimer un chemin de façonnage, vous devez le supprimer de chaque type de façonnage auquel il appartient. Ouvrez le type de façonnage auquel appartient le chemin que vous voulez supprimer, puis choisissez Supprimer dans la barre d'outils ou dans le menu contextuel.

Si vous supprimez un chemin qui n'appartient qu'à un seul type de façonnage, un message vous propose de le supprimer définitivement.

¨   Si vous répondez Oui, le chemin sera effacé et n'apparaîtra plus dans la liste des chemins possibles de la mise à jour des types de façonnage.

¨   Si vous répondez Non, le chemin sera conservé et pourra plus tard être à nouveau lié à un type de façonnage.

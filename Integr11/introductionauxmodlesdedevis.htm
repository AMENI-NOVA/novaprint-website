Introduction aux modèles de devis

Précédent   Suivant

Le bouton "Modèles de devis" vous permet d'enregistrer le devis en cours de saisie dans la liste des modèles de devis.

Les modèles de devis présentent les avantages suivants :

¨   Vous pouvez constituer une liste des devis les plus couramment saisis, triée selon vos types de produit et rapidement accessibles.

¨   Vous pouvez enregistrer un modèle de devis, même si toutes les zones de saisies ne sont pas remplies. Cela vous permet d'initialiser certaines informations pour éviter d'avoir à les ressaisir systématiquement ou de les oublier.

¨   Comme pour les devis enregistrés normalement, les choix effectués pour l'optimisation sont enregistrés dans les modèles de devis. Vous pouvez ainsi créer des modèles de devis correspondant à des processus de fabrication bien définis.

Lorsque vous rappelez un modèles de devis, l'écran de saisie est initialisé avec les valeurs enregistrées dans le modèle, exactement comme lorsque vous rappelez n'importe quel devis.

Seuls les utilisateurs possédant le droit d'accès au paramétrage, peuvent enregistrer des modèles de devis.

Les modèles de devis enregistrés sont accessibles par l'intermédiaire :

¨   du menu "Modèles de devis" depuis la liste des devis,

¨   du menu "Modèles de devis" depuis la liste des sociétés : dans ce cas, seuls les modèles de devis concernant la société sélectionnée apparaissent dans le menu.

¨   de la liste des modèles de devis

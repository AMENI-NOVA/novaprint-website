 Nombre de pistes

Trous Caroll, Crimp-lock, Encollage.

Sert à limiter le nombre de poses sens laize si un de ces façonnages est réalisé sur la machine. On indique ici le nombre maximum de poses sens laize qui pourra être travaillé (la perforation caroll est réalisée automatiquement par le logiciel si plusieurs feuillets sont à assembler même si le produit fini n'en possède pas).

Nombre de postes à fonction multiple

Définit le nombre d'opérations qui pourront être effectuées en même temps sur la machine. Ces opérations sont sélectionnées parmi les opérations particulières de façonnage (voir fichier façonnage). Jusqu'à 10 opérations peuvent être indiquées comme possibles sur la machine.

Type d’outil 

Type des outils de découpe qui peuvent être utilisés sur la machine. Voir le paragraphe « Types d’outils continu ».

Statistiques devis Optiplus

Précédent   Suivant

Les statistiques sur les devis permettent de connaître le positionnement d'un imprimeur par rapport aux autres pour les devis pour lesquels il a été consulté.

Pour accéder à l'écran de statistiques, sélectionnez Statistiques devis imprimeurs dans le menu Utilitaires de la liste des devis.

Pour consulter les statistiques d'un imprimeur, sélectionnez-le grâce à la liste déroulante, puis cliquez sur le bouton <Statistiques>.

Les statistiques sont calculées pour les douze derniers mois.

 

Nombre dans le top 15

Nombre de devis pour lesquels l'imprimeur a été consulté et pour lesquels le total de la solution se trouvait dans les 15 meilleures.

% dans le top 15

Pourcentage de devis dont la solution se trouvait dans les 15 meilleures par rapport au nombre de devis pour lesquels l'imprimeur a été consulté.

Rang moyen obtenu

Rang moyen obtenu par rapport aux imprimeurs pour les devis pour lesquels l'imprimeur a été consulté.

Classement obtenu

Classement du rang moyen par rapport aux autres imprimeurs.

Types de devis, Quantité, Nombre de pages

Classement du rang moyen obtenu par cet imprimeur pour les devis pour lesquels il a été consulté, par types de devis, quantité et nombre de pages.

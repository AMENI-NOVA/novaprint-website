Web2Print Pro est une application full-web dont l’interface utilisateur est complètement différente de Graphisoft.

Web2Print Pro nécessite un serveur Web sous Windows disposant d’une connexion à la base de données utilisée par l’imprimeur.

Pour l’installation de WebToPrint Pro, reportez-vous au guide d’installation spécifique.

Export des achats clients

Précédent   Suivant

Avec certaines interfaces comptables, vous avez la possibilité d’exporter les données comptables de vos factures de vente comme achats pour vos clients. Pour cela, la comptabilité du client doit être la même que la votre et se limiter pour les achats à un compte TVA et un compte d’achat en plus de votre compte fournisseur.

Dans le paramétrage vous pouvez définir les clients pour lesquels l’export est possible et paramétrer pour ceux-ci les comptes utilisés.

Attention : Seules les factures de vente non confirmées pour le client sélectionné sont exportées pour la période sélectionnée. Le nom du client pour lequel l’export sera réalisé est rappelé dans le titre de la feuille.

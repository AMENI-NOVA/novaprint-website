Print2B est une version simplifiée de Graphisoft.

Print2B nécessite la mise en place d’un serveur d’applications Windows accessible par vos clients.

Graphisoft doit être installé sur le serveur d’applications.

L’interface utilisateur simplifiée de Print2B Windows s’active automatiquement lorsque la connexion de l’utilisateur correspond à un nom de connexion défini dans le paramétrage Print2B de Graphisoft.

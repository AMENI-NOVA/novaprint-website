Sérigraphie

Précédent   Suivant

Les informations à saisir pour une machine de sérigraphie sont similaires à celles d’une machine offset feuilles.

Les machines de sérigraphie peuvent être utilisées pour imprimer à partir de supports conditionnés en bobine. Dans ce cas, on considère que le support est coupé avant l’impression et peut être imprimé dans un sens ou dans l’autre.

Les actions possibles

Précédent   Suivant

La barre d'outils en haut de chaque écran représente les actions que l'utilisateur peut effectuer. Parfois, certaines actions sont grisées. Cela signifie qu'elles ne sont pas accessibles dans le contexte actuel.

Pour utiliser une action, il faut appuyer sur la touche correspondante ([F1] à [F12]) ou cliquer dessus.

 

Quel que soit l'écran affiché, les premières actions sont toujours les mêmes :

¨   Aide      [F1] : Affiche ou masque, en bas de l'écran, une aide contextuelle (voir paragraphe suivant)

¨   Employés         [F2] : Affiche l'écran de sélection d'un employé

¨   Partir     [F3] : Permet à l'employé d'indiquer son départ

¨   Postes   [F4] : Affiche la liste des postes ainsi que les affectations de l'employé

¨   Fiches   [F5] : Affiche la liste des fiches de travail du poste actuel

¨   En cours          [F6] : Permet de revenir aux fiches en cours de l'employé

¨   Outils    [F8] : Affiche les outils suivants :

·    Contrôle atelier       [Ctrl]+[F9] : Affiche un récapitulatif de l'activité de l'atelier.

·    Messages     [Ctrl]+[F10] : Affiche l'écran de messagerie.

·    Options       [Ctrl]+[F12] : Affiche un écran permettant le paramétrage du module ATELIER.

 

Les autres actions sont spécifiques à chaque écran. Elles seront détaillées dans les chapitres suivants.

Etendue de l'impression

Précédent   Suivant

¨   Tous les messages : Tous les messages seront imprimés.

¨   Les messages sélectionnés : Seuls les messages sélectionnés seront imprimés.
Pour sélectionner plusieurs messages, cliquez sur le premier puis sur les suivants en maintenant la touche [Ctrl] enfoncée.

¨   Seulement les messages actifs : Seuls les messages actifs seront imprimés.
Un message actif dans votre boîte de réception est un message que vous n'avez pas traité complètement. Dans la boîte d'envoi, un message actif est un message qui n'a pas été traité par tous les destinataires.
Un message actif c'est :

·    Un message qui n'a pas été lu,

·    Un message où une action reste à faire,

·    Un message demandant une réponse qui n'a pas été faite.

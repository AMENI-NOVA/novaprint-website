Quadri

Si vous cochez « Quadri » et que le nombre de couleurs est inférieur à 4, il est automatiquement ramené à 4.

Cette option  est en relation avec le calcul du nombre de lavages (voir "Lavage quadri" dans le paragraphe "Offset feuilles : Caractéristiques" du chapitre "Paramétrages de calcul des devis").

Couleurs d'impression

Si vous changes le nombre de couleurs pour passer d’une valeur inférieure à 4 vers une valeur supérieure à 4, l’option "Quadri" est cochée automatiquement.
S'il s'agit d'un nouveau devis, le nombre de couleurs proposé au verso ainsi que l’option "Quadri" seront les mêmes que ceux qui ont été saisis au recto (sauf si devis d’affiche multi-morceaux).

Noir

En relation avec le prix du lavage du groupe noir pour les machines offset.

Couleurs identiques

Il s'agit des couleurs communes au recto et au verso, qui peuvent éventuellement être basculées. Ceci n'est qu'une indication. Graphisoft optimisera de toute façon entre la solution sans basculage et avec basculage. S'il s'agit d'un nouveau devis, la valeur proposée par défaut (de même pour les PMS) sera calculée en fonction des informations précédemment saisies (nombre de couleurs, quadri, noir).

Pages identiques

Pour un cahier de brochure, cette option permet d’indiquer que toutes les pages sont identiques. Dans ce cas, un seul jeu de plaques sera calculé, quel que soit le nombre de pages. Le calcul n’est possible avec cette option que si les couleurs d’impression sont identiques au recto et au verso.

Recto et verso identiques

Pour un devis d’affiche, cette option permet d’indiquer que l’impression est identique au recto et au verso. Dans ce cas, les plaques ne seront calculées que pour un seul côté. Le calcul n’est possible avec cette option que si les couleurs d’impression sont identiques au recto et au verso.

PMS

Cette informations est utilisée pour calculer le nombre de recherches de teinte.

Références PMS

Il s'agit des références Pantone. Vous pourrez les saisir ou les compléter au moment de transformer le devis en commande.

Changements de plaques

Les changements de plaques peuvent être utilisés dans 2 situations :

¨   Gestion de différentes versions, typiquement des versions dans différentes langues.

Dans ce cas, vous pouvez cocher la case « Activer les versions pour cet élément » pour saisir le détail des différentes versions. Dans ce cas, la saisie de devis passe automatiquement en mode multi-versions (voir « Devis avec plusieurs versions » plus loin dans ce chapitre).

¨   Gestion de l’usure des plaques dans le cas d’un tirage très important

Dans ce cas, vous devez décocher la case « Activer les versions pour cet élément ».

Vous devez saisir :

¨   Le nombre de changements

¨   Le nombre de couleurs recto pour les changements

¨   Le nombre de couleurs verso pour les changements

Le nombre de couleurs (recto ou verso) pour les changements, ne doit pas être supérieur au nombre de couleurs d'impression (recto ou verso) car dans ce cas les résultats pourraient être incohérents.

Le nombre de couleurs basculables pour les changements, est considéré comme étant maximum (étant donné qu'on ne dispose pas de cette information). Par exemple, si on a 1 changement à 4 couleurs recto et 2 couleurs verso, on aura 2 couleurs basculables parmi celles du changement. Les changements interviennent sur les calculs suivants :

¨   Nombre de films pour la photogravure à réaliser

¨   Nombre et coût des duplis et coût de la répétex

¨   Nombre d'ozalids

¨   Nombre de plaques : Plaques (matière), Copie, Montage des films, Calages, Gâche de calage

Dans le cas d’une impression recto seul, calculée sur 2 machines, les changements sont toujours comptés sur la 2ème machine.

Encre

Indiquez le pourcentage d’encrage à prendre en compte par rapport au calcul de base.

Par défaut, cette zone est initialisée avec une valeur standard provenant du paramétrage (voir « Encrage par défaut » dans le chapitre "Paramétrages de calcul des devis»), qui dépend du type de devis et du nombre de couleurs saisi pour l'élément de devis. Lorsque vous modifiez le nombre de couleurs, le pourcentage d'encrage est automatiquement ajusté en fonction de la nouvelle valeur, sauf si vous avez modifié directement le pourcentage d'encrage ou si le devis a été rappelé et que cet élément s'y trouvait déjà.

La consommation et le prix de l’encre (voir « Paramétrage des types d’encre » dans le chapitre « Paramétrages de calcul des devis ») doivent être rentrés pour une base de 100 %.
Le coût de l’encre est calculé par rapport au format d’impression et au nombre total de feuilles imprimées (gâche comprise).

Le type d’encre peut être imposé pour chaque élément.

NB : en cas d’imposition du type d’encre, il vous revient de vérifier qu’il soit compatible avec le papier de l’élément voir avec les machines d’impression  imposées.

Type d’encre

Par défaut, le devis est optimisé en prenant en compte tous les types d’encre paramétrés, suivant le type de devis.

Vous pouvez cependant imposer le type d’encre à utiliser pour chaque élément du devis.

Particularité de la saisie de l’impression pour les cahiers de brochure

Le nombre de couleurs à imprimer pour les cahiers de brochure étant généralement le même des deux côtés, la saisie des couleurs au verso n'est visible dans ce cas que dans la partie avancée.

D'autre part, toute saisie effectuée sur le recto est automatiquement reportée à l'identique pour le verso. Mais il est toutefois possible de saisir des couleurs au verso différentes du recto.

Eléments de devis sans impression

Le calcul des éléments sans impression ne nécessite aucune machine d’impression même virtuelle.

Graphisoft compte un coût de refente pour mettre le support au format automatiquement calculé dans la saisie de devis.

Le support utilisé peut être conditionné à plat (feuilles) ou en rouleau (bobines).

Devis de prépresse

La saisie de l'impression est possible pour les devis de prépresse, uniquement afin de permettre le calcul des opérations de prépresse dépendant du nombre de couleurs. Il n'y a dans ce cas aucun calcul de l'impression.

Surface imprimée (%)

Utilisé pour le calcul de la vitesse d’impression des machines numérique grand format, le coût de tirage étant variable suivant la surface effectivement imprimée qui peut être plus petite que la surface du support du document fini.

Cette zone de saisie n’apparaît que s’il y a au moins une machine numérique grand format dans le paramétrage.

Qualité d’impression (dpi)

Utilisé pour le calcul de la vitesse d’impression des machines numérique grand format.

Cette zone de saisie n’apparaît que s’il y a au moins une machine numérique grand format dans le paramétrage.

La valeur par défaut pour les nouveaux devis peut être paramétrée dans les préférences de saisie de devis.

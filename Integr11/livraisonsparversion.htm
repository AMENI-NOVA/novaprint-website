Livraisons par version

Précédent   Suivant

Lorsque le devis comporte plusieurs versions, vous pouvez saisir les points de livraison soit à partir de la rubrique « Livraison » de la version à livrer, soit à partir de la rubrique « Livraison » générale de la saisie de devis.

Saisie des livraisons à partir de la version

La saisie des livraisons à partir de la version est la manière la plus simple et la plus rapide pour saisir les livraisons d’une version. La saisie fonctionne comme dans le cas où le devis ne comporte qu’une seule version : vous avez juste à indiquer le point de livraison et la quantité à livrer.

Dans ce cas, chaque livraison ne concerne que la version à partir de laquelle elle a été saisie. Pour regrouper plusieurs versions dans un même, point de livraison, vous devez utiliser la rubrique « Livraison » générale (voir ci-dessous).

Saisie des livraisons à partir de la rubrique « Livraison » générale

La saisie des livraisons depuis la rubrique « Livraison » générale vous permet de grouper plusieurs versions dans la même livraison.

Dans ce cas, vous devez indiquer pour chaque point de livraison, les versions qui sont y seront livrées, la quantité (par version) à livrer et l’adresse éventuelle de livraison. La colonne « Détail » habituelle du tableau des livraisons est remplacée par une colonne « Versions ». Lorsque vous cliquez sur le bouton « … » de cette colonne, la fenêtre de saisie des versions à livrer s’ouvre.

Se référer au paragraphe « Adresse de livraison et conditionnement par point de livraison » pour le fonctionnement de la liste des destinataires.

Le bouton <Ajouter > de la fenêtre « Versions à livrer » permet de sélectionner les versions qui doivent être livrées à ce point.

La fenêtre affiche toutes les versions pour lesquelles il reste une quantité non encore attribuée à un point de livraison.

Après avoir ajouter une version, il est possible de modifier la quantité livrée pour cette version dans le tableau de la fenêtre « Versions à livrer ».

Pour chaque version livrée, vous pouvez modifier le conditionnement saisi par défaut pour la version, en cliquant sur le bouton <…> de la colonne « Conditionnement ».

Les quantités de chaque version livrée peuvent aussi être saisies en pourcentage.

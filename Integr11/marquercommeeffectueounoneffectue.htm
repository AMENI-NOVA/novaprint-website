Marquer comme "Effectuée" (ou "Non effectuée")

Précédent   Suivant

(Valable seulement pour les messages de la boîte de réception).

Si une action est liée au message, utilisez la commande "Marquer comme effectuée" pour indiquer que celle-ci est faite. "Marquer comme non effectuée" annule la commande précédente.

Cette commande est aussi disponible dans le programme de notification des messages.

 

Cas de points de contrôle de gestion de production :

Si le message est lié à un point de contrôle, la fenêtre de validation d'un point de contrôle s'ouvre quand vous cliquez sur "Marquer comme effectuée". L'action sera marquée comme effectuée si le point de contrôle est validé.

Pour plus d'informations sur les points de contrôle, reportez-vous au chapitre "Points de contrôle".

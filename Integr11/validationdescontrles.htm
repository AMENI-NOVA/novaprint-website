Validation des contrôles

Précédent   Suivant

On ne peut effectuer un contrôle que lorsqu'il est à l'état "Prêt" ou "Non Validé".

La validation peut être faite depuis le programme principal de différentes manières ou depuis le programme atelier.

A chaque fois qu'un contrôle est effectué (validé ou non), une note liée à ce contrôle est enregistrée dans l'historique du dossier. L'auteur de cette note est la personne qui a effectué le contrôle.

La validation d'un contrôle doit être effectuée par l'utilisateur défini conjointement par le paramétrage du contrôle et par le dossier (voir "Paramétrage de validation du contrôle" dans ce chapitre).

 

Validation d'un contrôle depuis le programme principal

Plusieurs fonctionnalités permettent de valider un contrôle

¨   La planification du dossier : sélectionnez le contrôle à valider, puis cliquez sur Contrôles – Effectuer…

¨   Le tableau de charge : sélectionnez le contrôle à valider, puis cliquez sur Valider le contrôle

¨   La messagerie si une notification a été envoyée à la personne à valider : sélectionnez le message de notification correspondant au contrôle, puis cliquez sur Effectuée dans le menu Edition ou dans le menu contextuel.

Le processus de validation est ensuite le même quelle que soit la fonctionnalité utilisée.

La boîte de dialogue permet d'indiquer :

¨   la date à laquelle le contrôle a été effectué

¨   le résultat du contrôle : Validé ou Non validé

¨   un commentaire qui vient compléter la note créée automatiquement et associée au contrôle.
En cliquant sur le bouton <Commentaire> la fenêtre de saisie d'une note apparaît. Cela permet aussi possible d'envoyer cette note à un ou plusieurs destinataires.

 

L'utilisateur courant de l'application doit être autorisé à effectuer le contrôle, suivant le paramétrage (voir "Paramétrage de validaton du contrôle" dans ce chapitre).

 

Validation d'un contrôle depuis le programme atelier

Ce mode de validation ne concerne que les contrôles liés aux fiches de travail (les contrôles planifiés manuellement n'apparaissent pas sur le terminal atelier car il ne sont liés à aucun fiche).

L'utilisateur autorisé à effectuer le contrôle dépend du paramétrage du contrôle (voir "Paramétrage de validaton du contrôle" dans ce chapitre) :

¨   Validation par une personne désignée :

Le programme affiche la liste des personnes autorisées à effectuer le contrôle.

Le contrôleur valide son nom et doit alors saisir son mot de passe. Il peut ensuite indiquer si le contrôle est validé ou non.

¨   Validation par le poste :

Le programme affiche la liste des employés présents sur le poste. L'employé qui effectue le contrôle doit sélectionner son nom dans la liste puis indiquer si le contrôle est validé ou non.

Voir "Valider un point de contrôle" dans le chapitre "Atelier" pour plus d'informations.

Caractéristiques d’une machine numérique grand format

Support

Type de support

Impression sur des supports à plat ou en rouleau. Les 2 modes sont possibles pour une même machine.

Format minimum et maximum

La 1ère valeur correspond toujours à la largeur, c’est-à-dire le sens perpendiculaire à l’impression (laize).

La 2ème valeur correspond toujours à la profondeur, c’est-à-dire le sens de défilement du support dans la machine. Cette valeur peut être laissée à 0 même dans le cas d’une machine avec alimentation à plat uniquement. Dans ce cas, c’est la taille du support utilisé qui fixe la limite.

Impression
Nombre de couleurs

On considère que l’impression en quadri est toujours possible si le nombre de couleurs est au moins égal à 4.

Tons directs

Permet d’autoriser ou d’interdire l’utilisation de la machine pour des devis avec des PMS.

Vernis

Même type de paramétrage que pour les presses offset feuilles : liaison de la machine a une famille d’opérations de surface pour indiquer que la machine peut effectuer en ligne les opérations de cette famille.

Lorsqu’une opération de vernis est faite sur la machine :

¨   Le coût de tirage est majoré du pourcentage saisi dans cet écran

¨   Un coût de calage spécifique est compté : à saisir dans l’onglet « Calage/Tirage »

Pour les machines numériques grand format, le vernis avec réserve est toujours possible et n’engendre aucun coût supplémentaire.

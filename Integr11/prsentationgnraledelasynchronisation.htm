Présentation générale de la synchronisation

Précédent   Suivant

Les fonctions de synchronisation servent à faire fonctionner Graphisoft sur des ordinateurs qui ne sont pas connectés en permanence au serveur de base de données. C'est le cas des ordinateurs portables de commerciaux qui font les devis directement chez les clients, mais aussi d'agences commerciales qui ne peuvent pas être connectées au serveur de base de données.

 

Les fonctions de synchronisation sont accessibles par la commande Synchronisation dans Utilitaires du menu Graphisoft. Le droit d'accès "Synchronisation" est obligatoire pour les utiliser.

 

Les données synchronisées sont les suivantes :

¨   Les nouvelles notes avec leurs pièces jointes

¨   Les nouveaux clients et les clients modifiés

¨   Les nouveaux devis avec leurs lettres de confirmations

 

Les informations sont transférées d'un ordinateur à l'autre soit directement (via une connexion au réseau), soit indirectement par l'intermédiaire d'un e-mail ou d'un fichier sur disquette.

Paramétrage entre Graphisoft et les services Web

Précédent   Suivant

Le paramétrage des accès aux services web papetiers se trouve dans les paramétrages généraux, sous la branche « Interface papiers Web ». Double-cliquez sur cette branche ou cliquez (après l’avoir sélectionné) sur le bouton « propriétés » de la barre d’outils.

 

Chaque fournisseur pour lequel l’interface papiers Web est possible avec Graphisoft, dispose d’un onglet permettant de paramétrer par utilisateur ses identifiants et ses droits (le seul droit relatif aux interfaces papiers web qui soit géré dans la fiche employé est le droit d’accéder à ce paramétrage).

 

Pour chaque fournisseur, afin de pouvoir accéder a son service web, vous devez activer le paramétrage en cochant la case « Service actif », puis

¨   Choisissez la société qui correspond à ce fournisseur, parmi les tiers « fournisseur de papier»

¨   Pour chaque utilisateur devant accéder à au moins une fonctionnalité de l’interface :

·    Cliquez sur le bouton « ajouter »

·    Sélectionnez l’utilisateur dans la liste déroulante (qui ne présente que les utilisateurs « classiques »)

·    Saisissez les identifiants à utiliser (nom et mot de passe pour Papyrus, nom, mot de passe et un numéro pour Antalis). Il est possible d’utiliser les même identifiants pour plusieurs utilisateurs.

·    Cochez le ou les droits autorisés pour cet utilisateur avec ce fournisseur.

¨   Le cadre « Journalisation des communications » vous permet d’indiquer les fichiers texte dans lesquels seront stockée les communications effectuées entre Graphisoft et les services Web :

·    Demandes de tarifs (actualisation des tarifs ou conditions d’achat depuis la saisie de devis, la gestion de stock ou avant de passer commande en ligne).

·    Commandes en ligne.

Si une de ces zones n’est pas renseignée, il n’y aura pas de conservation du type de communications concernées.

Pour activer cette journalisation :

§  Cliquez sur le bouton <…>.

§  Sélectionnez ensuite le répertoire ou devra être stockée la trace.

NB : le répertoire devra être accessible à tous les utilisateurs devant accéder aux servces Web.

§  Confirmez la création du fichier.

§  L’application va ensuite vérifier que vous avez le droit de créer/modifier ce fichier.

 

Remarque :

¨   En décochant « Service Actif », vous ne perdrez pas votre paramétrage, mais il n’y plus d’accès au service web tant que cette case ne sera pas cochée.

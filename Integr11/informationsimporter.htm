Informations à importer

Précédent   Suivant

Cette étape permet d'indiquer les informations que l'on souhaite importer dans Graphisoft.

Pour les fichiers à largeur fixe, il faut, dans un premier temps, indiquer la taille de chaque colonne.

Les boutons "<<" et ">>" servent à se déplacer d'une colonne à l'autre. Il est aussi possible de choisir une colonne directement en cliquant dessous dans l'aperçu du fichier.

Début et taille de la colonne

Ce paramétrage est visible uniquement pour les fichiers à largeur fixe.

Le début de la colonne est calculé en fonction du début et de la taille de la colonne précédente.

La taille correspond au nombre de caractères contenu dans la colonne.

Vous ne pouvez  définir la colonne suivante que si la colonne actuelle à une taille différente de 0.

Dès que vous saisissez une taille, l'aperçu du fichier en tient compte. Vous pouvez ainsi vérifier votre saisie.

Après avoir défini la largeur d'une colonne, cliquez sur le bouton ">>", pour définir la colonne suivante. Procédez ainsi jusqu'à ce que toutes les colonnes soient définies.

Information correspondante dans Graphisoft

Si vous voulez importer une colonne, il faut indiquer à quelle information elle correspond dans Graphisoft. Pour cela, il faut sélectionner l'information dans la liste. Par défaut, toutes les colonnes sont sur "(Information non importée)".

La liste affiche le nom et la taille de l'information dans Graphisoft. Les informations ne correspondant à aucune colonne du fichier d'importation sont affichées en gras.

Pour lier une colonne du fichier d'importation à une information de Graphisoft :

1. Sélectionnez la colonne du fichier d'importation avec les boutons "<<" et ">>"

2. Déroulez la liste "Information correspondante dans Graphisoft" et sélectionnez l'information de Graphisoft qui doit être alimentée par la colonne du fichier d'importation que vous avez sélectionnée en 1.

3. Choisissez si les données lues sont importées tel qu’elle (cliquez sur « Valeurs du fichier ») ou s’il faut importer des données spécifiques (choisir « Valeurs spécifiques »). Plus de détails dans la section ci-après.

4. Procédez ainsi pour toutes les colonnes que vous voulez importer

Vous n'êtes pas obligé de lier toutes les colonnes à une information de Graphisoft. Les colonnes qui restent sur "(Information non importée)" ne seront pas importées.

 

Valeurs du fichier ou valeurs spécifiques

Pour toutes les colonnes importées, il est possible de choisir comment sont importées les données.

Deux méthodes sont possibles :

·      Valeurs du fichier

·      Valeurs spécifiques

Valeurs du fichier

Quand cette option est choisi, les données lue dans le fichier sont enregistrées tel quelles dans la base de données.

Dans ce cas, une colonne du fichier alimente un et un seul champ dans la base de données.

Valeurs spécifiques

Tous les fichiers ne sont pas toujours importables directement dans Graphisoft. Parfois, certaines colonnes complexes (ex : mode de paiement, conditions de règlements, …) doivent être adaptées ou modifiées dans un outil tiers (ex : Excel) avant d’être importées dans Graphisoft.

Afin de minimiser ces modifications hors de Graphisoft, il est possible de saisir des correspondances entre les valeurs lues dans le fichier et celles qui seront enregistrées dans la base de données.

 

Pour définir ces correspondances, il faut choisir « Valeurs spécifiques » dans « Données à importer » et cliquer sur le bouton […].

 

La colonne « Valeurs du fichier » correspond aux données se trouvant dans le fichier à importer.

Vous pouvez saisir manuellement les valeurs ou cliquer sur le bouton « Lire les valeurs du fichier » pour remplir automatiquement la colonne en fonction des valeurs trouvées dans le fichier.

 

La colonne « Valeurs à importer » sert à renseigner les valeurs à enregistrer dans la base en fonction de valeurs lues dans le fichier.

Certain champ (ex : Mode de règlement) ont des valeurs imposées dans Graphisoft. Dans ce cas, la colonne « Valeurs à importer » propose une liste de choix avec les valeurs acceptées. Si vous saisissez une valeur qui n’est pas dans cette liste, la saisie sera ignorée.

 

Correspondances multiples

Il est possible de mettre à jour plusieurs champs dans Graphisoft en fonction d’une seule colonne dans le fichier d’importation.

Pour cela, il faut cliquer que le bouton  pour ajouter la colonne souhaitée.

Pour supprimer une colonne, il faut la sélectionner en cliquant sur son entête puis cliquer sur le bouton

 

Une fois que toutes les colonnes souhaitées sont ajoutées, il faut saisir les données à importer dans chacune d’elles en fontion de la valeurs lues dans le fichier.

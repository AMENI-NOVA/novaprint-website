Types de tiers

Précédent   Suivant

La gestion des stocks fait intervenir les tiers suivants :

¨   Les fournisseurs

¨   Les clients possesseurs de stocks

¨   Les ateliers

¨   Les sous-traitants

¨   Les départements de l'imprimerie (interne) et les abonnés

¨   Les interlocuteurs divers (Exemple: Dépréciation, Récupération)

Les fournisseurs, les sous-traitants et les clients possesseurs de stocks sont des sociétés gérées dans la mise à jour des sociétés (cf. le chapitre « Sociétés ») lorsque vous cochez l’option « Tiers » de la société.

Les autres tiers (ateliers, bureaux et divers) sont gérés dans l’écran de  paramétrage sous le titre « Noms (Interlocuteurs )».

Note : Si un interlocuteur entre dans plusieurs catégories, entrez-le dans chaque catégorie concernée.

Exemple: Si un possesseur vous fourni lui-même son papier, il faudra le rentrer à la fois dans le type ‘Possesseur’ ET dans le type ‘Fournisseur’ pour pouvoir le sélectionner à la fois dans les possesseurs d’un article et en tant que fournisseur pour définir les conditions d’achat.

Dans la liste arborescente, cliquez sur le < + > devant le type  d’interlocuteurs à mettre à jour. Les noms déjà saisis dans ce type apparaissent en dessous de celui-ci.

En cliquant avec le bouton droit de la souris sur un interlocuteur on a accès aux opérations suivantes :

¨   Nouveau ajoute un nouvel interlocuteur de même type

¨   Propriétés permet de modifier les données de l’interlocuteur sélectionné

¨   Supprimer supprime l’interlocuteur sélectionné

Le clic droit sur le type de nom ne donne accès qu’au menu Nouveau.

Afficher / Cacher les périodes non travaillées

Précédent   Suivant

 

Par défaut EmaPlan montre les périodes non travaillées (par exemple les nuits, les weekends). L’affichage des tâches prend en compte ces périodes.

 

 

Il est possible de cacher les périodes non travaillées afin de visualiser uniquement les périodes de planification. Pour ce faire, il faut cliquer sur le menu « Affichage » → « Cacher les périodes non travaillées ».

Les zones non travaillées disparaissent. L'échelle de temps est fractionnée et les zones travaillées se rejoignent.

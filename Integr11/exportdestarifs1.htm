Export des tarifs

Précédent   Suivant

En plus de l’export de la liste, fonction commune à toutes les listes, il est aussi possible d’exporter vos tarifs papier afin les manipuler à l’extérieur de Graphisoft (dans Excel par exemple) pour les réimporter ensuite (avec l’Importation des tarifs papetiers). Cette fonction d’export se trouve dans le menu Affichage.

 

Cet export, s’il impose les colonnes du fichier généré, ne porte que sur les papiers actuellement affichés dans la liste.

Deux paramètres influent sur les tarifs exportés :

¨   Les filtres actifs (filtre de la macro ou bien filtres express) :

·    Si aucun filtre n’est appliqué (cas de la macro <Standard> par exemple), tous les tarifs seront exportés.

·    Si un filtre est actif (dans la macro ou filtre express), seuls les papiers affichés le seront.

¨   Les colonnes affichées :

·    Si seules, des colonnes relatives aux catégories sont affichées, tous les papiers de ces catégories seront exportés.

·    Si au moins une colonne relative aux papiers est affichée, seuls les papiers correspondants seront exportés.

Vous avez donc la possibilité de choisir finement quels tarifs vous voulez exporter.

Tableaux récapitulatifs des presses feuilles

Précédent   Suivant

Ces tableaux regroupent tous les tarifs saisis pour toutes les presses feuilles. Vous pouvez ici, comparer, modifier et enregistrer les tarifs de toutes les presses feuilles.

Pour accéder à ces tableaux, sélectionnez Presses feuilles puis cliquez sur Propriétés dans le menu Paramétrages ou dans le menu contextuel.

Le tarifs peuvent être traités en coût ou en temps : la s élection se fait dans la liste « Paramétrage en :».

Si vous avez plus de 1 tarif, vous pouvez cacher les colonnes de tarifs qui ne vous interessent pas pour faciliter la comparaison entre les différents machines ( non disponible en gamme Estiprint).

Les tableaux fonctionnent de la même façon que dans les écrans de mise à jour des presses feuilles.

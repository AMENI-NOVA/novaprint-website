Importation d'une note

Précédent   Suivant

Lors de l'importation d'une société, le programme peut créer automatiquement une note dans son historique. Le contenu de la note peut provenir des informations du fichier à importer ou être saisie avant l'importation.

Créer une note …

Cette option permet de définir si une note doit être créé lors de l'importation (ou de la mise à jour) d'une société.

·      Oui, toujours : A chaque importation une note sera créée, soit avec les données provenant du fichier, soit avec les valeurs par défaut (définies en dessous).

·      Oui, seulement si la note est importée : La note sera créée si, et seulement si, un des champs suivant est importé : Auteur, Date, Titre ou Texte. Si tous les champs ne sont pas importés, le programme utilise les valeurs par défaut définies en dessous.

·      Non : Aucune note ne sera créer lors de l'importation.

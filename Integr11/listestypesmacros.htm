Listes types : macros

Précédent   Suivant

Une macro vous permet d’enregistrer une liste type qui comprend : les informations à afficher dans les colonnes ainsi que la largeur de chacune d’elles, l’ordre des éléments dans la liste et le filtre à appliquer. Lorsque vous chargez une macro, la liste reprend les caractéristiques définies pour la macro. Vous pouvez créer autant de macros que vous voulez pour chaque liste.


Pour créer une nouvelle macro, sélectionnez les informations à afficher, l’ordre des éléments et le filtre à appliquer. Réglez la largeur de chaque colonne, puis rappelez la fenêtre des options d’affichage. Au bas de la fenêtre, tapez un nom de macro dans la zone de texte de la liste déroulante, puis cliquez sur le bouton Enregistrer. La macro sera créée dans le groupe sélectionné dans la liste déroulante au dessus (voir plus bas pour la gestion des groupes de macros).

Pour rappeler une macro, utilisez une des deux méthodes suivantes :

¨   Sélectionnez la macro à charger dans la liste déroulante se trouvant en bas de la fenêtre de la liste.

¨   Faites afficher la fenêtre des options d’affichage, et dans la liste déroulante sélectionnez la macro à charger et cliquez sur le bouton Charger. Cliquez ensuite sur le bouton Valider pour fermer la fenêtre en confirmant votre choix.

Lorsque vous fermez une fenêtre de liste, le nom de la macro courante est enregistré automatiquement sur le poste. Lorsque vous ouvrez une des listes (sociétés, devis, commandes, factures etc.) sur le poste, la derniere macro utilisée pour l’affichage de cette liste est rappelée automatiquement.

Gestion des groupes de macros

Les macros d’une liste peuvent être organisées dans des groupes de macros, une macro pouvant appartenir à un groupe ou à aucun.

Un groupe n’étant qu’un contenant, il n’y a aucune contrainte sur les macros pouvant y être rattachées. Un groupe peut aussi contenir d’autres groupes (eux même pouvant contenir des macros et/ou des groupes).

Pour accéder à la gestion des groupes, cliquer sur le bouton situé sur le côté de la première liste déroulante du cadre « Macros » .

L’écran se divise en trois parties :

¨   La liste déroulante présentant les groupes et sous-groupes ainsi que les boutons permettant d’en créer un, d’enregistrer les modifications sur le groupe actuel et enfin de supprimer un groupe et ses ‘enfants’.

¨   La liste des macros du groupe courant (rafraîchie lorsque vous sélectionnez un groupe dans la liste déroulante)

¨   La hiérarchie existante (« Macros disponibles ») : les groupes existants et les macros et sous-groupes qui leur sont rattachées.

 

Création d’un groupe

¨   Groupe « principal » : Dans la liste déroulante, sélectionnez la 1ère ligne (celle qui n’a pas de texte), cliquez sur le bouton ‘Créer’, entrez le nom désiré puis validez en cliquant sur le bouton ‘Ok’.

¨   « Sous-groupe » : Dans la liste déroulante, sélectionnez le groupe « père » désiré, cliquez sur le bouton ‘Créer’, entrez le nom désiré puis validez en cliquant sur le bouton ‘Ok’.

Votre groupe est créé. Il est maintenant possible de lui affecter des macros.

Dans la liste déroulante, les groupes enfants apparaissent sous la forme père\enfant. Ainsi, la ligne ‘par type\liasses\commandes’ signifie groupe groupe ‘commandes’ appartenant au groupe ‘liasses’, lui-même appartenant au groupe ‘par type’.

Modification d’un groupe

¨   L’information « Groupe avancé » permet de restreindre l’accès aux macros de ce groupe (et des ses sous-groupes) aux seuls utilisateurs ayant le droit « Accès aux groupes de macros avancés ». Lorsqu’un groupe est marqué comme avancé, ses sous-groupes le seront aussi (l’inverse n’étant pas vrai). Cette zone n’est pas accessible lorsqu’un groupe appartient à groupe marqué comme « avancé ». Enfin, dans les différentes listes déroulantes affichant ces groupes de macro, lorsqu’un libellé est suivi des caractères (*), ceci désigne un groupe avancé.

¨   Affecter des macros à un groupe (quelque soit son niveau hiérarchique) : sélectionnez le groupe dans la liste déroulante, sélectionnez (une à la fois) chaque macro à y affecter dans la liste ‘macros disponibles’ puis cliquez sur le bouton  pour faire passer la macro sous le groupe (double cliquer sur la macro dans la liste aura le même effet). Si vous le faites sur une macro rattachée à un groupe, elle sera désaffectée de ce groupe pour être rattaché au ‘nouveau’ groupe.

¨   Désaffecter des macros d’un groupe (quelque soit sont niveau hiérarchique) : sélectionnez le groupe dans la liste déroulante, sélectionnez (une à la fois) chaque macro à enlever dans la liste ‘macros du groupe’ puis cliquez sur le bouton   pour faire passer la macro du groupe (double cliquer sur la macro dans la liste aura le même effet) à la liste des macros disponibles. La macro sera maintenant ‘<sans groupe>’.

Cliquer sur le bouton ‘Enregistrer’ pour sauvegarder vos modifications.

Suppression d’un groupe

Sélectionnez le groupe (quelque soit son niveau hiérarchique) puis cliquez sur le bouton ‘Supprimer’. Vous devez confirmer la suppression car TOUS les enfants (macros, sous-groupes, et les enfants de ceux-ci, etc..) seront supprimés définitivement ! Si vous voulez conserver des macros avant de supprimer le ou les groupes, pour chaque macro à conserver, sélectionnez son groupe, faites passer la macros dans les macros disponibles et enregistrez. Puis supprimez le ou les groupes.

Attention : Un groupe ne peut être supprimé que s’il ne contient aucun groupe marque comme avancé.

Les macros dans les listes

Les macros ainsi définies sont ensuite disponibles dans les liste via le menu déroulant se trouvant dans la barre d’état des ces listes.

Lorsque vous cliquez sur ce menu, la hiérarchie se déploie progressivement : à chaque fois qu’un groupe est sélectionné, un nouveau « panneau » est affiché, panneau qui contient les macros et sous-groupes de ce « niveau ».

L’organisation de chaque « panneau » est toujours la même : les macros, un séparateur puis les groupes de ce niveau.

Dans l’exemple suivant, nous avons un certain nombre de macros sans groupes (de « <Standard> » à « PrintToB », puis un groupe (« Devis du jour »). Ce groupe contient lui-même deux macros et un groupe : « Devis réservés ». Enfin, ce dernier, ne contient qu’une macro et aucun sous-groupe.

Ici, la macro « Commandes » est cochée. Cela signifie qu’il s’agit de la macro actuellement utilisée pour filtrer la liste. Le groupe « Devis reservés », quant à lui, est un groupe avancé, son libellé est donc suivi des caractères (*).

 

Certains écrans disposent des mêmes capacités de macro mais ont une présentation différente. Dans ce cas, la liste des macros apparait sous la forme d’un cadre avec un fond blanc. Un clic dans ce cadre fait apparaître un menu (bien que d’aspect légèrement différent) de fonctionnement identique : un « panneau » par groupe, séparation entre macros et groupes via un séparateur, coche sur la macro active, etc…

 

Attention : Seuls les utilisateurs ayant le droit « Accès aux groupes de macros avancés » verront les groupes (et leurs macros) marqués comme avancé.

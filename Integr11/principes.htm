Les affectations multiples sont une nouveauté du module ATELIER sous Windows.

 

Définition : "Affectation multiple"

Pouvoir s'affecter sur plusieurs postes de travail afin de réaliser plusieurs travaux simultanément.

Afin de limiter les erreurs de saisies, l'affectation multiple n'est pas autorisée pour tous les employés. Seuls ceux ayant le droit "Affectations multiples" dans le paramétrage des employés, peuvent s'affecter à plusieurs postes en même temps.

 

Travaux multiples :

Une personne ayant accès aux affectations multiples peut donc effectuer plusieurs travaux en même temps :

¨   Sur des postes différents,

¨   Sur le même poste (s'il s'agit d'un poste multiple),

¨   Sur le même dossier,

¨   Sur la même fiche (si le poste est détaillé par opération)

Le nombre de travaux multiples n'est pas limité par le programme.

 

Calcul du prix de revient:

Attention, le calcul du prix de revient d'un dossier ne tient pas compte des affectations multiples. Le coût d'un employé est le même qu'il travaille sur un ou sur plusieurs dossiers en même temps.

Liaison du Fiery au poste Graphisoft

Précédent   Suivant

Cette dernière étape consiste à lier le Fiery à un poste de gestion de production de Graphisoft.

Choissiez simplement le poste d’impression correspondant à votre Fiery.

 

Si le poste n’existe pas encore dans Graphisoft, vous pouvez utilisez les options « Ajouter une presse » ou « Dupliquer une presse existante ».

Dans ces deux cas, un nouveau poste de production est créé dans Graphisoft avec le paramétrage nécéssaire pour travailler avec le serveur Fiery.

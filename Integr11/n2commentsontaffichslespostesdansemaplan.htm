n°2 : Comment sont affichés les postes dans EmaPlan

Précédent   Suivant

Le paramétrage ci-contre permet de définir comment les postes envoyés à EmaPlan seront affichés (ordre et visibilité).

 

Un poste sélectionné dans le paramétrage précédent (n°1) mais non présent dans le modèle de regroupement sélectionné ici, ne sera pas visible dans EmaPlan, mais pris en compte pas le moteur d’optimisation.

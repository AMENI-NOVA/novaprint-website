Création d'une nouvelle note

Précédent   Suivant

"Nouvelle note" dans le menu "Notes" ou le menu contextuel.

 

Cette fonction vous permet de créer une nouvelle note. La nouvelle note aura les mêmes destinataires et les mêmes liaisons (devis, société, pièce jointe, ...) que la note sélectionnée dans l'historique.

Pour plus d'informations, voir la section "Création d'une note".

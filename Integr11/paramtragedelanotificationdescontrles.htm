Paramétrage de la notification des contrôles

Précédent   Suivant

Lors de l'activation d'un point de contrôle, une notification peut-être envoyée aux personnes concernées.

Cet écran de paramétrage permet de définir les options générales de cette notification.

Qui faut-il notifier ?

Les personnes à notifier dépendent de chaque contrôle. Pour les définir, il faut utiliser le paramétrage des contrôles.

Qui est l'auteur de cette notification ?

Ces options permettent de définir qui sera indiqué comme auteur (expéditeur) de la notification.

Les valeurs possibles sont :

¨   L'imprimerie.

¨   Commercial du dossier

¨   Information complémentaire du dossier. Vous devez dans ce cas sélectionner  l’information complémentaire dans la liste située contre cette option. Cette liste est la liste des informations complémentaires de type fonction. L'auteur sera l'employé qui aura été indiqué dans l'information complémentaire du dossier auquel le contrôle se rapporte.

¨   L'employé suivant : L'auteur sera l'employé sélectionné dans la liste.

Quel compte de messagerie faut-il utiliser ?

Seulement si Graph'e-mail est utilisé comme un programme de messagerie (voir "Paramétrage général" dans le chapitre "Messagerie").

Ces options indiquent quel compte de messagerie utiliser pour l'envoi de la notification.

Les valeurs possibles sont :

¨   Le compte par défaut de l'auteur.

¨   Le compte suivant. Dans ce cas vous devez sélectionner un compte dans la liste des comptes de messagerie existant.

Quel type de note faut-il utiliser ?

Cette option permet définir le type de la note.

Ce type peut-être utilisé pour filtrer les notes dans l'historique des commandes ou des sociétés.

Si la notification échoue…

Si la notification échoue, une note d'information sera envoyée automatiquement à un employé.

La note indiquant la source du problème, l'employé destinataire pourra ainsi traiter le problème et renvoyer la notification.

Qui faut-il prévenir ?

Cette liste permet de sélectionner l'employé à prévenir en cas d'échec.

Quel type de note faut-il utiliser ?

Cette option permet définir le type de la note.

Ce type peut-être utilisé pour filtrer les notes dans l'historique des commandes ou des sociétés.

Causes d'échecs possibles :

¨   L'auteur est le commercial du dossier, mais celui-ci n'est pas renseigné;

¨   L'auteur dépend d'une information complémentaire, mais celle-ci n'est pas renseignée pour ce dossier;

¨   L'auteur de la notification n'a pas de compte de messagerie;

¨   La personne à notifier est le commercial du dossier, mais celui-ci n'est pas renseigné;

¨   La personne à notifier dépend d'une information complémentaire, mais celle-ci n'est pas renseignée pour ce dossier;

¨   La personne à notifier est l'interlocuteur du dossier, mais celui-ci n'est pas renseigné;

¨   La personne à notifier n'a pas d'adresse e-mail.

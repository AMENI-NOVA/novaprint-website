Exportation du tableau de charge dans un fichier texte

Précédent   Suivant

La fonction Exporter… du menu Tableau de charge permet créer un fichier texte contenant les informations du tableau de charge. Le fichier créé sera utilisable dans tous les tableurs acceptant l’importation des fichiers textes.

Pour utiliser cette fonction, vous devez posséder le droit "Exportation du tableau de charge" (voir « Employés – droits » dans le chapitre « Paramétrages généraux »).

Il est possible d'agir sur l’exportation en modifiant les paramètres suivants :

¨   Nom du fichier d’exportation : Nom du fichier créé par l’exportation. Le format de ce dernier est de type ASCII. Pour faciliter la saisie, utilisez le bouton "...".

¨   Exporter tous les dossiers... : Permet de ne pas tenir compte des filtres en cours et d’exporter tous les dossiers du tableau de charge.

¨   Exporter les temps en heure centième : Si vous travaillez avec l’affichage des temps en heure minute (05h30mn), vous pouvez exporter les temps en heure centième(05,50). Ceci permet ensuite d’effectuer des calculs dans un tableur.

¨   Inclure le titre des colonnes... : Indique si la première ligne du fichier représente le titre des colonnes.

¨   Séparateur des données : Caractère de séparation des colonnes.

¨   Exporter avec le paramétrage actuel ou avec le modèle... : Choisissez "avec le paramétrage actuel" si vous souhaitez que le tableau soit exporté avec les mêmes regroupements de postes que ceux utilisés pour l’affichage à l’écran. Sinon, vous pouvez exporter le tableau de charge avec le modèle de regroupements de postes de votre choix en choisissant "avec le modèle :".

¨   Afficher cette fenêtre avant chaque exportation : Si vous ne souhaitez pas modifier ces paramètres a chaque exportation, décocher cette case. Vous pourrez de toute façon modifier ultérieurement ces informations en cliquant sur Préférences... dans le menu Tableau de charge.

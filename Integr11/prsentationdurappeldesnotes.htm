Présentation du rappel des notes

Précédent   Suivant

La fonction de rappel des notes permet de faire apparaître les notes importantes de l'élément sur lequel vous travaillez. Par exemple, si vous saisissez un devis, les notes concernant le client sont affichées.

Le rappel des notes est actif dans la plupart des fonctionnalités de Graphisoft. Ainsi, quoi que vous fassiez, les informations importantes ne peuvent plus être oubliées. Voir " Où le rappel est-il actif ?"

Les éléments de Graphisoft disposant du rappel de leurs notes sont les suivants:

¨   les sociétés,

¨   les devis,

¨   les commandes,

¨   les postes de gestion de production.

 

Le rappel des notes est toujours affiché sur fond jaune, en haut à droite de l'écran. Il indique le nombre de notes concernant l'élément sur lequel vous travaillez. S'il n'y à pas de note pour cet élément, le rappel ne s'affiche pas.

 

Si l'on déplace le curseur de la souris sur le message "X notes", l'écran du rappel s'agrandit pour faire apparaître la liste des notes. Pour réduire l'affichage, il suffit de déplacer le curseur de la souris en dehors du rappel.
